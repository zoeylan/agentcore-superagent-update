# 生产计划与排程优化

## 场景描述
当工厂面临订单交付延迟、产能与需求不匹配、计划频繁变更、在制品积压等生产计划问题时，运营总监需要从S&OP产销协同、主生产计划(MPS)、物料需求计划(MRP)、车间排程(Scheduling)等多层面进行系统性诊断和优化，建立柔性高效的计划体系。

## 触发条件
- 用户提到生产计划、排程、交期、产能规划、S&OP等关键词
- 用户反馈订单准时交付率(OTD)低、插单频繁、计划达成率差
- 用户询问APS系统选型、MRP运算逻辑、产能负荷分析
- 用户面临多品种小批量(HMLV)环境下的计划挑战

## 输出规范

### 分析框架
1. **计划体系诊断**：评估S&OP → MPS → MRP → 车间排程的四级计划体系完整性
2. **需求分析**：需求波动性分析(CV值)、ABC-XYZ分类、需求预测准确率评估
3. **产能建模**：瓶颈识别、产能负荷分析(CRP)、产能柔性评估
4. **排程策略**：确定推拉分界点(CODP)、选择排程规则(SPT/EDD/CR等)
5. **系统支撑**：APS/MES系统能力评估与选型建议

### 关键分析维度
- 计划冻结期(Frozen Zone)设置：一般为1-2周，冻结期内不允许变更
- 计划颗粒度：月度S&OP → 周MPS → 日排程 → 实时调度
- EPEI(Every Part Every Interval)：衡量生产批量和换型频率的合理性
- 安全库存设定：基于需求波动(σ)和补货周期(LT)的科学计算
- 约束排程(TOC-DBR)：识别瓶颈→利用瓶颈→从属非瓶颈→提升瓶颈→防止惰性

### 输出格式
- 计划体系成熟度评估（五级模型）
- 产能负荷甘特图分析
- 推拉策略设计方案（含CODP定位）
- KPI改善目标：OTD、计划达成率、库存周转率、EPEI
- 系统选型对比矩阵（适用场景、核心功能、投资量级）

## 专业知识要点
- '计划的敌人不是变化，而是没有管理变化的机制'——关键是建立需求变更的分级响应流程
- MRP的致命假设：无限产能和固定提前期，这在实际中几乎不成立，所以需要CRP和APS补充
- 多品种小批量环境的核心矛盾：换型时间vs.批量大小，SMED是破局关键
- APS不是万能药：数据质量(BOM准确率>98%、工艺路线完整)是APS上线的前提
- 真正的产能瓶颈往往不在设备，而在'隐性约束'——模具数量、检测设备、关键技工、物料供应
- S&OP会议的成功关键：销售必须给出约束性预测(而非'尽量多备')，运营必须给出明确产能边界
