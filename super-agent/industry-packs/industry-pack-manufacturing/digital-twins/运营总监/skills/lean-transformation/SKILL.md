# 精益转型规划与推进

## 场景描述
当企业需要系统性推进精益生产转型，或针对特定车间/产线进行精益改善时，运营总监需要评估现状成熟度、设计精益路线图、规划价值流改善方案，并指导变革管理策略以确保精益文化的可持续落地。

## 触发条件
- 用户提到精益生产、TPS、Lean、持续改善(Kaizen)、浪费消除等关键词
- 用户询问价值流分析(VSM)、标准作业、看板系统、SMED等精益工具
- 用户面临生产效率低、在制品(WIP)过多、交付周期(Lead Time)过长等问题
- 用户计划启动精益转型项目或评估精益成熟度

## 输出规范

### 分析框架
1. **成熟度评估**：用精益成熟度模型(5级)评估当前水平（基础5S → 标准化 → 流动化 → 拉动式 → 完美化）
2. **价值流分析**：绘制当前状态图(Current State Map)，识别七大浪费和改善机会点
3. **未来状态设计**：设计未来状态图(Future State Map)，明确目标DTD(Dock-to-Dock)时间
4. **实施规划**：制定12-18个月的精益推进路线图，分波次(Wave)推进
5. **变革管理**：设计组织保障、人才培养和激励机制

### 核心方法论
- 七大浪费(TIMWOODS)：运输、库存、动作、等待、过量生产、过度加工、缺陷（+未利用人才）
- 流动化三条件：连续流(One-Piece Flow) → 设备小型化 → U型布局
- 拉动系统设计：后补充看板 → 均衡化生产(Heijunka) → 节拍管理
- 标准作业三要素：节拍时间(TT)、作业顺序、标准在制品

### 输出格式
- 精益成熟度雷达图（5维度评分）
- 价值流现状图关键指标（PLT、VA ratio、WIP天数）
- 分阶段改善路线图（含里程碑和预期收益）
- 变革管理风险评估及应对策略
- 精益KPI仪表盘设计（DTD、FTT、BTS、EPEI等）

## 专业知识要点
- 精益不是工具箱，而是思维方式和管理体系，'工具先行'是最常见的失败模式
- 推进顺序很关键：先做流动化(物流改善)，再做均衡化(信息流改善)，最后拉动化(系统整合)
- 中国制造企业精益转型最大障碍：中层管理者的改善能力不足和一线员工流动率高
- 精益与MES/数字化的结合：先精益后数字化，用标准化为数字化奠基
- 模型线(Model Line)策略：选一条线做标杆，3个月见效后再横向展开(Yokoten)
- 精益办公室(Lean Office)和精益供应链往往被忽视，但价值流70%的Lead Time在工厂围墙之外
