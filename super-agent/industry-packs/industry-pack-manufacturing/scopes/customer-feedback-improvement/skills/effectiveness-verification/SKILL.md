# 改善效果量化验证技能

## 概述
本技能负责在改进措施落地后的观察期内（3个月），运用统计学方法持续监控和量化验证改善效果。制造业改善的有效性不能凭"感觉变好了"判定，必须用数据说话——同类投诉是否真正下降、下降幅度是否达标、下降是持续性的还是偶然波动。本技能是VOC闭环的最后关键环节，确保改善真正产生了效果。

## 适用场景
- 改进措施标记完成后启动3个月效果观察
- 观察期内的月度效果评估
- 观察期结束后的最终效果判定
- 效果未达标时的原因分析与后续建议
- 效果回弹时的异常检测

## 执行步骤

### 步骤1：基线数据确认
在观察期开始前确认基线数据：
- 改进前的同类投诉数量（取改进前3个月月均值作为基线）
- 基线期间的订单量/产量（用于归一化对比）
- 基线期间的外部因素记录（季节/客户结构变化等）
- 确认监控范围：同一缺陷模式的定义边界（关键词/分类代码）
- 确认监控数据源一致性（观察期数据来源与基线期一致）

### 步骤2：观察期持续监控
3个月观察期内每月执行：
- 统计当月同类投诉数量
- 计算投诉率（投诉数/当月产量或发货量，消除产量波动影响）
- 对比基线投诉率计算下降幅度
- 绘制投诉趋势折线图（基线期+观察期）
- 检测异常信号（投诉突然回升）

### 步骤3：统计显著性检验
观察期结束时执行统计检验：
- **方法选择**：
  - 计数数据（投诉次数）：泊松检验或卡方检验
  - 比率数据（投诉率）：比例检验（z-test for proportions）
- **零假设H0**：改善前后投诉率无显著差异
- **备择假设H1**：改善后投诉率显著下降
- **显著性水平**：α=0.05
- **输出**：p值、置信区间、效应量（下降百分比）

### 步骤4：效果判定
基于统计检验结果做出判定：

**达标（改善关闭）条件（需同时满足）：**
- 同类投诉下降>=50%（vs基线）
- 统计检验显著（p<0.05）
- 观察期内无明显回弹趋势（后2个月不比第1个月高）

**未达标处理：**
- 下降幅度在30-50%：部分有效，建议补充强化措施
- 下降幅度<30%：效果不明显，建议重新分析根因
- 无下降或上升：措施无效，需重新启动改善周期

**特殊情况：**
- 产量大幅变化导致绝对数量变化但比率未变：以比率为准
- 外部因素干扰（如主要客户停止订单）：标注并延长观察期
- 改善措施未完全执行：标注为"验证无效-措施执行不完整"

### 步骤5：结论输出与后续动作
- **达标**：
  - 输出效果验证报告
  - 通知责任部门在30天内完成标准化（更新SOP/作业指导书/检验标准）
  - 改善记录归档
  - 设置6个月后的回访检查点（防回弹）
- **未达标**：
  - 输出效果分析报告（含可能原因分析）
  - 建议后续行动方案
  - 保持任务"待改进"状态，不关闭

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| improvement_task_id | String | 是 | 改进任务ID |
| baseline_data | Object | 是 | 基线期投诉数据 |
| observation_data | Object | 是 | 观察期投诉数据 |
| production_volume | Object | 是 | 各期产量/发货量数据 |
| external_factors | Array | 否 | 外部干扰因素记录 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| effectiveness_score | Number | 效果评分（下降百分比）|
| statistical_test | Object | 统计检验结果（p值/置信区间）|
| verdict | String | 判定结论（达标/部分有效/未达标/无效）|
| trend_chart_data | Object | 趋势图数据 |
| recommended_actions | Array | 建议后续动作 |
| verification_report | Object | 完整验证报告 |

## 最佳实践
- 基线数据是一切对比的基础——改善启动前必须确认并冻结基线
- 不要只看绝对数量，要看比率（消除产量波动的影响）
- 对于季节性产品，基线应取去年同期数据做补充参考
- 效果验证需要"耐心"——3个月观察期不能缩短，短期波动不能代表长期效果
- 达标后的标准化步骤不可省略——否则改善效果会在人员更替后消失

## 约束条件
- 效果验证观察期固定为3个月，不可提前关闭（除非管理层特批）
- 观察期内数据追踪不得中断（中断超过2周需重新开始观察期）
- 效果判定必须有统计检验支撑，不接受主观判断
- 达标后的标准化必须在30天内完成，否则不予最终关闭
- 效果评判标准（>=50%下降）为刚性要求，不可协商降低
