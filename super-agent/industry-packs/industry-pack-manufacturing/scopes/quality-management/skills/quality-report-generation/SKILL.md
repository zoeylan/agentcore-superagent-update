# 质量报告生成 (Quality Report Generation)

## 概述
本技能负责自动生成各类质量管理报告，包括检验日报/周报、质量趋势分析报告和异常汇总报告。通过结构化数据汇总和可视化呈现，为质量管理决策提供及时、准确的数据支撑。

## 适用场景
- 每日检验情况日报生成（IQC/IPQC/OQC）
- 每周质量趋势分析报告
- 供应商来料质量月度报告
- SPC状态汇总报告
- 质量异常统计与分布分析
- 管理层质量看板数据更新

## 执行步骤

### 步骤1：数据采集与汇总
1. 采集指定时间范围内的检验记录数据
2. 按维度分组统计：
   - 检验类型维度：IQC/IPQC/OQC各自的检验批数、合格率
   - 供应商维度：各供应商来料批次数、合格率、不良类型分布
   - 工序维度：各工序首件通过率、巡检合格率、CPK值
   - 产品维度：各产品/型号的质量表现
   - 缺陷类型维度：TOP N缺陷类型Pareto分布
3. 计算关键指标：直通率(FPY)、各类合格率、PPM水平

### 步骤2：趋势分析
1. 与前一周期（日/周/月）对比，计算变化率
2. 识别显著变化项（变化超过±2个标准差或超过阈值）
3. 标注趋势方向：改善↑、恶化↓、稳定→
4. 对于恶化趋势，关联可能原因（新供应商引入/设备维护/工艺变更等）

### 步骤3：异常项标注
1. 标注超出目标值的指标（红色标记）
2. 标注接近目标值的指标（黄色预警）
3. 关联未关闭的质量异常单
4. 列出新发现的异常模式

### 步骤4：报告编制与输出
1. 按报告模板组织内容结构
2. 生成关键指标仪表盘（KPI summary）
3. 附上数据明细和趋势图表描述
4. 添加关键发现和建议措施
5. 标注需要关注/决策的事项

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| report_type | enum | 是 | daily/weekly/monthly/custom |
| time_range | object | 是 | 报告时间范围（start_date, end_date） |
| scope | enum | 是 | IQC/IPQC/OQC/ALL |
| dimensions | array | 否 | 分析维度（供应商/工序/产品/缺陷类型） |
| target_metrics | object | 否 | 目标值（用于标注达标情况） |
| comparison_period | object | 否 | 对比周期 |
| recipients | array | 否 | 报告接收对象（决定内容详略） |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| report_id | string | 报告唯一编号 |
| summary | object | 核心指标摘要（合格率/FPY/CPK等） |
| kpi_dashboard | object | KPI仪表盘数据 |
| trend_analysis | array | 趋势分析结果列表 |
| anomaly_items | array | 异常项清单（含关联信息） |
| pareto_chart_data | object | Pareto图数据 |
| recommendations | array | 建议措施列表 |
| attention_items | array | 需管理层关注的事项 |
| detailed_data | object | 数据明细（附件） |

## 最佳实践
1. **金字塔原则**：报告结构遵循结论先行——摘要→关键发现→详细数据，管理层只看前两部分即可决策
2. **数据对比**：始终提供同比/环比对比，孤立的数字没有意义
3. **异常优先**：报告重点放在异常和变化上，正常项简要带过
4. **可操作性**：每个发现项都附带建议措施和责任方，避免"仅供参考"式的报告
5. **时效性**：日报在次日上午9:00前生成，周报在每周一上午生成，月报在每月3日前生成

## 约束条件
- 报告数据必须注明来源系统和提取时间，确保可追溯
- 合格率等指标必须注明计算口径（批次合格率vs件合格率）
- 涉及供应商具体名称的报告仅限内部使用，不得外发
- 趋势分析至少需要5个数据点才有统计意义
- 不在报告中做未经验证的因果推断，只描述相关性和可能方向

## 示例

### 示例：检验周报
```
输出结构：
═══════════════════════════════════════
 质量检验周报 | 2024年第15周 (04/08-04/12)
═══════════════════════════════════════

【核心指标摘要】
┌─────────────┬───────┬───────┬──────┐
│ 指标        │ 本周  │ 上周  │ 趋势 │
├─────────────┼───────┼───────┼──────┤
│ IQC合格率   │ 98.5% │ 99.1% │ ↓    │
│ IPQC直通率  │ 98.2% │ 98.0% │ →    │
│ OQC合格率   │ 99.8% │ 99.7% │ →    │
│ 综合CPK     │ 1.45  │ 1.48  │ →    │
└─────────────┴───────┴───────┴──────┘

【⚠️ 需关注事项】
1. 供应商A-012来料合格率连续2周下降(99.5%→98.2%→96.8%)
   - 不良类型：外径超差（占比70%）
   - 建议：发起SCAR，要求供应商排查磨削工序

2. 注塑工序#3 CPK降至1.28（低于1.33目标）
   - 特性：壁厚尺寸
   - 建议：检查模具磨损状况，排查料温波动

【本周检验统计】
- IQC: 检验68批，合格67批，不合格1批（供应商B-005，表面划伤）
- IPQC: 巡检412次，异常8次（已处理7次，1次跟踪中）
- OQC: 出货42批，合格42批（让步接收0批）

【TOP5缺陷类型 - Pareto】
1. 尺寸超差 (35%) - 累计35%
2. 表面缺陷 (22%) - 累计57%
3. 外观色差 (18%) - 累计75%
4. 装配干涉 (12%) - 累计87%
5. 功能异常 (8%)  - 累计95%
```
