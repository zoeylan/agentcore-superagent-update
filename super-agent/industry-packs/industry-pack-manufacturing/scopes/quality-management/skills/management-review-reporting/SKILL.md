# 管理评审报告 (Management Review Reporting)

## 概述
本技能负责编制月度/季度质量管理评审报告，为管理层提供全面的质量绩效数据、趋势分析、风险预警和决策建议。管理评审是ISO 9001质量管理体系的核心要求之一（条款9.3），通过定期回顾质量目标达成情况、识别系统性风险和改进机会，确保质量管理体系的持续适宜性和有效性。

## 适用场景
- 月度质量绩效报告编制（每月5日前完成上月报告）
- 季度质量管理评审会议材料准备
- 年度管理评审报告（含年度质量目标回顾与新年目标设定）
- 客户审核前的质量绩效数据准备
- 重大质量事件后的专项分析报告

## 执行步骤

### 步骤1：数据采集与汇总
1. 采集关键质量指标（KPI）数据：
   - 直通率（FPY）：各产线/产品的月度直通率
   - 客诉率：客诉数量/出货批次数
   - 来料合格率：IQC合格批次数/来料总批次数
   - 8D按时关闭率：30天内关闭数/总8D数
   - 关键工序CPK：各关键特性的CPK月度均值
   - 质量成本：预防成本/鉴定成本/内部失败成本/外部失败成本
2. 采集过程数据：
   - SPC失控事件数量和处理情况
   - 不合格品处置记录（返工/报废/让步接收数量和金额）
   - 供应商质量绩效（SCAR数量/关闭率/TOP不良供应商）
3. 确认数据来源可靠性和完整性

### 步骤2：趋势分析与对比
1. 绘制KPI趋势图（至少展示12个月数据用于识别长期趋势）
2. 与目标值对比：标注达标/未达标项，计算差距
3. 同比/环比分析：识别改善或恶化趋势
4. Pareto分析：识别TOP5缺陷类型、TOP5不良工序、TOP5问题供应商
5. 标注异常月份并关联原因（如新产品导入、设备大修、季节性因素等）

### 步骤3：问题分析与风险预警
1. 未达标指标的原因分析：
   - 是偶发性波动还是系统性恶化？
   - 根因是否已识别并在处理中？
   - 预计恢复达标的时间节点
2. 风险预警：
   - 趋势性恶化但尚未突破目标的指标（黄灯预警）
   - 外部风险信号（新法规要求、客户标准变更、供应商异常等）
   - 改进项目延期可能带来的风险
3. 质量成本分析：
   - 质量成本占营收比（目标≤3%）
   - 四类成本的比例变化（健康比例：预防>鉴定>内部失败>外部失败）

### 步骤4：改进项目状态汇总
1. 在推进中的六西格玛/改善项目清单：
   - 项目名称、当前阶段（DMAIC哪个阶段）
   - 本月进展和下月计划
   - 项目预期收益和实际进度对比
2. 已关闭项目的效果验证：
   - 改善前后数据对比
   - 收益量化（质量成本降低额/产能提升额）
3. 新识别的改进机会（建议立项）

### 步骤5：建议决策项输出
1. 需要管理层决策的事项：
   - 资源投入请求（增加检测设备/人员/培训等）
   - 供应商淘汰/新增决策
   - 重大投资项目审批（设备改造/自动化等）
   - 质量目标调整建议
2. 每个决策项必须附带：
   - 背景和数据支撑
   - 方案选项和利弊分析
   - 建议方案和预期效果

### 步骤6：报告编制与发布
1. 按标准模板编制报告
2. 关键数据交叉验证（确保同一指标在不同图表中一致）
3. 确保每页有明确的结论或行动项（避免纯数据堆砌）
4. 报告发布前经质量经理审核确认

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| report_type | enum | 是 | MONTHLY/QUARTERLY/ANNUAL/SPECIAL |
| period | string | 是 | 报告期间（如2024-Q1） |
| kpi_targets | object | 是 | 各指标目标值 |
| quality_data | object | 是 | 质量数据（来自QMS系统） |
| eight_d_records | array | 是 | 期间内8D记录 |
| improvement_projects | array | 是 | 改进项目清单 |
| cost_data | object | 否 | 质量成本数据 |
| previous_action_items | array | 否 | 上次评审决策事项跟踪 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| report_id | string | 报告编号 |
| executive_summary | string | 执行摘要（1页以内关键信息） |
| kpi_dashboard | object | KPI仪表盘（含趋势和RAG状态） |
| trend_analysis | object | 趋势分析（含Pareto图） |
| risk_alerts | array | 风险预警清单 |
| project_status | array | 改进项目状态汇总 |
| decision_items | array | 需管理层决策事项 |
| action_items | array | 行动项清单（含责任人/时限） |

## 最佳实践
1. **一页纸摘要**：管理层时间有限，报告第一页必须是执行摘要，一目了然展示整体状况（绿/黄/红）
2. **趋势优于快照**：不要只展示本月数据，必须有趋势线，管理层需要判断方向而非仅仅知道当前值
3. **结论先行**：每个分析板块先给出结论/判断，再展示支撑数据，而非让管理层自己从数据中找答案
4. **行动导向**：报告中每个问题都应有对应的行动计划或需要管理层做出的决策
5. **数据一致性**：同一指标在报告不同位置必须一致，避免因计算口径不同造成混淆

## 约束条件
- 月度报告必须在次月5日前完成，季度报告在季度结束后10日内完成
- 所有KPI数据必须有明确的数据来源和计算方法说明
- 报告中不出现未经验证的推测性结论，所有分析必须有数据支撑
- 决策建议必须有ROI分析，不提无依据的投资请求
- 不在报告中暴露客户机密信息或未授权的供应商商务信息
- 上次评审的决策事项必须100%跟踪闭环

## 示例

### 示例1：月度KPI仪表盘
```
2024年3月质量绩效摘要：

| 指标 | 目标 | 实际 | 趋势 | 状态 |
|------|------|------|------|------|
| 直通率(FPY) | ≥98% | 97.6% | ↓ | 🟡黄灯 |
| 客诉率 | ≤0.5% | 0.3% | → | 🟢绿灯 |
| 来料合格率 | ≥99% | 99.2% | ↑ | 🟢绿灯 |
| 8D按时关闭率 | ≥90% | 85% | ↓ | 🔴红灯 |
| 关键工序CPK | ≥1.33 | 1.41 | → | 🟢绿灯 |
| 质量成本占比 | ≤3% | 2.8% | → | 🟢绿灯 |

关键发现：
1. 直通率连续2个月低于目标，主因为B线注塑工序缩孔缺陷率上升（0.8%→1.2%）
2. 8D关闭率下降主因：本月新增8D数量激增（12件 vs 月均8件），其中5件来自新供应商来料问题
3. 建议决策：B线注塑工序缩孔问题建议立项为六西格玛改进项目（预估年化收益35万元）
```
