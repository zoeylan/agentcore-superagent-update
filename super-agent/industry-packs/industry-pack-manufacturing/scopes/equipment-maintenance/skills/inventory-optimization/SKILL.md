# 库存优化分析 (Inventory Optimization)

## 技能概述
对备件库存进行系统性优化分析，运用ABC-XYZ分类、库存周转率分析、经济订购批量（EOQ）计算等方法，在备件可用性和库存成本之间找到最佳平衡点。目标是库存周转率>=2次/年，同时保障设备维修不因缺件停滞。

## 适用场景
- 季度库存策略评审和参数调整
- 新设备投用后的备件库存初始规划
- 库存周转率偏低时的优化分析
- 备件成本压降项目
- 设备淘汰后的库存清理规划

## 执行步骤

### 步骤1：ABC-XYZ分类分析
- ABC分类（按价值）：
  - A类：占库存总值70%的备件（通常数量占20%）→ 严格管控
  - B类：占库存总值20%的备件 → 适度管控
  - C类：占库存总值10%的备件 → 简化管理
- XYZ分类（按需求波动性）：
  - X类：需求稳定可预测（变异系数CV<0.5）→ 定量订货
  - Y类：需求有一定波动（CV 0.5-1.0）→ 定期订货
  - Z类：需求不规则难预测（CV>1.0）→ 按需订货
- 交叉矩阵策略：AX→精细管控高安全库存；CZ→最低库存按需采购

### 步骤2：库存周转率分析
- 计算整体备件库存周转率 = 年消耗金额 / 平均库存金额
- 按品类分析周转率差异
- 识别低周转备件（周转率<1次/年）和零周转备件（180天无动态）
- 分析低周转原因：安全库存过高/采购批量过大/需求减少/设备淘汰

### 步骤3：EOQ和库存参数优化
- 对需求稳定的备件计算经济订购批量（EOQ）：
  - EOQ = √(2×年需求量×单次订购成本 / 单位年持有成本)
- 重新计算安全库存：
  - 考虑服务水平目标（A类设备备件99%、B类设备备件95%）
  - 考虑采购提前期波动性
  - 安全库存 = Z值 × √(提前期方差×平均需求² + 平均提前期×需求方差)
- 调整再订货点和最高库存

### 步骤4：优化建议输出
- 库存参数调整建议（哪些备件的安全库存/批量需要调整）
- 呆滞备件处置建议（清理路径和预期释放金额）
- 采购模式优化建议（合并采购/框架协议/VMI等）
- 预期效果量化：库存金额降低X万元、周转率提升至Y次/年
- 风险提示：过度压降库存可能导致缺件风险增加

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| inventory_data | array | 是 | 完整备件库存数据（含金额）|
| consumption_history | array | 是 | 12个月消耗记录 |
| equipment_criticality | object | 是 | 设备关键程度映射 |
| lead_time_data | array | 否 | 各备件采购提前期 |
| holding_cost_rate | number | 否 | 年库存持有成本率（默认25%）|

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| abc_xyz_classification | array | ABC-XYZ分类结果 |
| turnover_analysis | object | 周转率分析 |
| parameter_adjustments | array | 库存参数调整建议 |
| obsolete_disposal_plan | array | 呆滞备件处置方案 |
| expected_savings | object | 预期节约金额和周转率提升 |
| risk_assessment | array | 风险评估 |

## 最佳实践
- 库存优化不等于简单压降库存——核心是"精准匹配需求"
- A类设备关键备件"宁可多备不可缺件"（缺件停机成本远高于库存持有成本）
- 与供应商谈VMI（供应商管理库存）可释放库存资金占用
- 通用件（轴承/密封件等多设备通用）应合并管理提高周转
- 优化方案实施需分阶段渐进式执行，避免激进调整导致缺件

## 约束条件
- 库存优化分析每季度执行一次
- A类设备关键备件的安全库存调整需设备经理审批
- 库存周转率提升不得以增加缺件风险为代价
- 呆滞备件报废金额>5万元需总监审批
- 优化建议需附风险评估和回退方案
