# ABC-XYZ物料分类管理

## 技能概述
运用ABC分类（按价值/金额）与XYZ分类（按需求波动性）的交叉分析方法，对全品类物料进行系统化分类，并为每个类别制定差异化的库存管理策略。这是库存精细化管理的基础方法论。

## 适用场景
- 年度物料分类更新（每年至少一次全面重分类）
- 新物料导入时的初始分类
- 库存策略优化项目中的分析基础
- 物料管理改善课题的数据支撑
- 供应链策略审查中的物料优先级排序

## 执行步骤

### 步骤1：ABC分类（价值维度）
- 提取12个月物料消耗数据（数量×单价=金额）
- 按年消耗金额降序排列
- 分类标准：
  - **A类**：累计金额前70%的物料（通常占SKU数的15-20%）
  - **B类**：累计金额70%-90%的物料（通常占SKU数的30%）
  - **C类**：累计金额后10%的物料（通常占SKU数的50-55%）
- 对分界处的物料进行合理性校验（避免机械切分）

### 步骤2：XYZ分类（波动维度）
- 计算每个物料12个月消耗量的变异系数CV = 标准差 / 平均值
- 分类标准：
  - **X类**（稳定）：CV < 0.5，需求可预测
  - **Y类**（波动）：0.5 <= CV < 1.0，需求有一定波动
  - **Z类**（极不稳定）：CV >= 1.0，需求难以预测
- 对新物料（<6个月数据）标注"数据不足"，暂按Y类管理

### 步骤3：交叉分类与策略制定
| 分类 | 策略建议 |
|------|---------|
| AX | 精准控制：低安全库存+高频补货+JIT |
| AY | 适度缓冲：中安全库存+滚动预测 |
| AZ | 谨慎管理：较高安全库存+多源备选+密切监控 |
| BX | 标准管理：正常安全库存+定期补货 |
| BY | 灵活调整：动态安全库存+按需调整 |
| BZ | 风险意识：较高安全库存+框架协议 |
| CX | 简化管理：批量采购+低管理投入 |
| CY | 最小化关注：寄售/VMI+定期review |
| CZ | 评估必要性：是否可删减/合并/标准化 |

### 步骤4：管理投入度差异化
- A类物料：每日监控，每周review，每月优化
- B类物料：每周监控，每月review，每季度优化
- C类物料：每月监控，每季度review，年度优化
- 盘点频率：A类月度盘点，B类季度盘点，C类年度盘点

### 步骤5：分类结果输出与建议
- 生成物料分类总表（物料编码/名称/ABC类/XYZ类/策略建议）
- 对比上期分类变化（升级/降级物料标注）
- 输出各类别物料数量和金额占比统计
- 标注与当前库存策略不匹配的物料（如AX物料库存过高）

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| material_consumption_data | Array | 是 | 12个月物料消耗明细 |
| material_master | Array | 是 | 物料主数据（编码/名称/规格/单价） |
| current_classification | Object | 否 | 当前分类（用于对比变化） |
| business_override_rules | Array | 否 | 业务覆盖规则（如客户指定料强制A类） |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| classification_result | Array | 物料分类结果明细 |
| category_statistics | Object | 各类别统计（数量/金额/占比） |
| strategy_recommendations | Array | 各类别管理策略建议 |
| change_highlights | Array | 与上期对比的分类变化 |
| misalignment_alerts | Array | 当前策略与分类不匹配项 |

## 最佳实践
- 分类不是一成不变的，至少年度更新一次
- 业务属性可覆盖统计分类（如安全物资不论金额均按A类管理）
- XYZ分类需排除异常事件影响（如一次性大单不计入CV计算）
- 与Kraljic矩阵结合使用效果更佳（ABC-XYZ管库存，Kraljic管采购策略）

## 约束与边界
- 分类是管理辅助工具，不是绝对标准
- 需要至少6个月历史数据才能进行可靠的XYZ分类
- 分类结果变更不自动触发参数修改（需人工审核确认后执行）
- 对季节性物料的CV计算需注明季节因素影响
