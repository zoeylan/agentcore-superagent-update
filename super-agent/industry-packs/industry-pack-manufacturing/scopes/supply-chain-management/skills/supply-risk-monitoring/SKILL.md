# 供应链风险信号监控

## 技能概述
持续监控影响供应链稳定性的多维度风险信号，包括原材料价格波动、供应商经营异常、物流中断、汇率变动和地缘政治风险，进行实时分级评估和早期预警，为供应链决策提供前瞻性情报支持。

## 适用场景
- 每日供应链风险信号自动扫描
- 突发事件（自然灾害/政策变动/供应商事故）快速影响评估
- 原材料价格大幅波动时的采购策略建议
- 物流通道中断时的替代路径评估
- 汇率大幅变动时的采购成本影响分析

## 执行步骤

### 步骤1：风险信号采集
- **大宗商品价格**：监控关键原材料（钢材/铜/铝/塑料粒子/芯片等）价格指数日变化
- **供应商舆情**：扫描供应商相关新闻（裁员/诉讼/事故/资金链问题）
- **物流信号**：港口拥堵指数、海运费率变化、主要运输通道状态
- **汇率变动**：采购相关币种（USD/EUR/JPY）兑人民币汇率变动
- **政策法规**：贸易制裁更新、关税调整、环保政策变化
- **气象灾害**：供应商所在区域的极端天气预警

### 步骤2：风险影响评估
对每个检测到的风险信号评估：
- **影响范围**：涉及哪些供应商/物料/在途订单
- **影响程度**：预估对成本/交期/供应连续性的量化影响
- **时间维度**：风险影响的开始时间和预计持续时间
- **连锁效应**：是否引发二次风险（如物流中断→库存断料→产线停产）

### 步骤3：风险分级与优先级
- **一级风险（紧急）**：已经或即将导致供应中断/产线停产，4小时内报告
- **二级风险（重要）**：短期内可能影响交期或成本>5%，24小时内报告
- **三级风险（关注）**：中长期可能演变的潜在风险，周度汇总报告
- 按「物料关键性×风险等级」排列处理优先级

### 步骤4：预警输出与建议
- 一级风险：立即推送给供应链总监和相关Agent，附带应急建议
- 二级风险：推送给供应商关系管理专员和采购计划专员
- 三级风险：纳入周度风险简报
- 建议内容：
  - 短期应对（提前备货/切换供应商/调整运输路线）
  - 中期对策（开发替代来源/调整安全库存水位）
  - 长期策略（供应商结构优化/本地化替代）

### 步骤5：风险演化跟踪
- 已识别风险持续跟踪直到解除或影响消退
- 定期更新风险状态（升级/持平/降级/关闭）
- 风险解除后复盘：预警是否及时、应对是否有效
- 更新风险知识库（新增风险类型/应对案例）

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| commodity_price_data | Object | 是 | 大宗商品价格数据 |
| supplier_news_feed | Array | 否 | 供应商相关新闻/舆情 |
| logistics_status | Object | 否 | 物流通道状态数据 |
| exchange_rate_data | Object | 否 | 汇率变动数据 |
| material_supplier_mapping | Object | 是 | 物料-供应商映射关系 |
| critical_material_list | Array | 是 | 关键物料清单 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| risk_alerts | Array | 风险预警列表（含级别/影响/建议） |
| price_volatility_report | Object | 价格波动分析报告 |
| logistics_risk_status | Object | 物流风险状态 |
| impact_assessment | Object | 对本企业供应链的影响评估 |
| action_recommendations | Array | 建议行动方案 |

## 最佳实践
- 建立关键物料的价格波动阈值（如±5%触发关注，±10%触发行动）
- 对高度依赖进口的物料，汇率和物流风险赋予更高权重
- 风险信号需多源交叉验证，避免单一信息源导致误判
- 与采购策略联动：风险升高时适当增加安全库存或提前采购

## 约束与边界
- 提供风险情报和建议，不直接执行采购/库存调整决策
- 价格预测是趋势性判断，不保证准确性
- 地缘政治风险评估存在高度不确定性，需标注置信度
- 舆情信息可能存在滞后或不准确，需标注来源可信度
