# 供应商风险持续监控

## 技能概述
对AVL中所有活跃供应商进行持续性风险监控，识别经营、财务、产能、合规等维度的风险信号，维护供应商风险地图，为供应商管理决策提供早期预警支持。

## 适用场景
- 每日/每周供应商风险信号扫描（自动化监控）
- 供应商重大事件发生后的快速风险评估
- 季度供应商风险地图更新
- 年度供应商结构优化中的风险维度分析
- 关键物料供应商的深度尽职调查

## 执行步骤

### 步骤1：多维度风险信号采集
- **财务风险**：供应商公开财务数据（营收变化/利润率/负债率）、工商信息变更、诉讼/被执行信息
- **产能风险**：供应商产能利用率变化、关键设备更新情况、人员流动率
- **质量风险**：来料质量趋势恶化、认证体系到期未续、客户投诉增多
- **交付风险**：交期达成率下降趋势、频繁延期确认、产能紧张信号
- **地缘/合规风险**：所在地区政策变动、环保/安全事故、贸易制裁

### 步骤2：风险信号评级
- 每个风险信号按发生概率（1-5）× 影响程度（1-5）计算风险值
- 风险值 >= 15：高风险（红色）
- 风险值 10-14：中风险（黄色）
- 风险值 < 10：低风险（绿色）
- 对高风险信号标注信息来源和时效性

### 步骤3：风险地图更新
- 按供应商维度汇总各供应商的综合风险等级
- 按物料维度分析各关键物料的供应风险暴露度
- 标注风险集中区域（如：某一地区多家供应商/某一行业集体风险）
- 更新风险地图可视化（供应商×风险维度矩阵）

### 步骤4：预警输出与建议
- 高风险供应商：24小时内通知供应商关系管理专员
- 新出现的风险信号：评估是否需要调整安全库存或启动替代供应商开发
- 风险趋势恶化：建议降低该供应商订单份额或加速替代方案
- 输出风险简报（核心风险事项+建议动作+优先级）

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| supplier_list | Array | 是 | 需监控的供应商列表 |
| financial_data_sources | Array | 否 | 财务数据源接入 |
| quality_performance_data | Object | 否 | 近期质量表现数据 |
| delivery_performance_data | Object | 否 | 近期交付表现数据 |
| external_news_signals | Array | 否 | 外部新闻/舆情信号 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| risk_alerts | Array | 新增/升级的风险预警 |
| risk_map | Object | 更新后的供应商风险地图 |
| high_risk_suppliers | Array | 高风险供应商列表及详情 |
| recommended_actions | Array | 建议的应对行动 |
| risk_trend_analysis | Object | 风险趋势分析 |

## 最佳实践
- 信息源多样化：不依赖单一渠道，交叉验证风险信号
- 区分"噪音"和"信号"：不是所有负面信息都构成实质风险
- 关注领先指标：财务恶化通常先于交付/质量问题出现
- 风险监控与绩效评分联动：风险信号应反映在下季度评分中

## 约束与边界
- 不做供应商淘汰决策（仅预警和建议）
- 风险评估基于公开可获取信息，不涉及商业机密
- 财务判断需标注信息时效性和数据完整度
- 地缘政治风险评估标注不确定性等级
