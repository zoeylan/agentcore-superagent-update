# 采购成本优化分析

## 技能概述
通过分析采购历史数据、市场价格趋势和批量效应，识别采购成本优化机会并提出具体降本策略。支持年度成本节降率>=3%的目标达成。

## 适用场景
- 年度采购预算编制时的成本分析
- 季度采购成本回顾与降本机会识别
- 供应商报价评估与比价分析
- 采购批量优化决策（EOQ计算）
- 采购策略调整建议（集中采购/分散采购）

## 执行步骤

### 步骤1：历史数据分析
- 提取12个月采购明细数据（物料/供应商/单价/数量/金额）
- 按物料类别计算采购支出Pareto分析（识别TOP20占80%金额的物料）
- 分析价格趋势（同一物料12个月价格走势）
- 对比同一物料不同供应商的价格差异

### 步骤2：成本结构拆解
- 对高金额物料进行成本结构分析：
  - 原材料成本占比（关联大宗商品指数）
  - 加工成本占比（关联供应商工时和良率）
  - 物流成本占比（运费/包装/关税）
  - 利润加成估算
- 识别成本驱动因素和可优化空间

### 步骤3：优化策略制定
- **批量优化**：计算EOQ，评估增加单次采购量的价格折扣收益 vs 额外库存持有成本
- **集中采购**：识别可合并的同类物料/同一供应商的采购量，获取规模折扣
- **供应商切换**：对比现有供应商与潜在替代供应商的价格竞争力
- **合约谈判**：基于市场价格走势和采购量预测，建议年度合约锁价策略
- **规格标准化**：识别功能相同但规格不一的物料，建议标准化降低品种数

### 步骤4：降本效果预估
- 每项优化策略量化预估节省金额
- 评估实施难度和周期
- 按ROI排序优化机会优先级
- 汇总年度降本目标达成预测

### 步骤5：风险评估
- 评估降本策略对质量风险的影响
- 评估对供应连续性的影响
- 评估对供应商关系的影响
- 标注需要质量验证的降本措施（替代物料/新供应商）

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| purchase_history | Array | 是 | 12个月采购明细数据 |
| commodity_indices | Object | 否 | 大宗商品价格指数 |
| supplier_quotes | Array | 否 | 供应商最新报价 |
| inventory_holding_cost_rate | Number | 否 | 库存持有成本率（默认25%/年） |
| annual_cost_target | Number | 否 | 年度降本目标金额 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| cost_analysis_report | Object | 采购成本分析报告 |
| optimization_opportunities | Array | 优化机会列表（含量化金额） |
| priority_ranking | Array | 按ROI排序的优先级 |
| risk_assessment | Array | 各策略的风险评估 |
| annual_savings_forecast | Number | 预估年度节省金额 |

## 最佳实践
- 降本分析必须结合质量和交期维度，纯价格导向可能导致综合成本上升
- TCO（总持有成本）视角：考虑隐性成本（质量问题处理成本、缺料停产损失）
- 降本策略需分期实施，避免同时变更过多引发风险
- 与供应商保持透明沟通，避免单纯压价损害合作关系

## 约束与边界
- 不直接与供应商进行价格谈判（由采购人员执行）
- 不做质量等效性判定（替代物料需工程和质量部门验证）
- 降本效果需在实施后追踪验证，预估值可能与实际有偏差
- 涉及合约变更的建议需法务审核
