# 产能规划 (Capacity Planning)

## 概述
基于设备实际产能数据、设备健康度预测和历史产出分析，评估产线当前和未来可用产能，为排产决策和订单承接提供产能约束信息。

## 适用场景
- 新订单评审时的产能可行性评估
- 设备维护/故障对产能影响的量化分析
- 中长期产能规划（季度/年度）
- 产能瓶颈预警和扩产建议
- 多产品组合下的产能分配优化

## 执行步骤

### 步骤1：当前产能盘点
- 计算各设备/产线的理论最大产能：
  - 理论产能 = 可用时间 / 理论CT × 良品率
- 计算实际有效产能（考虑OEE影响）：
  - 有效产能 = 理论产能 × OEE
- 盘点当前产能占用情况（已排订单占用的产能百分比）
- 计算剩余可用产能 = 有效产能 - 已占用产能

### 步骤2：产能影响因素评估
- **设备健康度影响**：健康度低的设备预期OEE会下降，需折减产能预期
- **维护计划影响**：已排期的PM将占用产能时间，需扣除
- **人员因素**：各班次人员配置差异导致的产能差异
- **产品组合影响**：不同产品CT不同，产品组合变化影响总体产能
- **换线损失**：频繁换线降低有效产能

### 步骤3：未来产能预测
- 基于设备退化趋势预测未来2-4周的可用产能
- 纳入已知的维护计划和设备升级计划
- 考虑季节性因素（如夏季高温导致设备降速运行）
- 输出产能日历（每日可用产能预测值）

### 步骤4：产能决策支持
- 新订单可行性评估：当前剩余产能是否能满足新订单需求
- 产能不足时的应对建议：
  - 加班排产（评估人工成本）
  - 委外加工（评估质量风险和交期风险）
  - 订单优先级调整（与客户协商延期交付）
  - 产能扩充投资建议（长期不足时）
- 输出产能利用率报告和瓶颈设备清单

## 输入规格
| 字段 | 类型 | 说明 |
|------|------|------|
| equipment_list | array | 设备清单及其参数（CT/可用时间） |
| current_oee | object | 各设备当前OEE数据 |
| committed_orders | array | 已排产订单列表 |
| maintenance_schedule | object | 维护计划 |
| health_predictions | object | 设备健康度预测 |
| new_order_request | object | 新订单需求（可选） |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| total_capacity | object | 总产能（理论/有效/剩余） |
| capacity_by_equipment | array | 按设备的产能明细 |
| capacity_calendar | array | 未来产能日历 |
| utilization_rate | number | 当前产能利用率 |
| bottleneck_equipment | array | 产能瓶颈设备 |
| feasibility_assessment | object | 新订单可行性评估（如适用） |
| recommendations | array | 产能优化建议 |

## 最佳实践
- 产能规划不能只看理论值，必须用OEE修正后的有效产能做计划
- 建议保留10-15%的产能缓冲应对紧急订单和设备故障
- 产能预测结合设备健康度趋势，比纯历史平均更准确
- 季度性的产能审视有助于提前识别产能缺口

## 约束条件
- 产能评估必须基于验证过的标准工时数据
- 产能利用率超过90%时应发出预警（余量不足）
- 产能扩充建议（投资类）需经过ROI分析后方可提出
- 委外加工建议需附带质量保障措施说明
