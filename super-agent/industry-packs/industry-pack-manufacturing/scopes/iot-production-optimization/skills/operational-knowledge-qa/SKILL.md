# 操作知识问答 (Operational Knowledge QA)

## 概述
基于设备操作手册、工艺文件、维护规程等知识库，提供智能问答服务，帮助操作人员快速获取设备操作、工艺参数、维护要求等方面的知识。

## 适用场景
- 设备操作规程查询（"注塑机开机步骤是什么"）
- 工艺参数标准查询（"这个产品的注塑温度标准是多少"）
- 维护规程查询（"液压系统多久换一次油"）
- 安全规程查询（"操作冲压机需要哪些防护装备"）
- 质量标准查询（"这个零件的公差要求是多少"）

## 执行步骤

### 步骤1：问题理解
- 解析用户问题，识别知识域：
  - 设备操作类：开关机步骤、操作顺序、模式切换
  - 工艺参数类：温度/压力/速度等标准值和范围
  - 维护保养类：保养周期、保养步骤、检查项目
  - 安全规程类：PPE要求、危险源、应急程序
  - 质量标准类：尺寸公差、外观标准、检验方法
- 提取上下文：具体设备型号、产品型号、工序名称

### 步骤2：知识检索
- 从知识库中检索相关文档：
  - 设备操作手册（按设备型号索引）
  - 作业指导书（SOP/WI按工序索引）
  - 维护规程（PM checklist按设备索引）
  - 工艺卡/控制计划（按产品+工序索引）
  - 安全管理规程（按区域/设备索引）
- 采用语义检索+关键词匹配混合策略，确保检索准确性

### 步骤3：答案生成
- 从检索到的文档中提取关键信息，组织为简洁的回答
- 回答格式根据问题类型适配：
  - 步骤类：编号列表，每步清晰具体
  - 参数类：表格形式（参数名/标准值/允许范围/单位）
  - 知识类：段落式说明，附带注意事项
- 标注信息来源（哪个文档/哪个版本）以便用户追溯验证
- 对于知识库未覆盖的问题，明确告知"未找到相关信息"并建议咨询途径

### 步骤4：补充与引导
- 根据用户问题预判可能的后续问题，主动提供相关信息
- 对于安全敏感信息，额外强调注意事项和风险
- 如发现知识库信息可能过时（文档版本>1年未更新），提示用户确认是否为最新版本
- 记录高频问题，用于知识库优化和培训改善

## 输入规格
| 字段 | 类型 | 说明 |
|------|------|------|
| user_question | string | 用户问题文本 |
| context | object | 上下文信息（设备/工序/产品） |
| user_role | enum | 用户角色 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| answer | string | 回答内容 |
| source_document | object | 信息来源文档（名称/版本/章节） |
| confidence | enum | 回答置信度（high/medium/low） |
| related_topics | array | 相关话题推荐 |
| safety_notes | array | 安全注意事项（如有） |

## 最佳实践
- 回答应简洁实用，避免照搬手册全文（用户需要的是精准答案不是整本手册）
- 安全规程相关的回答要完整不能省略，宁可详细也不能遗漏
- 对于"标准是什么"类问题，回答后建议用户到现场确认最新版本
- 高频问题整理成FAQ，提升响应速度

## 约束条件
- 回答必须基于有据可查的文档知识，不得凭推测回答工艺参数
- 安全规程问题不得给出简化或省略版回答
- 对于过时文档（超过2年未更新），需标注提醒用户确认
- 涉及商业机密或受限信息的查询需验证访问权限
