# 声学特征分析 (Acoustic Feature Analysis)

## 概述
从实时语音流中提取关键声学特征（基频、语速、音量、停顿模式、共振峰），为情绪分类提供客观的数据基础。以200ms为分析窗口进行滑动窗口处理，确保分析的实时性。

## 适用场景
- 每一帧语音输入的实时声学分析
- 用户说话过程中的声学特征变化监测
- 为情绪分类模型提供输入特征向量
- 异常声学事件检测（突然大声、长时间静默）

## 执行步骤

### 1. 音频预处理
- 接收原始PCM音频流（16kHz/16bit/单声道）
- 执行预加重滤波（系数0.97）增强高频成分
- 按200ms窗口进行分帧，帧移50ms（75%重叠）
- 应用汉明窗减少频谱泄漏

### 2. 基频(F0)提取
- 使用自相关法(ACF)或CREPE深度模型提取每帧基频
- 正常说话F0范围：男性85-180Hz，女性165-255Hz
- 计算F0统计特征：均值、标准差、变化率、最大最小值
- F0剧烈波动（标准差>正常值2倍）提示情绪激动

### 3. 语速计算
- 基于VAD标注的有效说话段落
- 计算每秒音节数(SPS)：正常4-6音节/秒
- 加速（>7 SPS）可能表示焦急/愤怒
- 减速（<3 SPS）可能表示犹豫/悲伤
- 计算语速变化率（当前窗口vs最近5秒均值）

### 4. 音量(RMS能量)分析
- 计算每帧的RMS能量值
- 归一化到dB标度
- 检测音量突变：短时间内变化>10dB标记为突变事件
- 音量持续升高+语速加快组合模式高度关联愤怒情绪

### 5. 停顿模式分析
- 检测语句间和语句内的静默段
- 短停顿(<300ms)：正常呼吸或思考
- 中停顿(300ms-1s)：可能犹豫或不确定
- 长停顿(>1s)：可能困惑、失望或准备放弃
- 频繁短停顿（>3次/10秒）：可能紧张或焦虑

### 6. 特征向量输出
组装多维特征向量：
```
[F0_mean, F0_std, F0_slope, speed_sps, speed_change_rate,
 rms_mean, rms_std, rms_delta, pause_count, pause_avg_duration,
 jitter, shimmer, spectral_centroid, spectral_rolloff]
```

## 输入规格
| 字段 | 类型 | 说明 |
|------|------|------|
| audio_frame | bytes | 200ms PCM音频帧 |
| sample_rate | int | 采样率（16000） |
| bit_depth | int | 位深度（16） |
| frame_index | int | 帧序号 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| feature_vector | float[] | 14维声学特征向量 |
| anomaly_flags | string[] | 异常事件标记列表 |
| frame_timestamp | int64 | 帧对应的时间戳 |

## 最佳实践
- 使用滑动窗口均值平滑单帧噪声，避免单帧异常值触发误告警
- 对不同性别/年龄段使用差异化的基线参数
- 环境噪声自适应：根据通话前3秒的环境噪声动态调整检测阈值
- 结合多个特征的组合模式进行判断，避免单一特征误判

## 约束
- 单帧分析处理时间<20ms（确保实时性）
- 特征提取延迟<50ms
- 内存占用：滑动窗口缓冲不超过10秒音频数据
