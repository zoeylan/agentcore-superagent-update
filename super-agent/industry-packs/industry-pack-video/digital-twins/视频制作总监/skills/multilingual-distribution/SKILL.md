# 多语种视频本地化与全球分发

## 技能描述
为用户设计视频内容的多语种本地化方案和全球分发策略，覆盖TTS配音、字幕翻译、文化适配、平台规范适配和分发排期。

## 触发场景
- 用户想将中文视频内容出海到海外市场
- 用户需要为视频添加多语种配音和字幕
- 用户想了解不同海外市场的内容偏好和平台规范
- 用户需要设计批量化的多语种内容生产流水线

## 输出规范

### 必须包含的内容
1. **目标市场分析**：各市场的内容消费习惯、付费意愿、竞争格局
2. **配音方案设计**：TTS工具选型（ElevenLabs/Azure TTS/Fish Audio等）、音色匹配策略、情感一致性控制
3. **字幕方案设计**：翻译质量控制（机翻+人工校对vs纯AI）、字幕排版规范、硬字幕vs软字幕选择
4. **口型同步方案**：是否需要口型适配、技术方案选择（Wav2Lip等）、质量与成本权衡
5. **文化适配策略**：需要修改的文化敏感内容、本地化梗的处理、视觉元素调整
6. **平台分发规范**：各平台的上传规格、标签策略、最佳发布时间、审核要点

### 市场优先级评估框架
```
市场优先级 = 市场规模 × 付费意愿 × 竞争强度(反向) × 本地化难度(反向)

各市场参考评级：
- 东南亚（印尼/泰国/越南）：规模★★★ 付费★★ 竞争★★ 本地化难度★★
- 中东（沙特/阿联酋）：规模★★ 付费★★★ 竞争★ 本地化难度★★★
- 拉美（巴西/墨西哥）：规模★★★ 付费★★ 竞争★★ 本地化难度★★
- 欧美（美国/英国）：规模★★★★ 付费★★★★ 竞争★★★★ 本地化难度★★★
- 日韩：规模★★ 付费★★★★ 竞争★★★ 本地化难度★★★★
```

### 单语种本地化成本参考
```
单集（10分钟）本地化成本：
- TTS配音：15-50元（取决于工具和音色质量要求）
- 字幕翻译：20-80元（纯AI vs AI+人工校对）
- 口型同步：50-200元（可选，剧情类建议加）
- 文化适配审核：50-100元（人工）
- 平台适配调整：10-30元
- 单语种总计：145-460元/集
```

### 关键策略
- **配音优先级**：东南亚市场必须配当地语言音（用户不习惯看字幕），欧美市场字幕可接受
- **先跑MVP**：首批出海建议只做2-3个语种验证ROI，再扩展
- **内容筛选**：不是所有内容都适合出海，优先选择"视觉驱动"而非"台词驱动"的内容
- **合规红线**：各市场的内容红线必须提前排查（中东的宗教禁忌、东南亚的政治敏感等）

## 注意事项
- TTS配音的情感自然度仍是行业痛点，高情感场景建议人工配音或AI+人工微调
- 口型同步技术在侧脸/遮挡场景效果差，需要在分镜阶段就规避
- 各平台对AI生成内容的标注要求不同（YouTube要求标注、TikTok部分地区要求标注）
- 字幕翻译的"信达雅"在短视频场景下更偏向"信"和"达"，避免过度文学化翻译
- 分发时间需考虑目标市场时区，建议使用定时发布功能
