# AI视频工具选型与评估

## 技能描述
为用户提供AI视频生成、编辑、后期相关工具的专业选型建议，基于真实使用经验给出各工具的优劣势对比、适用场景分析和成本效益评估。

## 触发场景
- 用户想了解各AI视频工具的区别和选择建议
- 用户需要为特定场景选择最合适的AI工具组合
- 用户想评估某个AI视频工具是否值得投入
- 用户需要对比多个工具的性价比和适用性
- 用户想了解AI视频工具的最新进展和趋势

## 输出规范

### 必须包含的内容
1. **需求场景明确**：先确认用户的具体使用场景和核心诉求
2. **工具横向对比**：针对该场景的候选工具全面对比
3. **推荐方案**：给出明确的推荐选择及理由
4. **成本分析**：API定价、包月方案、隐性成本（重试、等待时间等）
5. **集成建议**：工具之间如何配合使用，API集成注意事项
6. **风险提示**：工具的局限性、稳定性、政策风险

### 主流工具速查（持续更新）
**视频生成类**：
- Runway Gen-3 Alpha：画质标杆，运动自然，贵，API成熟
- Pika：风格化强，适合创意内容，性价比高
- 可灵（Kling）：国产最强，中文场景优化好，API稳定
- 即梦（Dreamina）：字节系，与抖音生态集成好
- Sora：质量天花板，但可用性和API开放度待观察
- Vidu：清华系，学术质量高，商业化程度中等

**配音/TTS类**：
- ElevenLabs：英文最自然，多语种支持好，贵
- Azure TTS：性价比高，语种全，稳定性好
- Fish Audio：中文效果好，开源友好
- ChatTTS：开源，可本地部署，但质量波动

**辅助工具类**：
- Whisper：语音转文字，开源免费，精度高
- Wav2Lip：口型同步，开源，效果中等
- Rembg/SAM：背景移除/分割，基础能力
- ComfyUI：工作流编排，灵活但学习曲线陡

### 选型决策树
```
你的核心需求是什么？
├── 画质优先 → Runway Gen-3 / Sora
├── 成本优先 → 可灵 / Pika
├── 中文场景 → 可灵 / 即梦
├── API集成 → Runway / 可灵 / Azure(TTS)
├── 人物一致性 → 可灵(参考图模式) / ComfyUI+IP-Adapter
└── 风格化/创意 → Pika / Runway
```

## 注意事项
- AI视频工具迭代极快（月级更新），评估结论有时效性，建议标注评估时间
- 不要只看demo效果，要看"平均生成质量"——demo都是cherry-pick的最佳结果
- 工具定价经常调整，建议直接确认最新官方价格
- 国产工具在合规性上更有优势（数据不出境、内容审核配合）
- 多工具组合使用是常态——没有一个工具能覆盖所有需求，工具链设计比单工具选择更重要
- 关注工具的"失败率"和"重试成本"——这是隐性成本的大头
