# 视频工业化流水线设计

## 技能描述
为用户设计规模化视频内容生产的工业化流水线架构，涵盖从需求输入到成片输出的全自动化/半自动化生产系统设计。

## 触发场景
- 用户需要搭建日产10+/50+/100+视频的生产体系
- 用户想从手工作坊式制作升级为工业化生产
- 用户需要设计视频生产的技术架构和工作流
- 用户想评估当前视频生产流程的瓶颈并优化

## 输出规范

### 必须包含的内容
1. **产能需求分析**：日产量目标、内容品类、质量标准、交付周期
2. **流水线架构设计**：各环节的输入/输出/工具/耗时/并发能力
3. **人机协作模型**：哪些环节全自动、哪些半自动、哪些必须人工
4. **质量控制体系**：各环节的质量卡点、自动化质检规则、人工审核标准
5. **系统架构建议**：技术栈选型、队列设计、存储方案、监控告警
6. **团队配置建议**：需要哪些角色、各角色工作量、招聘/外包建议

### 流水线标准模块
```
┌─────────────────────────────────────────────────────────┐
│  输入层：需求/脚本/素材 → 标准化格式转换                    │
├─────────────────────────────────────────────────────────┤
│  策划层：选题生成 → 脚本撰写 → 分镜规划 → 资源调度          │
├─────────────────────────────────────────────────────────┤
│  生产层：素材生成 → 智能剪辑 → 配音配乐 → 字幕特效          │
├─────────────────────────────────────────────────────────┤
│  质控层：自动质检 → 人工审核 → 修正重做 → 终审通过          │
├─────────────────────────────────────────────────────────┤
│  分发层：平台适配 → 封面生成 → 标签优化 → 定时发布          │
└─────────────────────────────────────────────────────────┘
```

### 产能阶梯模型
```
Level 1（日产10条）：
- 团队：2-3人（1策划+1审核+1运营）
- 工具：SaaS工具组合，月成本5000-15000元
- 自动化率：60-70%

Level 2（日产50条）：
- 团队：5-8人（2策划+1技术+2审核+2运营）
- 工具：自建Pipeline + API集成，月成本3-8万元
- 自动化率：75-85%

Level 3（日产100+条）：
- 团队：10-15人（3策划+3技术+4审核+3运营+1负责人）
- 工具：自建生产系统 + 自研质检，月成本10-30万元
- 自动化率：85-92%
```

### 关键瓶颈识别清单
1. **生成瓶颈**：API并发限制、生成速度、排队机制
2. **质量瓶颈**：一致性不达标的重试成本、审核人力不足
3. **存储瓶颈**：视频素材的存储成本和CDN费用
4. **调度瓶颈**：多任务优先级冲突、资源竞争
5. **人员瓶颈**：审核团队是最常见的产能天花板（非技术问题）

## 注意事项
- 不要一步到位设计Level 3系统——建议阶梯式演进，每级验证ROI后再升级
- 流水线最大的敌人是"过度定制"——保持模块化，方便替换单个环节的工具
- 监控告警是必须的——至少监控：生成成功率、平均耗时、成本/条、人工介入率
- 预留人工干预入口——完全无人值守的全自动流水线在当前技术条件下不现实
- 质量和数量的平衡：初期建议宁可降低产量也要保证单条质量，口碑一旦崩塌难以恢复
