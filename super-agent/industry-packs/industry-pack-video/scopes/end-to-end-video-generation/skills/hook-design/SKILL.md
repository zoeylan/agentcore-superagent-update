# 钩子设计 (Hook Design)

## 概述
为短视频剧本设计高吸引力的注意力钩子（Hook），包括开场钩子（前3秒）、中段次级钩子（每15秒）和片尾悬念钩子。钩子设计的质量直接影响视频的完播率和观众留存曲线，是短视频内容竞争力的核心要素。

## 适用场景
- 新剧本的钩子体系设计
- 已有剧本的钩子强化优化
- 投放素材的开场钩子专项设计
- 系列剧的跨集悬念设计

## 执行步骤

### Step 1: 分析内容类型与钩子策略匹配
- 甜宠类→情感冲突钩子（如：目击出轨/身份反转/意外表白）
- 悬疑类→信息缺口钩子（如：尸体出现/神秘线索/时间悖论）
- 都市类→共情/爽感钩子（如：职场打脸/逆袭反转/阶层碰撞）
- 古风类→视觉奇观钩子（如：仙境呈现/武打瞬间/服化道冲击）

### Step 2: 开场钩子设计（前3秒）
- **冲突直入法**：直接展示故事最核心的冲突瞬间
- **悬念抛出法**：用旁白/画面提出一个引人好奇的问题
- **结果前置法**：先展示结局的震撼画面，再倒叙
- **情绪冲击法**：用极端情绪（大哭/大笑/震惊）抓住注意力
- 评估标准：3秒内必须建立"为什么"——观众想知道接下来发生什么

### Step 3: 中段次级钩子布置（每15秒）
- 确保观众在每15秒节点有继续观看的理由
- 类型包括：新信息揭示、情绪转折、角色反应、小反转、悬念加深
- 控制节奏：不是每个次级钩子都要重磅，形成"轻→重→轻→重"的节奏感

### Step 4: 片尾悬念钩子设计
- 留下一个未解答的问题或冲突
- 给出下一集的预期价值（暗示更大的冲突/反转即将发生）
- 可用技法：中断法（在高潮处突然结束）、预告法（旁白暗示下集内容）、伏笔法（埋入一个细节引发猜测）

### Step 5: 钩子强度评估
- 对设计的每个钩子进行强度评分（1-10）
- 评估维度：好奇心激发度、情感冲击度、信息独特性、视觉可实现性
- 开场钩子评分须>=8，中段钩子>=6，片尾钩子>=7

```json
{
  "script_content": "剧本内容（场次和对白）",
  "content_genre": "sweet_romance|suspense|urban|fantasy|comedy",
  "target_platform": "douyin|kuaishou|youtube_shorts",
  "target_audience": "audience描述",

```

## 最佳实践
1. **3秒定生死**：前3秒决定70%以上的观众是否继续观看
2. **信息缺口理论**：最有效的钩子是让观众意识到"我不知道但我想知道"
3. **视觉优先**：短视频是视觉媒体，纯文字/对白钩子效果弱于视觉冲击钩子
4. **平台数据验证**：不同平台的观众对钩子类型有不同偏好，需参考数据
5. **避免标题党**：钩子承诺的内容必须在视频中兑现，否则影响完播率

## 约束条件
- 钩子内容不得违反平台合规要求
- 视觉钩子需考虑AI生成的可实现性
- 不得使用误导性信息作为钩子（承诺必须兑现）
- 片尾悬念不得在当集完全不给解答（需要部分满足+新悬念）
