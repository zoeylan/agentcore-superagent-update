# 内容合规检测 (Content Compliance Detection)

## 概述
对AI生成的图像和视频素材进行内容安全审核，检测暴力、色情、政治敏感、未成年人保护等维度的违规内容。该检测为一票否决制——任何素材只要命中合规红线即判定不通过，无论其他维度评分多高。合规通过率必须为100%。

## 适用场景
- 每个AI生成素材的常规合规质检
- 边缘案例的合规风险评估
- 新增合规规则后的批量回扫
- 发布前的最终合规确认

## 执行步骤

### Step 1: 视觉内容检测
- **暴力/血腥检测**：武器展示、伤害动作、血液/伤口、虐待行为
- **色情/裸露检测**：裸露身体部位、性暗示姿势、低俗构图
- **恐怖/惊悚检测**：极端恐怖画面、心理恐惧暗示
- 使用多模型交叉验证确保检出率

### Step 2: 人物安全检测
- **未成年人保护**：检测画面是否含未成年人形象+不当内容组合
- **公众人物检测**：是否生成了现实公众人物的肖像
- **种族/性别歧视**：画面是否存在刻板印象或歧视性表达

### Step 3: 政治/文化敏感检测
- **政治符号检测**：国旗/徽章/领导人形象的不当使用
- **领土争议**：地图/区域标注是否正确
- **宗教敏感**：宗教符号/场景是否存在冒犯性表达
- **文化禁忌**：针对目标市场的特定文化禁忌

### Step 4: 广告/商业合规
- **品牌侵权**：是否生成了未授权的品牌Logo/产品
- **虚假宣传**：画面是否包含可能被认为是虚假广告的内容
- **烟酒展示**：部分平台禁止烟酒画面

### Step 5: 风险评级与处置
- **CRITICAL（严重违规）**: 立即标记不通过，删除素材，记录告警
- **HIGH（高风险）**: 标记不通过，触发重新生成
- **MEDIUM（中风险）**: 标记为需人工复核
- **LOW（低风险）**: 通过但记录备注

### Step 6: 合规报告输出
- 生成详细的合规检测报告
- 对于不通过的素材，说明具体违规点和位置
- 提供修改建议（如"建议调整角色服装覆盖度"）

## 输入规范
```json
{
  "asset_id": "素材ID",
  "asset_type": "image|video",
  "asset_path": "文件路径",
  "target_platforms": ["douyin", "kuaishou", "youtube_shorts"],
  "content_context": "内容上下文描述（辅助判断）",
  "target_markets": ["CN", "US", "ME"]
}
```

## 最佳实践
1. **零容忍原则**：合规项不存在"差不多行"，任何疑似违规一律不通过
2. **平台差异化**：不同平台合规标准不同，需分平台出具结果
3. **上下文理解**：同一画面在不同叙事上下文中合规性可能不同（如：警匪片中的武器vs日常场景中的武器）
4. **持续更新**：平台审核标准经常变化，需保持规则库最新
5. **记录留痕**：所有检测结果需完整记录，便于审计和申诉

## 约束条件
- 合规检测必须100%执行（不得跳过任何素材）
- 严重违规的检出率>99.9%
- 检测延迟<3秒/素材
- 不对检测结果做"放水"处理
- 边缘案例宁可误判为违规也不放过
