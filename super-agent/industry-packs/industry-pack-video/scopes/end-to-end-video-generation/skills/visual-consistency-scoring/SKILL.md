# 视觉一致性评分 (Visual Consistency Scoring)

## 概述
对AI生成的图像/视频素材进行人物视觉一致性评估，通过将生成结果与角色参数库进行多维度对比，量化评估跨分镜人物一致性水平。该评分是整条生产线最关键的质量指标，直接决定素材是否合格（阈值>=0.85）。

## 适用场景
- 每个生成素材的常规一致性质检
- 批量素材的一致性统计分析
- 一致性问题的根因定位
- 提示词优化效果的量化验证

## 执行步骤

### Step 1: 人物检测与分割
- 在生成的图像/视频帧中检测人物区域
- 如果画面中有多人，需正确匹配哪个人物对应哪个角色
- 分割出人物的面部、身体、服装区域
- 处理遮挡/侧脸/背影等特殊情况

### Step 2: 面部一致性评分（权重50%）
评估维度：
- 脸型匹配度：与参数库中的脸型描述对比
- 五官相似度：眼型/鼻型/唇型与参数库的吻合程度
- 标志特征检测：参数库中标注的面部标志（如痣/疤/酒窝）是否存在
- 面部比例：与参考图的面部比例一致性
- 评分公式：face_score = 0.3*shape + 0.3*features + 0.25*landmarks + 0.15*proportions

### Step 3: 发型一致性评分（权重20%）
评估维度：
- 长度匹配：长/中/短与参数库一致
- 颜色匹配：发色与参数库描述的接近程度
- 造型匹配：直/卷/编发等与参数库一致
- 分线方式：中分/偏分/无分线与参数库一致
- 评分公式：hair_score = 0.3*length + 0.3*color + 0.25*style + 0.15*parting

### Step 4: 服装一致性评分（权重15%）
评估维度：
- 款式匹配：服装类型与参数库/分镜要求一致
- 颜色匹配：服装颜色准确
- 配饰检测：标志性配饰（如耳环/项链）是否存在
- 评分公式：outfit_score = 0.4*style + 0.35*color + 0.25*accessories

### Step 5: 体型一致性评分（权重15%）
评估维度：
- 身材比例：与参数库描述的体型一致
- 姿态自然度：人物比例是否合理（无AI畸形）
- 评分公式：body_score = 0.5*proportions + 0.5*naturalness

### Step 6: 综合评分与决策
- 总分 = face×0.5 + hair×0.2 + outfit×0.15 + body×0.15
- 判定标准：
  - >=0.85: PASS（通过）
  - 0.75-0.84: MARGINAL（边缘，可人工确认）
  - <0.75: FAIL（不通过，需重新生成）
- 生成详细评分报告，标注具体失分项

## 输入规范
```json
{
  "asset_id": "素材ID",
  "asset_type": "image|video_frame",
  "asset_path": "素材文件路径",
  "character_in_shot": "画面中应出现的角色",
  "character_profile": "对应角色的参数库数据",
  "reference_images": ["该角色的参考图（如有）"],
  "shot_context": "镜头上下文（用于判断有意变化）"
}
```

## 输出规范
```json
{
  "asset_id": "素材ID",
  "character": "角色名",
  "overall_score": 0.87,
  "verdict": "pass|marginal|fail",
  "breakdown": {
    "face_score": 0.91,
    "hair_score": 0.85,
    "outfit_score": 0.82,
    "body_score": 0.88
  },
  "issues": [
    {
      "dimension": "outfit",
      "detail": "服装颜色偏差：参数库为cream色毛衣，生成结果为白色毛衣",
      "severity": "minor",
      "impact_on_score": -0.03
    }
  ],
  "recommendation": "pass（总分达标，轻微服装色差可接受）"
}
```

## 最佳实践
1. **参考图对比**：有参考图时优先使用参考图做视觉对比，精度高于文本匹配
2. **上下文考量**：同一角色在不同光照/远近下的合理变化不应扣分
3. **关键锚点加权**：角色的标志性特征（如独特的痣/配饰）检测权重应更高
4. **批次趋势分析**：单次评分可能有偏差，关注批次内的平均水平
5. **人工校准**：定期用人工标注数据校准自动评分的准确性

## 约束条件
- 评分必须基于客观可量化的维度，不做主观美学判断
- 遮挡/侧脸等情况需降低置信度并标注
- 对于视频素材，取3帧（首/中/末）做平均评分
- 评分延迟<5秒/素材（不能成为流程瓶颈）
- 评分记录需要可追溯（便于后续分析和校准）
