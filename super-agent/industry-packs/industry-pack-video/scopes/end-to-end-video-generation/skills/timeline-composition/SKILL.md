# 时间线合成 (Timeline Composition)

## 概述
将通过质检的所有素材（视频片段、图像、AI配音）按照分镜脚本的时序编排进行精确组装，构建完整的视频时间线。该技能是将分散素材整合为连贯叙事的核心环节，需要确保每个素材在正确的时间点出现、正确的时长展示、并与相邻素材自然衔接。

## 适用场景
- 将一部短剧的所有分镜素材组装为完整时间线
- 素材替换后的时间线局部重建
- 转场效果的添加和调整
- 节奏调整（加速/减速/删减特定片段）

## 执行步骤

### Step 1: 素材完整性验证
- 核对所有shot是否都有对应的合格素材
- 验证每个素材的时长与分镜脚本要求匹配
- 标记缺失/不完整的素材位置
- 确认音频素材（配音）与视频素材一一对应

### Step 2: 时间线骨架构建
- 按照分镜脚本的shot顺序排列所有素材
- 设定每个素材的入点（in point）和出点（out point）
- 分配轨道层级：
  - V1轨道：主视频（视频片段/图像）
  - V2轨道：叠加层（字幕/特效/遮罩）
  - A1轨道：对白/旁白音频
  - A2轨道：背景音乐
  - A3轨道：音效

### Step 3: 转场设计与添加
根据相邻镜头的关系选择转场：
- **硬切（Cut）**: 同场景内的镜头切换，默认选择
- **淡入淡出（Dissolve）**: 时间流逝、回忆进出
- **闪白（Flash）**: 情绪爆发、梦境切换
- **叠化（Cross Dissolve）**: 场景平缓过渡
- **黑场（Fade to Black）**: 重大时间跳跃、章节分隔
- **推/拉转场**: 动作延续、空间位移
- 转场时长：0.3-0.8秒（快节奏0.3s，慢节奏0.8s）

### Step 4: 时间线微调
- 调整各片段的精确入出点（逐帧精度）
- 确保转场不会切掉重要画面内容
- 处理图像素材的静帧时长（配合Ken Burns效果）
- 验证总时长与目标一致（偏差<±5%）

### Step 5: 时间线导出
- 生成标准化的时间线描述文件（EDL/XML格式）
- 包含所有素材的精确时间码和处理指令
- 导出供下游音画同步和渲染使用

## 输入规范
```json
{
  "storyboard": "完整分镜脚本JSON",
  "approved_assets": [
    {
      "shot_ref": "SHOT-001",
      "video_path": "/assets/SHOT-001.mp4",
      "audio_path": "/audio/TTS-001.wav",
      "duration": 4.5,
      "type": "video|image"
    }
  ],
  "target_duration": 120,
  "pacing": "normal|fast|slow"
}
```

## 输出规范
```json
{
  "timeline_id": "TL-001",
  "total_duration": 118.5,
  "target_duration": 120,
  "duration_diff_percent": -1.25,
  "tracks": {

      {
        "shot_ref": "SHOT-001",
```

## 最佳实践
1. **先骨架后细节**：先确保所有素材位置正确，再微调转场和时间点
2. **节奏服务叙事**：快节奏段落用短镜头+硬切，慢节奏段落用长镜头+柔转场
3. **预留转场时间**：在素材时长规划时预留0.3-0.5秒给转场
4. **检查逻辑跳跃**：相邻镜头的内容应该有视觉/叙事上的连贯性
5. **总时长守恒**：添加转场会影响总时长，需要动态调整

## 约束条件
- 总时长偏差不超过目标的±5%
- 不允许出现素材位置空缺（黑帧）
- 转场时长0.3-0.8秒，不可超出此范围
- 素材不可重叠（除转场区域外）
- 时间线描述精度到帧级别（1/24秒）
