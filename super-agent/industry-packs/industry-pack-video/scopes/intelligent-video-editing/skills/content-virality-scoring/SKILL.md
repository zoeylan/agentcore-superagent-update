# 内容传播力评分 (Content Virality Scoring)

## 技能概述

内容传播力评分技能负责评估视频片段在社交媒体上的潜在传播价值。通过分析内容的共鸣性、可讨论性、情感感染力和形式创新性等维度，预测片段被用户主动分享和传播的可能性。该评分为投放素材优先级排序和内容运营决策提供数据支撑。

## 适用场景

- 投放素材的优先级排序（高传播力片段优先投放）
- 内容运营策略制定（什么类型的内容更容易火）
- A/B测试素材的预筛选
- 精华集锦片段的取舍决策

## 执行步骤

### 步骤1：内容特征提取
1. 情感共鸣特征：是否触及普遍情感（亲情/爱情/友情/正义感）
2. 话题讨论特征：是否涉及社会热点/争议话题/观点碰撞
3. 信息价值特征：是否提供实用信息/新知识/独特视角
4. 娱乐趣味特征：是否有幽默/反转/搞笑元素
5. 视觉奇观特征：是否有罕见画面/震撼场景/创意表现

### 步骤2：传播力维度评分
1. 共鸣指数（0-100）：内容引起情感共鸣的能力
   - 通用情感主题加分，小众/冷门主题减分
   - 情感表达到位加分，表达生硬减分
2. 可讨论指数（0-100）：内容引发评论和讨论的潜力
   - 开放性话题/争议观点加分
   - 封闭性陈述/无悬念结论减分
3. 分享动机指数（0-100）：用户主动分享的动机强度
   - "看了这个你会怎么想"型内容加分
   - 纯个人向内容减分
4. 平台适配指数（0-100）：内容形式是否匹配目标平台调性
   - 节奏快、信息密度高的内容在短视频平台加分
   - 深度长内容在中长视频平台加分

### 步骤3：综合传播力评分
1. 加权计算：共鸣(0.30) + 可讨论(0.25) + 分享动机(0.25) + 平台适配(0.20)
2. 输出0-100综合传播力分数
3. 标注主要传播驱动力类型
4. 与同类型视频的历史数据对比给出相对排名

### 步骤4：传播风险评估
1. 检测可能引起负面传播的风险因素
2. 评估内容是否可能被断章取义
3. 检测是否存在平台敏感内容
4. 输出传播安全评级

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| fragment | Object | 是 | 待评估的视频片段（含多模态分析数据） |
| target_platform | String | 否 | 目标发布平台 |
| target_audience | String | 否 | 目标受众画像 |
| historical_data | Object | 否 | 同类型内容的历史表现数据 |

## 最佳实践

- 传播力评分应结合平台算法特征使用（如抖音重完播率、微博重转发评论）
- 高传播力不等于高商业价值，需结合转化目标综合判断
- 建议将预测分数与实际投放数据定期对齐校准
- 季节性热点和社会事件会影响传播力，评分模型需定期更新

## 约束与限制

- 传播力预测本质上具有不确定性，准确率约70-80%
- 突发热点事件可能使预测模型暂时失效
- 不同地区/文化的传播偏好差异显著
- 平台算法调整可能导致历史数据失去参考价值
