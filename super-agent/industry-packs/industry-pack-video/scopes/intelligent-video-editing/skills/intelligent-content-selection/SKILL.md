# 智能内容选择 (Intelligent Content Selection)

## 技能概述

智能内容选择技能负责在给定目标时长约束下，从分段评分和高光评分中选择最优内容子集。这是从"分析"到"决策"的关键技能——需要在内容质量、叙事完整性和时长控制三者之间找到最优平衡点。

## 适用场景

- 从60分钟长视频中提取5分钟精华
- 为30秒投放素材选择最有效的内容片段组合
- 从一部短剧中产出20-50条不同的投放素材
- 在时长约束下最大化内容价值

## 执行步骤

### 步骤1：约束分析
1. 解析目标时长约束（绝对值或范围）
2. 计算可用候选内容总时长
3. 判断约束类型：
   - 内容过剩（需要取舍）：候选总时长 > 目标时长 × 1.5
   - 内容适中（需要精选）：目标时长 × 0.8 <= 候选总时长 <= 目标时长 × 1.5
   - 内容不足（需要扩展）：候选总时长 < 目标时长 × 0.8
4. 根据约束类型选择对应的选择策略

### 步骤2：内容过剩时的智能取舍
1. 将所有候选片段按综合评分排序
2. 执行贪心选择：从最高分片段开始累加
3. 当累加时长接近目标时，使用动态规划精确求解最优子集
4. 取舍原则：
   - 5星片段不得删除（除非5星片段总时长已超出目标）
   - 优先删除过渡性段落（1-2星）
   - 相邻保留片段之间的过渡段可被删除，但需确保上下文连贯
5. 输出取舍决策及理由

### 步骤3：内容不足时的扩展策略
1. 降低高光评分阈值（从80→70→60逐步降低）
2. 扩展候选范围（纳入更低评级的分段内容）
3. 考虑复用同一片段的不同部分
4. 如仍不足，标记为"无法满足目标时长"并输出最大可用时长

### 步骤4：批量差异化选择（多条素材产出）
1. 为每条素材维护"已使用内容池"
2. 计算新候选与已产出素材的内容相似度
3. 确保任意两条素材之间的内容差异度>=0.6
4. 通过组合策略实现同源视频的差异化产出
5. 当差异化候选耗尽时，通过不同钩子+不同结尾的排列组合扩展产量

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| segments_scored | Array | 是 | 带评分的分段列表 |
| highlights | Array | 是 | 高光片段列表 |
| target_duration_ms | Integer | 是 | 目标时长（毫秒） |
| duration_tolerance | Float | 否 | 时长容差（默认0.05，即±5%） |
| output_count | Integer | 否 | 需要产出的素材条数（默认1） |
| purpose | String | 否 | 用途（investment_material/highlight_reel/trailer） |

## 最佳实践

- 投放素材建议目标时长设为平台最优完播时长（抖音15-30秒，快手20-40秒）
- 批量产出时先生成"核心池"（必选高光）+"调配池"（可选替换片段）的策略效率更高
- 时长取舍时优先削减尾部内容而非头部（保护钩子的完整性）
- 当五星片段极多时，考虑分拆为多条素材而非强制压缩到一条中

## 约束与限制

- 选择算法不保证全局最优（NP问题的近似解）
- 批量差异化在源视频内容单一时难以保证（如独角视频）
- 叙事连贯性评分基于相邻片段语义相似度的启发式方法
