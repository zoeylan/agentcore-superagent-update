# 钩子片段识别 (Hook Fragment Identification)

## 技能概述

钩子片段识别技能专门用于定位视频中能在前3秒内抓住观众注意力的片段。这是短剧投放素材制作的核心能力——在信息流广告场景中，用户滑动决策时间仅2-3秒，钩子片段的质量直接决定了投放素材的完播率和转化率。

## 适用场景

- 短剧投放素材的开头片段选取
- 信息流广告创意的钩子设计
- 短视频封面动图（GIF）的选取
- 视频预告片/预览的吸引力片段选择

## 执行步骤

### 步骤1：钩子候选片段初筛
1. 从高光片段列表中筛选潜在钩子候选（评分>=60）
2. 从全视频中额外扫描"冷启动吸引力"片段：
   - 视觉冲击开头（强画面、冲突场面、异常画面）
   - 悬念对白开头（反问句、悬念句式、冲突对白）
   - 情感爆发开头（哭戏、怒吼、惊讶反应）
   - 动作起始点（追逐、打斗、冲突动作的起始瞬间）
3. 每个候选截取前3秒片段进行独立评估

### 步骤2：3秒吸引力评估
1. 对每个3秒片段执行以下评估：
   - 视觉冲击分（0-100）：画面是否有足够的视觉吸引力
   - 信息悬念分（0-100）：是否能引起"想知道接下来发生什么"的好奇心
   - 情感共鸣分（0-100）：是否能引起情感反应（紧张/好奇/同情/愤怒）
   - 认知冲突分（0-100）：是否存在反常规/反预期的元素
2. 综合钩子效果分 = 视觉冲击(0.25) + 信息悬念(0.30) + 情感共鸣(0.25) + 认知冲突(0.20)

### 步骤3：可用性验证
1. 验证片段的独立可理解性（无需前置上下文是否能看懂）
2. 验证内容合规性（无暴力/色情/违规内容）
3. 验证画面质量（无模糊/黑帧/花屏）
4. 验证音频可用性（有有效音频或适合静音播放）

### 步骤4：钩子片段排序与推荐
1. 按综合钩子效果分降序排列
2. 标注每个钩子的主要吸引力机制（视觉驱动/悬念驱动/情感驱动/冲突驱动）
3. 去除内容高度相似的重复钩子（保留最高分的）
4. 输出Top-N推荐列表及使用建议

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| highlights | Array | 是 | 高光片段列表 |
| semantic_graph | Object | 是 | 视频语义图谱 |
| target_platform | String | 否 | 目标平台（抖音/快手/YouTube等，影响评估标准） |
| top_n | Integer | 否 | 需要的钩子片段数量（默认10） |

## 最佳实践

- 抖音/快手平台用户决策时间约1.5-2秒，钩子需在前2秒内展现核心吸引力
- YouTube Shorts用户决策时间约3秒，可以有稍微缓慢的钩子节奏
- 悬念驱动的钩子通常比纯视觉冲击的钩子具有更高的完播率
- 每部短剧建议提取至少10-20个差异化钩子用于A/B测试投放
- 情感爆发类钩子在女性用户群体中的点击率通常更高

## 约束与限制

- 钩子效果评估基于历史投放数据训练的模型，对于全新品类内容可能不准确
- 片段独立可理解性的判断有一定主观性
- 不同平台的用户偏好差异显著，跨平台钩子效果可能差异很大
- 合规性检测依赖外部内容审核模型的准确率
