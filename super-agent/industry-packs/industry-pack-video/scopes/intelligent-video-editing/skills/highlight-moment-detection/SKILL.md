# 高光时刻检测 (Highlight Moment Detection)

## 技能概述

高光时刻检测技能负责从视频内容中自动识别具有高吸引力和传播价值的片段。该技能融合多模态分析结果，通过多维度综合评判来识别情绪高潮、关键转折、精彩对白和视觉冲击等不同类型的高光时刻。

## 适用场景

- 短剧投放素材中高吸引力片段的自动提取
- 直播回放中精彩时刻的自动标记
- 长视频精华集锦的素材筛选
- 活动/会议视频的高光快剪

## 执行步骤

### 步骤1：多维度高光信号检测
1. 情绪峰值检测：扫描情感曲线中的局部最大值点
   - 阈值：情感强度>0.7 且 持续时长>2秒
   - 额外加分：多模态一致（语音情感+表情+动作同时达到高峰）
2. 叙事转折检测：定位语义反差度突增的时刻
   - 阈值：前后5秒窗口的语义相似度<0.3
   - 额外加分：伴随背景音乐变化或静音
3. 精彩对白检测：识别具有传播潜力的语言片段
   - 通过语义分析检测：反问句式、排比修辞、犀利观点、幽默表达
   - 额外加分：该对白导致后续情绪变化
4. 视觉冲击检测：识别画面视觉吸引力突增的时刻
   - 阈值：视觉复杂度突增>50% 或 运动幅度>0.8
   - 额外加分：伴随音效强调

### 步骤2：高光片段范围确定
1. 以信号峰值点为中心，向前后扩展至语义自然边界
2. 确保片段是自包含的（观众无需额外上下文即可理解）
3. 片段最短3秒，最长60秒
4. 重叠片段合并处理（取并集或保留更高分的）

### 步骤3：高光分类与标签
1. 为每个高光片段标注类型（可多标签）：
   - emotional_peak（情绪高潮）
   - plot_twist（关键转折）
   - brilliant_dialogue（精彩对白）
   - visual_impact（视觉冲击）
   - hook_fragment（钩子片段）
2. 标注主要吸引力要素的描述

### 步骤4：综合评分
1. 按评分公式计算：
   - 情感强度 × 0.30
   - 叙事重要性 × 0.25
   - 视觉吸引力 × 0.20
   - 传播潜力 × 0.15
   - 钩子效果 × 0.10
2. 标准化为0-100分
3. 按评分降序排列

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| semantic_graph | Object | 是 | 视频语义图谱 |
| segments | Array | 是 | 语义分段结果（含评分） |
| narrative_structure | Object | 是 | 叙事结构分析结果 |
| target_count | Integer | 否 | 期望的高光数量（影响阈值调整） |
| highlight_types | Array | 否 | 指定关注的高光类型 |

## 最佳实践

- 对于短剧投放场景，优先关注hook_fragment类型，确保前3秒吸引力
- 当target_count > 实际高分候选时，逐步降低阈值并在输出中说明降档
- 高光片段之间应保证内容差异度>=0.6，避免提取出相似重复的片段
- 情绪高潮和关键转折通常共现，应避免重复计分

## 约束与限制

- 对于纯风景/美食等无情节内容，叙事转折类高光不适用
- 高光检测效果依赖于上游多模态分析和分段评分的质量
- 传播潜力评分基于历史数据模型，对于全新内容形式可能不准确
