# ASR语音识别 (ASR Speech Recognition)

## 技能概述

ASR语音识别技能负责将视频中的语音内容高精度转写为结构化文本，同时完成说话人分离（Speaker Diarization）和语音情感标注。这是语义图谱中文本信息的主要来源，其准确率直接影响下游语义分割和高光提取的质量。目标准确率>=95%。

## 适用场景

- 视频多模态分析管线中的语音转写环节
- 多语种混合内容的语种识别与分段转写
- 说话人分离用于区分不同角色的对白
- 语音情感标注用于情绪高潮检测

## 执行步骤

### 步骤1：音频预处理
1. 从视频中分离音轨（支持多音轨选择）
2. 音频降噪处理（消除背景音乐和环境噪声）
3. 声道分析与合并（立体声→单声道用于识别）
4. 音频切分为适合模型处理的片段（30秒/片段，2秒重叠）

### 步骤2：语种检测与模型选择
1. 对前30秒音频进行语种快速检测
2. 检测是否存在多语种混合（如中英混杂）
3. 根据检测结果选择对应的ASR模型
4. 多语种混合时启用代码切换（Code-Switching）模型

### 步骤3：语音转写
1. 对每个音频片段执行端到端语音识别
2. 输出带时间戳的逐字/逐词转写结果
3. 计算每个片段的置信度分数
4. 对低置信度片段（<0.8）标记为需复核
5. 执行标点符号恢复和断句优化

### 步骤4：说话人分离
1. 提取音频的声纹特征向量
2. 执行无监督聚类确定说话人数量
3. 为每段语音分配说话人ID
4. 处理重叠语音（两人同时说话的情况）
5. 生成说话人时间线（每位说话人的发言时间分布）

### 步骤5：情感标注
1. 对每段语音提取韵律特征（语速、音调、音量变化）
2. 执行6维情感分类（喜/怒/哀/乐/平静/激动）
3. 输出情感向量和主情感标签
4. 标记情感突变点（情绪转折时刻）

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| audio_source | File/Stream | 是 | 音频文件或从视频提取的音轨 |
| language_hint | String | 否 | 预期语种（zh/en/ja/ko等） |
| speaker_count_hint | Integer | 否 | 预期说话人数量（优化分离效果） |
| enable_emotion | Boolean | 否 | 是否启用情感标注（默认true） |

## 最佳实践

- 对于短剧内容，说话人数量通常在2-8人之间，可提供hint提升分离效果
- 背景音乐较强时，应先进行音源分离（人声/背景分离）再送入ASR
- 低置信度段落往往出现在多人重叠说话、背景噪音大或方言段落
- 建议对每批次结果进行5%的人工抽检以持续监控准确率

## 约束与限制

- 当前支持语种：中文（普通话/粤语）、英语、日语、韩语、西班牙语等15种
- 方言识别准确率可能低于普通话标准（约85-90%）
- 说话人分离在超过10人时准确率下降
- 极低音量段落（<-40dB）可能无法识别
