# 叙事结构分析 (Narrative Structure Analysis)

## 技能概述

叙事结构分析技能负责识别视频内容的宏观叙事框架，判断其属于何种叙事类型（短剧/演讲/直播/纪录片等），并据此应用对应的结构模板进行深度分析。该技能使系统能够"理解"视频的整体故事脉络，而非仅做局部分割。

## 适用场景

- 短剧内容的起承转合结构识别
- 演讲/教程的论述逻辑结构识别
- 直播回放的功能性段落划分
- 内容类型自动分类与模板匹配

## 执行步骤

### 步骤1：内容类型分类
1. 综合分析视频特征判断内容类型：
   - 短剧/影视：多角色对话、场景频繁切换、明确的情节推进
   - 演讲/教程：单人或少人、连续性话题、逻辑递进结构
   - 直播回放：长时间、话题随机切换、互动性强
   - 综艺/访谈：多人轮流、主持节奏、环节明确
2. 输出分类结果和置信度

### 步骤2：结构模板匹配
1. 根据内容类型选择对应的叙事结构分析模板：
   - 短剧模板：起（背景建立）→ 承（冲突发展）→ 转（高潮转折）→ 合（结局/悬念）
   - 演讲模板：引言（问题提出）→ 论点（核心观点）→ 论据（支撑材料）→ 总结（行动号召）
   - 直播模板：开场寒暄 → 正式内容 → 互动环节 → 商品推介 → 收尾
   - 通用模板：开头 → 主体（可含多个子段） → 结尾

### 步骤3：结构映射
1. 将语义边界检测的分段结果映射到结构模板的各个阶段
2. 识别每个阶段的起止时间和关键节点
3. 标注阶段间的转换信号（如高潮转折点的触发事件）
4. 评估结构完整性（是否有遗漏的关键阶段）

### 步骤4：叙事节奏分析
1. 计算各阶段的时长占比（如高潮是否过短/过长）
2. 绘制叙事张力曲线（tension curve）
3. 标记张力峰值点和谷值点
4. 与同类型视频的理想节奏模型对比，输出节奏健康度评分

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| semantic_graph | Object | 是 | 视频语义图谱 |
| boundaries | Array | 是 | 语义边界检测结果 |
| content_type_hint | String | 否 | 内容类型提示（可覆盖自动检测） |

## 最佳实践

- 对于系列短剧，建议积累同系列前几集的结构数据用于优化后续识别
- 直播类内容的结构通常更松散，不必强制匹配严格的模板
- 张力曲线对于高光提取具有直接指导价值，应确保其准确性
- 当自动分类置信度<0.7时，建议同时运行多个模板并输出最优匹配

## 约束与限制

- 实验性/先锋派叙事风格可能不匹配任何预设模板
- 短于5分钟的视频可能无法完整识别四幕结构
- 多线叙事（平行剪辑）会增加结构分析的复杂度
