# 情绪高潮分析 (Emotional Peak Analysis)

## 技能概述

情绪高潮分析技能专注于识别视频中观众情感达到峰值的时刻。通过融合语音情感、人物表情、背景音乐、叙事张力等多维信号，精确定位情感爆发点，为高光提取提供情绪维度的核心判据。

## 适用场景

- 短剧中的情感高潮点定位（哭戏/怒吼/告白等）
- 演讲中的激情段落识别
- 直播中粉丝情绪爆发时刻的捕捉
- 音乐MV中的情感高潮段落标记

## 执行步骤

### 步骤1：情感时间线构建
1. 提取ASR情感标注的时间序列数据（6维情感向量）
2. 提取人物表情分析的时间序列数据
3. 提取背景音乐情绪曲线（通过音频特征分析）
4. 构建综合情感时间线（多信号加权融合）

### 步骤2：峰值检测算法
1. 对综合情感曲线执行局部极大值检测
2. 应用峰值显著性过滤（峰值必须高于周围窗口平均值的1.5倍）
3. 计算峰值宽度（情感高潮的持续时长）
4. 计算峰值上升斜率（情感爆发的急剧程度——越陡越有冲击力）

### 步骤3：多模态一致性验证
1. 检验峰值点的多模态信号是否一致：
   - 语音情感强度是否>0.7？
   - 人物表情是否为非中性（强表情）？
   - 背景音乐是否处于高潮段？
   - 叙事张力曲线是否处于上升/峰值？
2. 计算多模态一致性分数（一致信号数/总信号数）
3. 高一致性（>=0.75）的峰值确认为真实情绪高潮
4. 低一致性的峰值降级为"情感波动"而非"高潮"

### 步骤4：情绪高潮分类
1. 根据主导情感向量对高潮进行分类：
   - 悲伤高潮（哭戏/分别/失去）
   - 愤怒高潮（争吵/反抗/爆发）
   - 惊喜高潮（反转/揭秘/意外）
   - 欢乐高潮（重逢/成功/浪漫）
   - 激动高潮（热血/竞技/突破）
2. 标注高潮的情感色彩标签

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| semantic_graph | Object | 是 | 包含多模态情感数据的语义图谱 |
| tension_curve | Array | 否 | 叙事张力曲线（来自结构分析） |
| sensitivity | Float | 否 | 峰值检测灵敏度（0-1，默认0.5） |

## 最佳实践

- 对于短剧投放，"惊喜高潮"和"悲伤高潮"通常具有最高的传播价值
- 情绪高潮前通常有一段"情绪酝酿期"，提取高光片段时应包含部分酝酿段
- 连续出现多个高潮（<30秒间隔）时，通常意味着大高潮场景，应合并处理
- 灵敏度参数应根据视频整体情感基调动态调整（情感丰富的内容用更高阈值）

## 约束与限制

- 对于情感表达含蓄的文化风格内容（如日式表演），检测灵敏度可能不足
- 纯叙事性旁白缺乏情感波动，该技能适用性降低
- 背景音乐情绪分析对于纯器乐比歌曲更具挑战性
