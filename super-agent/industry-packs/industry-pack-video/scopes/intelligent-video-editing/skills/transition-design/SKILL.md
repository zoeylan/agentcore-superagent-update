# 转场设计 (Transition Design)

## 技能概述

转场设计技能负责为剪辑方案中相邻片段之间的拼接点设计合适的转场方式。优秀的转场不仅连接画面，更传达叙事信息——时间流逝、空间切换、情绪变化等。该技能确保每个转场点位于语义自然断点，且转场方式与内容语境匹配。

## 适用场景

- 投放素材的片段拼接（通常使用快速硬切）
- 精华集锦的片段过渡（多种转场混合使用）
- 预告片的节奏型转场（闪白/快切/音频先行）
- 完整拆条的自然衔接

## 执行步骤

### 步骤1：拼接点分析
1. 分析每个拼接点前后片段的属性差异：
   - 场景差异度（同场景/不同场景/时空跨越）
   - 情绪差异度（相同情绪/情绪过渡/情绪突变）
   - 节奏差异度（同速/加速/减速）
   - 说话人连续性（同一人继续/不同人接话/对话轮换）
2. 计算拼接难度系数（差异越大转场越需要精心设计）

### 步骤2：转场方式选择
1. 基于拼接点属性选择转场类型：
   - **硬切（Cut）**：场景相近、节奏紧凑、对话连续时使用
   - **交叉溶解（Dissolve）**：时间流逝、记忆闪回、情绪渐变时使用
   - **淡入淡出（Fade）**：章节分隔、大时间跨越、段落结束时使用
   - **闪白/闪黑（Flash）**：高潮瞬间、冲击强调、梦境切换时使用
   - **音频先行（J-Cut）**：场景预告、悬念建立、对话引入时使用
   - **画面先行（L-Cut）**：对白延续、情感延伸、环境交代时使用
   - **动态遮罩（Wipe）**：平行叙事、时空对比、分屏效果时使用
2. 对于投放素材：优先使用硬切和闪白，保持节奏紧凑
3. 对于精华集锦：混合使用多种转场，创造视觉变化

### 步骤3：转场参数设计
1. 确定转场时长：
   - 硬切：0帧（即时切换）
   - 溶解/淡化：12-30帧（0.4-1秒）
   - 闪白：6-12帧（0.2-0.4秒）
   - 音频先行：15-45帧（0.5-1.5秒）
2. 确定转场锚点（精确到帧的切换起止位置）
3. 设计转场音频处理（淡入淡出/音效叠加/静音缓冲）

### 步骤4：转场质量验证
1. 检验转场点是否在语义自然断点（不在句子中间/动作中间）
2. 检验前后片段的亮度/色温差异是否需要调色过渡
3. 检验音频是否有突兀的断裂感
4. 输出转场自然度评分（0-1，目标>=0.85）

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| arrangement | Object | 是 | 叙事编排结果（片段序列） |
| semantic_graph | Object | 是 | 视频语义图谱（用于断点验证） |
| style_preference | String | 否 | 风格偏好（fast_paced/cinematic/smooth） |
| fps | Integer | 否 | 目标帧率（默认30） |

## 最佳实践

- 一条30秒投放素材中转场类型不宜超过3种，保持风格一致
- 硬切是最安全的转场方式——当其他转场效果不确定时默认使用硬切
- 音频转场（J-Cut/L-Cut）对观众感知的流畅度影响最大
- 转场时长应与整体节奏匹配：快节奏内容用短转场，慢节奏用长转场
- 避免在两个都有人说话的片段之间使用溶解（会产生"鬼影"效果）

## 约束与限制

- 复杂转场特效（如3D翻转、粒子效果）超出本技能范围
- 转场设计依赖于源片段的首尾帧质量
- 某些转场方式（如溶解）在两段色调差异极大时效果不佳
