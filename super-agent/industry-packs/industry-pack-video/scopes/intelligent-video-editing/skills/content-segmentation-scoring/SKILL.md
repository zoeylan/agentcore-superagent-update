# 内容分段评分 (Content Segmentation Scoring)

## 技能概述

内容分段评分技能负责对语义分割后的每个视频段落进行多维度质量评估和重要性评级。评分结果直接服务于下游的高光提取和剪辑编排，是从"理解"到"决策"的关键桥梁。

## 适用场景

- 为每个分段标注重要性级别（1-5星）
- 为剪辑编排提供内容取舍的量化依据
- 评估分段质量用于分割结果的自我校验
- 生成视频内容的热力图概览

## 执行步骤

### 步骤1：段落内容丰富度评估
1. 计算段落内的信息密度（单位时间内的语义事件数量）
2. 评估对白质量（语句完整性、信息量、修辞水平）
3. 评估视觉丰富度（场景复杂度、人物数量、动作密度）
4. 综合得出内容丰富度分数（0-1）

### 步骤2：叙事重要性评估
1. 基于叙事结构分析，判断段落在整体故事中的功能角色
2. 评估是否包含关键剧情推进（因果链中的必要环节）
3. 评估是否可被安全删除而不影响故事理解
4. 输出叙事重要性分数（0-1）

### 步骤3：情感强度评估
1. 提取段落内的情感曲线峰值
2. 计算情感波动幅度（标准差）
3. 判断是否包含情感转折点
4. 输出情感强度分数（0-1）

### 步骤4：综合评级
1. 加权综合三维评分：内容丰富度(0.3) + 叙事重要性(0.4) + 情感强度(0.3)
2. 将综合分数映射为1-5星级别
3. 输出评级理由（一句话说明）
4. 生成全视频的段落重要性分布图

## 输入规格

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| segments | Array | 是 | 分段列表（含时间戳和内容摘要） |
| semantic_graph | Object | 是 | 视频语义图谱 |
| narrative_structure | Object | 是 | 叙事结构分析结果 |
| scoring_weights | Object | 否 | 自定义评分维度权重 |

## 最佳实践

- 对于投放素材制作场景，应提高情感强度的权重（如0.4）
- 对于精华集锦制作，应提高叙事重要性权重确保内容完整性
- 1-2星段落通常是过渡性内容，可优先作为取舍候选
- 5星段落应确保不被意外裁切，设置保护标记

## 约束与限制

- 评分模型基于大量标注数据训练，对于非常规内容类型可能存在偏差
- 评分结果应定期与人工判断进行相关性校准（目标相关系数>=0.8）
- 叙事重要性评估对于非线性叙事内容准确度降低
