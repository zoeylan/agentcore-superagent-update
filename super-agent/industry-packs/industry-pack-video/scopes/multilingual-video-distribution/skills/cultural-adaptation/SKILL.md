# 文化适配 (Cultural Adaptation)

## 技能概述

文化适配技能负责对翻译内容进行深度文化本地化处理，确保视频内容在目标市场不仅语言通顺，更在文化层面被当地受众接受和理解。该技能涵盖文化禁忌检测、俚语/梗的本地化转化、表达习惯适配、以及内容合规审核。

## 适用场景

- 视频内容出海时的文化合规审核（中东/东南亚/欧美等不同市场）
- 俚语、网络梗、文化特指表达的本地化转化
- 宗教/政治/习俗敏感内容的检测和替代表达建议
- 品牌/营销视频的跨文化表达策略调整

## 执行步骤

### 步骤1：文化禁忌扫描
1. 加载目标市场的文化禁忌词库（按地区细分：中东/南亚/东南亚/东亚/欧美/拉美/非洲）
2. 对翻译文本执行全量关键词匹配和语义匹配
3. 对画面描述文本（如涉及着装/手势/符号）执行视觉文化敏感度检测
4. 对命中项进行风险分级（高危/中危/低危）
5. 高危项必须标记为"阻断性问题"——不处理则不允许进入下一步

### 步骤2：俚语与梗的本地化
1. 识别源文本中的俚语、网络流行语、文化特指表达
2. 评估直译是否能被目标市场受众理解
3. 不能直译的表达，提供目标市场的等效表达替代方案
4. 对于无法找到等效表达的内容，提供解释性翻译方案
5. 记录所有替换决策及理由（用于审计追溯）

### 步骤3：表达习惯适配
1. 检查翻译文本的语体是否符合目标市场偏好（正式/非正式/年轻化）
2. 调整数字/日期/货币/度量单位的本地化格式
3. 人名/地名的音译规范检查
4. 敬语体系适配（日语尊敬语/韩语敬语等级/泰语皇室用语等）

### 步骤4：合规报告生成
1. 汇总所有文化适配操作的清单
2. 列出所有被标记的文化禁忌项及处理结果
3. 计算文化禁忌检测覆盖率（目标100%）
4. 对无法自动处理的项标记为"需人工审核"

## 最佳实践

1. **地区细分**：同一语种在不同地区有不同禁忌（如西班牙语在西班牙vs墨西哥vs阿根廷的差异）
2. **宁严勿松**：文化禁忌检测宁可误报也不能漏报，误报可通过人工解除
3. **保留原意**：文化适配的目标是让目标受众理解原意，而非重新创作
4. **动态更新**：文化禁忌词库需定期更新（跟踪国际政治/社会事件变化）

## 约束与限制

- 文化禁忌检测覆盖率必须达到100%（不允许跳过任何段落）
- 高危文化禁忌必须阻断流程，不得自动通过
- 所有适配决策必须有记录和理由（可追溯）
- 不得对原始创作意图进行过度改写
