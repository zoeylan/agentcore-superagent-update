# 去水印处理 (Watermark Removal)

## 技能概述

去水印处理技能负责自动检测视频中的水印（含固定位置水印、动态水印、半透明水印），并使用AI图像修复算法对水印区域进行内容填充，使修复后的区域自然融入周围画面。目标修复自然度>=0.9，水印检测召回率>=95%。

## 适用场景

- 源视频包含平台水印需要去除后再分发（如从抖音下载的视频去除抖音水印）
- 品牌logo水印的替换（去除原水印后添加新水印）
- 文字水印的去除与区域修复
- 动态/移动水印的追踪和逐帧去除

## 执行步骤

### 步骤1：水印检测
1. 对视频首帧执行静态水印检测（固定位置、固定内容的水印）
2. 对全视频执行动态水印检测（周期性出现/位置移动的水印）
3. 水印类型分类：文字水印/Logo水印/半透明水印/动画水印
4. 输出水印位置信息（bounding box + 逐帧mask）
5. 计算水印区域占画面比例（过大的水印区域标记告警）

### 步骤2：修复策略选择
1. 根据水印类型和所在区域的画面复杂度选择修复策略
2. 简单背景（纯色/渐变）→ 使用颜色插值修复
3. 复杂纹理背景 → 使用AI Inpainting修复
4. 动态画面区域 → 使用视频修复模型（考虑时序一致性）
5. 水印面积过大（>画面20%）→ 标记为"建议保留/人工处理"

### 步骤3：AI修复执行
1. 对每一帧的水印区域执行修复算法
2. 确保时序一致性（相邻帧的修复结果不能有闪烁）
3. 边缘羽化处理（修复区域与周围画面的过渡自然）
4. 颜色一致性校验（修复区域的色温/亮度与周围一致）

### 步骤4：修复质量评估
1. 计算修复区域的自然度评分（SSIM/LPIPS等指标）
2. 目标：修复自然度>=0.9
3. 检测修复伪影（模糊/色块/重复纹理等）
4. 自然度<0.9的区域标记为"需人工确认"
5. 自然度<0.7的区域保留原始水印并标记

## 输入规格

```json
{
  "video_file": "path/to/source_video.mp4",
  "detection_config": {
    "sensitivity": "high|medium|low",
    "watermark_hints": {
      "known_positions": ["top_right", "bottom_center"],
      "known_types": ["text", "logo"]
    },
    "skip_if_area_exceeds": 0.2
  }
}
```

## 最佳实践

1. **先检测后确认**：水印检测结果应先确认再修复，避免误将视频内容当作水印去除
2. **时序一致性最重要**：视频去水印比图片去水印多一个维度——帧间一致性，修复结果闪烁比略微模糊更不可接受
3. **保守策略**：宁可保留一个可疑水印也不要误删真实内容
4. **分级处理**：简单水印快速处理，复杂水印精细处理，降低整体处理时间

## 约束与限制

- 修复自然度低于0.7时必须保留原始水印区域
- 水印面积超过画面20%时建议人工处理
- 不得在去水印过程中引入新的伪影
- 去水印处理不得改变视频的分辨率/帧率/编码格式
