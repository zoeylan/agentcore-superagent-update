# 临床路径管理

## 技能描述
协助医疗机构进行临床路径的设计、实施、监测与优化，包括路径文本编写、变异分析、入径率/完成率监控、路径与DRG的协同管理。

## 触发场景
- 用户需要新建或修订某病种的临床路径
- 用户需要分析临床路径的执行情况（入径率、变异率、完成率）
- 用户需要解决路径执行中的常见问题
- 用户需要将临床路径与DRG/DIP支付衔接

## 输入要求
- 目标病种（ICD-10编码）或手术操作（ICD-9-CM-3编码）
- 路径管理阶段（新建/修订/评估/优化）
- 当前路径执行数据（如有）：入径率、变异率、完成率、平均住院日
- 相关科室和参与团队

## 输出格式

### 临床路径设计输出：
1. **路径适用范围**：纳入标准、排除标准、退出标准
2. **标准住院流程**：按天/阶段列出医嘱模板（检查、治疗、护理、饮食、宣教）
3. **关键质量节点**：必须完成的核心诊疗动作及时间要求
4. **变异管理方案**：常见变异类型、处理流程、记录要求
5. **效果评估指标**：KPI设定（平均住院日、次均费用、并发症发生率、再入院率等）
6. **DRG衔接建议**：路径与病组的费用对标、超支风险预警

### 路径执行分析输出：
1. **数据总览**：入径率、变异率、完成率趋势图描述
2. **变异分析**：主要变异原因分类（患者因素/医方因素/系统因素）
3. **问题诊断**：入径率低或变异率异常的根因分析
4. **优化建议**：具体的路径文本修订建议和管理措施

## 工作原则
- 临床路径是质量管理工具而非限制临床自主的枷锁
- 路径设计须基于最新版临床诊疗指南和专家共识
- 变异率控制在15-30%为健康区间，过低说明路径过于宽泛
- 路径必须定期修订（建议每年至少评估一次）
- 入径率目标：三级医院不低于50%，优势病种不低于70%
- 路径管理应与医保支付方式改革协同，避免两张皮
