# 医疗制度合规审查

## 技能描述
协助医疗机构进行医疗管理制度的制定、审查、修订和合规性评估，确保医院各项制度符合最新法律法规要求，并具备可操作性和可追溯性。

## 触发场景
- 用户需要新建或修订某项医疗管理制度
- 用户需要审查现有制度是否符合最新法规要求
- 用户需要解读新发布的医疗相关法规政策
- 用户需要建立核心制度的执行监督机制
- 用户需要准备制度相关的评审资料

## 输入要求
- 制度类型（十八项核心制度/专项管理制度/部门规章）
- 具体制度名称或涉及领域
- 审查目的（新建/修订/合规检查/评审准备）
- 相关法规变化信息（如有）
- 医院等级和实际运行情况

## 输出格式

### 制度制定/修订输出：
1. **法规依据梳理**
   - 上位法依据（法律→行政法规→部门规章→规范性文件）
   - 最新政策要求和变化点
   - 地方性法规和行业标准补充
2. **制度框架建议**
   - 目的和适用范围
   - 核心条款和关键要求
   - 职责分工和权限划分
   - 执行流程和操作规范
   - 监督考核和持续改进
   - 附则（生效日期、修订记录、解释权）
3. **关键条款审查意见**
   - 合法性审查：是否与上位法冲突
   - 合理性审查：是否符合临床实际
   - 可操作性审查：是否有明确的执行标准和时限
   - 完整性审查：是否覆盖全流程、无管理盲区
4. **配套措施建议**
   - 培训计划（覆盖范围、频次、考核方式）
   - 执行记录模板（表单设计要点）
   - 监督检查机制（频次、方式、结果应用）

### 合规审查输出：
1. **合规风险清单**：逐条对照法规要求，标注合规/部分合规/不合规
2. **风险等级评估**：高风险（可能导致行政处罚）/中风险（评审扣分）/低风险（管理建议）
3. **整改建议**：按优先级排序的整改措施和完成时限
4. **法规更新追踪**：近期需要关注的政策变化和制度修订需求

## 涉及的核心法规体系
- 《基本医疗卫生与健康促进法》
- 《医师法》（2022年施行）
- 《医疗机构管理条例》及实施细则
- 《医疗质量管理办法》（国家卫计委令第10号）
- 《医疗纠纷预防和处理条例》
- 《医疗质量安全核心制度要点》（十八项核心制度）
- 《医疗技术临床应用管理办法》
- 《处方管理办法》《抗菌药物临床应用管理办法》
- 《病历书写基本规范》《电子病历应用管理规范》
- 《医疗器械临床使用管理办法》
- 各专科诊疗规范和技术操作标准

## 工作原则
- 制度不是越多越好，关键在于体系化、层次化、可执行
- 核心制度（十八项）是底线，必须100%建立并有效运行
- 制度修订必须留痕：修订原因、修订内容对比、审批流程、生效日期
- "制度有"只是第一步，"知晓→执行→监督→改进"才是完整闭环
- 制度间不能矛盾，新制度出台前必须审查与现有制度的一致性
- 关注"放管服"改革趋势，制度设计要兼顾规范性和便利性
- 涉及患者权益的制度（知情同意、隐私保护等）需特别审慎
- 制度的生命力在于与时俱进，建议每2-3年进行一次全面修订
