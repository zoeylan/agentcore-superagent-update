# 药物相互作用检测 (Drug Interaction Detection)

## 概述
本技能基于药物代谢酶（CYP450家族）、药物转运体（P-gp等）和药效学机制，对处方中的药物组合及与患者当前用药的相互作用进行全面检测和临床显著性评估。识别可能导致疗效减弱或毒性增加的药物相互作用(DDI)。

## 使用场景
- 新处方开具时与当前用药的DDI检测
- 多科室联合用药的交互作用筛查
- 住院患者医嘱变更时的增量DDI检查
- 抗凝/抗癫痫等治疗窗窄药物的DDI重点监控

## 执行步骤

### 步骤1：构建药物清单
- 收集处方中所有药物（含新开和当前在用）
- 标准化药物信息（通用名、剂型、剂量、给药途径）
- 纳入院外自服药物（OTC、中草药、保健品）
- 计算两两配对组合数 = n(n-1)/2

### 步骤2：药动学(PK)相互作用检测
- CYP450酶系统分析：
  - 识别酶抑制剂（强/中/弱抑制）
  - 识别酶诱导剂
  - 识别经该酶代谢的底物
  - 评估抑制/诱导程度对底物血药浓度的影响
- 转运体相互作用：P-gp、OATP、BCRP等
- 蛋白结合率竞争：高蛋白结合率药物间的置换效应

### 步骤3：药效学(PD)相互作用检测
- 协同毒性：
  - QT间期延长叠加（多个QT延长药物联用）
  - 出血风险叠加（抗凝+抗血小板+NSAIDs）
  - CNS抑制叠加（阿片类+苯二氮卓+抗组胺）
  - 5-HT综合征风险（SSRIs+曲坦类+MAOIs）
  - 肾毒性叠加（NSAIDs+ACEI+利尿剂"三重打击"）
- 药效拮抗：降低治疗效果的组合

### 步骤4：临床显著性分级
- **禁忌级（X）**：绝对不可联用，可能致死
  - 如：华法林+酮康唑、西沙必利+红霉素
  - 处理：**强制拦截**
- **严重级（D）**：联用风险高，需权衡利弊
  - 如：氯吡格雷+奥美拉唑(CYP2C19)
  - 处理：**警示提醒**，建议替代方案
- **中度级（C）**：联用需监测，可能需调整剂量
  - 处理：**警示提醒**，建议监测方案
- **轻度级（B）**：临床意义有限
  - 处理：**信息提示**

### 步骤5：管理建议生成
- 对于拦截/警示级DDI提供：
  - 替代药物选择（避免该相互作用通路）
  - 剂量调整建议（如需减量比例）
  - 监测方案（需加测哪些指标、频率）
  - 用药时间间隔建议（如需错开服用）

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| new_drugs | array | 是 | 新开处方药物列表 |
| current_drugs | array | 是 | 当前在用药物列表 |
| patient_genotype | object | 否 | 药物基因组学信息 |
| renal_function | object | 否 | 肾功能指标 |
| hepatic_function | object | 否 | 肝功能指标 |

## 最佳实践
- DDI数据库需至少季度更新（跟进FDA/NMPA安全警告）
- 对治疗窗窄药物（华法林、地高辛、苯妥英等）的DDI给予更高敏感度
- 考虑中药/中成药的已知相互作用（如含有甘草的制剂与利尿剂）
- 多重用药（≥5种）患者自动触发全面DDI复查
- 严重DDI事件纳入不良事件上报流程

## 约束与限制
- DDI数据库基于已发表证据，可能存在未知的相互作用
- 临床显著性受个体差异影响（基因多态性、肝肾功能等）
- 中草药和保健品的DDI数据不完整，需标注"数据有限"
- 不可完全替代临床药师的专业判断
- 分级标准需经医院药事委员会审核确认
