# 证据分级标注 (Evidence Grading)

## 概述
本技能为诊断建议和临床推荐标注循证医学证据等级，确保医生了解每条建议的可靠性程度。遵循GRADE证据分级体系和Oxford循证医学中心的证据等级标准，为临床决策提供透明的证据质量参考。

## 使用场景
- 鉴别诊断列表中每个诊断的证据等级标注
- 检查检验推荐的证据支持标注
- 治疗方案建议的证据强度标注
- 临床指南推荐强度的说明

## 执行步骤

### 步骤1：证据来源检索
- 检索与诊断/推荐相关的临床证据
- 数据来源层次：
  1. 系统性综述/Meta分析（Cochrane Library等）
  2. 随机对照试验(RCT)
  3. 队列研究/病例对照研究
  4. 病例系列/病例报告
  5. 专家意见/临床经验
  6. 国家级临床指南/专家共识

### 步骤2：证据质量评估
- 评估维度：
  - 研究设计类型
  - 样本量是否充足
  - 是否存在偏倚风险
  - 结果一致性
  - 直接性（是否与当前临床问题直接相关）
  - 精确性（置信区间宽度）

### 步骤3：等级标注
- **A级（高质量证据）**：多个高质量RCT结果一致，或高质量Meta分析
  - 标注为：⭐⭐⭐⭐ 证据充分
  - 意味着：进一步研究不太可能改变对该效果估计的信心
- **B级（中等质量证据）**：单个大型RCT，或多个有一定局限性的RCT
  - 标注为：⭐⭐⭐ 证据较充分
  - 意味着：进一步研究可能对该效果估计有重要影响
- **C级（低质量证据）**：观察性研究结果一致，或专家共识
  - 标注为：⭐⭐ 证据有限
  - 意味着：进一步研究很可能对该效果估计有重要影响
- **D级（极低质量证据）**：专家意见、临床经验推断、病例报告
  - 标注为：⭐ 证据不足
  - 意味着：任何效果估计都很不确定

### 步骤4：标注输出
- 在诊断/推荐条目旁以简洁标识呈现证据等级
- 提供证据来源的简要说明（如"基于2020年ESC指南"）
- 对于证据等级较低(C/D)的推荐，增加提示说明
- 提供参考文献链接供医生深入查阅

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| clinical_item | object | 是 | 需要标注的诊断/推荐 |
| item_type | enum | 是 | 类型（诊断/检查/治疗/预防） |
| patient_context | object | 否 | 患者上下文（用于直接性评估） |

## 输出规格
```json
{
  "evidence_grade": "A|B|C|D",
  "grade_display": "⭐⭐⭐⭐",
  "grade_label": "证据充分",
  "confidence_statement": "对该效果估计的信心描述",
  "key_references": [
    {
      "source": "文献/指南来源",
      "year": 2023,
      "type": "Meta-analysis|RCT|Guideline|Expert consensus"
    }
  ],
  "limitations": "当前证据的主要局限",
  "applicability_note": "对当前患者的适用性说明"
}
```

## 最佳实践
- 证据等级标注准确率目标≥90%
- 对于同一诊断，选择最高质量的可用证据进行标注
- 当指南之间存在矛盾推荐时，注明不同指南的立场
- 定期更新证据库（至少季度更新），跟踪最新临床研究
- 对于中国特有的疾病谱（如肝癌HBV相关），优先参考中国指南

## 约束与限制
- 证据等级不等同于临床推荐强度——低证据等级不意味着不应采用
- 不对证据做选择性呈现——呈现所有相关证据包括不一致的结果
- 标注时效性——注明证据产生年份，超过5年的标注"可能需要更新"
- 承认知识库的局限性——对于尚无循证证据的领域如实标注
