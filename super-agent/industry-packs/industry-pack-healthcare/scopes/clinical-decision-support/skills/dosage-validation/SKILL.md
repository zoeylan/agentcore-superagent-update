# 剂量合理性校验 (Dosage Validation)

## 概述
本技能对处方药物的剂量进行个体化合理性校验，综合考虑患者年龄、体重、体表面积(BSA)、肝肾功能、基因多态性等个体化参数，验证给药剂量是否在安全有效范围内，识别过量或不足的风险。

## 使用场景
- 处方审核中的剂量校验环节
- 特殊人群（儿童、老年人、肝肾功能不全）的剂量计算辅助
- 治疗药物监测(TDM)指导下的剂量调整建议
- 化疗药物BSA/AUC计算剂量校验

## 执行步骤

### 步骤1：获取患者个体化参数
- 基础参数：年龄、性别、体重、身高
- 计算衍生参数：
  - BSA（体表面积）= Du Bois公式
  - BMI（体质量指数）
  - 理想体重IBW / 调整体重ABW（肥胖患者）
- 器官功能参数：
  - 肾功能：Scr → eGFR (CKD-EPI公式)
  - 肝功能：Child-Pugh评分 / MELD评分
- 药物基因组学参数（如有）：
  - CYP2C19代谢表型
  - CYP2D6代谢表型
  - HLA-B*5801等

### 步骤2：确定标准剂量范围
- 查询药品说明书规定的常规剂量范围
- 获取各适应症对应的推荐剂量
- 获取特殊人群的剂量调整建议
- 确定日极量/单次极量

### 步骤3：个体化剂量计算与比对
- 根据患者参数计算推荐剂量：
  - 按体重计算：mg/kg × 体重
  - 按BSA计算：mg/m² × BSA（肿瘤药物）
  - 按肾功能调整：eGFR对应的剂量比例
  - 按肝功能调整：Child-Pugh对应的剂量建议
- 与处方实际剂量比对
- 计算偏差比例

### 步骤4：剂量安全性评估
- **超极量10倍以上**→**强制拦截**（可能为书写错误）
- **超极量但≤10倍**→**警示提醒**（可能为特殊情况需确认）
- **超常规剂量50%以上**→**警示提醒**（建议确认适应症和理由）
- **低于常规剂量50%以上**→**信息提示**（可能为亚治疗剂量）
- **肾功能不全未调整剂量**→**警示提醒**（标注建议调整剂量）

### 步骤5：给药频次与途径验证
- 验证给药频次是否符合药物半衰期和PK/PD特性
- 验证给药途径的合理性（如：某些药物仅限静脉/口服）
- 检查特殊给药要求（如：需空腹、避光输注、输注速率限制）
- 检查给药间隔是否合理

### 步骤6：输出校验结果
- 给出校验通过/警示/拦截结果
- 对于异常剂量提供：
  - 推荐的个体化剂量
  - 计算依据和参考来源
  - 需要医生确认的临床理由

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| drug_name | string | 是 | 药物通用名 |
| prescribed_dose | object | 是 | 处方剂量（单次量、频次、途径） |
| indication | string | 是 | 用药适应症 |
| patient_params | object | 是 | 患者个体化参数 |
| lab_results | object | 否 | 相关检验结果（肝肾功能等） |
| genotype | object | 否 | 药物基因组学信息 |

## 最佳实践
- 对于儿童患者，按年龄/体重双重校验
- 对于化疗药物，剂量偏差>10%即触发警示
- TDM药物（万古霉素、氨基糖苷类等）结合血药浓度调整
- 老年患者（≥65岁）启动低剂量策略提醒
- 所有剂量计算保留过程记录供追溯

## 约束与限制
- 无法校验的情况明确标注（如说明书无该适应症的剂量推荐）
- 基因多态性数据依赖患者是否做过基因检测
- 对于新药或说明书更新，需确保数据库及时同步
- 不可替代临床药师对复杂用药方案的专业判断
- eGFR计算依赖最近一次Scr（需标注数据时效性）
