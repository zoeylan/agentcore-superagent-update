# 费用结构优化 (Fee Structure Optimization)

## 概述
分析医院和科室层面的费用构成比例，对比DRG标准和同级医院标杆数据，识别费用结构异常（如药占比过高、耗占比超标），提供结构优化建议以提升DRG盈余能力。

## 适用场景
- 科室费用构成比例的定期分析
- 药占比/耗占比超标时的深入分析
- 新DRG政策实施后的费用适应性调整
- 经营分析中的费用效率评估

## 执行步骤

### 1. 费用构成数据采集
- 按科室/病种汇总费用构成（药品/耗材/检查/治疗/床位/护理）
- 计算各项目占比
- 获取DRG支付标准中的费用构成基准
- 获取同级医院标杆数据

### 2. 对标分析
- 科室费用构成与DRG标准对比
- 与同级医院平均水平对比
- 识别偏差超过15%的费用项
- 分析偏差原因（收治结构差异/用药习惯/检查偏好）

### 3. 优化机会识别
- 药品费用优化：仿制药替代、集采药品使用率、辅助用药占比
- 耗材费用优化：国产替代进口、集中采购议价空间
- 检查费用优化：重复检查识别、检查必要性评估
- 住院日优化：日间手术转化、术前住院日压缩

### 4. 建议生成与影响评估
- 针对每个优化机会估算潜在节省金额
- 评估实施难度（低/中/高）
- 评估对医疗质量的影响（确保不降低质量）
- 生成优先级排序的优化建议清单

### 5. 效果追踪
- 跟踪已实施优化措施的效果
- 月度对比费用构成变化
- 计算DRG盈余率变化
- 定期更新优化建议

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| departmentId | string | 是 | 科室代码 |
| feeComposition | object | 是 | 当前费用构成数据 |
| drgBenchmark | object | 是 | DRG标准费用构成 |
| peerBenchmark | object | 否 | 同级医院标杆数据 |
| timeRange | object | 是 | 分析时间范围 |

## 输出规范
| 字段 | 类型 | 说明 |
|------|------|------|
| currentComposition | object | 当前费用构成占比 |
| deviations | array | 与标杆的偏差项 |
| optimizationOpportunities | array | 优化机会列表 |
| estimatedSavings | number | 预估节省金额 |
| implementationPlan | array | 实施建议（含优先级） |
| qualityImpactAssessment | object | 质量影响评估 |

## 最佳实践
- 费用优化必须以不影响医疗质量为前提
- 药占比目标≤30%，耗占比目标≤20%
- 优先推荐政策导向的优化措施（集采/仿制药替代）
- 建议的可操作性和临床接受度需评估

## 约束条件
- 不得以费用控制为由限制必要的医疗服务
- 优化建议不等于削减质量
- 涉及药品调整需经过药事委员会审核
- 分析数据不可用于不当考核压力
