# 基因候选优先级排序 (Gene Prioritization)

## 概述
基于患者临床表型和基因组数据，对候选致病基因进行综合评分和优先级排序，缩小罕见病诊断的搜索范围，输出Top10候选基因列表。

## 适用场景
- 全外显子/全基因组测序后的候选基因筛选
- 多基因变异场景下的优先级判定
- Panel检测阴性后扩展分析的基因排序
- 家系分析中需要在多个候选变异间做出选择

## 执行步骤

### 1. 表型特征提取与权重分配
- 提取患者核心表型和次要表型（HPO术语）
- 为核心表型分配高权重（×2.0），次要表型标准权重（×1.0）
- 区分特异性表型（高诊断价值）和非特异性表型

### 2. 表型-基因关联评分
- 查询OMIM/Orphanet数据库的基因-表型映射
- 计算每个候选基因的表型覆盖度（覆盖核心表型比例）
- 基于HPO语义相似度计算表型匹配分数
- 考虑表型的年龄依赖性（某些表型可能尚未显现）

### 3. 变异致病性权重
- 整合变异解读Agent的ACMG分类结果
- P/LP变异 → 高权重（×3.0）
- VUS → 中等权重（×1.5）
- LB/B → 低权重（×0.1）/排除
- 考虑变异类型：LoF（PVS1适用基因）> missense > synonymous

### 4. 遗传模式一致性验证
- 确认候选基因的已知遗传模式（AD/AR/XL）
- 验证变异数量与遗传模式一致性（AR需双等位基因变异）
- 考虑不完全外显率和可变表达度

### 5. 综合评分与排序输出
- 加权综合评分 = 表型匹配×0.4 + 变异致病性×0.3 + 遗传模式一致性×0.2 + 文献证据×0.1
- 输出Top10候选基因排序列表
- 每个基因附带：评分、证据摘要、建议的验证方案

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| patientPhenotypes | array | 是 | HPO术语列表（区分核心/次要） |
| candidateVariants | array | 是 | 候选变异列表（含ACMG分类） |
| inheritancePattern | string | 否 | 疑似遗传模式 |
| familyHistory | object | 否 | 家族史信息 |
| patientAge | number | 否 | 患者年龄（考虑年龄依赖性表型） |

## 输出规范
| 字段 | 类型 | 说明 |
|------|------|------|
| rankedGenes | array | Top10排序基因列表 |
| geneScores | object | 每个基因的细分评分 |
| validationSuggestions | array | 建议的验证方案 |
| differentialDiagnoses | array | 鉴别诊断列表 |

## 最佳实践
- Top3候选基因应推荐Sanger验证
- AR疾病注意复合杂合变异的识别
- 考虑基因的突变不耐受性（pLI/LOEUF评分）
- 负结果（所有候选评分均低）也应报告并建议进一步检测

## 约束条件
- 排序结果仅为辅助参考，不等同于确诊
- 表型信息不完整时需标注"排序可靠性受限"
- 结果需在完整报告出具时限内提供
