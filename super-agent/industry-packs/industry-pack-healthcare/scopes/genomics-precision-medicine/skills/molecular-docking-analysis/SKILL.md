# 分子对接分析 (Molecular Docking Analysis)

## 概述
对候选药物分子与目标蛋白进行分子对接模拟计算，评估结合亲和力和结合模式，为药物再利用候选筛选提供结构生物学层面的证据支持。

## 适用场景
- 药物再利用候选的靶点结合力评估
- 多个候选药物的比较排序
- 已知药物对新靶点的结合可能性预测
- 先导化合物的初步虚拟筛选

## 执行步骤

### 1. 靶点蛋白准备
- 获取靶点蛋白3D结构（PDB数据库/AlphaFold预测）
- 结构预处理（加氢、电荷分配、溶剂分子处理）
- 活性位点识别（已知位点/预测位点）
- 构建对接网格（Grid Box）

### 2. 配体分子准备
- 获取候选药物3D结构（DrugBank/PubChem）
- 能量最小化和构象搜索
- 电荷计算和原子类型分配
- 柔性键设定

### 3. 分子对接计算
- 选择对接引擎（AutoDock Vina/Glide/GOLD）
- 执行多构象对接（exhaustiveness≥32）
- 记录结合自由能（ΔG, kcal/mol）
- 输出Top10对接构象

### 4. 结果分析与评分
- 计算对接得分排名
- 分析关键相互作用（氢键、疏水、π-π堆积）
- 评估结合模式合理性
- 与已知配体（如有）的结合模式比较

### 5. 筛选与排序
- 对接得分排名前5%纳入候选池
- 综合考虑ADMET预测数据
- 标注可能的选择性问题（脱靶风险）
- 输出最终排序和详细分析报告

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| targetProtein | string | 是 | 靶点蛋白PDB ID或序列 |
| bindingSite | object | 否 | 活性位点坐标（如已知） |
| candidateDrugs | array | 是 | 候选药物列表（SMILES/SDF） |
| referenceCompound | string | 否 | 参照化合物（已知活性配体） |

## 输出规范
| 字段 | 类型 | 说明 |
|------|------|------|
| dockingScores | array | 每个候选的对接得分 |
| bindingMode | object | 结合模式描述 |
| keyInteractions | array | 关键相互作用列表 |
| ranking | array | 综合排序结果 |
| confidence | enum | high/medium/low |

## 最佳实践
- 优先使用高分辨率实验结构（<2.5Å）
- AlphaFold预测结构需标注置信度（pLDDT）
- 使用多种对接软件交叉验证提高可靠性
- 对接结果需结合MD模拟进一步验证（高优先级候选）

## 约束条件
- 分子对接为预测性方法，准确度有限
- 结果需实验验证方可确认
- 不考虑蛋白柔性的刚性对接可能产生假阴性
- 计算结果不可直接用于临床决策
