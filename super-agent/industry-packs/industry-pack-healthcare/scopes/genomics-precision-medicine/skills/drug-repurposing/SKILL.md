# 药物再利用分析 (Drug Repurposing Analysis)

## 概述
基于基因网络分析和分子对接评分，对已有药物进行再利用潜力评估，筛选可能对目标疾病/靶点有效的候选药物，输出TOP5候选药物清单。

## 适用场景
- 罕见病确诊后缺乏特效药物时的候选药物筛选
- 已知致病机制但无现有治疗方案的疾病
- 现有治疗方案疗效不佳需寻找替代药物
- 临床研究团队需要药物再利用候选列表

## 执行步骤

### 1. 靶点信息收集与验证
- 确认目标疾病/通路的关键分子靶点
- 查询UniProt/PDB获取靶点蛋白结构信息
- 验证靶点的成药性（druggability评估）
- 构建靶点相关的PPI（蛋白-蛋白相互作用）网络

### 2. 基因网络分析
- 基于STRING/BioGRID构建疾病基因网络
- 通路富集分析（KEGG/Reactome/GO）
- 识别网络中的hub基因和关键通路节点
- 扩展至相邻通路中可能的间接靶点

### 3. 候选药物库筛选
- 查询DrugBank/ChEMBL已批准药物数据库
- 基于靶点-药物已知关联初步筛选
- 扩展至网络邻居靶点对应的已批准药物
- 排除已知严重安全性问题的药物

### 4. 分子对接评分
- 对候选药物与靶点蛋白进行分子对接模拟
- 计算结合自由能和对接得分
- 筛选对接得分排名前5%的候选
- 分析结合模式和关键相互作用残基

### 5. 文献证据验证
- PubMed系统检索每个候选药物的相关证据
- 查询ClinicalTrials.gov已有临床试验数据
- 评估证据质量（体外→动物→人体数据）
- 仅保留有至少1篇支持性文献的候选

### 6. TOP5候选输出
- 综合对接得分+文献证据生成最终排序
- 每个候选药物附带完整评估报告
- 包含：作用机制假说、分子对接得分、支持文献、已有临床证据、安全性信号

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| targetDisease | string | 是 | 目标疾病名称/OMIM编号 |
| targetGenes | array | 是 | 目标基因/靶点列表 |
| proteinStructures | array | 否 | 靶点蛋白PDB结构ID |
| excludeDrugs | array | 否 | 需排除的药物（如已失败的候选） |
| patientGenotype | object | 否 | 患者基因型（用于排除代谢禁忌） |

## 输出规范
| 字段 | 类型 | 说明 |
|------|------|------|
| top5Candidates | array | TOP5候选药物清单 |
| mechanismHypothesis | string | 每个候选的作用机制假说 |
| dockingScore | number | 分子对接得分 |
| literatureEvidence | array | 支持性文献列表 |
| clinicalTrialStatus | string | 相关临床试验状态 |
| safetySignals | array | 已知安全性信号 |

## 最佳实践
- 双重筛选：对接得分前5% AND 有文献支持
- 优先推荐已有人体安全性数据的已批准药物
- 标注再利用用途与原批准适应症的差异
- 建议与临床药理学专家讨论后再进入实验验证

## 约束条件
- 药物再利用分析结果仅用于科研参考，不可直接用于临床处方
- 必须明确标注"研究性建议，非临床推荐"
- 涉及专利药物需提示知识产权考量
- 每个候选必须有≥1篇支持性文献方可纳入推荐
