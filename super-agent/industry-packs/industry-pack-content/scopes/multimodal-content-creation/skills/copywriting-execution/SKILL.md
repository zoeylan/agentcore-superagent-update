# 文案创作执行 (Copywriting Execution)

## 概述

文案创作执行技能负责根据创作Brief产出高质量的品牌文案内容，覆盖营销文案、产品描述、社交媒体帖子和SEO内容四大文案类型。该技能的核心要求是在遵循品牌调性指南的前提下，实现高效率、高质量、可规模化的文案产出，同时确保每篇文案都附带品牌合规自检结果。

## 适用场景

- 营销Campaign文案创作（广告语/活动文案/落地页文案）
- 产品描述撰写（电商详情页/产品手册/规格说明）
- 社交媒体内容创作（微信/抖音/小红书/微博帖子）
- SEO内容生产（博客文章/知识百科/行业白皮书）
- 品牌传播文案（品牌故事/企业新闻稿/CEO社交账号内容）

## 执行步骤

### 步骤1：Brief解析与规范加载

1. 解析创作Brief中的核心要素：
   - 目标受众画像（决定用词和表达方式）
   - 核心信息点（决定内容重点和说服逻辑）
   - 调性要求（决定语气、句式和修辞风格）
   - 渠道规格（决定篇幅、格式和互动方式）
2. 加载品牌调性指南：推荐词库/禁用词库/语气定义/句式偏好
3. 参考案例分析：提取参考案例中的写作手法和调性元素

### 步骤2：内容结构规划

根据文案类型选择对应的创作框架：

- **营销文案**：AIDA（注意力→兴趣→欲望→行动）或PAS（痛点→激化→解决方案）
- **产品描述**：FABE（特征→优势→利益→证据）
- **社交帖子**：Hook（开头吸引）→Story（内容展开）→CTA（行动引导）
- **SEO文章**：关键词布局规划→大纲生成→正文撰写→SEO元素优化

### 步骤3：文案撰写

1. 按照规划的结构框架进行内容创作
2. 创作过程中实时对照品牌调性要求：
   - 语气是否匹配（正式/活泼/专业/亲和）
   - 用词是否在推荐词库范围内
   - 是否避开了禁用词/敏感表达
   - 句式是否符合品牌风格（短句利落型/长句优雅型等）
3. 如Brief要求多版本（A/B测试），产出差异化版本并标注变量

### 步骤4：品牌调性自检

对产出的文案进行系统化自检评分：

| 自检维度 | 评分标准 | 目标分数 |
|----------|----------|----------|
| 语气匹配度 | 与调性指南一致 | ≥8/10 |
| 用词合规性 | 无禁用词，推荐词使用率高 | ≥8/10 |
| 核心信息覆盖 | Brief要求的卖点全部覆盖 | 100% |
| 受众适配度 | 表达方式匹配目标人群 | ≥8/10 |
| 渠道适配度 | 篇幅/格式/互动元素符合平台要求 | 合格 |

### 步骤5：交付与修改

1. 输出文案正文+自检报告
2. 如自检评分<8分，自动优化后重新输出
3. 接收品牌审核反馈后，按照修改指导进行迭代
4. 修改记录保留，支持版本对比

## 输入/输出规格

### 输入
- 创作Brief（JSON结构化格式）
- 品牌调性指南
- 参考案例（可选）
- 修改反馈（如为修改迭代时）

### 输出
- 文案正文（Markdown格式）
- 品牌调性自检报告（评分+逐项检查结果）
- 关键词覆盖情况（SEO文案适用）
- A/B测试版本差异说明（如为多版本时）

## 最佳实践

1. **开头决定阅读率**：社交媒体文案前3秒/前15字决定用户是否继续阅读，开头必须有Hook
2. **CTA要具体**：不写"了解更多"，写"点击领取限时8折优惠券"
3. **数字增强说服力**：将抽象描述转为具体数字（"超长续航"→"42天不用充电"）
4. **SEO不能牺牲可读性**：关键词密度控制在2-3%，保持文章自然流畅
5. **一稿多改不如多想框架**：结构对了内容自然对，结构错了改多少遍都不对

## 约束条件

- 所有文案必须通过品牌调性自检评分≥8/10方可提交
- 禁用词零容忍——出现禁用词自动标红重写
- 不得抄袭/洗稿已有内容，不得使用竞品品牌名称
- 营销文案须符合《广告法》规定（不使用绝对化用语"最""第一""顶级"等）
- SEO文案须通过原创度检测（>95%原创）
