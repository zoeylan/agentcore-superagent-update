# 品牌资产管理 (Brand Asset Management)

## 概述

品牌资产管理技能负责企业品牌资产库的全生命周期管理，包括品牌规范的录入、更新、版本控制和分发。该技能确保所有内容创作活动在统一的品牌框架下进行，是实现规模化生产下品牌一致性的基础设施能力。

## 适用场景

- 新品牌/子品牌接入时的品牌规范初始化录入
- 品牌升级或VI改版时的规范更新与版本切换
- 品牌资产库的季度Review和合规审计
- 创作任务启动前的品牌规范引用与下发
- 品牌规范变更后的全局通知与影响评估

## 执行步骤

### 步骤1：品牌规范录入/更新

1. 收集品牌规范文档（VI手册/调性指南/禁用元素清单）
2. 结构化解析品牌规范，提取以下核心要素：
   - **色彩体系**：主色（HEX/RGB/CMYK）、辅色、点缀色、使用比例规则
   - **字体规范**：标题字体/正文字体/字号层级/行间距/字间距
   - **Logo规范**：标准版/横版/竖版/最小尺寸/安全空间/禁止变形方式
   - **调性指南**：语气类型（正式/活泼/专业/亲和）、推荐词库、禁用词库、句式风格
   - **模板库**：各类内容模板（海报/banner/社交帖子/视频片头片尾）
   - **禁用元素清单**：竞品元素/已废弃视觉/文化敏感元素
3. 将结构化数据写入品牌资产库，分配唯一版本号
4. 设置规范生效时间和适用范围（全品牌/特定子品牌/特定市场）

### 步骤2：版本管理

1. 每次规范变更创建新版本，保留历史版本完整记录
2. 标注变更内容摘要（变更字段/变更原因/审批人）
3. 设置版本状态：草稿→审批中→已生效→已归档
4. 规范生效后锁定版本，不允许直接修改（需创建新版本）

### 步骤3：变更通知与影响评估

1. 品牌规范变更生效时，自动扫描所有进行中的创作任务
2. 评估变更对各任务的影响程度（高/中/低/无影响）
3. 向受影响任务的执行Agent发送变更通知
4. 高影响任务须暂停并引用新规范重新审核

### 步骤4：季度Review

1. 每季度末触发品牌资产库全面Review
2. 检查项包括：规范完整性/模板可用性/禁用清单时效性
3. 收集创作过程中的规范使用反馈（哪些规范描述模糊需澄清）
4. 输出Review报告，提交品牌负责人审批

## 输入/输出规格

### 输入
- 品牌规范文档（PDF/Word/Figma链接）
- 变更申请（包含变更内容/原因/申请人/期望生效时间）
- Review触发信号（定时触发或手动触发）

### 输出
- 结构化品牌资产库记录（JSON格式）
- 版本变更日志
- 变更影响评估报告
- 季度Review报告

## 最佳实践

1. **规范粒度要细化**：色彩不能只写"蓝色"，必须精确到HEX值和使用场景
2. **禁用清单要及时更新**：竞品新素材/过期代言人/社会敏感事件相关元素需及时加入
3. **调性指南需附加案例**：纯文字描述不够，须配合正例/反例说明
4. **版本切换设缓冲期**：新版本生效后旧版本保留7天兼容期，避免进行中任务断裂
5. **权限分级管理**：查看权限开放给所有创作Agent，修改权限仅限品牌内容策略师

## 约束条件

- 品牌规范变更必须经过品牌负责人审批后方可在系统中生效
- 新品牌接入必须完成完整规范录入（调性指南+VI规范+禁用元素清单）三项缺一不可
- 已发布版本不允许直接修改，必须创建新版本
- 品牌资产库每季度至少Review一次
