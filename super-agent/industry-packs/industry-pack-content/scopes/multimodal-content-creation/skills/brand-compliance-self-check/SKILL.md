# 品牌合规自检 (Brand Compliance Self-Check)

## 概述

品牌合规自检技能是多模态创作执行师在内容产出后、提交审核前执行的内部质量自查流程。该技能通过系统化的检查清单和量化评分，帮助创作者在提交前发现并修正品牌规范偏差，显著降低审核退回率（目标：首次审核通过率>80%）。这是一种"质量左移"策略——将品牌合规检查从审核环节前移到创作环节。

## 适用场景

- 文案创作完成后的调性自检
- 图像生成后的VI合规自检
- 视频制作完成后的品牌元素检查
- 修改迭代后的变更点复查
- 批量创作时的抽样自检

## 执行步骤

### 步骤1：加载品牌检查标准

1. 根据内容类型确定适用的检查维度集合：
   - 文案类：调性/用词/句式/信息/受众 五维度
   - 图像类：色彩/字体/Logo/布局/禁用元素 五维度
   - 视频类：片头片尾/字幕/水印/配色/音乐 五维度
   - 综合类：全部维度
2. 加载当前品牌规范版本中的具体参数值
3. 确认是否有本次任务的特殊约束（Brief中的额外要求）

### 步骤2：逐维度自动检查

**文案自检流程：**
```
1. 禁用词扫描 → 命中则标红（零容忍）
2. 推荐词覆盖率计算 → 低于50%则预警
3. 语气分析 → 与品牌调性关键词匹配度评分
4. 广告法合规扫描 → 绝对化用语检测
5. 核心信息点覆盖确认 → 与Brief逐条对照
```

**图像自检流程：**
```
1. 色彩提取 → 与品牌色板计算ΔE偏差
2. 字体识别 → 与品牌字体库比对
3. Logo检测 → 尺寸/位置/安全空间/变形度
4. 禁用元素扫描 → 图像内容安全检查
5. 尺寸规格验证 → 与目标平台规格比对
```

**视频自检流程：**
```
1. 片头片尾模板匹配 → 与品牌标准模板对比
2. 字幕样式检查 → 字体/颜色/位置/阴影
3. Logo水印检测 → 全片可见性/位置/尺寸
4. 技术参数验证 → 分辨率/帧率/码率/时长
5. 音频合规 → 版权确认/音量标准
```

### 步骤3：评分与决策

1. 各维度独立评分（1-10分）
2. 计算加权总分：
   - 硬性指标（禁用词/Logo/安全）：不达标=一票否决
   - 软性指标（调性/美学/创意）：加权计算总分
3. 自检结论：
   - **可提交**（总分≥8且无硬性指标不达标）
   - **需优化**（总分6-7.9或有轻微硬性问题）→ 自动修正后重检
   - **需重做**（总分<6或有严重硬性问题）→ 标注问题返回创作步骤

### 步骤4：问题修正

1. 对"需优化"级别的问题尝试自动修正：
   - 禁用词替换为推荐同义词
   - 色彩微调至ΔE<3
   - Logo尺寸/位置自动校正
2. 修正后重新执行检查（最多2轮自动修正）
3. 自动修正仍不通过则标注问题点，人工介入

### 步骤5：生成自检报告

输出结构化自检报告：
```json
{
  "checkTime": "检查时间",
  "contentType": "文案|图像|视频",
  "brandVersion": "品牌规范版本号",
  "overallScore": 8.5,
  "verdict": "可提交|需优化|需重做",
  "dimensions": [
    {
      "name": "维度名称",
      "score": 9,
      "issues": [],
      "autoFixed": false
    }
  ],
  "hardRequirements": {
    "prohibitedWords": "pass|fail",
    "logoCompliance": "pass|fail",
    "safetyCheck": "pass|fail"
  },
  "autoFixApplied": ["修正项描述"],
  "remainingIssues": ["未能自动修正的问题"]
}
```

## 输入/输出规格

### 输入
- 创作产出内容（文案/图像/视频文件）
- 品牌规范参数（当前有效版本）
- 创作Brief（用于核心信息覆盖验证）

### 输出
- 品牌合规自检报告（JSON格式）
- 自动修正后的内容（如有修正）
- 未解决问题清单（如有需人工处理的问题）

## 最佳实践

1. **先查硬性后查软性**：禁用词/Logo/安全是一票否决项，先检查可以快速发现严重问题
2. **自动修正要保守**：只修正确定性高的问题（禁用词替换/尺寸调整），不自动修改创意表达
3. **记录修正历史**：每次自动修正都记录原始值和修正值，便于审核时追溯
4. **批量时做抽检**：100+篇批量创作时全量自检效率低，可全量扫硬性+抽样查软性
5. **反馈闭环**：将高频不通过项反馈给创作流程，从源头减少问题产生

## 约束条件

- 自检报告必须附在每次内容提交中，不允许跳过自检直接提交
- 硬性指标不达标不允许提交，即使总分≥8
- 自动修正不得改变内容核心语义和创意方向
- 自检评分标准必须与品牌一致性审核标准对齐（避免自检通过但审核不通过）
