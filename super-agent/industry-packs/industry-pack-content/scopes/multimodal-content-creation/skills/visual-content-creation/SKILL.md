# 视觉内容创作 (Visual Content Creation)

## 概述

视觉内容创作技能负责通过AI生成工具（DALL-E/Midjourney/Stable Diffusion/Flux等）制作海报、Banner、产品图、信息图等视觉素材。该技能的核心差异化在于严格遵循品牌VI规范——每一张产出的图像都必须在配色、字体、布局、Logo使用等方面完全合规，实现"创意自由度"与"品牌规范约束"之间的最优平衡。

## 适用场景

- 营销海报设计（电商大促/节日营销/活动宣传）
- Banner广告制作（网站首屏/APP开屏/信息流广告）
- 产品图制作（白底图/场景图/详情图/对比图）
- 信息图设计（数据可视化/流程图/对比图表）
- 社交媒体视觉素材（封面图/配图/头图）

## 执行步骤

### 步骤1：设计需求解析

1. 从Brief中提取视觉创作要求：
   - 内容主题和关键信息
   - 目标平台和尺寸规格
   - 视觉风格参考（参考图/moodboard）
   - 品牌VI约束条件
2. 确认品牌VI参数：
   - 主色调（HEX值）和使用比例
   - 字体规范（标题/正文/辅助文字）
   - Logo版本选择和放置规则
   - 布局栅格和安全边距

### 步骤2：Prompt工程

1. 根据设计需求构建生成Prompt：
   - 主体描述（场景/物体/人物）
   - 风格定义（写实/插画/扁平/3D等）
   - 构图指导（视角/布局/焦点）
   - 色彩约束（引用品牌色彩体系）
   - 负向提示（排除不需要的元素/风格）
2. 选择最优生成模型：
   - 写实风格 → Midjourney/Flux
   - 插画风格 → DALL-E/Stable Diffusion
   - 产品图 → 专用产品图模型
3. 设置生成参数（分辨率/宽高比/迭代步数/CFG scale）

### 步骤3：图像生成与筛选

1. 执行生成（通常生成4-8张候选）
2. 从候选中筛选最佳结果，评估维度：
   - 主题表达准确度
   - 美学质量
   - 技术质量（无伪影/无畸变）
   - 品牌VI初步符合度
3. 如无满意结果，调整Prompt重新生成（最多3轮）

### 步骤4：后处理与品牌元素叠加

1. 对选中的图像进行后处理：
   - 色彩校正（确保与品牌色板一致，ΔE<3）
   - 尺寸裁切（适配目标平台规格）
   - 添加品牌元素（Logo/字体排版/装饰图形）
   - 合成文字层（使用品牌指定字体）
2. 格式转换（根据用途选择PNG/JPG/WebP/SVG）
3. 文件大小优化（满足平台上传限制）

### 步骤5：VI合规自检

逐项检查品牌VI合规性：

| 检查项 | 标准 | 结果 |
|--------|------|------|
| 主色使用 | 与品牌主色HEX值偏差ΔE<3 | ✓/✗ |
| 辅色使用 | 在品牌辅色范围内 | ✓/✗ |
| 配色比例 | 符合品牌规定比例（如60:30:10） | ✓/✗ |
| 标题字体 | 使用品牌指定标题字体 | ✓/✗ |
| 正文字体 | 使用品牌指定正文字体 | ✓/✗ |
| Logo尺寸 | ≥最小尺寸要求 | ✓/✗ |
| Logo安全空间 | 满足最小安全空间 | ✓/✗ |
| Logo无变形 | 无拉伸/旋转/裁切 | ✓/✗ |
| 禁用元素 | 未使用禁用元素清单中的元素 | ✓/✗ |
| 尺寸规格 | 符合目标平台要求 | ✓/✗ |

## 输入/输出规格

### 输入
- 创作Brief（视觉部分）
- 品牌VI规范参数
- 参考图/Moodboard（可选）
- 产品素材（产品图创作时）

### 输出
- 生成的视觉素材文件（指定格式/尺寸）
- 使用的Prompt记录（便于复现和迭代）
- VI合规检查清单（逐项确认结果）
- 设计说明（创意思路/元素选择理由）

## 最佳实践

1. **Prompt要精确而非冗长**：核心描述控制在50词内，用负向提示排除不想要的元素
2. **先构图后细节**：先确定整体布局和构图合理性，再细化元素质量
3. **品牌色先于创意色**：生成时就在Prompt中约束色彩范围，而非后期大幅修色
4. **预留合成空间**：如需叠加文字/Logo，生成时预留留白区域
5. **多候选多筛选**：一次生成多张，从中选最优，比反复修改一张效率更高

## 约束条件

- 色彩偏差必须控制在ΔE<3范围内
- 字体使用必须100%在品牌字体库内
- Logo使用必须符合品牌手册全部规则
- 不得生成含有真实人脸的素材（肖像权风险）除非有明确授权
- 不得使用有版权争议的视觉风格（如模仿特定艺术家风格）
- 生成的素材必须经过安全扫描（无违规内容）
