# 多语言本地化 (Multilingual Localization)

## 概述

多语言本地化技能负责将内容从源语言翻译为目标语言，并进行深度文化适配。该技能超越简单的机器翻译——执行"创译"（Transcreation）策略，确保翻译后的内容在目标文化中自然流畅、无文化禁忌、保持品牌调性，同时融入本地表达习惯。支持10+主要语言，翻译准确率目标≥95%。

## 适用场景

- 营销文案的多语言版本生产（出海品牌Campaign）
- 产品描述的本地化（跨境电商多语言listing）
- 社交媒体内容的多语言适配
- 品牌Slogan/标语的创译
- 视频字幕的多语言翻译
- 法律/合规声明的精确翻译

## 执行步骤

### 步骤1：源内容分析

1. 确认源语言和目标语言列表
2. 分析源内容特征：
   - 内容类型（营销/信息/法律/技术）
   - 文化特定元素（双关语/成语/网络用语/数字梗）
   - 品牌调性关键词
   - 不可翻译元素标注（品牌名/产品型号/专有名词）
3. 评估本地化难度等级：
   - 简单（信息类，无文化特定元素）
   - 中等（营销类，有修辞但可替代）
   - 困难（强文化依赖，需要创译）

### 步骤2：翻译与文化适配

1. 执行核心翻译，策略选择：
   - 信息型内容 → 直译+润色（Faithful Translation）
   - 营销型内容 → 创译（Transcreation，保留情感意图而非字面含义）
   - 法律型内容 → 精确翻译（Exact Translation，不可意译）
2. 文化适配处理：
   - **颜色象征**：白色在西方=纯洁，中国=丧事；红色中国=喜庆，中东=危险
   - **数字禁忌**：4在中日韩=不吉利，13在西方=不吉利
   - **手势/肢体语言**：竖大拇指在中东=冒犯
   - **动物象征**：猫在日本=吉祥，某些中东地区=不洁
   - **计量单位**：公制/英制转换，温度℃/℉
   - **日期格式**：YYYY/MM/DD vs MM/DD/YYYY vs DD/MM/YYYY
   - **货币符号**：本地货币前置/后置规则
3. SEO本地化：
   - 目标语种关键词研究（非直译源语言关键词）
   - 本地搜索习惯适配
   - Meta标签本地化

### 步骤3：质量校验

1. **回译校验（Back Translation）**：
   - 将翻译结果翻译回源语言
   - 对比回译结果与原文的语义一致性
   - 语义保留率≥95%为合格
2. **文化适配清单确认**：

| 检查项 | 状态 |
|--------|------|
| 无颜色/数字文化禁忌 | ✓/✗ |
| 无宗教/政治敏感引用 | ✓/✗ |
| 计量单位已本地化 | ✓/✗ |
| 日期/时间格式正确 | ✓/✗ |
| 货币标注正确 | ✓/✗ |
| 本地习惯表达自然 | ✓/✗ |
| 幽默/修辞适配得当 | ✓/✗ |
| 人名/地名本地化 | ✓/✗ |

3. **语言质量评估**：
   - 语法正确性
   - 拼写/标点准确
   - 语体适切性（正式/非正式与品牌调性匹配）
   - 术语一致性

### 步骤4：品牌调性验证

1. 确认翻译版本保持品牌核心调性特征
2. 品牌名/Slogan的翻译策略确认：
   - 保留原文（如Apple/Nike）
   - 音译（如可口可乐）
   - 意译（如宝马=Bayerische Motoren Werke）
3. 核心信息点覆盖确认（是否在翻译过程中丢失关键卖点）

### 步骤5：审核分级

1. 根据语种成熟度决定审核模式：
   - **新语种**（首次进入市场）：母语reviewer全量审核
   - **成熟语种**（有历史积累）：抽检模式（抽检率20%）
   - **高风险内容**（法律/医疗/政治相关）：无论语种均全量审核
2. 创建外审工单（如需母语reviewer）
3. 收集审核反馈，更新翻译记忆库

## 输入/输出规格

### 输入
- 源内容（文案/字幕/界面文本）
- 目标语言列表
- 品牌调性指南（目标语言版本，如有）
- 术语表/翻译记忆库
- 本地化特殊要求（如特定市场的法规要求）

### 输出
- 翻译后内容（原文-译文对照格式）
- 回译校验结果（语义保留率评分）
- 文化适配清单完成确认
- 本地化质量评分
- 不可翻译元素处理说明
- 外审工单（如需人工reviewer）

## 最佳实践

1. **术语一致性优先**：建立并维护品牌专属术语库，确保同一术语全渠道翻译一致
2. **不要直译营销文案**：好的营销翻译是"用目标语言重新创作"，而非逐字对照
3. **长度膨胀要预估**：英译中通常缩短30%，中译德通常膨胀40%，影响排版布局
4. **本地化不仅是文字**：图片中的文字、按钮文案、表单标签都需要本地化
5. **持续积累翻译记忆**：历史优质翻译入库，提升后续翻译效率和一致性

## 约束条件

- 翻译准确率（回译校验）必须≥95%
- 文化适配清单所有必检项必须逐一确认
- 品牌名/Slogan翻译策略须经品牌内容策略师审批
- 法律/医疗/金融类内容翻译须经专业人员复核
- 首次进入新语种市场须全量母语审核，不允许跳过
