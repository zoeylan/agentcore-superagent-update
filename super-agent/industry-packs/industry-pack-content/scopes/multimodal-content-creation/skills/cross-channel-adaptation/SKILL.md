# 跨渠道适配 (Cross-Channel Adaptation)

## 概述

跨渠道适配技能负责将同一内容自动调整为各目标平台的最优呈现形式，解决企业多平台运营中"一稿多改"的低效问题。该技能不仅确保适配版本在格式/尺寸/时长上符合平台技术规格，还针对各平台的内容生态和算法偏好进行内容策略调整，使内容在目标平台上获得最大化传播效果。

## 适用场景

- 一条内容需要同时发布到多个平台时
- 横版视频需要适配竖版平台时
- 长文需要适配短内容平台时
- 图文内容需要适配纯图/纯文平台时
- 新平台接入时的适配模板开发

## 执行步骤

### 步骤1：平台规格确认

加载目标平台的最新规格要求：

**文本内容规格：**
| 平台 | 标题限制 | 正文限制 | 格式特点 | 标签要求 |
|------|----------|----------|----------|----------|
| 微信公众号 | ≤64字 | 不限 | 富文本/长文/多图穿插 | 无 |
| 抖音 | ≤55字 | ≤300字 | 短文案/表情包/话题标签 | #话题 |
| 小红书 | ≤20字 | ≤1000字 | 笔记体/干货向/emoji多 | #标签 |
| Instagram | 无 | ≤2200字符 | 短caption/换行分段 | #hashtag |
| Twitter/X | 无 | ≤280字符 | 精炼/话题性/Thread | #hashtag |
| LinkedIn | ≤150字 | ≤3000字符 | 专业/故事性/数据驱动 | #hashtag |

**图片规格：**
| 平台 | 推荐尺寸 | 格式 | 文件大小 | 图片数量 |
|------|----------|------|----------|----------|
| 微信公众号封面 | 900×383px (2.35:1) | JPG/PNG | ≤5MB | 单图/多图 |
| 抖音封面 | 1080×1920px (9:16) | JPG | ≤5MB | 单图 |
| 小红书 | 1080×1440px (3:4) | JPG/PNG | ≤20MB | 1-9图 |
| Instagram帖子 | 1080×1080px (1:1) | JPG | ≤30MB | 1-10图 |
| Twitter/X | 1200×675px (16:9) | JPG/PNG | ≤5MB | 1-4图 |

**视频规格：**
| 平台 | 分辨率 | 比例 | 时长 | 格式 | 大小限制 |
|------|--------|------|------|------|----------|
| 抖音 | 1080×1920 | 9:16 | 15-600s | MP4 | ≤4GB |
| 小红书 | 1080×1440 | 3:4 | 60-300s | MP4 | ≤1GB |
| B站 | 1920×1080 | 16:9 | 不限 | MP4 | ≤8GB |
| Instagram Reels | 1080×1920 | 9:16 | 15-90s | MP4 | ≤4GB |
| YouTube Shorts | 1080×1920 | 9:16 | ≤60s | MP4 | ≤256MB |

### 步骤2：内容策略适配

针对各平台内容生态进行策略调整（非仅格式调整）：

**微信公众号适配策略：**
- 深度长文+故事性叙事
- 封面图要有信息量（标题+核心卖点可视化）
- 正文穿插配图（每300字至少1图）
- 结尾引导关注+引流

**抖音适配策略：**
- 前3秒Hook决定完播率
- 文案短平快，直击痛点
- 竖版构图，人物居中
- 热门BGM/话题标签使用

**小红书适配策略：**
- 封面图决定点击率（高颜值/信息量/对比图）
- 笔记体表达，真诚分享感
- 分享干货/教程/测评/种草
- 大量emoji+结构化排版

**Instagram适配策略：**
- 视觉优先（高质量图片/统一色调）
- Caption简洁有力
- 精准hashtag（15-30个）
- Stories/Reels互动性强

### 步骤3：格式转换执行

1. **图片适配**：
   - 裁切原图到目标比例（智能裁切，保留主体）
   - 调整分辨率到目标平台要求
   - 文件格式转换和压缩（控制大小在限制内）
   - 添加平台特定元素（水印位置调整等）

2. **文案适配**：
   - 根据字数限制进行内容精简/扩充
   - 调整语言风格匹配平台调性
   - 添加平台特定元素（话题标签/emoji/CTA）
   - 关键信息保留优先级处理

3. **视频适配**：
   - 画面比例转换（横转竖/竖转横）
   - 时长剪辑（长视频→短视频精华剪辑）
   - 字幕位置/大小调整
   - 平台特定包装（封面/片头片尾调整）

### 步骤4：平台规格校验

对每个适配版本执行自动化规格校验：
```
✓ 图片尺寸：1080×1080px = 符合Instagram要求
✓ 文件格式：JPG = 符合
✓ 文件大小：2.3MB ≤ 30MB限制 = 符合
✓ 文案长度：185字符 ≤ 2200字符限制 = 符合
✓ Hashtag数量：22个 ∈ [15,30]推荐范围 = 符合
```

校验不通过则自动调整并重新校验，直至全部通过。

### 步骤5：最佳实践建议附加

为每个适配版本附加发布建议：
- 推荐发布时间（基于平台活跃时段数据）
- 标签/话题建议（基于当前平台热度）
- 互动引导语建议
- 预估内容表现区间

## 输入/输出规格

### 输入
- 原始内容（文案/图像/视频）
- 目标平台列表
- 品牌规范约束
- 内容优先级（哪些信息必须保留/可以删减）

### 输出
- 各平台适配版本文件
- 平台规格校验报告（逐项确认）
- 内容策略调整说明（与原版差异标注）
- 发布最佳实践建议

## 最佳实践

1. **先确定信息优先级再裁剪**：缩短文案时根据重要程度决定保留什么，而非机械截断
2. **视觉焦点要在安全区域内**：裁切时确保主体不被UI元素遮挡
3. **平台规格库要实时更新**：各平台规则变化频繁，过期规格可能导致发布失败
4. **不是所有内容都适合所有平台**：有些深度内容只适合长文平台，强行缩短反而影响品质
5. **测试发布验证**：新平台/新规格首次适配建议先小范围测试验证效果

## 约束条件

- 每个适配版本必须通过平台规格自动校验（100%通过才可提交）
- 适配不得改变品牌核心视觉元素（Logo/品牌色/字体）
- 核心营销信息不得因适配而被删除（可精简表达但不可缺失）
- 新平台接入须完成适配模板开发+10条样例验证
- 平台规格变更须在24小时内更新规格库
