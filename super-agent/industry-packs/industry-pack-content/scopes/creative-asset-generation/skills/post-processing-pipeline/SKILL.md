# 后处理流水线（Post-Processing Pipeline）

## 技能概述

本技能负责对AI生成的原始素材进行后处理操作，使其满足最终交付的技术规格和业务需求。后处理是生成到交付之间的关键桥梁，包括尺寸适配、格式转换、质量增强、水印叠加、画布合成等操作。

## 适用场景

- 生成的原始图片需要适配到多种平台尺寸
- 需要添加品牌水印或logo叠加
- 需要将多个生成元素合成到一张画布上
- 文件格式和大小需要满足特定平台要求
- 图片质量增强（超分辨率/降噪/锐化）

## 执行步骤

### Step 1: 规格校验
检查生成原图是否满足基本要求：
- 分辨率是否达到最低要求（通常≥1024px短边）
- 色彩空间是否正确（sRGB for web, CMYK for print）
- 是否存在明显的技术瑕疵需要修复

### Step 2: 尺寸适配

常见平台尺寸规格：
| 平台 | 场景 | 尺寸(px) | 宽高比 |
|------|------|----------|--------|
| 淘宝 | 商品主图 | 800×800 | 1:1 |
| 淘宝 | 详情页 | 750×不限 | - |
| 京东 | 商品主图 | 800×800 | 1:1 |
| 抖音 | 商品卡片 | 720×960 | 3:4 |
| 小红书 | 笔记配图 | 1080×1440 | 3:4 |
| 微信 | 公众号封面 | 900×383 | 2.35:1 |
| Instagram | Feed | 1080×1080 | 1:1 |
| Facebook | Ad | 1200×628 | 1.91:1 |

适配策略：
- **智能裁切**：基于主体检测的中心裁切（避免裁掉重要元素）
- **画布扩展**：使用AI outpainting扩展背景以适配更宽的比例
- **缩放+留白**：缩小内容+添加纯色/渐变背景填充

### Step 3: 格式转换

| 格式 | 适用场景 | 注意事项 |
|------|----------|----------|
| PNG | 需要透明背景 | 文件较大，适合设计稿交付 |
| JPG | 照片类素材 | 质量设80-95，平衡大小和质量 |
| WebP | Web展示 | 体积最小，部分老浏览器不支持 |
| AVIF | 新一代Web | 压缩率最高，兼容性待提升 |
| PSD | 设计师二次编辑 | 保留图层信息 |
| SVG | 矢量图形 | 仅适用于AI生成的矢量内容 |

### Step 4: 质量增强（按需）
- **超分辨率放大**：Real-ESRGAN/GFPGAN，适用于需要大尺寸输出的场景
- **降噪处理**：去除AI生成常见的噪点和伪影
- **锐化处理**：增强文字和边缘的清晰度
- **色彩校正**：统一同系列素材的色温和饱和度

### Step 5: 品牌元素叠加
- Logo水印位置（通常右下角，不遮挡主体）
- 水印透明度控制（30-50%透明度）
- 品牌色条/边框添加
- 固定文案区域预留

### Step 6: 批量打包交付
- 按命名规范重命名文件（如：`{品牌}_{场景}_{尺寸}_{版本号}.{格式}`）
- 生成预览缩略图（用于资产管理系统展示）
- 写入EXIF元数据（生成时间/模型/版权信息）
- 按文件夹结构组织（按平台/按尺寸/按主题）
- 压缩打包（ZIP格式，附带清单文件）

## 输入规格

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| raw_assets | array | 是 | 原始生成文件列表 |
| target_specs | array | 是 | 目标平台规格列表 |
| brand_overlays | object | 否 | 品牌水印/logo配置 |
| enhancement_config | object | 否 | 质量增强配置 |
| naming_convention | string | 否 | 文件命名规范 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| processed_assets | array | 处理完成的文件列表 |
| delivery_package | object | 打包交付物 |
| processing_report | object | 处理报告（含错误信息） |
| thumbnail_gallery | array | 预览缩略图 |

## 最佳实践

1. **无损优先**：中间处理步骤尽量使用无损格式，最终输出再压缩
2. **主体检测先行**：裁切前先检测主体位置，避免裁掉关键内容
3. **批量规格统一**：同一系列的素材使用统一的后处理参数
4. **保留原图**：所有后处理操作基于原图副本，原图永远保留
5. **自动化校验**：处理后自动校验文件尺寸/格式/大小是否符合目标规格

## 常见陷阱

- JPG压缩过度导致画质损失（特别是渐变和细节区域）
- 超分辨率放大过度导致"塑料感"
- 水印遮挡了主体重要信息
- 裁切后构图失衡（主体偏移到边缘）
- 忽略了平台的文件大小限制（某些平台限制单图<2MB）
