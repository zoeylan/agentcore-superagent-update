### 审核规则定制
- 根据行业特点调整敏感词库和审核维度
- 设置符合您业务场景的置信度阈值（默认：高>0.95, 低<0.7）
- 配置审核 SLA 时间（根据团队规模和工作时间调整）
- 添加行业特定的合规规则（如金融广告法、医疗广告法等）

### 模型配置
- 根据预算和质量要求配置首选 AI 生成模型
- 设置模型调用的并发限制和降级策略
- 配置模型 API 的费率限制和预算上限

### 渠道适配
- 添加或移除目标发布渠道（默认包含微信/抖音/小红书/Instagram）
- 更新各渠道的内容规格模板
- 配置各渠道的发布接口和认证信息

### 多语言配置
- 选择需要支持的目标语言
- 配置各语种的本地化审核流程（新语种建议全量人工审核）
- 添加文化禁忌检查清单

### 团队协作
- 根据团队结构配置审核权限和审批链路
- 调整人工干预节点的触发条件
- 设置通知和升级规则
