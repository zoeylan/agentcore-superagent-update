# 多模态内容创作指导

## 技能描述
针对图文、短视频、音频、互动内容等多种模态，提供从创意构思到执行落地的全流程创作指导，包括创意Brief撰写、脚本大纲、视觉方向建议、文案优化等。

## 适用场景
- 短视频脚本撰写（抖音/快手/视频号）
- 小红书图文笔记策划
- 品牌宣传片创意方案
- 直播脚本与话术设计
- 公众号深度长文撰写
- H5互动内容策划
- 海报/Banner视觉创意方向
- 播客/音频内容策划

## 输入要求
请提供以下信息：
1. **内容模态**：图文/短视频/长视频/音频/互动/海报等
2. **目标平台**：将在哪个平台发布
3. **内容目标**：品牌曝光/种草转化/教育科普/情感共鸣/热点借势
4. **目标受众**：面向什么人群
5. **核心卖点/信息点**：需要传递的关键信息（1-3个）
6. **品牌调性要求**：风格/语气/视觉调性
7. **参考案例**（可选）：喜欢的同类内容案例
8. **限制条件**：预算、时长、禁忌词、合规要求等

## 输出格式

### 短视频类输出
```
【创意概念】一句话创意核心
【内容钩子】前3秒抓眼方案
【分镜脚本】
| 时间 | 画面 | 文案/旁白 | 音乐/音效 | 字幕 |
|------|------|-----------|-----------|------|
| 0-3s | ... | ... | ... | ... |

【发布建议】标题、标签、发布时间
【AB测试建议】可测试的变量
```

### 图文类输出
```
【选题角度】
【标题备选】（3-5个，含不同风格）
【内容大纲】结构化的内容框架
【关键文案段落】金句/hook/CTA
【配图方向】视觉风格、构图建议
【SEO/标签建议】
```

### 创意Brief输出
```
【项目背景】
【创意目标】
【目标受众洞察】
【核心信息】Key Message
【创意方向】2-3个方向供选择
【调性参考】Moodboard描述
【执行要求】规格、时长、交付物
【时间节点】
```

## 创意方法论
- **SCAMPER法**：替代、合并、改造、转用、消除、逆转、重排
- **强制关联法**：将不相关元素强行组合产生新创意
- **情绪坐标法**：在情绪维度（开心-感动-惊喜-共鸣）中定位
- **爆款公式**：痛点共鸣 + 信息增量 + 情绪价值 + 行动召唤
- **3B原则**：Beauty（美感）、Beast（猎奇）、Baby（萌系）

## 平台创作规则
- 抖音：前3秒定生死，完播率优先，评论区互动设计
- 小红书：封面图>标题>正文，真实感优先，避免硬广感
- 视频号：社交推荐逻辑，适合情感/知识类，转发率是核心
- B站：长内容友好，信息密度要高，弹幕互动设计
- 公众号：深度内容，标题党有效但需克制，打开率持续下降需创新

## 注意事项
- 所有创意建议会标注合规风险等级（低/中/高）
- 会指出哪些环节可以用AI工具辅助（如AI生成初稿、AI配图等）
- 会考虑内容的可复制性和规模化生产可能性
- 涉及肖像、音乐、素材版权问题会特别提醒
