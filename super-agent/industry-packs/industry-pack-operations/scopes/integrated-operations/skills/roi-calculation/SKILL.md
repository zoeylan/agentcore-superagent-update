# ROI核算与分析 (ROI Calculation)

## 技能概述

ROI核算技能负责计算各渠道、活动、商品线的投入产出比，识别高效和低效的资源配置，为运营资源的重新分配提供量化决策依据。该技能贯穿预算分配的前期规划和事后评估环节。

## 适用场景

- 渠道ROI排名：对比各投放渠道的投入产出效率
- 活动效果评估：大促/营销活动的投入产出核算
- 资源重配建议：基于ROI数据建议预算在渠道间重新分配
- 边际收益分析：评估追加投入的边际ROI是否仍有正收益

## 执行步骤

### 步骤1：成本归集
- 按分析维度（渠道/活动/商品线）归集直接成本
- 直接成本：投放费用、活动补贴、物流费用、人力直接工时
- 间接成本分摊：系统费用、管理费用按合理基准分摊
- 确认成本口径一致性（含税/不含税、是否含退款）

### 步骤2：收益量化
- 直接收益：GMV、订单收入、会员费收入
- 间接收益估算：品牌曝光价值（CPM折算）、新客获取价值（按LTV估算）
- 归因逻辑：多触点归因时采用末次互动/线性归因/时间衰减模型
- 时间窗口：确认收益统计的归因窗口（7天/15天/30天）

### 步骤3：ROI计算与对比
- 基础ROI = 收益 / 成本
- 增量ROI = (有投入的收益 - 无投入基线收益) / 投入成本
- 按维度汇总排名，标注高于/低于基准线（ROI=3）的项目
- 计算各维度的边际ROI趋势（追加投入时ROI变化方向）

### 步骤4：分析与建议
- 识别Top3高效渠道/活动（建议加大投入）
- 识别Bottom3低效渠道/活动（建议缩减或优化）
- 评估资源重配方案的预期收益提升
- 输出ROI趋势预测（基于近期数据预测未来ROI走向）

## 输入规格

| 输入项 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| 成本数据 | 结构化数据 | 是 | 各维度的投入明细 |
| 收益数据 | 结构化数据 | 是 | 对应的营收/转化数据 |
| 分析维度 | 枚举 | 是 | 渠道/活动/商品线/时间 |
| 归因模型 | 枚举 | 否 | 默认末次互动归因 |
| ROI基准线 | 数值 | 否 | 默认3.0 |

## 输出规格

| 输出项 | 格式 | 说明 |
|--------|------|------|
| ROI排名表 | 表格 | 各维度ROI值及排名 |
| 效率象限图描述 | 文本 | 高投入高回报/高投入低回报等四象限 |
| 资源重配建议 | 列表 | 从低效向高效迁移的具体方案 |
| 边际ROI分析 | 图表描述 | 各渠道追加投入的边际收益曲线 |
| 预期收益提升 | 数值 | 实施建议后的ROI预测值 |

## 最佳实践

1. **统一口径**：确保各渠道ROI计算的成本和收益口径一致，避免苹果比橘子
2. **区分短期和长期ROI**：品牌投放短期ROI低但长期价值大，需分开评估
3. **考虑边际效应**：ROI高的渠道追加投入后ROI可能下降，需评估边际曲线
4. **多维交叉分析**：渠道×人群×时段的交叉分析可发现更精细的优化空间
5. **ROI<1自动暂停规则**：投放类渠道ROI持续<1超过30分钟应建议暂停

## 约束条件

- 间接收益估算需明确标注假设条件和计算方法
- ROI计算结果受归因模型选择影响，需标注所用模型
- 不对ROI低的渠道直接做关停决策，仅提供分析和建议
- 品牌建设类投入不适用短期ROI评估标准
