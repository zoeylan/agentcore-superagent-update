# 商品文案生成 (Product Copywriting)

## 概述
本技能用于店铺运营管家基于商品基础信息自动生成符合各电商平台规范的高质量商品文案，包括标题、卖点描述、详情页文案和SEO关键词优化。文案需兼顾搜索流量获取和用户转化引导。

## 使用场景
- 新商品上架前的文案生成
- 存量商品文案优化（转化率低/搜索排名下降）
- 大促活动文案调整（加入促销利益点）
- 多平台差异化文案适配
- 竞品分析后的文案改写

## 执行步骤

### Step 1: 商品信息解析
- 提取商品核心信息：品类/品牌/型号/规格/材质/功效/适用人群
- 分析商品定位：价格带（低/中/高）、差异化优势、目标客群
- 获取竞品文案参考（TOP10同类商品标题和卖点）
- 获取平台搜索热词（近7天该品类搜索TOP关键词）

### Step 2: SEO关键词策略
- 筛选核心关键词（搜索量高+竞争适中+高度相关）
- 组合长尾关键词（核心词+属性词+场景词+人群词）
- 各平台关键词适配：
  - 淘宝/天猫：标题限30字，前10字最重要（影响搜索权重）
  - 抖音小店：标题限20字，重视内容属性词
  - 拼多多：标题限60字，适合堆砌关键词
  - 快手小店：标题限30字，口语化表达优先

### Step 3: 标题生成
- 按平台规范生成多版本标题
- 标题结构：[品牌词] + [核心关键词] + [属性词] + [卖点词] + [场景词]
- 确保核心卖点词在前半段（用户注意力集中区）
- 生成3-5个备选标题供选择

### Step 4: 卖点描述（五点描述）
- 按用户关注优先级排序撰写5条卖点：
  1. 核心功效/价值主张（解决什么问题）
  2. 产品成分/材质/技术（为什么有效）
  3. 适用场景/人群（谁适合用）
  4. 使用体验/口碑（效果怎么样）
  5. 售后保障/性价比（买了不亏）
- 每条卖点控制在50-80字，包含关键词

### Step 5: 详情页文案
- 按标准结构生成详情页内容框架：
  - 首屏：核心卖点海报文案（一句话打动用户）
  - 痛点：目标人群的痛点/需求描述
  - 方案：产品如何解决痛点
  - 证据：成分/技术/检测报告/用户评价
  - 细节：产品细节展示文案
  - 使用：使用方法/教程文案
  - 保障：售后/退换/正品承诺

### Step 6: 合规检查与输出
- 广告法合规检查（无绝对化用语/无虚假宣传/无对比贬低）
- 平台规范检查（无违禁词/无敏感词/无引导站外交易）
- 输出各平台版本的最终文案包
- 标注可选替换词和A/B测试建议

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| product_basic_info | object | 是 | 商品基础信息 |
| product_category | string | 是 | 商品品类 |
| target_platforms | array | 是 | 目标上架平台列表 |
| competitor_analysis | object | 否 | 竞品文案分析结果 |
| search_keywords | array | 否 | 平台搜索热词 |
| brand_tone | string | 否 | 品牌调性要求 |
| promotion_info | object | 否 | 促销活动信息 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| titles | object | 各平台标题（3-5版本/平台） |
| bullet_points | array | 五点卖点描述 |
| detail_page_copy | object | 详情页文案框架 |
| seo_keywords | object | SEO关键词策略 |
| compliance_check | object | 合规检查结果 |
| ab_test_variants | array | A/B测试备选版本 |

## 最佳实践
1. **数据驱动选词**：基于平台搜索数据选词，不靠直觉
2. **竞品差异化**：避免与TOP竞品文案雷同，突出差异化卖点
3. **用户语言**：使用目标客群的日常表达，不用行业术语
4. **转化优先**：标题要兼顾搜索和点击，不能只堆关键词不考虑可读性
5. **定期优化**：根据搜索排名和转化数据每2周优化一次文案

## 约束条件
- 所有文案必须通过广告法合规检测
- 标题长度严格遵守各平台字数限制
- 功效描述必须有对应的产品资质支撑
- 文案生成时效：商品信息采集后2小时内完成初稿
- 禁止抄袭竞品文案（相似度需<30%）
