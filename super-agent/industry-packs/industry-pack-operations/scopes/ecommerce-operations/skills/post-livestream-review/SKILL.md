# 直播复盘分析 (Post-Livestream Review)

## 概述
本技能用于直播结束后快速生成结构化复盘报告，通过多维数据分析提炼本场直播的成功因素和改进点，输出可执行的优化建议。复盘报告需在收播后60分钟内完成生成。

## 使用场景
- 每场直播结束后的标准复盘
- 大促活动后的专项复盘
- 周度/月度直播趋势分析
- 异常直播场次的深度复盘
- 新策略/新品的效果评估

## 执行步骤

### Step 1: 数据采集与整理（收播后0-15分钟）
- 拉取本场直播全量数据（平台后台+自有监控系统）
- 核心数据：总GMV、UV、PV、平均在线、峰值在线、停留时长、互动率、GPM、转化率、客单价
- 分品数据：每品曝光/点击/加购/下单/GMV/转化率
- 流量数据：自然流量/付费流量/关注流量占比及各渠道ROI
- 互动数据：弹幕总量、评论关键词、互动高峰时段

### Step 2: 目标达成分析（15-30分钟）
- 对比计划目标与实际达成：GMV达成率、UV达成率、GPM达成率
- 分析差距原因（超额/达标/未达分别分析）
- 对比历史同期表现：环比（上一场）和同比（上周同时段）
- 标注异常数据点并初步归因

### Step 3: 分维度深度分析（30-45分钟）

**商品维度：**
- TOP5爆款分析：为什么卖得好（话术/价格/时段/流量）
- BOTTOM5滞销品分析：为什么表现差
- 提炼爆款规律：品类/价格带/话术风格/讲解时长的共性

**话术维度：**
- 高转化话术片段标注和分析
- 弹幕互动高频时段与对应话术关联
- 逼单话术效果评估（哪种逼单方式转化最高）

**流量维度：**
- 各渠道流量贡献和质量对比
- 付费流量ROI逐计划分析
- 自然流量波动原因分析（与内容/互动的相关性）

**时段维度：**
- 分时段GMV/转化率/在线人数热力图
- 识别黄金时段和低效时段
- 与排品策略的关联分析

### Step 4: 优化建议输出（45-60分钟）
- 下场直播的具体优化建议（5-10条，可执行）
- 排品调整建议：提升/保持/淘汰/调整位置
- 话术优化方向：哪些话术加强/哪些需要改写
- 投放优化建议：预算分配/人群调整/素材更换
- 长期策略建议：选品方向/内容风格/时段选择

### Step 5: 报告生成与分发
- 按标准模板生成复盘报告
- 发送给相关角色：场控调度师、店铺运营管家、流量投放操盘手
- 关键数据更新到直播数据资产库
- 爆款规律归入知识库

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| livestream_raw_data | object | 是 | 本场直播全量原始数据 |
| livestream_plan | object | 是 | 本场直播计划（含目标） |
| historical_data | object | 是 | 历史同类直播对比数据 |
| director_log | array | 是 | 场控调度日志 |
| traffic_report | object | 是 | 投放效果数据 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| review_report | object | 完整复盘报告文档 |
| key_findings | array | 关键发现（Top5） |
| optimization_suggestions | array | 优化建议列表 |
| product_performance_rank | array | 商品表现排名 |
| best_practice_extracts | array | 可复制的最佳实践 |
| next_plan_inputs | object | 下场计划输入建议 |

## 最佳实践
1. **时效性**：60分钟内必须完成，趁热打铁发现问题
2. **数据准确**：所有数据需与平台后台100%核对，不允许估算
3. **可执行性**：建议必须具体（具体到哪个品/哪个话术/哪个时段），不说空话
4. **趋势视角**：不只看单场，要放在趋势中看（本场是否比上场改善）
5. **归因谨慎**：区分相关性和因果性，多因素时标注不确定性

## 约束条件
- 报告必须在收播后60分钟内完成
- 数据准确率100%（必须与平台后台核对）
- 优化建议可执行率≥80%（不给无法落地的建议）
- 报告长度控制（核心内容2-3页，详细数据附录）
- 敏感数据（ROI/利润率）需标注查看权限
