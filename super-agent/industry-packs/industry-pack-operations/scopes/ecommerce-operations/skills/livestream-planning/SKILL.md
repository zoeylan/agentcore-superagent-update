# 直播策划 (Livestream Planning)

## 概述
本技能用于直播运营策划师制定完整的直播计划，包括主题确定、排品策略、时间规划、目标设定和资源协调。输出的直播计划是指导场控调度师和数字主播执行的核心依据文档。

## 使用场景
- 日常带货直播的周度/场次策划
- 新品首发专场策划
- 节日大促直播策划（618/双11/年货节等）
- 品牌专场/联名活动策划
- 达人合作直播策划

## 执行步骤

### Step 1: 目标设定与分解
- 确定本场直播核心目标（GMV/涨粉/清库存/新品推广）
- 基于历史同类型直播数据设定目标值
- 将总目标分解到时段和品次：
  - 时段分解：前30min（考核期）/黄金时段/平稳期/收尾期
  - 品次分解：每品预估GMV = 预计UV × 转化率 × 客单价

### Step 2: 选品与排品策略
- 从商品库中选择本场商品（通常15-30个品/场）
- 商品角色分类：
  - 引流款（20%）：低价高性价比，拉人气和停留
  - 利润款（40%）：正价核心商品，贡献利润
  - 爆款（20%）：历史高转化品，贡献GMV主力
  - 福利款（10%）：超低价/赠品，拉互动和关注
  - 测试款（10%）：新品/待验证品，收集数据
- 排品节奏编排：引流→利润→爆款→福利→利润→爆款（循环）
- 考虑平台算法特性（如抖音前30分钟决定后续流量分配）

### Step 3: 话术框架设计
- 为每个品准备话术大纲（卖点+促销+逼单要点）
- 设计过渡话术（品与品之间的自然衔接）
- 准备互动话术模板（抽奖/问答/口令/PK）
- 设计应急话术（故障/冷场/违规时使用）

### Step 4: 互动玩法设计
- 确定互动节点（通常每30分钟一次大互动）
- 选择互动形式：整点抽奖/秒杀/红包雨/PK/集赞
- 设定参与门槛和奖品配置
- 计算互动成本并纳入预算

### Step 5: 资源协调与确认
- 确认商品库存充足（各品至少有预估销量150%的库存深度）
- 与流量投放操盘手确认投放计划和预算
- 与店铺运营管家确认商品链接和价格设置
- 确认技术设备和数字主播实例就绪

### Step 6: 计划文档输出
- 输出完整直播计划书（含时间线/排品表/话术/目标/应急预案）
- 将排品策略传递给场控调度师
- 将商品知识需求传递给店铺运营管家
- 将预热投放需求传递给流量投放操盘手

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| livestream_type | enum | 是 | 直播类型（daily/newproduct/promotion/brand） |
| target_gmv | number | 是 | 目标GMV（元） |
| available_products | array | 是 | 可选商品池 |
| historical_data | object | 是 | 历史同类直播数据 |
| platform | enum | 是 | 目标平台 |
| time_slot | object | 是 | 计划直播时段 |
| budget | object | 是 | 本场预算（含投放+互动） |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| plan_document | object | 完整直播计划书 |
| product_schedule | array | 排品时序表 |
| script_framework | object | 话术框架 |
| interaction_plan | array | 互动玩法计划 |
| target_breakdown | object | 目标分解表 |
| resource_checklist | array | 资源确认清单 |

## 最佳实践
1. **数据驱动选品**：优先选择历史转化率TOP商品，搭配新品测试
2. **平台适配排品**：抖音前30分钟安排高互动品（影响推流），淘宝首小时安排爆款（影响搜索权重）
3. **备选方案**：每个品位准备1-2个备选品，应对临时调整
4. **时间缓冲**：预留10-15%的时间弹性，应对互动超时/加品等情况
5. **复盘闭环**：每场计划需标注与上场复盘建议的对应关系

## 约束条件
- 直播计划需在开播前T-3天完成初版，T-1天完成终版
- 排品数量控制在15-30个/场（8小时直播）
- 每品预计讲解时间需与库存深度匹配
- 互动成本不超过本场预估利润的10%
- 计划需通过合规审查（无违规品/违规话术）
