# 直播商品讲解 (Live Product Presentation)

## 概述
本技能用于在直播间进行结构化的商品讲解，通过专业的话术框架和卖点提炼，最大化商品的转化率。适用于数字主播在直播过程中对每个排品进行标准化但不失个性的产品介绍。

## 使用场景
- 直播间按排品顺序进入新商品讲解环节
- 观众弹幕要求重新讲解某商品
- 场控调度师指令切换商品时的衔接讲解
- 福利品/秒杀品的快速讲解

## 执行步骤

### Step 1: 商品信息加载
- 从商品知识库获取当前商品的结构化信息卡片
- 提取核心卖点Top3、价格信息、促销规则、库存状态
- 确认该商品在当前平台的合规话术要求

### Step 2: 开场Hook设计（5秒）
- 根据商品品类选择hook类型：
  - 痛点型："家人们，你们是不是也有XX困扰？"
  - 效果型："看我手上这个，用了XX天后的变化..."
  - 价格型："今天这个价格我只能说两个字——疯了！"
  - 场景型："上班族必备！每天早上3分钟搞定XX"

### Step 3: 核心卖点讲解（60-90秒/品）
- 按「功效→成分/材质→使用场景→对比竞品」顺序展开
- 每个卖点配合使用场景描述，增强代入感
- 穿插真实用户评价/销量数据增强信任感
- 适时展示产品细节（引导画面切换）

### Step 4: 价格锚定与促销信息（15-30秒）
- 先报原价/专柜价建立价格锚点
- 再公布直播间专属价，强调差价优势
- 说明赠品/优惠券等额外福利
- 强调限时限量制造紧迫感

### Step 5: 逼单话术（15-30秒）
- 倒计时催促："还剩最后XX单，拍完就没了"
- 从众引导："已经有XXX位家人下单了"
- 售后保障："7天无理由退换，不满意随时退"
- 行动指令："现在点击下方小黄车第X个链接"

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| product_id | string | 是 | 商品ID |
| product_knowledge_card | object | 是 | 商品知识卡片（含卖点/参数/FAQ） |
| platform | enum | 是 | 当前平台（douyin/taobao/kuaishou） |
| promotion_rules | object | 是 | 当前促销规则（价格/赠品/限量） |
| presenter_persona | string | 是 | 主播人设风格 |
| time_slot | enum | 是 | 时段类型（peak/normal/low） |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| hook_script | string | 开场hook话术 |
| main_presentation | string | 核心讲解话术 |
| price_anchor_script | string | 价格锚定话术 |
| closing_script | string | 逼单话术 |
| estimated_duration | number | 预计讲解时长（秒） |
| compliance_check | boolean | 是否通过合规检测 |

## 最佳实践
1. **品类差异化**：美妆品重感受描述，3C品重参数对比，食品重安全溯源
2. **节奏控制**：引流款快讲（2-3分钟），利润款精讲（5-8分钟），爆款反复讲
3. **平台适配**：抖音话术更口语化/活泼，淘宝更专业/详细，快手更亲民/实在
4. **违规规避**：绝对不用"最""第一""100%有效"等绝对化用语

## 约束条件
- 单品讲解时长控制在2-8分钟内
- 话术必须基于商品知识库真实信息，禁止夸大
- 必须通过平台违规词实时检测
- 响应场控切品指令时，需在10秒内完成话术衔接
