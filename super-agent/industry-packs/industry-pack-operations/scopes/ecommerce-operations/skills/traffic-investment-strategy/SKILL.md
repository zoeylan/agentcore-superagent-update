# 流量投放策略 (Traffic Investment Strategy)

## 概述
本技能用于流量投放操盘手制定跨平台付费流量投放的整体策略，包括平台选择、预算分配、出价策略和投放节奏规划。策略目标是在预算约束下最大化GMV产出，同时保持ROI≥3的基础红线。

## 使用场景
- 新直播场次的投放计划制定
- 大促期间的预算分配和策略调整
- 日常投放策略的周度迭代优化
- 新平台/新工具的投放策略试水
- 预算追加/缩减时的策略重新规划

## 执行步骤

### Step 1: 投放目标确认
- 明确本次投放目标：
  - 直播引流（目标：进入直播间人数）
  - 商品推广（目标：商品点击/加购/下单）
  - 品牌曝光（目标：CPM最优覆盖）
  - 粉丝增长（目标：关注成本最优）
- 确认预算总额和分日/分时预算上限
- 确认目标ROI底线和理想ROI

### Step 2: 平台策略选择
根据目标和商品特性选择投放平台组合：

| 平台工具 | 适合场景 | 核心优势 | 注意事项 |
|----------|----------|----------|----------|
| 巨量千川 | 抖音直播引流 | 兴趣推荐精准 | 素材消耗快，需大量创意 |
| 磁力金牛 | 快手直播引流 | 复购人群强 | 人群包质量关键 |
| 直通车 | 淘宝搜索截流 | 精准关键词 | CPC竞争激烈 |
| 超级推荐 | 淘宝信息流 | 货找人逻辑 | 冷启动需要耐心 |
| 腾讯广告 | 私域引流 | 社交关系链 | 转化链路长 |

### Step 3: 预算分配策略
- 按平台分配：根据历史ROI和GMV贡献比例
- 按时段分配：直播前预热20% + 直播中60% + 直播后追投20%
- 按目标分配：转化目标70% + 曝光目标20% + 测试预算10%
- 预留机动预算（总预算的10-15%）应对突发需求

### Step 4: 出价策略设定
- **手动出价**：适合经验丰富、数据充足的成熟计划
- **自动出价**：适合新计划冷启动、快速探索最优出价
- **ROI目标出价**：适合预算充足、追求稳定回报的场景
- 初始出价参考：行业均值×1.2（确保起量）
- 设置出价上限（不超过目标ROI倒推的最大CPA）

### Step 5: 投放节奏规划
```
T-2小时：开启预热计划（低出价广覆盖，积累曝光）
T-0开播：提升出价20%，启动转化计划
T+30分钟：观察数据，表现好的计划加码
T+1小时：稳定期，按ROI规则自动优化
T+N（高峰期）：全量加码优质计划
收播前1小时：控制消耗，不新增投放
收播后：关闭直播引流计划，保留商品推广计划
```

### Step 6: 策略文档输出
- 输出完整投放计划书（含平台/预算/出价/节奏/创意）
- 设定自动化规则（ROI阈值触发的加码/暂停规则）
- 明确异常处理预案（消耗过快/ROI暴跌/无法起量）
- 与场控调度师同步投放计划，确保协同

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| campaign_objective | enum | 是 | 投放目标类型 |
| total_budget | number | 是 | 总预算（元） |
| target_roi | number | 是 | 目标ROI |
| livestream_plan | object | 是 | 直播计划信息 |
| historical_performance | object | 是 | 历史投放效果数据 |
| audience_profiles | array | 否 | 可用人群包列表 |
| creative_assets | array | 否 | 可用素材列表 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| strategy_document | object | 完整投放策略文档 |
| budget_allocation | object | 预算分配方案 |
| bidding_config | object | 出价策略配置 |
| schedule_plan | object | 投放节奏时间表 |
| automation_rules | array | 自动优化规则 |
| contingency_plan | object | 应急预案 |

## 最佳实践
1. **分层测试**：新策略用10%预算测试验证后再全量
2. **数据积累**：至少3天数据才做策略判断，不根据1天数据调整
3. **组合投放**：不依赖单一平台，分散风险
4. **创意储备**：每个平台至少准备5套创意，应对素材衰减
5. **竞争感知**：关注竞品投放动态（出价变化/素材趋势）

## 约束条件
- 整体ROI不得低于3（低于1的单计划30分钟内暂停）
- 日预算消耗偏差≤10%（不过度超支也不严重欠投）
- 投放素材必须通过平台广告审核
- 禁止使用虚假宣传/对比贬低竞品的投放素材
- 每日9点前完成当日投放计划调整
