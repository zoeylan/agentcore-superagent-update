# 社交内容生成 (Social Content Generation)

## 技能概述

社交内容生成技能负责基于品牌调性和各平台特性，自动生成差异化的多形态社交媒体内容。涵盖图文、短视频脚本、直播预告、互动话题等内容形态，适配微博、抖音、小红书、知乎等中国主流社交平台的不同内容规范和用户偏好。

## 适用场景

- 每日常规内容生产（T-1生产次日内容）
- 会员活动传播内容的定制生成
- 热点事件的快速借势内容创作
- 挽留场景下的定向社交内容
- 品牌官方回应和正面引导内容
- 节日/促销节点的主题内容生产

## 执行步骤

### 步骤1：内容需求分析
1. 确认内容来源：常规排期/活动需求/热点借势/舆情回应
2. 明确目标平台和对应调性：
   - 微博：短平快、话题感、适合事件传播（字数≤140字/带话题标签/配图1-9张）
   - 抖音：娱乐性、节奏快、视觉冲击（视频脚本15-60秒/BGM建议/转场设计）
   - 小红书：真实感、种草风、精致图文（标题≤20字/正文≤1000字/配图6-9张）
   - 知乎：专业度、深度感、知识价值（回答体≥500字/数据引用/逻辑链路）
3. 确认目标受众画像和期望互动形式
4. 检查品牌合规红线（敏感词库/广告法限制/平台规则）

### 步骤2：内容创作
1. 基于主题和平台要求生成内容框架
2. 填充品牌核心信息和卖点（确保一致性）
3. 融入平台原生语言风格：
   - 微博：热搜体/吃瓜体/互动体
   - 抖音：口语化/梗文化/挑战体
   - 小红书：闺蜜体/测评体/攻略体
   - 知乎：分析体/科普体/亲历体
4. 添加互动引导元素（提问/投票/征集/挑战）
5. 配置话题标签（主话题+长尾话题，3-5个）
6. 编写配图/视频需求描述

### 步骤3：合规检查
1. 对照品牌敏感词库进行全文扫描
2. 检查广告法合规性（禁用"最"/"第一"/"国家级"等绝对化用语）
3. 检查平台社区规范（无引导关注/无诱导分享/无违规外链）
4. 检查版权风险（图片来源/引用内容/BGM版权）
5. 标注可能引起争议的表述并提供替代方案

### 步骤4：内容优化
1. SEO优化：嵌入搜索关键词（小红书/知乎尤为重要）
2. 发布时间建议：根据平台用户活跃时间和竞争内容密度推荐最佳发布窗口
3. 互动率预测：基于历史同类内容表现预估互动数据
4. 生成A/B版本（标题/首图/引导语的2个版本供选择）

### 步骤5：提交审核
1. 整理内容包（正文+配图描述+标签+发布时间+平台）
2. 提交至人工审核队列
3. 标记内容类型和优先级（常规/紧急/敏感）
4. 若审核未通过，根据反馈修改并记录修改原因供学习

## 输入规格

| 输入项 | 类型 | 说明 |
|--------|------|------|
| 内容主题/需求 | 文本 | 来自排期或临时需求的主题描述 |
| 目标平台 | 枚举 | 微博/抖音/小红书/知乎（可多选） |
| 品牌调性指南 | 文档 | 品牌语言风格、禁忌、核心信息 |
| 目标受众画像 | 标签集 | 年龄/性别/兴趣/消费层级 |
| 会员活动信息 | 结构化数据 | 来自会员增长引擎的活动详情（可选） |
| 热点话题数据 | 列表 | 当日热搜/趋势话题（可选） |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| 内容正文 | 文本 | 符合平台规范的完整内容 |
| 配图/视频描述 | 文本 | 视觉素材的详细需求描述 |
| 话题标签集 | 列表 | 3-5个相关话题标签 |
| 发布时间建议 | 时间 | 建议的最佳发布时间窗口 |
| 合规检查结果 | 报告 | 潜在风险点和处理建议 |
| 互动预测 | 数值 | 预估阅读/点赞/评论范围 |

## 最佳实践

1. **平台原生感**：内容应像该平台的原生用户创作，避免"广告感"和"机器感"
2. **热点时效性**：热点内容需在发现后2小时内完成创作，超过6小时不再追热点
3. **一源多用**：同一核心素材应适配多平台，但绝不能直接复制粘贴
4. **UGC素材利用**：优质用户口碑内容二次加工比纯品牌内容互动率高30%+
5. **避免过度营销**：内容价值感:营销感 比例应保持在7:3以上

## 约束条件

- 所有内容100%原创，抄袭/洗稿是红线
- 内容生成后必须经过人工审核，不得自动发布
- 严格遵守《广告法》和各平台社区规范
- 不在非工作时间发布争议性内容
- 涉及价格/优惠的内容需与运营部门确认准确性
- 审核未通过内容需在2小时内完成修改重新提交
