# 内容效果分析 (Content Performance Analysis)

## 技能概述

内容效果分析技能负责追踪社交媒体内容发布后的互动数据表现，进行多维度效果评估，识别爆款规律和低效原因，为内容策略迭代提供数据驱动的优化建议。该技能将内容运营从"凭感觉"升级为"数据驱动"。

## 适用场景

- 单条内容发布后的实时互动监控
- 周度内容效果综合分析报告
- 爆款内容的成功因素拆解
- 低效内容的原因诊断与优化建议
- 跨平台内容策略的ROI对比评估
- 内容互动率异常检测与预警

## 执行步骤

### 步骤1：数据采集
1. 采集各平台内容互动数据：
   - 曝光量/阅读量（内容触达范围）
   - 点赞数（内容认可度）
   - 评论数（内容话题度）
   - 转发/分享数（内容传播力）
   - 收藏数（内容价值度）
   - 完播率/阅读完成率（内容吸引力）
2. 采集内容带来的转化数据：链接点击、商品加购、会员注册
3. 采集评论区情感倾向（正面/中性/负面占比）
4. 记录内容属性标签：平台/主题/形态/发布时间/目标人群

### 步骤2：效果评估
1. 计算核心效果指标：
   - 互动率 = (点赞+评论+转发+收藏) / 曝光量 × 100%
   - 传播系数 = 转发量 / 阅读量
   - 内容转化率 = 目标行为数 / 阅读量
   - 内容ROI = 内容带来的GMV / 内容制作成本
2. 与同类型历史内容对比（同主题/同平台/同时段）
3. 与平台平均水平对比（是否超过平台baseline）
4. 标记异常表现内容：
   - 爆款：互动率超过近30日均值200%
   - 低效：互动率低于近30日均值50%

### 步骤3：归因分析
1. 爆款归因（正向分析）：
   - 话题因素：是否蹭中热点？是否触发平台推荐算法？
   - 内容因素：标题/首图/开头是否有强钩子？
   - 时间因素：是否在最佳发布窗口？
   - 互动因素：评论区是否有优质互动引发二次传播？
2. 低效诊断（负向分析）：
   - 曝光低但互动率正常：平台推荐/分发问题→调整标签和发布时间
   - 曝光正常但互动率低：内容质量/主题匹配问题→优化内容策略
   - 评论情感偏负面：内容引发争议→标记并转舆情流程

### 步骤4：异常预警处理
1. 发布后2小时互动率低于预期50%：
   - 检查内容是否正常展示（无审核限流/无技术问题）
   - 建议调整推送时间（错峰重新推送）或追加投放预算
2. 内容引发争议评论（负面评论占比>30%）：
   - 立即标记并暂停推广/付费投放
   - 转入舆情监控流程评估
   - 通知社交内容策划师准备应对

### 步骤5：周度报告输出
1. 本周内容总量和各平台分布
2. 核心指标汇总：总曝光/总互动/平均互动率/最佳表现内容
3. TOP3爆款内容分析：成功因素和可复制点
4. BOTTOM3低效内容诊断：问题原因和改进建议
5. 平台效果对比：各平台投入产出对比
6. 下周优化建议：主题方向/时间调整/形态偏好

## 输入规格

| 输入项 | 类型 | 说明 |
|--------|------|------|
| 平台互动数据 | 结构化数据 | 各内容的阅读/点赞/评论/转发/收藏数据 |
| 内容属性标签 | 标签集 | 平台/主题/形态/发布时间等属性 |
| 历史基线数据 | 时序数据 | 近30日各类型内容的平均表现 |
| 转化追踪数据 | 事件数据 | 内容带来的点击/加购/转化 |
| 评论情感数据 | 分析结果 | 评论区NLP情感分析结果 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| 单条内容效果卡片 | 报告 | 各内容的效果数据和简评 |
| 周度效果分析报告 | 报告文档 | 全面的周度内容效果评估 |
| 异常预警通知 | 告警 | 互动率异常或争议内容的实时预警 |
| 优化建议清单 | 列表 | 基于数据的具体优化动作建议 |
| 爆款规律库更新 | 知识条目 | 新发现的爆款模式和规律 |

## 最佳实践

1. **及时性**：发布后2小时内关注首波数据，24小时后做初步评估，7天后做最终评估
2. **对照实验**：始终保持A/B版本对比意识，单一变量分析效果差异
3. **长期趋势 > 单点数据**：不因单条内容表现做重大策略调整，关注趋势性变化
4. **平台差异化评估**：各平台基线不同，不能横向简单对比绝对值
5. **关注负面信号**：评论区负面情感比正面数据下降更值得关注

## 约束条件

- 发布后2小时内必须完成首次展示确认
- 互动率低于平均50%时必须触发预警机制
- 争议内容必须在发现后30分钟内暂停推广
- 周度报告每周一上午输出，覆盖上周全部已发布内容
- 数据分析必须基于统计学显著性，避免过度解读随机波动
