# 品牌声誉分析 (Reputation Analysis)

## 技能概述

品牌声誉分析技能负责对品牌在全网的声誉状况进行系统性评估和趋势分析，输出月度/季度品牌健康度报告，为品牌战略调整提供数据支撑。不同于实时监控的"发现问题"导向，声誉分析偏重"趋势洞察"和"战略建议"，服务于中长期品牌建设决策。

## 适用场景

- 月度品牌声誉健康度评估报告
- 品牌活动/事件后的声誉影响评估
- 竞品声誉对标分析
- 品牌定位调整前的舆情基线评估
- 年度品牌传播效果总结

## 执行步骤

### 步骤1：数据整理与清洗
1. 汇总评估周期内的全量舆情数据（通常为月度）
2. 清洗数据：去除水军/广告/无关内容
3. 按维度分类整理：
   - 按平台：各平台声量分布和情感倾向
   - 按话题：品牌被讨论的核心主题
   - 按人群：不同人群（KOL/KOC/普通用户）的态度差异
   - 按时间：声量和情感的时间线变化
4. 标记关键舆情事件节点（用于解释趋势变化）

### 步骤2：品牌健康度评分
1. 构建品牌健康度指标体系（满分100）：
   - 知名度维度（30%）：品牌提及总量/环比变化/搜索指数
   - 美誉度维度（35%）：正面占比/NPS相关评论分析/推荐意愿
   - 忠诚度维度（20%）：复购提及/品牌忠诚表达/流失表达
   - 活跃度维度（15%）：UGC创作量/互动率/社群活跃度
2. 计算各维度得分和综合得分
3. 与上一周期对比，标注显著变化的维度
4. 与行业基准/竞品对比，确定品牌所处位置

### 步骤3：话题热力图与情感分析
1. 提取高频讨论话题（词频分析+语义聚类）
2. 生成话题词云（区分正面/中性/负面话题）
3. 分析话题情感倾向：
   - 品牌哪些方面获得最多好评？（品牌优势）
   - 品牌哪些方面争议最大？（品牌痛点）
   - 消费者最关注品牌的哪些特质？（品牌认知焦点）
4. 对比竞品话题分布：差异化优势和劣势识别

### 步骤4：趋势分析与预警
1. 声量趋势：是否有持续上升/下降的趋势？
2. 情感趋势：正面占比是否稳定？是否有恶化倾向？
3. 话题迁移：消费者关注点是否在发生变化？
4. 危机风险评估：当前是否有潜在的声誉风险在积累？
5. 竞品动态：竞品是否有策略调整可能影响品牌？

### 步骤5：策略建议输出
1. 品牌传播建议：
   - 应强化传播的品牌优势点
   - 需修复/改善的品牌弱势点
   - 可借势的外部机会
2. 内容方向建议：
   - 向社交内容策划师输出话题方向建议
   - 推荐重点发力的平台和人群
3. 会员运营建议：
   - 向会员增长引擎输出正面口碑案例（忠诚度培养素材）
   - 识别品牌大使候选人（高质量UGC创作者）
4. 风险提示：
   - 需关注的潜在风险点
   - 建议的预防性措施

## 输入规格

| 输入项 | 类型 | 说明 |
|--------|------|------|
| 月度舆情原始数据 | 大数据集 | 全量品牌提及内容和互动数据 |
| 历史健康度评分 | 时序数据 | 往期品牌健康度评分记录 |
| 竞品舆情数据 | 对比数据 | 主要竞品的同期舆情表现 |
| 品牌活动日历 | 事件数据 | 评估期内的品牌营销活动记录 |
| 行业基准数据 | 参考值 | 行业平均水平的各项指标 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| 品牌健康度评分 | 评分卡 | 总分+各维度分+环比变化 |
| 月度声誉报告 | 报告文档 | 完整的声誉分析和策略建议 |
| 话题词云 | 可视化 | 高频话题的情感标注词云 |
| 竞品对标分析 | 对比报告 | 品牌vs竞品的多维对比 |
| 风险预警清单 | 列表 | 潜在风险点和预防建议 |
| 内容方向建议 | 策略文档 | 给社交内容策划师的选题建议 |

## 最佳实践

1. **长期主义视角**：声誉建设是长期工程，避免因短期波动做出过激调整
2. **多维度交叉分析**：单一指标可能误导，需要多维度交叉验证
3. **竞品动态纳入考量**：品牌声誉是相对概念，需要在竞争格局中评估
4. **定性+定量结合**：数据分析+典型案例深度解读相结合
5. **策略可执行性**：建议必须具体可落地，避免空泛的"加强品牌建设"

## 约束条件

- 月度声誉报告每月5号前完成
- 分析数据量需覆盖该周期95%以上的品牌提及
- 竞品分析至少覆盖3个主要竞品
- 健康度评分模型参数每季度校准一次
- 报告中的数据需可追溯到原始来源
- 策略建议需附带预期效果和实施优先级
