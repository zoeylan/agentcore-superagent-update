# 舆情实时监控 (Sentiment Monitoring)

## 技能概述

舆情实时监控技能负责对全网品牌相关内容进行7×24小时不间断监控和情感分析，是品牌声誉管理的第一道防线。通过多平台数据采集和NLP情感分析，及时发现品牌相关的正面口碑、中性讨论和负面舆情，为后续的预警响应和策略调整提供实时数据基础。

## 适用场景

- 7×24小时全网品牌关键词监控
- 新品/活动上线后的舆论反馈实时追踪
- 竞品动态和行业舆情趋势监控
- 正面口碑素材发现和收集
- 日常舆情摘要生成（每日/每周）

## 执行步骤

### 步骤1：监控配置与数据采集
1. 配置监控关键词矩阵：
   - 品牌词：品牌名/缩写/常见别称/创始人名字
   - 产品词：核心产品名称/型号/系列名
   - 竞品词：主要竞品品牌名（对比监控）
   - 负面预警词：投诉/骗子/假货/维权/曝光+品牌词组合
2. 覆盖平台清单及采集频率：
   - 微博：每5分钟扫描（含热搜/超话/品牌提及）
   - 抖音：每10分钟扫描（含评论区/视频标签/话题）
   - 小红书：每10分钟扫描（含笔记标题/正文/评论）
   - 知乎：每15分钟扫描（含问答/文章/评论）
   - 百度贴吧：每30分钟扫描
   - 黑猫投诉：每15分钟扫描
3. 数据清洗：去重、去广告、去无关内容

### 步骤2：情感分析与分类
1. 对每条内容进行NLP情感分析：
   - 正面（分数>0.6）：好评/推荐/赞美/满意体验
   - 中性（分数0.3-0.6）：客观描述/提问/分享信息
   - 负面（分数<0.3）：投诉/不满/批评/差评
2. 细分负面类型：
   - 产品质量类：质量问题/功能缺陷/包装损坏
   - 服务体验类：客服态度/物流问题/售后推诿
   - 价格争议类：涨价/虚假促销/价格歧视
   - 品牌形象类：企业价值观/社会责任/负面联想
3. 标注内容影响力指标：发布者粉丝量/内容互动量/传播趋势

### 步骤3：水军/恶意攻击识别
1. 账号特征筛查：
   - 注册时间<30天的新号批量发布负面内容
   - 账号无头像/无历史动态/关注列表异常
   - 多个账号内容高度雷同（相似度>80%）
2. 行为模式识别：
   - 集中时间段（1小时内）批量出现
   - 内容模板化（仅替换关键词）
   - 评论区相互配合/接力
3. 标记处理：确认为水军/恶意攻击的内容标记为"非自然舆情"，不纳入常规统计，但单独归档作为证据

### 步骤4：实时状态维护
1. 维护实时舆情热力图（各平台正/中/负面占比）
2. 维护24小时舆情趋势线（监控突变点）
3. 更新活跃话题词云（当前品牌被讨论的核心话题）
4. 维护正面口碑素材库（优质UGC内容收集）

### 步骤5：日常摘要输出
1. 每日09:00输出前日舆情摘要：
   - 总提及量和情感分布变化
   - 核心话题和关键事件
   - 新增负面事件摘要
   - 值得关注的正面口碑
2. 异常信号立即触发预警流转（交给crisis-alert-response技能处理）

## 输入规格

| 输入项 | 类型 | 说明 |
|--------|------|------|
| 品牌关键词库 | 词库 | 需监控的品牌/产品/竞品关键词 |
| 平台API接口 | 系统配置 | 各平台数据采集接口 |
| 水军特征库 | 规则库 | 识别异常账号和行为的规则 |
| 历史舆情基线 | 统计数据 | 用于判断当前舆情是否异常的基准 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| 实时舆情数据流 | 流式数据 | 标注情感标签的品牌提及内容流 |
| 日度舆情摘要 | 报告 | 每日舆情概况和关键事件 |
| 正面口碑素材库 | 素材库 | 收集的优质正面内容 |
| 负面预警触发 | 事件 | 达到预警阈值时的触发信号 |
| 水军标记记录 | 日志 | 被标记为非自然舆情的内容存档 |

## 最佳实践

1. **关键词持续迭代**：每周根据新出现的讨论方式更新关键词库（如谐音梗/缩写）
2. **误报控制**：调优情感分析模型阈值，维持误报率≤10%
3. **竞品对比**：监控竞品舆情可预判行业风险，也可发现差异化优势
4. **正面素材即时捕获**：优质口碑内容有时效性，需快速传递给内容团队二次传播
5. **节假日特殊监控**：大促/节假日期间加密监控频率

## 约束条件

- 监控覆盖率 ≥ 95%（不遗漏任何主流平台的品牌提及）
- 情感分析准确率 ≥ 85%
- 误报率 ≤ 10%
- 系统可用性 ≥ 99.5%（7×24不间断）
- 数据延迟 ≤ 10分钟（从内容发布到被系统捕获）
- 个人隐私数据不采集不存储（仅分析公开内容）
