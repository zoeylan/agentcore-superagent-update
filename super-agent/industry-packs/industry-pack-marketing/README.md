## 🏢 业务场景

### 1. 🎨 创意生产与内容创作 (`creative-production`)

智能体批量创作广告文案、图片和视频，支持多语言、多风格、个性化内容生产。覆盖从创意构思、素材生成、质量评估到多渠道适配的全流程，基于多智能体协作架构实现规模化高质量素材产出。

**核心价值：** 将传统需要设计师团队数天完成的创意工作压缩到分钟级别，单张图片生成<30秒，单条视频<5分钟，批量支持100+素材并发处理。

**智能体团队：**

| 智能体 | 角色 |
|--------|------|
| 创意总监Agent (`creative-supervisor`) | 创意生产流程的总调度者和质量把关人 |
| 视觉生成Agent (`visual-generation`) | 负责产品摄影、场景图和广告主图的AI生成 |
| 内容编辑Agent (`content-editor`) | 负责广告文案撰写、多语言翻译和图文排版 |
| 质量评估Agent (`quality-assessor`) | 素材质量的自动化评审官和品牌合规守门人 |

**技能清单：** brief解析、图片生成、文案撰写、多语言本地化、风格适配、多尺寸渲染、排版合成、品牌合规检查、视觉质量评分、文案质量审核、质量决策、任务编排

---

### 2. 🎯 广告投放与智能优化 (`ad-delivery-optimization`)

跨平台智能投放管理，实时监控广告效果，自动调整出价、预算和受众定向。支持主流广告平台（Google Ads、Meta、TikTok、Amazon Ads等）统一管理，通过数据驱动的智能决策实现ROI最大化。

**核心价值：** 将分散在各平台的投放操作统一到智能决策层，实现CPM降低≥15%、CPA降低≥15%，异常预警响应<5分钟。

**智能体团队：**

| 智能体 | 角色 |
|--------|------|
| 投放策略Agent (`delivery-strategist`) | 跨平台投放策略的制定者和全局优化决策者 |
| 预算调控Agent (`budget-controller`) | 跨渠道预算的动态分配和消耗节奏管控者 |
| 受众定向Agent (`audience-targeting`) | 广告受众的智能细分和高价值人群发现者 |
| 效果监控Agent (`performance-monitor`) | 投放效果的实时监控者和异常预警响应者 |

**技能清单：** 跨平台策略规划、智能出价优化、预算动态分配、消耗节奏控制、预算异常检测、受众细分分析、Lookalike扩量、受众测试验证、实时效果监控、效果诊断分析、异常诊断、告警管理

---

### 3. 📊 数据洞察与决策智能 (`data-insights-intelligence`)

自动化竞品监测、市场趋势分析、爆款预测、营销效果归因。整合多维度营销数据，通过AI分析生成可执行的营销决策建议，支持自然语言查询营销数据。

**核心价值：** 为所有Scope提供数据智能支撑，爆款预测准确率≥70%，自然语言查询响应<10秒，实现从数据到决策的自动化闭环。

**智能体团队：**

| 智能体 | 角色 |
|--------|------|
| 市场情报Agent (`market-intelligence`) | 竞品动态和市场趋势的自动化监测与分析者 |
| 效果归因Agent (`attribution-analyst`) | 营销效果的多触点归因分析和ROI核算者 |
| 数据顾问Agent (`data-query-advisor`) | 营销数据的自然语言查询接口和决策建议生成者 |

**技能清单：** 竞品监测、趋势分析、爆款预测、归因建模、ROI分析、自然语言查询、异常检测、数据告警、决策报告生成

### 4. 📞 用户获客与智能外呼 (`customer-acquisition`)

AI智能外呼全天候触达客户，对话式销售与个性化推荐，自动化受众细分与商机发现。覆盖从潜在客户识别、多渠道触达、对话转化到线索评分的获客全流程。

**核心价值：** 智能外呼7×24小时运转（日处理10000+通话），A级线索1小时内分配销售，流失预警提前7天识别，获客成本目标环比降低5%。

**智能体团队：**

| 智能体 | 角色 |
|--------|------|
| 智能外呼Agent (`outbound-caller`) | 全天候自动外呼触达和对话式初步转化执行者 |
| 线索评分Agent (`lead-scorer`) | 潜在客户价值评估和线索分级流转管理者 |
| 私域运营Agent (`private-domain-operator`) | 客户生命周期管理和私域会员差异化运营者 |

**技能清单：** 外呼执行、对话意图识别、通话摘要生成、线索评分模型、线索路由管理、评分模型验证、客户生命周期分层、流失预测响应、挽留活动设计

## 📦 导入指南

### 前置条件

1. 确保平台已部署并运行 Super Agent 系统
2. 确认具备管理员权限以创建 Scope、Agent 和 Skill

### 导入步骤

1. **导入业务场景（Scope）**
   - 依次导入 `scopes/` 目录下的四个子目录
   - 每个子目录中的 `scope.json` 为场景定义文件

2. **导入智能体（Agent）**
   - 每个 Scope 的 `agents/` 目录包含该场景所有智能体定义
   - 按 JSON 文件逐个导入

3. **导入技能（Skill）**
   - 每个 Scope 的 `skills/` 目录包含技能定义（SKILL.md）
   - 按目录逐个导入

4. **导入工作流（Workflow）**
   - 每个 Scope 的 `workflow/workflow-plan.json` 为工作流定义

5. **导入 SOP**
   - 每个 Scope 的 `sop/sop.md` 为标准作业流程

6. **导入数字孪生**
   - 导入 `digital-twins/cmo-首席营销官/twin.json`
   - 导入对应 `skills/` 目录下的专属技能

7. **导入初始记忆**
   - 每个 Scope 的 `memories/initial-memories.json` 为初始知识库

### 推荐导入顺序

```
数据洞察与决策智能 → 创意生产与内容创作 → 广告投放与智能优化 → 用户获客与智能外呼 → CMO数字孪生
```

## ⚙️ 定制化建议

以下内容需要根据您的具体业务情况进行调整：

### 1. 广告平台对接
- 根据实际使用的广告平台修改投放相关 Agent 的平台列表（默认覆盖 Google Ads、Meta、TikTok、Amazon Ads）
- 如果主要面向国内市场，需增加巨量引擎、腾讯广告、小红书聚光等平台配置

### 2. 业务指标阈值
- `ROAS目标 > 3:1` — 根据行业和品类调整（高毛利品类可设更高）
- `CPM/CPA降低≥15%` — 根据当前基线水平调整
- `质量评估合格率≥80%` — 根据品牌标准严格程度调整
- `A级线索1小时分配` — 根据销售团队实际响应能力调整

### 3. 品牌规范
- 在创意生产 Scope 的品牌合规检查技能中配置您的品牌指南（logo规范、色彩体系、字体要求、安全区规则）
- 根据不同产品线/子品牌设置差异化的合规标准

### 4. 受众与市场
- 根据目标市场调整多语言支持范围（默认140+语言全覆盖）
- 根据业务类型调整线索评分模型的权重维度（B2B vs B2C 差异较大）
- 根据客户行业调整流失预警的时间窗口（默认7天）

### 5. 外呼与获客
- 根据当地法规调整外呼时段限制和频率限制
- 根据业务类型调整话术模板和对话策略
- 根据销售团队规模调整线索分配规则

### 6. 数据源集成
- 配置实际使用的数据源连接（GA4、CRM、CDP等）
- 调整归因模型选择（首次/末次/线性/时间衰减），根据业务特点选择主模型
- 设置竞品监测的具体竞争对手列表
