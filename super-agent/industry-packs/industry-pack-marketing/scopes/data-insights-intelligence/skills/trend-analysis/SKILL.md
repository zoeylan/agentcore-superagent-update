# 市场趋势分析 (Trend Analysis)

## 概述

市场趋势分析技能用于从多维度数据源中捕捉和研判行业趋势、消费者偏好变化和新兴渠道机会。通过系统化的趋势信号采集和分析框架，输出每周市场趋势报告，指导营销策略方向调整。

## 适用场景

- 每周市场趋势报告的自动化生成
- 消费者偏好迁移方向的早期识别
- 行业热点话题和营销机会的捕捉
- 新兴渠道和流量洼地的发现与评估
- 季节性/节日性营销节奏的提前规划

## 执行步骤

### 步骤1：趋势信号采集
- **搜索趋势**：采集百度指数、Google Trends、巨量算数的品类关键词搜索量变化
- **社交热度**：监控抖音热榜、微博热搜、小红书热门话题中的品类相关内容
- **电商趋势**：追踪品类销量排行、新品上架速度、价格带变化
- **内容趋势**：分析UGC内容的主题聚类、情感极性、高互动内容特征
- **渠道趋势**：各平台DAU/MAU变化、新功能上线、流量政策调整

### 步骤2：信号筛选与聚类
- 对采集的原始信号进行去噪处理（过滤一次性热点、刷量数据）
- 基于语义相似度进行信号聚类，归纳为趋势主题
- 计算每个趋势主题的"信号强度"：数据源覆盖面 × 增长速率 × 持续时间
- 标记趋势阶段：萌芽期 → 增长期 → 成熟期 → 衰退期

### 步骤3：趋势研判
- **增长潜力评估**：基于历史相似趋势的发展轨迹，预测当前趋势的峰值时间和持续周期
- **品牌相关性评分**：评估趋势与当前品牌定位、产品线、目标受众的契合度
- **竞争窗口判断**：分析竞品是否已切入该趋势，评估差异化空间
- **风险评估**：识别趋势反转信号、舆论风险、合规风险

### 步骤4：报告输出
- 按"数据摘要 → 关键洞察 → 建议行动项"三段式结构输出
- 每个趋势洞察包含：趋势描述、数据支撑、阶段判断、营销切入建议、最佳时间窗口
- 趋势优先级排序：按"品牌相关性 × 增长潜力 × 竞争窗口"综合打分

## 输入规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| industry_category | String | 是 | 行业品类（如美妆、3C、食品） |
| brand_profile | Object | 是 | 品牌信息（定位、目标受众、产品线） |
| monitoring_keywords | Array | 是 | 核心监测关键词列表 |
| time_range | String | 是 | 分析时间范围（默认近7天） |
| data_sources | Array | 否 | 指定数据源（默认全部） |

## 输出规格

| 字段 | 类型 | 说明 |
|------|------|------|
| report_title | String | 报告标题（含时间范围） |
| executive_summary | String | 核心摘要（200字内） |
| trend_insights | Array | 趋势洞察列表（按优先级排序） |
| trend_insights[].title | String | 趋势名称 |
| trend_insights[].description | String | 趋势详细描述 |
| trend_insights[].stage | Enum | 趋势阶段（emerging/growing/mature/declining） |
| trend_insights[].signal_strength | Number | 信号强度（1-10） |
| trend_insights[].relevance_score | Number | 品牌相关性（1-10） |
| trend_insights[].action_suggestion | String | 建议营销动作 |
| trend_insights[].optimal_window | String | 最佳切入时间窗口 |
| emerging_channels | Array | 新兴渠道机会 |
| risk_alerts | Array | 风险提示 |

## 最佳实践

1. **多源交叉验证**：单一数据源的趋势信号需至少2个其他数据源佐证才确认为有效趋势
2. **基线对比**：所有增长判断基于同比（去年同期）和环比（上期）双维度，避免季节性误判
3. **行动导向**：每个趋势洞察必须附带具体的营销行动建议，避免纯描述性分析
4. **时效分层**：区分"立即行动"（增长期趋势）和"持续观察"（萌芽期趋势）的不同紧迫度
5. **复盘校准**：每月回溯上月趋势判断的准确性，持续优化研判框架

## 约束条件

- 周报交付时间：每周一10:00前
- 趋势洞察数量：每份报告5-10条，聚焦最高价值信息
- 数据时效性：报告中的数据不超过48小时时延
- 结论置信度：所有趋势判断必须标注置信度等级（高确定性/中等/待观察）
- 不得将短期热点（<3天）直接判定为趋势，需观察持续性
