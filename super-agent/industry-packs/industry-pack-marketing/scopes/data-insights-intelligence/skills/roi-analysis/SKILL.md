# ROI分析与报告 (ROI Analysis)

## 概述

ROI分析技能用于系统化计算和呈现营销投资回报率，涵盖各渠道ROAS、各广告组CPA、各创意素材效率等多维度ROI指标。自动生成月度营销ROI分析报告，为预算决策和策略优化提供量化依据。

## 适用场景

- 月度营销ROI分析报告的自动化生成
- 各渠道ROAS排名和趋势追踪
- 预算效率分析：当前分配 vs 最优分配的差距量化
- 投放策略调整前后的效果对比验证
- 广告主/CMO的投资回报总览仪表板数据支撑

## 执行步骤

### 步骤1：成本数据汇总
- 汇总各广告平台投放花费：Google Ads、Meta Ads、TikTok/巨量引擎、快手磁力引擎、Amazon Ads等
- 纳入非直接广告成本：创意制作成本、代理商服务费、工具订阅费（按月均摊）
- 建立成本维度树：总成本 → 渠道 → 广告系列 → 广告组 → 广告/创意
- 处理跨币种统一：按日均汇率统一为报告币种

### 步骤2：收入/转化数据归集
- 从归因分析结果获取各渠道/广告组的归因转化数和归因收入
- 区分直接收入（首购金额）和间接价值（客户LTV预估、品牌认知提升等）
- 对于非电商类转化（线索、注册、下载），使用预设的线索价值系数转化为等效收入
- 处理退货/退款扣减：在归因收入中扣除对应退款金额

### 步骤3：ROI指标计算
- **ROAS（广告支出回报率）** = 归因收入 / 广告花费
- **CPA（单次获客成本）** = 广告花费 / 归因转化数
- **CPL（单线索成本）** = 广告花费 / 归因线索数
- **利润ROI** = (归因收入 - 商品成本 - 广告花费) / 广告花费
- 按渠道、广告系列、广告组、创意四个维度分别计算
- 计算环比/同比变化率，标记显著变化（>15%）

### 步骤4：效率分析与优化建议
- **渠道效率排名**：按ROAS从高到低排序，标记高效渠道（ROAS>目标值150%）和低效渠道（ROAS<目标值50%）
- **预算利用率分析**：各渠道实际消耗vs分配预算的比率，识别"花不出去"和"超支"情况
- **边际效益分析**：评估各渠道追加投入的边际ROAS是否递减
- **优化建议生成**：具体到渠道和金额比例的预算调整建议，附带预期ROAS变化

### 步骤5：报告生成
- 按「数据摘要 → 关键洞察 → 建议行动项」三段式输出
- 数据摘要：总花费、总收入、整体ROAS、环比变化
- 关键洞察：效率最高/最低渠道、最大改善机会、风险预警
- 建议行动项：按优先级排列的具体优化动作和预期收益

## 输入规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| cost_data | Object | 是 | 各渠道/广告组花费数据 |
| revenue_data | Object | 是 | 归因收入/转化数据 |
| attribution_model | String | 否 | 使用的归因模型（默认主模型） |
| time_range | String | 是 | 分析时间范围 |
| target_roas | Number | 否 | 目标ROAS（默认3.0） |
| dimensions | Array | 否 | 分析维度（channel/campaign/adgroup/creative） |

## 输出规格

| 字段 | 类型 | 说明 |
|------|------|------|
| summary | Object | 总体数据摘要 |
| summary.total_spend | Number | 总花费 |
| summary.total_revenue | Number | 总归因收入 |
| summary.overall_roas | Number | 整体ROAS |
| summary.roas_change_mom | Number | ROAS月环比变化 |
| channel_breakdown | Array | 渠道维度明细 |
| efficiency_ranking | Array | 效率排名 |
| optimization_suggestions | Array | 优化建议列表 |
| optimization_suggestions[].action | String | 具体操作 |
| optimization_suggestions[].expected_impact | String | 预期收益 |
| optimization_suggestions[].priority | Enum | 优先级（high/medium/low） |
| risk_alerts | Array | 风险预警 |

## 最佳实践

1. **归因一致性**：ROI报告使用的归因模型应与团队达成共识，避免不同口径的数字混淆
2. **全成本视角**：仅计算媒体花费的ROAS会高估回报，应纳入创意、人力、工具等完整成本
3. **LTV视角补充**：短期ROAS可能低估品牌广告和新客获取的价值，需用LTV/CAC比率补充
4. **季节性修正**：大促期间ROAS的提升不能直接类推到日常，需区分"自然转化"和"广告增量"
5. **可执行性优先**：报告中的每条建议必须具体到可执行的操作粒度（哪个渠道、调多少比例）

## 约束条件

- 月度报告交付时间：每月5日前
- ROAS计算口径统一使用归因收入（非总收入）
- 预算调整建议单次幅度不超过当前分配的30%（渐进式优化原则）
- 低数据量渠道（月转化<30）的ROAS标记为"样本不足，仅供参考"
- 报告数据必须可追溯：每个数字可回溯到原始数据源
