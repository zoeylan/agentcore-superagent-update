# 爆款预测 (Viral Prediction)

## 概述

爆款预测技能通过多维数据信号融合和预测模型，提前识别具有高传播潜力和高转化潜力的商品、内容方向或营销主题。目标预测准确率≥70%，为创意生产和投放策略提供前瞻性指导。

## 适用场景

- 识别即将爆发的商品趋势，指导选品和备货策略
- 预测高传播潜力的内容方向，指导创意生产优先级
- 发现营销主题和话题的最佳切入时机
- 评估已投放内容的爆款潜力，辅助追投决策

## 执行步骤

### 步骤1：爆款信号特征提取
- **早期增长信号**：内容发布初期（前6小时）的互动增速、完播率、分享率是否超过品类均值3倍以上
- **跨平台扩散信号**：单平台内容是否开始在其他平台被二次创作或讨论
- **搜索回流信号**：相关关键词搜索量是否出现显著跳升（>50% day-over-day）
- **供给侧信号**：电商平台同类商品上架数量激增、KOL密集种草
- **情感极性信号**：UGC评论情感从中性转向强正面，且分享动机从"好奇"转为"推荐"

### 步骤2：预测模型运行
- 输入当期捕获的信号特征向量
- 与历史爆款案例库进行模式匹配（相似度计算）
- 运行时间序列预测模型，预估未来7-14天的增长轨迹
- 输出爆款概率评分（0-100分）和预计峰值时间

### 步骤3：预测结果验证与校准
- 每月回溯预测结果：统计准确率（预测为爆款且实际爆发 / 总预测数）
- 分析误判案例：假阳性（预测爆但未爆）和假阴性（未预测但实际爆了）的原因
- 根据误判模式调整模型权重和阈值
- 目标：整体预测准确率≥70%，连续两月<60%触发模型重训练

### 步骤4：建议生成
- 对高置信度预测（>80分）生成即时行动建议：
  - 创意方向建议：参考该趋势的内容特征，建议素材风格和切入角度
  - 投放建议：最佳投放平台、目标受众、建议追投预算比例
  - 时间窗口：建议最迟切入时间点（通常在预计峰值前3-5天）
- 对中等置信度预测（60-80分）生成观察清单和触发条件

## 输入规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| category | String | 是 | 品类/行业领域 |
| signal_data | Object | 是 | 多维度信号数据（搜索、社交、电商、内容） |
| historical_cases | Array | 否 | 历史爆款案例库（用于模式匹配） |
| prediction_horizon | String | 否 | 预测时间窗口（默认14天） |
| confidence_threshold | Number | 否 | 输出阈值（默认60分） |

## 输出规格

| 字段 | 类型 | 说明 |
|------|------|------|
| predictions | Array | 爆款预测列表 |
| predictions[].subject | String | 预测对象（商品/内容方向/话题） |
| predictions[].confidence_score | Number | 置信度评分（0-100） |
| predictions[].predicted_peak | String | 预计峰值时间 |
| predictions[].growth_trajectory | String | 预测增长轨迹描述 |
| predictions[].similar_cases | Array | 历史相似爆款案例 |
| predictions[].creative_suggestion | String | 创意方向建议 |
| predictions[].optimal_entry_window | String | 最佳切入时间窗口 |
| predictions[].risk_factors | Array | 风险因素 |
| model_accuracy_last30d | Number | 模型近30天准确率 |

## 最佳实践

1. **多信号融合**：单一信号的爆款判断准确率通常<50%，至少需要3个独立信号同时触发
2. **品类差异化**：不同品类的爆发周期差异大（快消品3-7天 vs 耐用品14-30天），模型参数需品类适配
3. **排除干扰因素**：过滤人为刷量、平台算法推荐导致的虚假增长信号
4. **及时性优先**：爆款预测的价值随时间快速衰减，宁可置信度稍低也要提早预警
5. **持续学习**：每个已验证的爆款案例（无论预测正确与否）都应回流模型训练集

## 约束条件

- 预测准确率目标≥70%，连续两月<60%触发模型重训练流程
- 每次预测输出不超过10个高潜力项目，聚焦最高价值
- 预测结果的有效期为14天，超期自动标记为"过期"
- 不对已进入成熟期/衰退期的趋势做爆款预测
- 涉及敏感话题（政治、争议性事件）的趋势不纳入推荐，标记为"高风险"
