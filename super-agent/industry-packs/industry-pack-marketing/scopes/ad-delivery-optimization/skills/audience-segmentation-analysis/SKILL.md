# 受众细分分析 (Audience Segmentation Analysis)

## 技能概述

该技能用于基于转化数据深度分析高价值受众的共同特征，从人口属性、兴趣标签、行为序列、设备偏好等维度建立高转化受众画像，为广告定向策略提供数据驱动的受众推荐。

## 适用场景

- 新Campaign的受众策略制定
- 现有受众表现评估与优化
- 高价值转化客户的特征提取
- 受众饱和后的新人群探索
- 跨平台受众策略统一规划

## 执行步骤

### Step 1: 转化数据收集
- 收集过去30天的转化用户数据
- 维度包括：年龄、性别、地域、设备、兴趣标签、行为路径、转化时段
- 区分高价值转化（高客单/高LTV）和普通转化
- 同时收集未转化用户（曝光但未转化）的数据作为对比

### Step 2: 特征分析
- **人口属性分析**：年龄段分布、性别比例、城市线级
- **兴趣标签分析**：Top20高频兴趣标签、标签组合模式
- **行为序列分析**：转化路径（如搜索→浏览→加购→购买的典型序列）
- **时间维度分析**：高转化时段、星期效应
- **设备分析**：设备类型、操作系统、网络环境
- 计算各特征的转化率提升倍数（vs大盘基准）

### Step 3: 受众画像建模
- 提取转化率提升>2倍的核心特征组合
- 构建受众画像标签体系：核心标签（必选）+ 扩展标签（可选）
- 估算各画像组合的覆盖人群规模
- 计算画像之间的重叠度，避免受众包互相竞争

### Step 4: 定向策略推荐
- 为每个广告平台翻译为平台原生的定向条件
- 推荐出价系数调整（高价值画像溢价20-50%）
- 设计受众排除列表（已转化、低质量来源、竞品员工）
- 标注需要A/B测试验证的新假设受众

### Step 5: 报告输出
- 生成受众分析报告（数据图表+结论+推荐动作）
- 标注数据统计显著性（样本量是否充足）
- 给出受众策略有效期建议（通常2-4周后需更新）

## 输入规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| conversion_data | array | 是 | 转化用户数据集 |
| impression_data | array | 是 | 曝光未转化用户数据（采样） |
| time_range | string | 是 | 分析时间范围 |
| platform | string | 是 | 目标广告平台 |
| current_targeting | object | 否 | 当前定向设置 |
| business_category | string | 否 | 业务类别 |

## 最佳实践

1. **样本量第一**：特征分析需基于足够样本（至少100个转化），小样本结论不可靠
2. **避免过拟合**：不要选择过于精细的特征组合导致受众包过小（<1万人）
3. **动态更新**：受众画像每2-4周更新一次，消费者兴趣会变化
4. **交叉验证**：用不同时间窗口的数据交叉验证特征稳定性
5. **排除已转化**：一定要排除已转化用户，避免重复触达浪费预算

## 约束条件

- 受众包规模不低于1万人（否则无法起量）
- 单个广告组的受众定向不超过5个核心标签（过多反而限制算法优化）
- 分析结论需标注统计显著性水平
- 涉及用户隐私数据需符合GDPR/个人信息保护法
- 受众画像有时效性，超过4周需重新验证
