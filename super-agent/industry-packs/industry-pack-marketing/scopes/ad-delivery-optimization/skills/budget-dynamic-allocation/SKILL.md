# 预算动态分配 (Budget Dynamic Allocation)

## 技能概述

该技能用于在多个广告渠道之间动态分配和再平衡预算，基于各渠道实时ROAS表现、消耗进度和转化趋势，自动将预算从低效渠道流转到高效渠道。核心目标是在总预算约束下实现全局ROAS最大化，确保预算利用率>95%。

## 适用场景

- 每日例行的跨渠道预算再分配评估
- 某渠道ROAS显著超目标/低于目标时的紧急调配
- 新增预算的最优分配方案
- 预算削减时的优先级排序
- 大促/活动期间的预算临时倾斜

## 执行步骤

### Step 1: 渠道效果评估
- 拉取各渠道当日/近7日ROAS数据
- 计算各渠道ROAS vs 目标ROAS的比值
- 标注高效渠道（ROAS>目标值150%）和低效渠道（ROAS<目标值50%连续2天）
- 评估各渠道的增量潜力（展现份额、竞争程度）

### Step 2: 再分配方案计算
- 高效渠道追加规则：ROAS>目标值150% → 可追加预算，上限30%
- 低效渠道削减规则：ROAS<目标值50%连续2天 → 削减50%
- 追加来源：先从低效渠道削减的预算中调配，不足则从应急池补充
- 确保调配后各渠道日预算不低于平台最低门槛
- 单次大规模调配（>总预算30%）需上报审批

### Step 3: 影响预测
- 预测追加预算后的边际ROAS（通常追加后边际ROAS递减）
- 预测削减预算后的消耗和转化变化
- 评估调配对日内消耗节奏的影响
- 输出调配前后的全局ROAS预测对比

### Step 4: 执行与监控
- 生成各平台的预算调整指令
- 设置调配生效时间（建议非高峰时段执行）
- 调配后2小时内密切监控消耗节奏变化
- 设定回滚条件：追加预算后ROAS下降>20%则回滚

## 输入规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| channel_performance | array | 是 | 各渠道效果数据（ROAS、消耗、转化量） |
| total_budget | number | 是 | 当日总预算 |
| target_roas | number | 是 | 目标ROAS |
| current_allocation | object | 是 | 当前预算分配 |
| reserve_pool | number | 否 | 应急预算池金额 |

## 最佳实践

1. **避免频繁调配**：每日最多执行1次大规模再分配，避免平台算法不稳定
2. **边际收益递减意识**：追加预算不等于等比例增加转化，需评估增量CPA
3. **保留基础预算**：任何渠道不应削减至平台最低预算以下
4. **考虑滞后效应**：部分平台（如Meta）预算变更后需1-2天重新校准
5. **大促预留**：重大促销活动前3天锁定预算分配，不做大幅调整

## 约束条件

- 单次追加上限：渠道当前预算的30%
- 削减触发条件：ROAS<目标值50%需连续2天确认
- 大规模调配（>总预算30%）需上报投放策略Agent审批
- 各渠道预算不低于平台最低日预算门槛
- 预算变更需在非高峰时段执行（避免影响竞价）
