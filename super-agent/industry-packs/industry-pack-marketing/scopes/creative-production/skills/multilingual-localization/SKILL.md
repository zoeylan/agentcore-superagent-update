# 多语言本地化与文化适配

## 技能概述
本技能负责将广告文案从源语言翻译并适配到140+目标语言市场，不仅是语言转换更是文化重创作。确保目标市场的消费者看到的文案具有native-level的自然度和说服力，同时保持品牌核心信息的一致传达。

## 适用场景
- 全球化广告Campaign的多语言文案生产
- 跨境电商商品详情页的本地化
- 不同语言市场的广告文案A/B测试
- 品牌口号/slogan的文化适配
- 节日营销的地域化定制（各地不同节日）

## 执行步骤

### Step 1: 源文案分析
分析源文案的核心要素：
- **核心信息**：必须保留且不可偏离的关键信息
- **情感调性**：需要在目标语言中复现的情感效果
- **文化元素**：可能需要替换的文化相关内容（谚语/比喻/数字含义）
- **品牌术语**：有官方译法必须使用的专有名词

### Step 2: 目标市场文化调研
每个目标市场的文化考量：
```
文化适配检查清单：
- 颜色含义差异（红色：中国=喜庆，西方=警示/激情）
- 数字禁忌（4在东亚、13在西方）
- 幽默风格差异（自嘲型/讽刺型/夸张型/冷幽默）
- 直接vs间接表达偏好
- 性别/宗教/政治敏感度
- 本地竞品用语习惯
- 货币/度量衡/日期格式
```

### Step 3: 文化重创作
不是逐字翻译，而是文化重创：
- 保留核心卖点和情感效果
- 使用目标语言的自然表达方式
- 替换文化不通用的比喻/典故
- 适配本地消费者的购买决策心理
- 调整语气强度（如日语需更委婉，美式英语可更直接）

### Step 4: 本地化质量验证
- 语法正确性验证
- 自然度检查（是否像native speaker写的）
- 品牌术语一致性
- 字符长度适配（各语言字符宽度不同影响排版）
- 文化敏感性审核

### Step 5: 排版适配
不同语言的排版考量：
- 阿拉伯语/希伯来语：RTL（从右到左）排版
- 日语/中文：竖排/横排选择
- 德语/芬兰语：单词较长，需要更多水平空间
- 泰语/缅甸语：特殊字符渲染
- 字体可用性：确认目标语言有对应品牌字体

## 输入规格
| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| source_copy | Object | ✓ | 源文案（含标题/正文/CTA） |
| source_language | String | ✓ | 源语言 |
| target_languages | Array | ✓ | 目标语言列表 |
| brand_glossary | Object | × | 品牌术语表（含各语言官方译法） |
| cultural_notes | Object | × | 文化注意事项 |
| max_length | Object | × | 各部分最大字符数 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| localized_copy | Object | 本地化后的文案 |
| target_language | String | 目标语言代码 |
| adaptation_notes | String | 适配说明（做了哪些文化调整） |
| character_count | Object | 字符数统计 |
| confidence_score | Number | 本地化质量信心分（1-10） |
| cultural_risks | Array | 潜在文化风险提示 |

## 最佳实践
1. 永远先理解原文想传达什么"效果"，而不是翻译"文字"
2. 高频市场（英/日/韩/西/法/德/阿）应有专门的文案风格指南
3. 品牌术语表是底线——任何场景都必须使用官方译法
4. 长尾语言质量可能较低，标注confidence_score供人工review优先级
5. 注意"假朋友"（看似相同但含义不同的跨语言词汇）

## 约束与限制
- 品牌核心信息不可在翻译中丢失或扭曲
- 品牌术语必须使用官方glossary中的标准译法
- 不确定的文化敏感内容标注"需人工确认"
- 字符长度超出限制时不得简单截断，需改写缩短
- RTL语言的排版必须完整反转，不是简单的文字替换
