# 图文排版合成

## 技能概述
本技能负责将文案内容精确叠加到视觉素材上，完成最终可投放的图文合成素材制作。涵盖字体排版、文字层级设计、品牌元素放置和多尺寸适配排版等能力，确保最终产出既美观又符合品牌规范。

## 适用场景
- 广告主图上叠加标题和CTA文案
- 产品图添加促销信息标签
- Banner设计中的图文层级处理
- Logo和品牌水印的标准化放置
- 多语言版本的排版适配（字符宽度差异处理）

## 执行步骤

### Step 1: 排版规范确认
从品牌指南中提取排版规范：
```
字体规范：
- 主标题字体：[品牌指定字体] / Bold / [字号范围]
- 副标题字体：[品牌指定字体] / Medium / [字号范围]
- 正文字体：[品牌指定字体] / Regular / [字号范围]
- 字间距：标准 / 宽松 / 紧凑
- 行距：1.2x-1.5x字号

色彩规范：
- 文字主色：[HEX值]（确保与背景对比度≥4.5:1）
- 文字辅色：[HEX值]
- 强调色：[HEX值]（用于CTA）
- 禁用色组合：[列表]

Logo规范：
- 标准位置：右下角 / 左上角（视画面而定）
- 最小尺寸：不小于画面宽度的1/6
- 净空间：Logo周围至少1个Logo高度的空白
- 颜色版本：全彩 / 白色 / 黑色（视背景明暗而定）
```

### Step 2: 视觉层级设计
确定画面中各元素的视觉优先级：
1. **第一层级**（最先被看到）：主标题 或 产品主体
2. **第二层级**（引导阅读）：副标题 / 利益点
3. **第三层级**（行动引导）：CTA按钮 / 促销标签
4. **第四层级**（品牌识别）：Logo / 品牌名

### Step 3: 排版布局规划
常用广告排版模式：
- **居中型**：文案居中于画面上方/下方，产品居中
- **左图右文**：产品在左，文案在右（适合横版）
- **上文下图**：标题在上，产品在下（适合竖版信息流）
- **叠加型**：文案直接叠加在产品图上（需半透明底色/阴影保证可读性）

### Step 4: 文字渲染与合成
- 字体渲染：抗锯齿、次像素渲染确保清晰
- 文字可读性保障：
  - 深色背景用浅色文字
  - 复杂背景加文字投影/描边/半透明底色条
  - 对比度检查（WCAG AA标准：≥4.5:1）
- 多行文案的对齐方式（左对齐/居中/右对齐）

### Step 5: 最终输出检查
- [ ] 所有文字清晰可读（缩放到实际展示尺寸检查）
- [ ] 字体使用正确（品牌授权字体）
- [ ] 文字未超出安全区域
- [ ] Logo位置、大小、净空间符合规范
- [ ] 画面整体平衡，无杂乱感
- [ ] 色彩对比度达标
- [ ] 输出格式和分辨率符合要求

## 输入规格
| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| background_image | File | ✓ | 底层视觉素材 |
| text_content | Object | ✓ | 文案内容（标题/正文/CTA） |
| brand_typography | Object | ✓ | 品牌字体规范 |
| logo_asset | File | ✓ | Logo文件 |
| layout_template | String | × | 排版模板类型 |
| target_dimensions | Object | ✓ | 最终输出尺寸 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| composed_asset | File | 最终合成素材（PNG/JPG） |
| layout_spec | Object | 排版规格记录（坐标/字号/颜色） |
| contrast_ratio | Number | 文字对比度值 |
| font_list | Array | 使用的字体列表 |

## 最佳实践
1. 排版设计遵循"少即是多"——信息密度过高反而降低阅读率
2. 移动端优先——所有文字在手机屏幕上必须清晰可读
3. CTA按钮/文字使用品牌强调色，从画面中跳出
4. 多语言版本先排字符最长的语言（如德语），再适配其他语言
5. 保存分层源文件，方便后续修改而无需重做

## 约束与限制
- 只能使用品牌授权字体（不得使用未授权的商用字体）
- 文字最小字号不低于各渠道规范（通常≥12px实际显示尺寸）
- Logo使用必须严格遵循品牌指南（不可拉伸/变色/遮挡）
- 合成后不得降低底层图片的分辨率
- 所有输出必须附带完整的排版参数记录用于复现
