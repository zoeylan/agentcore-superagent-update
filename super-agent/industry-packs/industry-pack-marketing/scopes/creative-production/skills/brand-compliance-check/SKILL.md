# 品牌合规检查

## 技能概述
本技能对生成素材执行严格的品牌合规审核，确保所有产出严格遵循品牌视觉规范体系。这是一个"一票否决"型检查——任何一项品牌合规检查不通过，整个素材即判定为不合格，无论视觉质量多高。

## 适用场景
- 所有素材上线前的强制品牌合规审核
- 新品牌指南发布后的存量素材复查
- Logo使用规范性检查
- 品牌色彩体系符合度验证
- 字体使用合规性确认

## 执行步骤

### Step 1: Logo使用规范检查
```
Logo检查清单：
✓ Logo存在性：是否已放置Logo（必须存在，缺失即Critical）
✓ Logo版本：使用正确的颜色版本（全彩/白色/黑色vs背景明暗）
✓ Logo尺寸：不小于规定的最小尺寸（通常≥画面宽度1/6）
✓ Logo位置：在品牌指南规定的标准位置
✓ Logo净空间：周围空白区域≥规定值（通常1个Logo高度）
✓ Logo完整性：未被裁切、遮挡、变形、旋转
✓ Logo清晰度：在最终尺寸下清晰可辨

严重违规（Critical）：Logo缺失、Logo错误（放了旧版/他品牌）、Logo严重变形
```

### Step 2: 主色调符合度检查
```
色彩检查方法：
1. 提取素材中占比前5的颜色
2. 与品牌色彩体系对比：
   - 主色调：必须出现且占比≥30%（或按品牌规范）
   - 辅助色：使用范围内的辅色
   - 禁用色：完全不出现品牌禁用色

量化标准：
- 主色调Delta E偏差≤5（可接受）
- Delta E 5-10：Warning（偏差较大但可能通过）
- Delta E>10：Fail（色彩偏离过大）

常见问题：
- AI生成时色彩漂移（需在prompt中强化色彩指导）
- 背景色与品牌辅色冲突
- 产品图背景色抢主色调风头
```

### Step 3: 字体规范检查
```
字体检查清单：
✓ 字体类型：使用品牌授权字体列表中的字体
✓ 字体权重：正确使用Bold/Medium/Regular
✓ 字体大小：在规定的字号范围内
✓ 字间距/行距：符合品牌排版规范
✓ 字体颜色：使用品牌允许的文字颜色

严重违规：
- 使用未授权的商用字体（版权风险）
- 使用品牌明确禁用的字体
```

### Step 4: 安全区域检查
```
安全区检查：
✓ 核心内容（产品主体+文案+Logo）在安全区内
✓ 各平台的特殊遮挡区域已规避
✓ 重要信息未被放置在可能被裁切的边缘

各平台安全区参考：
- Instagram: 上下各5%为危险区
- TikTok: 底部20%被互动栏遮挡
- Facebook Feed: 图片上方可能叠加广告标签
- YouTube: 右下角有时间戳
```

### Step 5: 分辨率达标检查
```
分辨率标准：
✓ 输出分辨率≥目标平台要求
✓ 放大后无明显像素化
✓ 文字在目标显示尺寸下清晰可读

最低要求：
- 社交媒体投放：≥1080px（短边）
- 展示广告：≥所需尺寸的1x
- 印刷物料：≥300dpi
- 视频帧：≥720p
```

### Step 6: 合规结论输出
```
合规判定逻辑：
- 5项全部通过 → COMPLIANT（合规）
- 任何1项Fail → NON-COMPLIANT（不合规，需修改后重审）
- 任何1项Critical → CRITICAL VIOLATION（严重违规，中止并告警）

输出格式：
{
  "compliance_status": "compliant | non_compliant | critical_violation",
  "checklist": {
    "logo": "pass | fail | critical",
    "color": "pass | fail",
    "typography": "pass | fail",
    "safe_zone": "pass | fail",
    "resolution": "pass | fail"
  },
  "issues": [...],
  "remediation": "具体修复指引"
}
```

## 输入规格
| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| asset_file | File | ✓ | 待检查素材 |
| brand_guidelines | Object | ✓ | 品牌规范（logo/色彩/字体） |
| target_platform | String | ✓ | 目标平台 |
| asset_type | String | ✓ | 素材类型 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| compliance_status | String | 合规状态 |
| checklist_result | Object | 5项逐项结果 |
| issues_detail | Array | 问题详情 |
| severity | String | info/warning/critical |
| remediation_guide | String | 修复指引 |

## 最佳实践
1. 品牌合规是"硬线"——不存在"差一点就过"的情况
2. 建立品牌素材合规样例库——合格样例+常见违规样例
3. 品牌指南更新时必须同步更新检查规则
4. Critical违规需要立即通知，不等批次完成
5. 频繁出现的合规问题应反馈到生成环节的prompt中预防

## 约束与限制
- Logo检查为强制项，不可跳过
- Critical违规不可自动重试，必须人工确认
- 品牌指南是唯一标准——不接受"看起来还行"的主观判断
- 每次规范更新后需重新校准检查逻辑
- 合规检查结果必须可追溯（谁检查的/何时/结果）
