# 视觉质量评分

## 技能概述
本技能对AI生成的视觉素材进行多维度质量评估，输出1-10分的综合质量分数和各维度细分评分。通过模拟专业设计师和广告投放优化师的双重视角，判定素材是否达到商业投放标准。

## 适用场景
- AI生成图片的质量评估
- 广告素材上线前的质量关卡
- 批量素材的快速筛选（合格/不合格分流）
- 不合格素材的问题诊断和改进指导
- 素材质量趋势监控

## 执行步骤

### Step 1: 清晰度评估（权重25%）
```
评分标准：
10分：专业摄影级清晰度，细节锐利
8-9分：高清晰度，放大后无明显模糊
6-7分：基本清晰，局部可能有轻微模糊
4-5分：存在明显模糊区域或压缩失真
1-3分：严重模糊，不具备商用价值

检查项：
- 整体锐度（focus sharpness）
- 边缘清晰度（无明显锯齿）
- 压缩伪影（无块状失真）
- 噪点水平（ISO噪声控制）
- 分辨率充足度（像素密度足够）
```

### Step 2: 构图质量评估（权重25%）
```
评分标准：
10分：大师级构图，视觉引导完美
8-9分：专业构图，主体突出，平衡感好
6-7分：基本合理，但不够精致
4-5分：构图有问题（主体偏移/比例失调/空间拥挤）
1-3分：构图混乱，主体不明确

检查项：
- 主体位置（遵循三分法/黄金比例）
- 视觉平衡（左右/上下重量感）
- 视觉引导线（眼睛自然流向CTA）
- 留白合理性（呼吸感vs空洞感）
- 元素层次（前景/中景/背景分明）
```

### Step 3: 美观度评估（权重25%）
```
评分标准：
10分：视觉震撼，高级感强烈
8-9分：美观度高，色彩和谐，吸引力强
6-7分：基本美观，无明显审美问题
4-5分：平庸或有不协调元素
1-3分：视觉不适，色彩冲突严重

检查项：
- 色彩和谐度（色相搭配合理性）
- 光影自然度（光源方向一致，无不自然阴影）
- 整体吸引力（是否能在信息流中脱颖而出）
- 质感表现（材质真实感）
- 氛围感（是否传递了目标情绪）
```

### Step 4: AI伪影检测（权重25%）
```
评分标准：
10分：完全看不出AI生成痕迹
8-9分：极少量微小瑕疵，不影响商用
6-7分：有轻微AI痕迹，但不显著
4-5分：有明显AI伪影，影响专业度
1-3分：严重AI伪影，不可商用

检查项：
- 手部/手指形态（多余手指/变形）
- 文字/数字渲染（乱码/不可读）
- 物理合理性（重力/透视/阴影方向）
- 面部/人体比例（五官错位/肢体比例失调）
- 重复图案（AI特有的纹理重复）
- 边缘处理（物体边缘的不自然融合）
```

### Step 5: 综合评分输出
```
综合分 = 清晰度×0.25 + 构图×0.25 + 美观度×0.25 + AI伪影×0.25

通过标准：
- 综合分≥7.0：Pass（通过）
- 综合分6.5-6.9：Conditional Pass（条件通过，附改进建议）
- 综合分<6.5：Fail（不通过，需重做）

附加规则：
- 任何单项≤3分：一票否决，直接Fail
- AI伪影项≤5分：直接Fail（AI痕迹过重不可商用）
```

## 输入规格
| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| image_asset | File | ✓ | 待评估图片 |
| intended_use | String | ✓ | 用途（产品图/主图/场景图） |
| target_channel | String | × | 目标渠道 |
| reference_standard | Object | × | 参考质量标准 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| overall_score | Number | 综合分（1-10） |
| clarity_score | Number | 清晰度分 |
| composition_score | Number | 构图分 |
| aesthetics_score | Number | 美观度分 |
| artifact_score | Number | AI伪影分（越高越好） |
| pass_status | String | pass/conditional_pass/fail |
| issues_found | Array | 发现的问题列表 |
| improvement_suggestions | Array | 改进建议 |

## 最佳实践
1. 评分要客观一致——建立评分标杆样例库（什么样的图是7分/8分/9分）
2. 关注"商用标准"而非"完美标准"——广告素材追求有效而非艺术
3. AI伪影检测重点关注人物相关的伪影（最容易被消费者识别）
4. 对于产品图，清晰度和AI伪影权重应提高
5. 评分结果附带具体改进建议才有实际价值

## 约束与限制
- 评分必须有明确的依据，不可模糊判定
- 同一素材多次评估结果差异不超过0.5分
- 评估时间≤5秒/张（批量处理效率要求）
- 不对创意方向进行主观好坏判断，只评估执行质量
- Conditional Pass的素材必须说明具体需要改进哪里
