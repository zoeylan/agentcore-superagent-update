# 文案质量评审

## 技能概述
本技能对广告文案进行多维度质量评审，评估文案的语法正确性、语义通顺度、情感基调匹配度和广告法合规性。确保文案既有创意吸引力和商业说服力，又完全符合法规要求和品牌调性。

## 适用场景
- AI生成文案的质量把关
- 多语言文案的本地化质量验证
- 广告法合规性审核
- 文案A/B测试变体的质量一致性确认
- 品牌调性匹配度评估

## 执行步骤

### Step 1: 语法与语义检查
```
检查维度：
- 语法正确性：无错别字、无语法错误、标点规范
- 语义通顺度：逻辑连贯、无歧义、易于理解
- 表达自然度：是否像"人写的"（vs机器翻译腔）
- 信息完整性：核心卖点是否完整传达

评分标准（1-10）：
10：完美无瑕，native-level自然
8-9：高质量，极少量可商榷之处
6-7：基本合格，有小瑕疵但不影响理解
4-5：存在明显问题，影响专业度
1-3：严重语法/语义问题，不可商用
```

### Step 2: 情感基调评估
```
评估方法：
1. 提取Brief指定的目标情感基调
2. 分析文案实际传达的情感
3. 判断匹配程度

情感基调类型：
- 激励型（"你值得更好的"）
- 紧迫型（"限时特惠，仅剩XX"）
- 信任型（"百万用户的选择"）
- 温暖型（"为家人，选最好"）
- 理性型（"对比数据，高下立判"）
- 趣味型（"拒绝无聊，来点不一样的"）

匹配度评分：
- 完全匹配：8-10分
- 基本匹配但力度不够：6-7分
- 方向偏离：4-5分
- 完全不匹配：1-3分
```

### Step 3: 广告法合规审核
```
强制检查项（中国广告法）：
✗ 绝对化用语：最佳、最好、第一、唯一、首选、顶级、极品
✗ 虚假承诺：100%有效、永不反弹、保证XX
✗ 限时虚假：清仓大甩卖（非真实清仓）
✗ 医疗相关：非医疗产品不得暗示疗效
✗ 金融相关：保本保息、稳赚不赔
✗ 教育相关：保过、保就业

特殊品类额外检查：
- 食品：不得宣称治疗效果
- 化妆品：不得承诺功效
- 保健品：需标注"非药品"
- 酒类：不得诱导过量饮酒

违规严重性：
- Critical：使用明确违法用语
- Warning：使用边界表述（可能被投诉）
- Info：建议优化的表述
```

### Step 4: 商业说服力评估
```
评估AIDA要素完整性：
A(Attention) - 标题是否抓眼球？（3秒法则）
I(Interest) - 正文是否激发兴趣？（与受众相关性）
D(Desire) - 是否创造了购买/行动的欲望？（利益点清晰度）
A(Action) - CTA是否明确有力？（下一步清晰度）

说服力评分要素：
- 卖点清晰度（读完知道为什么要买）
- 利益具体性（具体数字优于模糊描述）
- 受众相关性（针对目标人群的语言）
- CTA有效性（行动指引的明确性）
```

### Step 5: 综合质量判定
```
文案综合质量 = 语法语义×0.3 + 情感匹配×0.2 + 合规性×0.3 + 说服力×0.2

注意：广告法合规为一票否决——Critical违规直接Fail

通过标准：
- 综合分≥7.0 且 无合规Critical → Pass
- 综合分6.5-6.9 且 无合规问题 → Conditional Pass
- 综合分<6.5 或 有Critical违规 → Fail
```

## 输入规格
| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| copy_content | Object | ✓ | 文案内容（headline/body/cta） |
| target_language | String | ✓ | 目标语言 |
| brand_tone | String | ✓ | 品牌调性要求 |
| target_emotion | String | ✓ | 目标情感基调 |
| product_category | String | ✓ | 产品品类（影响合规规则） |
| target_platform | String | × | 目标平台 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| overall_score | Number | 综合分（1-10） |
| grammar_score | Number | 语法语义分 |
| emotion_match_score | Number | 情感匹配分 |
| compliance_result | Object | 合规审核结果 |
| persuasion_score | Number | 说服力分 |
| pass_status | String | pass/conditional_pass/fail |
| issues | Array | 问题列表 |
| rewrite_suggestions | Array | 改写建议 |

## 最佳实践
1. 合规审核是底线——先过合规再看质量
2. 情感基调评估要结合受众画像——对年轻人和中年人的"激励"方式不同
3. 不同语言的合规规则不同——每个市场有自己的广告法
4. 给出的改写建议要具体可执行（不只说"需要优化"）
5. 对边界表述（可能合规也可能违规）标注Warning让人工判断

## 约束与限制
- 广告法合规检查不可跳过，即使时间紧张
- 合规判定以法规文本为准，不接受"行业惯例"作为抗辩
- 多语言文案需按目标市场法规审核（不以源语言法规为准）
- 评审意见必须附带具体条款引用
- Critical违规文案不得以任何形式发布
