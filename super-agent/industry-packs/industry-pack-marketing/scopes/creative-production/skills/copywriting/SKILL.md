# 广告文案撰写与变体生成

## 技能概述
本技能负责撰写高转化率的广告文案，包括标题、正文、CTA（行动号召）等完整文案组件。支持基于同一核心卖点生成多版本变体用于A/B测试，确保文案既有创意吸引力又符合平台规范和广告法要求。

## 适用场景
- 新产品/新活动的广告文案创作
- 同一核心信息的多版本变体生成（A/B测试用）
- 不同渠道的文案长度适配
- 不同受众群体的文案调性调整
- 季节性/节日营销文案批量生产

## 执行步骤

### Step 1: 文案策略确定
根据Brief确定文案创作策略：
- **营销目标**：品牌认知 / 兴趣激发 / 购买转化 / 复购促进
- **核心卖点**：提炼1个主USP（独特销售主张）+ 2-3个支撑点
- **情感定位**：理性说服 / 情感共鸣 / 紧迫感驱动 / 社会认同
- **受众语言风格**：正式商务 / 轻松口语 / 潮流网感 / 专业术语

### Step 2: 文案架构设计
标准广告文案架构（AIDA模型）：
```
[Headline/标题] - 3-5秒抓住注意力（≤15字）
  └── Hook类型：提问式/数据式/痛点式/悬念式/利益式

[Body/正文] - 传递核心信息
  └── 利益点 + 场景化描述 + 信任背书

[CTA/行动号召] - 明确下一步动作
  └── 动词开头：立即抢购 / 免费试用 / 限时领取 / 了解更多
```

### Step 3: 变体生成策略
单一核心信息的多版本变体：
| 变体维度 | 变体A（基准） | 变体B | 变体C |
|----------|--------------|-------|-------|
| 情感基调 | 理性利益驱动 | 情感故事化 | 紧迫感制造 |
| 标题策略 | 直述卖点 | 提问引导 | 数据冲击 |
| CTA强度 | 标准引导 | 强行动号召 | 低压力建议 |

### Step 4: 渠道适配
各平台文案长度限制：
- Google Ads标题：≤30字符 × 3行
- Google Ads描述：≤90字符 × 2行
- Meta主文案：≤125字符（折叠前可见）
- Meta标题：≤40字符
- 抖音信息流：≤55字符
- 微信朋友圈：≤40字

### Step 5: 合规检查
文案自检清单：
- [ ] 无绝对化用语（最好、第一、唯一、全网最低等）
- [ ] 无虚假承诺（100%有效、保证瘦10斤等）
- [ ] 无违禁词（参考广告法负面清单）
- [ ] 品牌名称使用标准写法
- [ ] 无未授权的第三方品牌提及
- [ ] 特殊品类合规（医疗/金融/教育类需额外审核）

## 最佳实践
1. 标题是决定用户是否继续阅读的关键——花60%的精力在标题上
2. 文案要"说人话"——避免过度书面化，模拟目标受众的表达方式
3. 每个文案只传递一个核心信息——信息过载=零转化
4. CTA必须具体——"立即抢购"优于"了解更多"（转化目标场景）
5. 利用数字增强说服力——"3天见效"优于"快速见效"

## 约束与限制
- 严格遵守各平台字符数限制
- 不得使用广告法禁止的绝对化用语
- 不得做无法验证的效果承诺
- 品牌专有名词必须使用官方标准写法
- 每个变体之间差异必须明显可区分（不是换个同义词）
