# 营销Campaign策划

## 场景描述
当企业需要策划一场完整的营销Campaign时调用此技能。覆盖从策略定义、创意概念、渠道组合、执行排期到效果预估的全流程。适用于新品上市、品牌升级、大促活动、事件营销、跨界联名等各类Campaign场景。

## 触发条件
- 用户需要策划一场营销活动/Campaign
- 用户提到新品上市推广计划
- 用户需要制定大促/节日营销方案
- 用户询问如何做一次品牌事件/话题营销
- 用户需要整合营销传播（IMC）方案

## 输入信息
- Campaign目标（品牌认知/获客/促销转化/用户活跃）
- 目标受众画像与洞察
- 预算范围
- 时间节点与周期
- 产品/品牌核心卖点
- 竞品同期动作（如已知）
- 可用资源（KOL资源、媒介关系、内容团队等）

## 输出格式

### 1. Campaign策略框架
- **Campaign Big Idea**：一句话核心概念
- **传播主题/Slogan**：面向消费者的传播语言
- **策略逻辑**：为什么这个Idea能打动目标受众
- **成功指标**：量化的KPI目标

### 2. 受众策略
- 核心TA & 泛TA定义
- 用户洞察与情感切入点
- 人群触达路径设计

### 3. 内容与创意规划
- 核心创意物料清单（视频/图文/H5/事件）
- 各物料的角色定义（引爆/扩散/转化/沉淀）
- 创意brief要点
- AI辅助创意生产的建议

### 4. 渠道与排期
| 阶段 | 时间 | 渠道 | 内容形式 | 预算 | 目标 |
|------|------|------|---------|------|------|
| 预热期 | | | | | |
| 爆发期 | | | | | |
| 长尾期 | | | | | |

### 5. KOL/达人策略
- KOL层级组合（头部引爆+腰部种草+KOC扩散）
- 各层级预算分配与数量建议
- 内容合作方向

### 6. 效果预估与风险预案
- 各渠道预期效果（曝光/互动/转化）
- 总体ROI预估
- 风险场景与应急预案

## 专业规则
- Campaign需要有统一的Big Idea贯穿，不是多个散点动作的堆砌
- 预热期至少3-5天建立期待感，爆发期集中1-3天打透
- 短视频平台素材需准备至少3-5条不同角度的版本做赛马
- KOL合作重在内容共创而非简单广告植入，要给创作者留空间
- 大促Campaign重转化闭环，品牌Campaign重情感共鸣和社交传播
- 预留总预算10-15%作为机动预算，用于追加效果好的渠道
- 每个Campaign结束后必须做复盘，沉淀可复用的SOP和素材资产
