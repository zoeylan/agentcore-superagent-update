# 方言识别与标准化转换

## 概述
本技能负责将老年用户的方言语音输入识别并转换为标准普通话文本，覆盖中国5种以上主要方言（粤语、四川话、上海话、东北话、闽南语等），为后续意图理解提供可靠的文本基础。

## 适用场景
- 老年用户使用方言进行语音控制指令
- 方言与普通话混杂的日常表达
- 带有浓重方言口音的普通话语音
- 方言特有的词汇和句式结构转换

## 执行步骤

### 步骤1：语音接收与预处理
- 接收ASR（自动语音识别）引擎输出的原始文本
- 识别语音片段的方言类型（基于语音特征和词汇特征）
- 若ASR引擎输出置信度低于60%，标记为需要二次处理

### 步骤2：方言类型判定
- 基于词汇特征匹配判定方言分类（粤语特征词如"嘅"/"唔"、四川话"要得"/"安逸"）
- 基于语法结构特征辅助判定（如粤语的"动词+先"结构）
- 输出方言类型标签和判定置信度

### 步骤3：方言到标准文本转换
- 应用方言词汇映射表进行直接替换（如"冇"→"没有"、"啥子"→"什么"）
- 处理方言特有的语序调整（如粤语倒装结构）
- 保留语义完整性，不做过度标准化（保留用户意图的所有信息）

### 步骤4：输出与置信度标注
- 输出标准化后的普通话文本
- 附带方言类型标签、转换置信度评分
- 低置信度（<70%）区段标记，供下游模块重点处理

## 输入规格
```json
{
  "asr_text": "原始ASR识别文本",
  "audio_features": {
    "duration_ms": 3000,
    "confidence": 0.75,
    "language_hint": "cantonese|sichuan|shanghai|northeast|minnan|mandarin"
  }
}
```

## 输出规格
```json
{
  "standardized_text": "标准化后的普通话文本",
  "dialect_type": "detected_dialect",
  "conversion_confidence": 0.88,
  "low_confidence_segments": ["可能有歧义的片段"],
  "original_text": "原始方言文本"
}
```

## 最佳实践
- 维护持续更新的方言词汇映射库，定期补充新发现的方言表达
- 对同一用户的方言类型进行记忆，避免每次重新判定
- 混合方言场景（如四川话夹杂普通话）按片段分别处理
- 老年人特有的用词习惯（年代词汇、旧称谓）需纳入映射库

## 约束条件
- 方言识别准确率目标：核心5种方言 >= 85%
- 转换延迟：< 200ms（不含ASR耗时）
- 不可篡改用户原始语义，仅做语言形式转换
- 遇到完全无法识别的方言时，保留原文并标记，而非猜测
