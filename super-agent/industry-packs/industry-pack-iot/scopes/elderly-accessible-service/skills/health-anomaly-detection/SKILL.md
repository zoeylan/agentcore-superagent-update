# 健康与行为异常检测

## 概述
本技能通过持续分析老年用户的生理指标和生活行为数据，执行多级异常检测（紧急/重要/轻度），在偏离正常基线时及时识别异常并触发相应的响应流程。核心价值在于早期发现健康隐患和生活规律变化，实现「被动治疗」向「主动预防」的转变。

## 适用场景
- 穿戴设备生理指标持续监测（心率/血压/血氧/体温）
- 生活作息规律变化检测（起床/睡觉/外出时间偏移）
- 活动量趋势分析（日步数/移动范围变化）
- 用水用电等间接行为指标异常
- 长时间无活动信号检测

## 执行步骤

### 步骤1：数据采集与基线建立
- 采集周期：穿戴设备每5分钟/家居传感器实时/用水用电每小时汇总
- 建立个人基线：基于连续7天以上数据计算个人正常范围
- 基线动态更新：每周滑动窗口重新计算（排除已知异常数据点）

### 步骤2：实时异常评估
- 紧急指标判定（立即响应）：
  - 心率 > 基线+40bpm 或 < 基线-30bpm
  - 血氧 < 90%
  - 体温 > 39°C
  - 48小时内无任何活动信号
- 重要指标判定（当日通知）：
  - 连续2天作息偏移 > 2小时
  - 连续2次用药未确认
  - 日活动量 < 基线的30%
  - 血压连续3天超出正常范围
- 轻度指标判定（记录观察）：
  - 单日作息偏移 > 1小时
  - 睡眠质量评分下降 > 20%
  - 单日活动量低于基线50%
  - 用水/用电量显著偏离

### 步骤3：异常确认与分级
- 紧急异常：跳过确认，直接触发响应
- 重要异常：检查是否存在合理解释（如当天有外出记录/家属来访等）
- 轻度异常：累计记录，连续出现则升级为重要异常

### 步骤4：输出与联动
- 紧急：输出至紧急呼叫技能 + 家属协同联络器
- 重要：输出异常报告至家属协同联络器（APP推送）
- 轻度：记录至健康日志，纳入周报
- 所有异常数据同步至健康趋势分析模块

## 最佳实践
- 基线建立期间（前7天）不输出轻度异常，避免冷启动噪音
- 季节变化时适当放宽阈值（如冬季活动量自然减少）
- 区分「传感器异常」和「真实健康异常」（如血氧仪佩戴不当导致的低读数）
- 对慢性病患者使用专属阈值（如高血压患者的血压判定标准不同）
- 外出/旅行/家属来访等特殊事件应作为背景因素纳入判定

## 约束条件
- 异常检测准确率 >= 85%
- 紧急异常检出延迟 < 30秒
- 基线计算至少需要7天连续数据
- 不可基于单一数据点判定重要或轻度异常（需趋势确认）
- 不可输出医学诊断结论，仅描述数据异常现象
