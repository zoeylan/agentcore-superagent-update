# 光伏储能协同优化 (Solar-Storage Optimization)

## 技能概述

光伏储能协同优化技能负责管理家庭光伏发电系统和储能电池的充放电策略，实现"光伏发电优先自用、余电储能、峰电放电"的最优能量管理。随着户用光伏和家庭储能系统的普及，该技能帮助用户最大化新能源自用率、降低从电网购电量、并在峰电时段通过储能放电替代高价电力。

### 适用场景
- 家庭光伏系统日常充放电管理
- 储能电池SOC（荷电状态）区间管理
- 光伏余电的消纳策略选择（储能 vs 馈电）
- 阴雨天/夜间的储能放电时机优化
- 光伏+储能+电网三方的实时功率平衡

## 执行步骤

### 步骤1：光伏发电预测
1. 获取当日/次日天气预报（日照时长、云量、温度）
2. 基于光伏板参数（装机容量、朝向、倾角）和天气预测发电量曲线
3. 结合历史同季节同天气的实际发电数据修正预测
4. 输出24小时逐时发电功率预测（kW）
5. 计算预测置信度（晴天>0.9，多云0.6-0.8，阴雨<0.5）

### 步骤2：负载预测
1. 基于历史用电模式预测当日逐时用电负载
2. 结合已知的调度计划（EV充电/热水器等已排程设备）
3. 考虑特殊因素（周末用电高于工作日、节假日模式）
4. 输出24小时逐时预测负载（kW）

### 步骤3：储能策略制定
1. 获取储能电池当前状态：
   - SOC当前值(%)
   - 可用容量(kWh)
   - 最大充放电功率(kW)
   - SOC管理区间（建议保持20%-90%，避免过充过放）
2. 制定充放电策略：
   - **光伏充电**：发电>负载时，余电充入储能
   - **谷电充电**：谷电时段从电网补充充电（如果光伏当日预计不足）
   - **峰电放电**：峰电时段释放储能覆盖家庭负载
   - **应急保留**：始终保留20% SOC作为停电应急备用
3. 计算充放电时间表和功率曲线

### 步骤4：实时功率平衡
1. 每5分钟更新一次实时数据：
   - 光伏实际发电功率
   - 家庭实际负载功率
   - 储能电池实际SOC
2. 计算功率平衡：
   - 光伏 > 负载 → 余电充储能 → 储能满则考虑馈网
   - 光伏 < 负载 → 判断当前电价时段 → 峰电则储能放电 → 谷电则电网供电
   - 光伏 = 0（夜间）→ 峰电储能放电 → 谷电电网充电
3. 下发实时功率调度指令给逆变器/储能BMS

### 步骤5：经济收益计算
1. 计算自用比例 = 光伏直接消耗 / 光伏总发电量
2. 计算储能收益 = 峰电放电量 × (峰电价 - 谷电价)（充电成本）
3. 计算馈网收益 = 馈网电量 × 馈网电价（通常较低）
4. 对比无储能场景的电费，计算投资回报周期
5. 输出日度/月度收益报告

## 最佳实践

1. **保守SOC管理**：不追求100%充满或放到0%，保持20%-90%延长电池寿命
2. **天气预测修正**：实际发电量与预测偏差>20%时动态调整后续时段策略
3. **停电应急保留**：始终保留至少20%SOC（约2-3小时基本照明/冰箱用电）
4. **冬夏策略差异**：冬季日照短、负载高（暖气），需更多依赖谷电充电；夏季日照长、光伏充裕
5. **电池寿命优先**：避免一天内超过2次完整充放循环，日历衰减和循环衰减取平衡

## 约束条件

- SOC管理区间：硬限制10%-95%，推荐20%-90%
- 最大充放电功率：不超过电池额定功率的1C
- 策略更新频率：每5分钟根据实时数据微调
- 光伏发电预测误差目标：晴天<10%，多云<25%
- 储能控制指令响应时间 < 2秒
- 电池日循环次数限制：≤1.5次等效完整循环
