# 自动化营销跟进 (Automated Marketing Followup)

## 概述

自动化营销跟进技能负责对识别出的中意向商机客户执行精准的营销触达策略，通过适时发送促销信息、优惠券、限时折扣和个性化推荐，提升商机转化率。该技能在服务场景中以自然、不打扰的方式嵌入营销元素，将客服对话从单纯的问题解决扩展为转化引导，同时严格尊重客户的沟通偏好。

## 适用场景

- 中意向商机客户的自动化促销触达
- 犹豫型客户的限时优惠推送
- 浏览未购买客户的精准推荐
- 老客户的复购激励
- 大促期间的主动营销消息群发
- 购物车放弃后的挽回触达

## 详细执行步骤

### 步骤1：策略选择
1. 根据商机信号类型和客户特征选择营销策略：

| 信号类型 | 推荐策略 | 触达时机 |
|----------|----------|----------|
| 询价犹豫 | 限时折扣码 | 对话结束后30分钟 |
| 竞品比较 | 差异化卖点+优惠 | 对话中即时嵌入 |
| 批量需求 | 阶梯价格方案 | 对话中即时提供 |
| 复购意向 | 老客专属优惠 | 对话结束时 |
| 浏览未购 | 相似商品推荐 | 次日同一时段 |
| 购物车放弃 | 限时包邮/降价通知 | 放弃后2小时 |

2. 检查客户营销接受度：
   - "免打扰"标记 → 不执行任何主动营销
   - 近7天已收到2次营销 → 暂停触达（避免骚扰）
   - 客户偏好渠道 → 按偏好渠道触达

### 步骤2：内容生成
1. 基于选定策略生成营销内容：
   - 优惠券：确定面值（基于客户历史客单价的10-15%）、有效期（通常48-72小时）、使用条件
   - 限时折扣：确定折扣力度、截止时间、适用商品范围
   - 商品推荐：基于客户浏览/咨询历史，推荐3-5个相关商品
   - 卖点话术：针对竞品比较场景，准备差异化卖点说明
2. 内容本地化：
   - 翻译为客户偏好语言
   - 适配文化习惯（如东南亚市场强调包邮、欧美市场强调退换保障）
   - 金额展示使用当地货币

### 步骤3：触达执行
1. 对话内嵌入式营销（即时）：
   - 在回答客户问题的同时，自然地附加推荐信息
   - 话术模式："关于您咨询的XX，目前有限时优惠..."
   - 确保营销内容不喧宾夺主，服务解答仍是主体
2. 延迟触达式营销（对话结束后）：
   - 通过平台消息/邮件/短信按预设时机发送
   - 选择客户活跃时段发送（基于历史在线时间分析）
   - 内容简洁，包含明确的行动号召（CTA）
3. 执行记录：记录触达时间、渠道、内容，用于后续效果分析

### 步骤4：效果追踪
1. 追踪指标：
   - 触达送达率（消息是否成功送达）
   - 优惠券领取率
   - 优惠券使用率（领取→使用转化）
   - 推荐商品点击率
   - 最终下单转化率和成交金额
2. 按策略类型统计ROI：营销成本（优惠券面值）vs 带来的增量订单
3. 追踪窗口：48小时（优惠券使用）/ 7天（最终转化归因）

### 步骤5：策略迭代
1. 低转化策略识别：连续7天转化率<5%的策略标记为待优化
2. A/B测试：对新策略进行小流量测试（10%客户），验证后再扩大
3. 最佳时机学习：基于历史数据优化最佳触达时机（小时/星期几）
4. 频次控制：维护客户级别的营销疲劳度模型，避免过度触达

## 输入规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| opportunity | object | 是 | 商机信息（评分、信号类型、产品） |
| customer_profile | object | 是 | 客户画像 |
| available_promotions | array | 否 | 可用的促销资源（优惠券、折扣活动） |
| channel_preference | string | 否 | 客户偏好触达渠道 |
| marketing_history | array | 否 | 客户近期营销触达历史 |

## 输出规格

| 字段 | 类型 | 说明 |
|------|------|------|
| marketing_action | object | 营销动作详情 |
| action_type | string | 动作类型（coupon/discount/recommendation/message） |
| content | object | 营销内容（多语言版本） |
| timing | string | 触达时机 |
| channel | string | 触达渠道 |
| tracking_id | string | 效果追踪ID |

## 最佳实践

1. **自然嵌入**：营销信息必须以服务延伸的形式呈现，避免生硬推销
2. **稀缺性设计**：限时优惠标注明确的截止时间，制造适度紧迫感
3. **个性化优先**：基于客户历史行为推荐，而非千人一面的群发
4. **退出机制**：每条营销消息包含"不再接收此类信息"的选项
5. **合规优先**：遵守各平台的营销消息发送规则和频次限制

## 约束条件

- 同一客户7天内最多接收3次主动营销触达
- 客户标记"免打扰"后30天内不进行任何主动营销
- 优惠券面值不超过客户历史客单价的20%
- 营销消息必须遵守各平台反垃圾消息政策
- 大促期间营销频次上限可临时提升至每3天1次
- 所有营销内容不得包含虚假承诺或误导性信息
