# PIP流程管理 (PIP Management)

## 技能概述

管理绩效改进计划（Performance Improvement Plan）的全生命周期，从启动条件判断到期满评估决策。PIP是组织对连续低绩效员工的最后改进机会，同时也是企业依法解除劳动合同的必要前置程序。PIP管理需要HR专业知识、法律合规意识和人文关怀的三重平衡。

## 适用场景

- 连续两个考核周期绩效评级为D的员工
- 单次绩效严重不达标且情节特殊的员工（需HR总监特批）
- PIP过程中的定期跟踪和进度评估
- PIP期满后的决策（达标/未达标/延期）

## 执行步骤

### 第一步：PIP启动评估
1. 确认PIP启动条件已满足（连续2次D评级，且有完整书面记录）
2. 核实历史合规性：
   - 考核标准是否事先告知员工（入职时签署）
   - 考核过程是否公正客观（有数据支撑）
   - 前次D评级是否已书面通知并提供改进机会
3. 提请HR总监审批PIP启动
4. 与法务沟通确认PIP方案的法律合规性

### 第二步：PIP计划制定
1. **改进目标设定**：具体、可衡量、可实现的30/60/90天阶段目标
   - 30天：基础能力恢复/态度改善，设定最低底线要求
   - 60天：独立完成核心工作任务，达到B级标准的80%
   - 90天：全面恢复到B级绩效水准
2. **支持措施明确**：
   - 培训安排（针对能力不足的技能培训）
   - 导师指导（指定资深同事进行工作辅导）
   - 资源支持（是否需要调整工作量或提供额外工具）
3. **跟踪机制**：每两周一次正式check-in，管理者+HR+员工三方参加
4. **评估标准**：明确"达标"和"未达标"的量化界定

### 第三步：PIP执行跟踪
1. 每两周组织check-in会议，记录：
   - 阶段目标完成情况（量化数据）
   - 员工态度和投入度变化
   - 支持措施的有效性（是否需要调整）
   - 下一阶段重点和预期
2. 所有check-in记录需三方签字存档
3. 重大进展或问题即时上报HR总监
4. 员工表示放弃或主动提出离职时，记录并转入协商流程

### 第四步：PIP期满评估
1. **达标**：改进目标全部或大部分达成→PIP结束，回归正常绩效管理
   - 出具PIP结束通知书
   - 后续1-2个考核周期保持关注
2. **部分达标**：核心目标达成但存在个别不足→可延期PIP（最多延期30天）
   - 需HR总监审批延期决定
   - 明确延期期间的具体改进要求
3. **未达标**：改进目标未达成→启动法务评估流程
   - 整理全部PIP过程材料（启动审批→计划制定→check-in记录→期满评估）
   - 法务评估解除合同的法律风险
   - 制定处理方案（协商解除/调岗降薪/按程序解除）
   - 所有处理方案需HR总监+法务+业务三方会签

## 输入要求

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 员工绩效历史记录 | 连续评级和评估详情 | 是 |
| 前期面谈和改进记录 | 之前的沟通和改进措施 | 是 |
| 岗位胜任力标准 | PIP目标设定的参考基线 | 是 |
| HR总监审批 | PIP启动和方案审批 | 是 |
| 法务合规意见 | PIP方案和退出方案的法律审核 | 是 |

## 输出规格

| 输出项 | 格式 | 说明 |
|--------|------|------|
| PIP计划书 | 正式文档（签字版） | 包含目标、时间线、支持、评估标准 |
| 双周check-in记录 | 签字表格 | 每次跟踪的进展记录 |
| PIP期满评估报告 | 正式文档 | 综合评估结论和建议处理方案 |
| PIP全套归档材料 | 文件包 | 全流程书面材料（合规存档用） |

## 最佳实践

1. **程序正义优先**：确保每一步都有书面记录和员工签字确认，这是合规底线
2. **改进目标合理**：PIP目标不能故意设定为无法完成（否则员工可主张PIP走过场）
3. **支持措施实质化**：培训和辅导必须真实提供，不能只是文件上的"支持"
4. **保护员工尊严**：PIP过程保密，不在团队中公开，避免羞辱性对待
5. **留有出路**：即使预判员工可能未达标，也要提供真实的改进机会和支持

## 约束条件

- PIP启动必须经HR总监书面审批
- PIP全程保留完整书面记录（劳动仲裁的关键证据）
- PIP期限通常为90天（30+60结构或30+30+30结构）
- 每两周至少一次正式check-in（不可省略）
- PIP期间员工的合法权益（薪资、社保、假期）不受影响
- 未达标后的处理方案必须法务审核后方可执行
