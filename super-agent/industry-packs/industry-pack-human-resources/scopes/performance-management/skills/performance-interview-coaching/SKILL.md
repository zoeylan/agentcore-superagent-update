# 绩效面谈辅导 (Performance Interview Coaching)

## 技能概述

为管理者提供绩效面谈技巧培训和实操辅导，帮助其有效传递绩效评估结果，特别是低绩效面谈中的沟通策略。同时监控全公司面谈完成时效，确保评估结果出来后5个工作日内100%完成面谈。高质量的绩效面谈是绩效管理闭环中最关键的"最后一公里"。

## 适用场景

- 年度/季度绩效评估结果出炉后的面谈组织
- 新晋管理者的面谈能力培训
- 低绩效员工（C/D等级）面谈前的辅导准备
- 面谈后员工情绪异常（如申诉、抵触、消极）的应对

## 执行步骤

### 第一步：面谈准备辅导
1. 为管理者提供面谈准备Checklist：
   - 回顾员工全周期工作表现（非只看结果）
   - 准备2-3个具体的正面事例和待改进事例
   - 明确面谈目标（传递结果+达成共识+约定行动）
   - 预判员工可能的反应并准备应对方案
2. 低绩效面谈特别准备：
   - 提前与HR BP沟通法律风险点
   - 准备客观数据和事实支撑
   - 设计面谈话术（事实描述→影响分析→期望明确→支持承诺→行动约定）
   - 预备员工情绪应急预案

### 第二步：面谈结构化框架
1. **开场（5分钟）**：营造轻松氛围，说明面谈目的
2. **回顾（10分钟）**：回顾目标完成情况，先肯定亮点
3. **反馈（15分钟）**：传递评估结果，用"事实+影响+期望"模型
4. **对话（10分钟）**：听取员工观点，回应疑问和不同看法
5. **行动（10分钟）**：共同制定下一步行动计划
6. **总结（5分钟）**：确认双方共识，约定后续跟踪节点

### 第三步：面谈时效监控
1. 评估结果确认后Day 0发布面谈要求通知
2. Day 3监控完成率，对未开始的管理者发送提醒
3. Day 4对仍未开始的管理者升级催办（通知其上级）
4. Day 5截止日确认100%完成，未完成者纳入管理者自身绩效扣分

### 第四步：面谈质量确认
1. 收集面谈记录表（双方签字确认版本）
2. 抽查面谈记录质量（是否有具体反馈内容、是否有行动计划）
3. 收集员工面谈满意度反馈（匿名调研）
4. 对面谈质量不达标的管理者提供一对一辅导

## 输入要求

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 评估结果清单 | 各员工最终绩效等级 | 是 |
| 目标完成度数据 | 各目标的实际完成情况 | 是 |
| 管理者名单 | 需要执行面谈的管理者清单 | 是 |
| 历史面谈记录 | 上周期面谈行动计划的执行情况 | 否 |

## 输出规格

| 输出项 | 格式 | 说明 |
|--------|------|------|
| 面谈进度报告 | 仪表盘 | 实时面谈完成率（按部门/按管理者） |
| 面谈记录档案 | 签字文档 | 双方签字的面谈记录表 |
| 管理者面谈能力评估 | 报告 | 各管理者的面谈质量评分和改进建议 |
| 面谈满意度报告 | 统计报告 | 员工对面谈过程的满意度分析 |

## 最佳实践

1. **先肯定后改进**：即使是D等级员工，也要先找到值得肯定的点再谈问题
2. **事实不评价**：用"你上季度3个项目延期了"而非"你工作态度有问题"
3. **不比较**：避免"你比XXX差"的表述，聚焦在个人进步
4. **给出路径**：低绩效面谈必须包含"如何改进"的具体路径，不能只说问题
5. **确保隐私**：面谈在封闭空间进行，不在开放区域讨论绩效结果

## 约束条件

- 面谈必须在评估结果确认后5个工作日内完成
- 面谈记录需双方签字，员工拒签需HR在场见证并记录
- C/D等级员工面谈建议HR BP旁听或事后确认
- 面谈过程中不允许修改已确认的评估结果（员工有异议走申诉渠道）
- 面谈记录归入员工档案保存至少3年（合规要求）
