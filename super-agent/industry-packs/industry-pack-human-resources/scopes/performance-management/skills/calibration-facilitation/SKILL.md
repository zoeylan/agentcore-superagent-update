# 校准会组织 (Calibration Facilitation)

## 技能概述

组织和引导绩效校准会议（Calibration Session），确保不同管理者之间、不同部门之间的评估标准一致性，消除系统性偏差，实现同一绩效等级在全公司范围内含义一致。校准会是强制分布落地前的关键质量把关环节。

## 适用场景

- 年度绩效评估后的部门内校准会
- 跨部门校准会（HR总监主持）
- 强制分布配额调整讨论
- 边界案例（介于两个等级之间）的定级讨论

## 执行步骤

### 第一步：校准会前准备
1. 汇总各管理者的评估结果和初步分布情况
2. 计算各部门当前分布与目标分布的偏差（S10%/A20%/B50%/C15%/D5%）
3. 标注需要重点讨论的案例：边界人员、极端评分、与历史大幅偏离者
4. 准备校准会议程和讨论规则（每案例讨论时间限制、表决方式）
5. 发送预读材料给参会者（不含个人详细评分，仅含部门分布概况）

### 第二步：部门内校准会（评估截止后3天内）
1. 由部门负责人主持，HR BP协助，部门内所有管理者参加
2. 各管理者逐一陈述S和D评级员工的理由（用具体事实和数据支撑）
3. 其他管理者可质疑或补充跨项目协作中的表现
4. 对边界案例进行讨论，达成部门内共识
5. 确认部门内分布是否符合配额要求

### 第三步：跨部门校准会（部门校准后5天内）
1. 由HR总监主持，各部门负责人和HR BP参加
2. 对比各部门分布情况，识别明显异常（如某部门0个D或S占比超标）
3. 要求异常部门负责人提供说明和支撑材料
4. 讨论部门间公平性问题（如支撑部门vs创收部门的标准差异）
5. 必要时进行跨部门调配（将某部门多出的S配额调配给确有突出表现的其他部门）

### 第四步：校准结果确认
1. 记录所有校准决议和调整理由
2. 更新最终绩效评级（校准后结果为最终结果）
3. 对被调整评级的员工，准备管理者沟通要点
4. 产出校准会纪要，各方签字确认

## 输入要求

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 各管理者初步评估结果 | 评分和评级建议 | 是 |
| 强制分布目标 | S/A/B/C/D的比例要求 | 是 |
| 员工关键业绩数据 | 目标完成度、量化业绩 | 是 |
| 历史绩效数据 | 员工历史评级用于对比 | 否 |
| 部门业务指标达成 | 部门整体业绩参考 | 否 |

## 输出规格

| 输出项 | 格式 | 说明 |
|--------|------|------|
| 校准后最终评级 | 数据表 | 每位员工的最终绩效等级 |
| 校准会纪要 | 文档 | 讨论要点、调整记录、决议及理由 |
| 分布达标报告 | 统计表 | 校准后各部门分布与目标的偏差 |
| 异常案例备忘录 | 文档 | 需特别关注的人员及后续行动 |

## 最佳实践

1. **用数据说话**：校准讨论必须基于具体事实和业绩数据，不接受"感觉"
2. **聚焦头尾**：重点讨论S和D，中间的B/C争议较小可批量确认
3. **时间管控**：每个讨论案例限时5分钟，避免陷入个别争议
4. **避免政治博弈**：HR作为中立方确保讨论聚焦在业绩而非关系
5. **保密原则**：校准会讨论内容不得向被评估员工透露（只告知结果不告知过程）

## 约束条件

- 部门内校准会必须在评估截止后3个工作日内完成
- 跨部门校准会必须在部门校准后5个工作日内完成
- 校准后强制分布偏差>5%的部门需上报HR总监审批
- 校准会纪要需各参会管理者签字确认，作为合规存档文件
- 已校准确认的结果原则上不得再次修改（除发现重大事实错误）
