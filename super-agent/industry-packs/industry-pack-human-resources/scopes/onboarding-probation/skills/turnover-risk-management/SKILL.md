# 离职风险管理 (Turnover Risk Management)

## 技能概述

本技能用于在试用期内识别新员工的离职风险信号（主动离职和被动离职），并根据风险级别采取对应的预防和干预措施。对于主动离职风险，通过改善体验和沟通留人；对于绩效不胜任的被动离职风险，确保合规流程和证据完备。

## 适用场景

- 试用期内新员工表现出离职倾向的预警
- 直属上级反馈新员工适应困难的干预
- 试用期绩效不达标员工的分级处理
- 试用期不通过决策的合规评估
- 试用期内劳动关系解除的流程管理

## 执行步骤

### 步骤1：风险信号识别
- 主动离职风险信号：
  - 出勤异常（频繁请假、迟到增多）
  - 工作投入下降（产出减少、参与度降低）
  - 社交退缩（不参加团队活动、与同事互动减少）
  - 直接表达不满（对工作内容、团队关系或公司文化的负面反馈）
- 被动离职风险信号：
  - 试用期目标严重偏离（1/3节点完成度低于50%）
  - 直属上级多次反映能力不匹配
  - 导师评估持续低分
  - 出现严重违纪或价值观冲突事件

### 步骤2：风险评估分级
- 低风险：个别信号，可能是临时性困难→关注观察
- 中风险：多个信号叠加或持续存在→主动干预
- 高风险：明确离职意向或严重不胜任→启动应对流程

### 步骤3：主动离职风险的干预
- 安排HR与新员工的非正式沟通，了解真实原因
- 区分可解决问题（岗位调整、团队关系、工作内容）和不可解决问题（薪资竞争力、地理位置）
- 对可解决问题制定针对性改善方案：与上级沟通改善管理方式、调整工作安排、提供额外支持
- 对确定要离职的员工，尊重决定，做好知识交接

### 步骤4：被动离职风险的合规处理
- 确认证据链完整性：
  - 录用条件是否明确告知且有签字确认
  - 试用期目标是否双方确认
  - 绩效不达标的事实是否有客观数据支撑
  - 改进机会是否已提供且有记录
  - 各次沟通是否有书面记录
- 评估法律风险：咨询法务确认可以以"不符合录用条件"为由解除
- 准备书面通知：明确不通过原因、引用具体证据、符合格式要求
- 确保提前15天通知（法定要求）

### 步骤5：离职执行与善后
- 正式面谈告知决定（HR+直属上级共同在场）
- 送达书面通知并由员工签收
- 办理离职手续（工作交接、设备归还、系统权限关闭）
- 结算工资和经济补偿（如适用）
- 开具离职证明
- 归档全部过程文件

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 新员工信息 | 对象 | 是 | 基本信息、合同信息 |
| 风险信号描述 | 文本 | 是 | 发现的具体异常表现 |
| 试用期目标完成情况 | 对象 | 是 | 各项目标当前进度 |
| 回访评估记录 | 列表 | 是 | 历次回访的结果 |
| 沟通记录 | 列表 | 否 | 已进行的改进沟通 |

## 输出规范

- 风险评估报告（含级别判断和建议）
- 干预行动方案（针对可挽留情况）
- 证据完备性检查清单（针对不通过决策）
- 书面通知文本（法务审核版）
- 离职手续办理清单

## 最佳实践

1. **早发现早干预**：不要等到试用期末才处理，1/3节点发现问题就要行动
2. **证据先行**：任何可能导致试用期不通过的情况，第一时间开始证据留痕
3. **善意推定**：不要轻易放弃，真正不胜任的情况往往经过充分沟通和帮助后才能确认
4. **法务前置**：在正式做出不通过决定前，必须经法务审核确认合规性
5. **体面离场**：即使是不通过的员工，也要确保流程尊重和体面，保护雇主品牌

## 约束条件

- 试用期解除合同的举证责任在用人单位，证据不充分时不得轻率决定
- 提前15天书面通知是法定要求，计算日历日而非工作日
- 不得以"不符合录用条件"之外的理由在试用期解除（如部门裁员不适用试用期解除）
- 员工处于医疗期、孕期、产期、哺乳期的，试用期解除需特别谨慎
- 试用期不通过不等于无需支付任何补偿，需根据具体情况由法务判断
