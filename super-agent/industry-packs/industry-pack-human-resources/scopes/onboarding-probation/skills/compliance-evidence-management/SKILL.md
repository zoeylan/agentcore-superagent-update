# 合规证据管理 (Compliance Evidence Management)

## 技能概述

本技能用于在试用期管理全过程中系统性地收集、整理和保管法律合规所需的各类证据材料，确保在发生劳动争议时公司拥有完整的举证链条。根据中国劳动合同法，用人单位以"不符合录用条件"解除试用期员工的举证责任完全在用人单位一方。

## 适用场景

- 试用期全过程的日常证据归档
- 试用期不通过决策前的证据完备性审查
- 劳动仲裁/诉讼中的证据准备和应诉支持
- 试用期管理流程合规性的定期审计

## 执行步骤

### 步骤1：证据类型清单维护
- 录用条件告知证据：
  - Offer letter中载明的录用条件
  - 入职时签署的《录用条件确认书》
  - 试用期目标设定书（双方签字）
- 考核过程证据：
  - 试用期各阶段目标完成的客观数据
  - 1/3和2/3节点回访评估记录
  - 直属上级的书面评估意见
  - 导师的辅导记录和评估反馈
- 沟通改进证据：
  - 绩效问题沟通面谈记录（双方签字）
  - 改进计划和目标（书面确认）
  - 改进措施的跟进记录
  - 公司提供支持和帮助的记录
- 程序合规证据：
  - 提前15天书面通知的送达记录
  - 不通过原因的书面说明
  - 员工签收或拒签的记录

### 步骤2：日常归档规范
- 所有正式文件要求双方签字，拒签则请第三方见证
- 电子沟通记录（邮件、IM消息）定期导出备份
- 系统数据（考勤、产出、评分）定期截屏或导出
- 按员工建立独立的证据档案文件夹
- 每份材料标注：形成时间、当事人、内容摘要

### 步骤3：证据完备性检查
- 在做出试用期不通过决策前，执行证据完备性检查清单：
  - ☐ 录用条件已明确告知且有签字
  - ☐ 试用期目标已设定且双方确认
  - ☐ 不达标有客观数据/事实支撑
  - ☐ 已给予充分的改进机会和帮助
  - ☐ 改进无效有后续跟踪记录
  - ☐ 各次正式沟通有书面记录
  - ☐ 程序符合法定要求（提前通知等）
- 任何一项不满足，暂不推进决策，先补齐证据

### 步骤4：争议应对准备
- 维护证据索引表（按时间线梳理）
- 形成不通过决策的《事实认定和法律依据报告》
- 预判员工可能提出的质疑点，准备回应
- 与法务共同审核证据链的完整性和说服力
- 保存所有原件，复印件加盖公章

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 员工信息 | 对象 | 是 | 基本信息、合同信息 |
| 证据材料 | 列表 | 是 | 各类需归档的证据文件 |
| 证据类型 | 枚举 | 是 | 录用条件/考核过程/沟通改进/程序合规 |
| 归档时间 | 日期 | 是 | 材料形成或收集的日期 |

## 输出规范

- 结构化证据档案（分类归档）
- 证据完备性检查报告
- 证据索引时间线
- 《事实认定和法律依据报告》（决策前出具）

## 最佳实践

1. **事前建档**：从入职第一天就建立证据档案，而非出问题后才补收集
2. **同步归档**：每次正式沟通、评估后24小时内完成归档，避免事后回忆性补录
3. **客观记录**：证据记录使用客观事实描述，避免主观评价性语言
4. **双重备份**：纸质原件+电子扫描件双重保存，防止材料丢失
5. **定期审查**：每月对在试用期员工的证据档案进行完备性检查

## 约束条件

- 证据收集不得侵犯员工隐私权（如偷录、未经授权的监控）
- 电子证据需注意保全方式（时间戳、未篡改证明）
- 证据材料的保存期限至少为劳动关系终止后2年（仲裁时效）
- 涉密信息在劳动争议场景中的使用需脱敏处理
- 拒签情况必须有第三方见证人签名和情况说明
