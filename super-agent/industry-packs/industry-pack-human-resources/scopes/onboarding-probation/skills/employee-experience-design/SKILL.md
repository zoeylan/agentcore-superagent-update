# 员工体验设计 (Employee Experience Design)

## 技能概述

本技能用于设计和优化新员工入职首周的沉浸式体验，通过精心设计的触点管理、融入活动和关怀措施，帮助新员工快速建立归属感和组织认同。研究表明入职首周体验对员工6个月内的留任意愿有决定性影响。

## 适用场景

- 入职首日的欢迎仪式和环境引导设计
- 首周融入活动的策划与组织
- 新员工关怀触点的设计和执行
- 入职体验满意度调研与持续优化
- 批量入职（校招）的集体融入活动设计

## 执行步骤

### 步骤1：首日体验设计
- 设置欢迎仪式：工位欢迎卡/小礼品、团队欢迎邮件/群消息
- 办公环境引导：工位区域、会议室、茶水间、洗手间、打印区、健身房
- 实用信息传达：WiFi密码、内部通讯工具使用、常用系统入口
- 午餐安排：由直属上级或buddy陪同午餐，避免新员工第一天独自用餐
- 首日结束前的check-in：询问感受、解答疑惑、确认次日安排

### 步骤2：首周融入活动规划
- Day 2-3：部门团队午餐/茶歇，认识团队成员
- Day 3-4：与直属上级首次一对一沟通（期望对齐、首月重点）
- Day 4-5：如有同期入职的新员工，组织新员工互相认识交流
- 首周末：发送首周体验简短问卷（3-5题，快速收集反馈）

### 步骤3：关怀触点管理
- 入职第1天：欢迎礼+环境引导+温暖的首日总结
- 入职第3天：主动询问是否遇到任何困难或不便
- 入职第7天：首周体验反馈收集+后续安排说明
- 入职第14天：两周适应情况跟进
- 入职第30天：一月融入效果回顾

### 步骤4：满意度收集与优化
- 首周问卷回收率目标≥90%
- 分析满意度数据，识别体验短板（如IT设备问题、环境指引不足等）
- 每月汇总新员工体验反馈，形成改进建议
- 每季度更新入职体验方案，持续迭代优化

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 新员工信息 | 对象 | 是 | 姓名、部门、岗位、兴趣偏好 |
| 团队情况 | 对象 | 是 | 团队规模、文化风格 |
| 同期入职人数 | 数字 | 是 | 决定是否组织集体活动 |
| 特殊需求 | 文本 | 否 | 如饮食禁忌、身体状况等 |

## 输出规范

- 个性化首周日程安排表
- 欢迎物料准备清单
- 首周体验满意度调研报告
- 体验优化建议（基于反馈数据）

## 最佳实践

1. **个性化触点**：了解新员工的偏好（如内向/外向），相应调整融入方式的intensity
2. **Buddy制度**：除正式导师外，安排一位同级别的buddy作为日常问题的求助对象
3. **信息不过载**：首日信息量已经很大，避免在首周前3天安排过多正式培训
4. **领导可见度**：确保新员工入职首周内至少与隔级上级有一次简短交流
5. **仪式感创造**：入职欢迎邮件、团队自我介绍round、首周小里程碑庆祝等

## 约束条件

- 体验设计不得增加新员工的压力感（如过于密集的社交活动安排）
- 欢迎礼品预算遵循公司标准，不同岗位级别统一标准
- 融入活动不得安排在非工作时间（除非员工自愿参加）
- 收集的个人偏好信息仅用于体验优化，不得挪用
- 关注特殊群体需求（如有社交焦虑的员工提供替代方案）
