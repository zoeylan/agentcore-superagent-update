# 结构化回访评估 (Structured Review Assessment)

## 技能概述

本技能用于在试用期关键节点（1/3和2/3节点）执行标准化的多方回访评估，通过结构化问卷和访谈获取新员工适应情况的全方位信息，及早发现问题并启动干预。

## 适用场景

- 试用期1/3节点（通常入职后1-2个月）的首次回访
- 试用期2/3节点（通常入职后2-4个月）的二次回访
- 发现异常信号后的加密回访
- 转正评估前的综合信息收集

## 执行步骤

### 步骤1：回访准备
- 确认回访时间节点（根据入职日期和试用期时长计算）
- 提前3天通知回访对象（新员工、直属上级、导师）并预约时间
- 准备结构化回访问卷（针对不同对象的版本）
- 收集前置信息：试用期目标设定表、上次回访记录（如有）、近期工作产出

### 步骤2：新员工回访
- 工作适应维度：对工作内容、节奏、难度的感受
- 团队融入维度：与同事的关系、团队氛围感受、沟通顺畅度
- 支持满意度：导师辅导是否有帮助、上级指导是否充分、资源是否充足
- 困难和需求：当前遇到的最大困难、需要公司提供的支持
- 自我评估：对自己目标完成进度的判断
- 职业期望：是否与入职前预期一致、未来发展想法

### 步骤3：直属上级回访
- 任务完成度：各项目标的进度和质量评估
- 工作能力：学习速度、问题解决能力、独立性
- 工作态度：主动性、责任心、时间管理
- 团队协作：配合度、沟通效率、对团队的贡献
- 综合判断：目前是否在转正轨道上、主要顾虑
- 支持需求：需要HR提供什么协助

### 步骤4：导师回访
- 辅导进展：辅导计划执行情况、沟通频率和质量
- 学习成长：知识吸收速度、技能提升情况
- 潜力评估：学习能力、成长潜力、发展方向
- 融入观察：从导师视角看到的适应情况
- 改进建议：对新员工的发展建议

### 步骤5：结果汇总与行动
- 将三方回访信息整合为《回访评估报告》
- 综合判断分类：
  - 绿灯（一切顺利）→记录存档，继续观察
  - 黄灯（存在可控风险）→制定改进措施+增加关注频率
  - 红灯（严重问题）→24小时内上报HR经理，启动干预流程
- 将行动计划同步给相关方
- 回访记录48小时内归档

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 新员工信息 | 对象 | 是 | 姓名、部门、入职日期 |
| 回访节点 | 枚举 | 是 | 1/3节点 / 2/3节点 / 加密回访 |
| 试用期目标 | 对象 | 是 | 已确认的考核目标 |
| 上次回访记录 | 对象 | 否 | 如为2/3节点回访则需 |
| 已知问题 | 列表 | 否 | 此前发现的待关注事项 |

## 输出规范

- 《试用期回访评估报告》（含三方意见汇总）
- 风险等级判断（绿灯/黄灯/红灯）
- 行动计划（针对黄灯和红灯情况）
- 异常上报单（红灯情况24小时内提交）

## 最佳实践

1. **一对一访谈**：三方回访应分别进行，避免新员工在上级面前不敢表达真实感受
2. **结构化+开放式结合**：标准问卷确保覆盖关键维度，开放式提问捕捉意外信息
3. **对比分析**：关注三方回答中的一致性和差异，差异点往往是问题所在
4. **及时反馈**：回访发现的积极表现及时反馈给新员工，增强信心和动力
5. **趋势追踪**：将1/3节点和2/3节点的评估对比，关注变化趋势而非单点数据

## 约束条件

- 回访时间不得延迟超过3个工作日（法律合规需要时间节点明确）
- 回访记录48小时内完成归档，超期需说明原因
- 红灯异常情况24小时内上报，不得拖延
- 回访内容涉及员工隐私，严格控制知情范围
- 回访结论需客观呈现，不得隐瞒或夸大问题
