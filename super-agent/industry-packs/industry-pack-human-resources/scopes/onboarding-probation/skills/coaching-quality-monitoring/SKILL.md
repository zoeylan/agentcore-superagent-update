# 辅导质量监控 (Coaching Quality Monitoring)

## 技能概述

本技能用于持续监控导师对新员工的辅导过程，确保辅导频率达标、内容有效、双方满意，及时发现和解决辅导过程中的问题，保障导师制度的实际效果而非流于形式。

## 适用场景

- 导师辅导频率的定期检查（每两周一次）
- 辅导质量的月度评估
- 辅导异常（如长期缺乏沟通）的预警和干预
- 导师更换需求的评估和决策
- 导师体系整体效果的季度分析

## 执行步骤

### 步骤1：辅导频率监控
- 检查标准：每周至少1次一对一沟通，每次不少于30分钟
- 数据来源：导师辅导记录系统/周报提交情况
- 频率检查周期：每两周汇总一次
- 预警规则：
  - 黄色预警：连续2周未有辅导记录→发送提醒
  - 红色预警：连续3周未有辅导记录→升级至导师直属上级

### 步骤2：辅导内容质量审核
- 审核维度：
  - 辅导主题是否与发展计划对齐
  - 内容是否有实质性帮助（vs 闲聊/走形式）
  - 是否有明确的行动项和跟进
  - 是否覆盖了计划中的关键能力模块
- 审核方式：
  - 辅导记录抽检（每月抽查30%的辅导记录）
  - 新员工满意度问卷（月度）
  - 阶段性产出对比（新员工能力提升速度）

### 步骤3：双向满意度收集
- 新员工维度（5分制评估）：
  - 导师可及性（是否容易约到时间）
  - 指导有效性（建议是否有帮助）
  - 耐心和支持度（是否感到被支持）
  - 专业水平（专业知识是否过硬）
  - 综合满意度
- 导师维度（5分制评估）：
  - 新员工学习主动性
  - 辅导投入的时间合理性
  - 辅导成就感
  - HR的支持到位度

### 步骤4：问题诊断与干预
- 满意度<3.0时启动诊断面谈：
  - 与新员工单独沟通了解原因
  - 与导师单独沟通了解困难
  - 判断问题根因（匹配问题/方法问题/投入问题/期望差距）
- 制定干预措施：
  - 匹配问题→启动导师更换
  - 方法问题→为导师提供辅导技巧培训
  - 投入问题→与导师上级协调工作量
  - 期望差距→调整双方预期并明确辅导边界

### 步骤5：整体效果分析
- 季度汇总指标：
  - 辅导频率达标率（目标≥85%）
  - 新员工平均满意度（目标≥4.0/5.0）
  - 导师更换率（目标<10%）
  - 新员工能力提升速度（对比无导师组）
- 产出改进建议：优化匹配规则、改进辅导模板、调整激励措施

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 辅导配对信息 | 列表 | 是 | 当前所有在进行的辅导配对 |
| 辅导记录 | 列表 | 是 | 导师提交的辅导记录 |
| 满意度评分 | 列表 | 是 | 双方满意度数据 |
| 辅导计划 | 对象 | 是 | 各配对的辅导计划 |

## 输出规范

- 辅导频率达标情况报告（双周）
- 辅导质量评估报告（月度）
- 异常预警通知和干预建议
- 导师体系效果分析报告（季度）

## 最佳实践

1. **轻量化记录**：辅导记录模板简洁（5分钟即可完成），避免增加导师负担
2. **正向激励为主**：对表现优秀的导师及时认可，避免仅在出问题时才沟通
3. **趋势重于单点**：关注满意度变化趋势，偶尔的低分可能是暂时性波动
4. **保护隐私**：新员工反馈的具体内容不直接转达导师，以保护信任关系
5. **系统化支持**：为导师提供辅导工具包（模板、话题库、资源推荐）

## 约束条件

- 监控不等于微观管理，保持对导师的信任和尊重
- 满意度数据仅用于改进目的，不直接作为负面考核依据
- 辅导记录属于内部管理文件，不对外公开
- 导师更换需同时征得新员工和新导师同意
- 频率监控宽限：因导师出差/休假导致的临时中断不计入预警
