# 试用期目标跟踪 (Probation Goal Tracking)

## 技能概述

本技能用于协助设定、追踪和评估新员工试用期考核目标，确保目标设定科学合理（SMART原则）、过程跟踪有据可依、结果评估客观公正。试用期目标是转正评估的核心依据，也是试用期不通过时的法律举证基础。

## 适用场景

- 新员工入职首周的试用期目标制定
- 目标执行过程中的阶段性检查
- 目标调整（因业务变化或岗位职责变更）
- 转正前的目标完成度评估汇总
- 试用期不通过时的证据准备

## 执行步骤

### 步骤1：目标设定框架设计
- 与直属上级沟通岗位核心产出要求
- 将试用期分为3个阶段（各1/3时长），设定阶段性里程碑：
  - 第一阶段（适应期）：了解业务流程、掌握基础工具、完成入门任务
  - 第二阶段（成长期）：独立处理常规任务、达到基本产出标准
  - 第三阶段（验证期）：独立处理复杂场景、达到岗位胜任标准
- 每个阶段设定2-3个可量化的关键指标

### 步骤2：SMART原则审核
- Specific（具体）：目标描述清晰无歧义，不使用"尽量""争取"等模糊词
- Measurable（可衡量）：有明确的量化标准或可观察的行为描述
- Achievable（可达成）：参考同岗位历史新人数据，目标难度适中
- Relevant（相关）：与岗位录用条件直接挂钩
- Time-bound（有时限）：每个目标有明确的完成时间节点

### 步骤3：目标确认与存档
- 形成《试用期考核目标书》
- 直属上级、新员工、HR三方签字确认
- 明确告知员工：这些目标是转正评估的核心依据
- 原件存入员工档案，电子版录入HR系统

### 步骤4：过程追踪
- 在1/3和2/3节点对照目标检查完成进度
- 记录目标完成的客观数据（系统数据、工作产出物、客户反馈等）
- 目标偏离时分析原因并记录（员工能力不足 vs 外部客观因素）
- 若因业务变化需调整目标，走书面变更流程并重新确认

### 步骤5：目标完成度评估
- 转正评估前汇总各项目标的完成情况
- 对每项目标给出"达标/部分达标/未达标"的结论
- 附上支撑材料（数据、产出物、360反馈等）
- 形成《试用期目标完成情况评估报告》

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 新员工信息 | 对象 | 是 | 姓名、岗位、入职日期、试用期时长 |
| 岗位录用条件 | 文本 | 是 | Offer中明确的录用条件 |
| 岗位职责描述 | 文本 | 是 | JD中的核心职责 |
| 同岗位历史基准 | 对象 | 否 | 过往同岗位新人的表现数据 |

## 输出规范

- 《试用期考核目标书》（三方签字版）
- 阶段性进度检查记录
- 《试用期目标完成情况评估报告》
- 目标调整变更记录（如有）

## 最佳实践

1. **与录用条件对齐**：试用期目标必须与Offer中的录用条件一致，这是法律合规的关键
2. **难度梯度设计**：三个阶段难度递增，第一阶段不设过高要求，给适应期足够空间
3. **避免主观指标**：尽量用可量化指标，减少"态度好""配合度高"等主观评价
4. **定期沟通**：不要到评估节点才告知问题，过程中发现偏差应及时反馈
5. **证据留痕**：每次沟通和评估都要有书面记录，这是试用期不通过时的法律保障

## 约束条件

- 目标设定必须在入职后7个工作日内完成
- 目标不得包含与岗位无关的要求（如与工作无关的私人要求）
- 目标变更必须经过三方重新确认，单方面变更无效
- 录用条件必须是入职前已告知或入职时书面确认的，不能事后补设
- 评估结论必须有客观数据支撑，不能仅凭主观印象
