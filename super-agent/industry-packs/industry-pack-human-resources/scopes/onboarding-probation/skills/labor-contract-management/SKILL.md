# 劳动合同管理 (Labor Contract Management)

## 技能概述

本技能用于管理新员工入职时的劳动合同签署全过程，确保合同条款合法合规、签署程序规范、双方权利义务明晰，同时完成相关附件（保密协议、竞业限制协议等）的签署和归档。

## 适用场景

- 新员工入职首日的劳动合同签署
- 试用期相关条款的解释和确认
- 保密协议和竞业限制协议的签署
- 合同签署异常情况的处理（员工对条款有疑问/异议）
- 电子合同和纸质合同的双轨管理

## 执行步骤

### 步骤1：合同准备
- 根据Offer审批信息生成劳动合同（已审批的薪资、岗位、试用期时长等）
- 核实合同关键条款：
  - 合同期限与试用期时长的合法性对应（1-3年合同→试用期≤2个月；3年以上→≤6个月）
  - 试用期工资≥转正工资的80%且≥当地最低工资标准
  - 工作地点、工作时间、劳动保护等必备条款完整
- 准备附件：保密协议、知识产权归属协议、竞业限制协议（如适用）
- 合同打印两份（公司一份、员工一份）

### 步骤2：条款讲解
- 逐一向新员工讲解合同核心条款：
  - 合同期限和试用期说明
  - 薪资结构（基本工资+绩效奖金+其他）
  - 工作职责和岗位说明
  - 劳动纪律和解除条件
  - 保密义务和竞业限制（如有）
- 特别提醒试用期相关权益：试用期工资标准、试用期解除的条件和程序
- 留出时间回答员工疑问

### 步骤3：签署执行
- 确认员工已充分理解合同内容（可询问确认）
- 员工在合同每页签章处签名/按手印
- 合同尾页签署完整签名+日期
- 附件协议逐一签署
- 公司法定代表人或授权人签章

### 步骤4：异常处理
- 员工对条款有疑问：耐心解释，必要时请法务支持
- 员工要求修改条款：记录诉求，报HR经理和法务评估
- 员工当日拒绝签署：不得强迫，约定次日签署，超过3天未签署需上报
- 签署笔误或错误：作废重新打印签署，不得涂改

### 步骤5：归档与交付
- 合同原件一份存入员工档案（HR保管）
- 合同原件一份交付员工（当面签收确认）
- 电子扫描件上传至HR系统
- 合同签署情况录入台账（合同编号、签署日期、到期日期、试用期截止日期）

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Offer审批信息 | 对象 | 是 | 含薪资、岗位、合同期限等 |
| 新员工信息 | 对象 | 是 | 身份证信息、联系方式 |
| 合同类型 | 枚举 | 是 | 固定期限/无固定期限 |
| 附件要求 | 列表 | 是 | 需签署的附件协议清单 |

## 输出规范

- 已签署的劳动合同（双方各一份）
- 已签署的附件协议
- 合同签署确认回执（员工已领取合同的签收单）
- 合同信息台账更新记录

## 最佳实践

1. **充分告知**：不走过场式签署，确保员工真正理解关键条款，特别是试用期和竞业限制
2. **不催促**：给员工充分的阅读和思考时间，被证明是在"被催促"下签署的合同可能被认定无效
3. **版本管控**：合同模板由法务统一审定，HR不得擅自修改标准条款
4. **及时交付**：合同签署后当天交付员工一份原件，延迟交付属违法行为
5. **预警管理**：建立合同到期预警（提前60天），避免事实劳动关系风险

## 约束条件

- 劳动合同必须在入职当日签署，最迟不超过入职后1个月（超期需支付双倍工资）
- 试用期时长严格遵守劳动合同法规定，不得约定超法定上限
- 试用期工资不低于合同约定工资的80%，且不低于当地最低工资标准
- 同一用人单位与同一劳动者只能约定一次试用期
- 竞业限制协议必须约定经济补偿条款，否则该限制条款可能无效
- 合同条款不得违反法律强制性规定（如约定"工伤自理"等条款无效）
