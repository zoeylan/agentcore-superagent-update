# 入职材料审核 (Document Verification)

## 技能概述

本技能用于对新员工提交的入职材料进行合规性、完整性和真实性审核，确保入职手续符合法律法规要求，同时保护公司免受虚假材料带来的用工风险。

## 适用场景

- 新员工入职前的标准材料收集与审核
- 特殊情况下的材料豁免/延期审批
- 材料真伪存疑时的深入核查
- 材料归档和电子化管理

## 执行步骤

### 步骤1：材料清单发放与收集
- 根据岗位类型发放《入职材料清单》
- 区分必交材料（入职前必须提供）和可补交材料（入职后限期提供）
- 必交材料清单：身份证复印件、最高学历证书复印件、离职证明原件、近3个月体检报告、银行卡复印件
- 可补交材料：一寸证件照（5张）、社保转移单、公积金转移函、学位证书认证

### 步骤2：形式审核
- 检查材料完整性：是否缺件
- 检查材料有效性：证件是否在有效期内、体检报告是否在3个月内
- 检查格式规范性：复印件是否清晰可辨、是否需要原件核验
- 离职证明重点审核：是否有竞业限制约定标注、离职日期与入职日期是否衔接

### 步骤3：实质审核
- 学历验证：通过学信网验证学历证书真伪
- 离职证明核实：如有疑问，征得候选人同意后联系前单位HR确认
- 体检报告审核：确认必检项目齐全（传染病筛查、肝功能、常规检查等）
- 特殊岗位附加审核：财务岗查询征信记录（需本人授权）、管理岗查询竞业限制状态

### 步骤4：问题处理
- 材料缺失：明确补交截止日期（入职后5个工作日内），发送书面提醒
- 材料存疑：暂不下结论，启动进一步核实流程
- 关键材料无法提供（如离职证明）：评估风险，报HR经理决策是否暂缓入职
- 发现虚假材料：立即上报HR总监，暂停入职流程

### 步骤5：归档管理
- 审核通过的材料进行电子扫描归档
- 建立员工入职材料电子档案
- 原件核验后退还本人，保留复印件存档
- 材料审核结论和异常处理记录一并存档

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 新员工基本信息 | 对象 | 是 | 姓名、身份证号、岗位 |
| 提交材料列表 | 列表 | 是 | 已提交的各项材料 |
| 岗位类型 | 枚举 | 是 | 决定审核标准和附加项 |
| 材料提交方式 | 枚举 | 是 | 线上提交/线下提交/混合 |

## 输出规范

- 《入职材料审核报告》：各项材料的审核状态和结论
- 《材料补交通知书》：缺失材料的补交要求和截止日期
- 《材料异常报告》：发现问题时的详细说明和处理建议
- 电子档案：审核通过材料的标准化归档

## 最佳实践

1. **分级管理**：将材料分为"入职阻断项"和"可后补项"，避免因非关键材料影响入职时间
2. **提前沟通**：Offer发放时即附带材料清单，给候选人充足准备时间
3. **线上化收集**：推动材料线上提交和审核，减少纸质材料传递的时间损耗
4. **隐私保护**：材料收集和保管严格遵循《个人信息保护法》，最小必要原则
5. **标准化命名**：电子档案统一命名规范（员工号_材料类型_日期），便于检索

## 约束条件

- 离职证明是唯一能证明候选人已与前单位解除劳动关系的法定文件，缺失时风险极高
- 体检报告必须在3个月有效期内，超期需重新体检
- 学历验证以学信网官方查询结果为准，不接受其他渠道证明
- 材料审核过程中获取的个人信息仅用于入职审核目的，不得挪作他用
- 背景调查必须取得候选人书面授权后方可进行
