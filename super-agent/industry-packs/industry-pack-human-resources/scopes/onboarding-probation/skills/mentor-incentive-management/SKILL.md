# 导师激励管理 (Mentor Incentive Management)

## 技能概述

本技能用于设计和运营导师激励体系，通过物质激励与精神认可相结合的方式，确保资深员工愿意投入时间和精力担任新员工导师，维持导师制度的可持续运转。

## 适用场景

- 导师辅导工作量的绩效加分核算
- 年度/半年度优秀导师评选
- 导师能力提升培训的组织
- 导师资源池的扩充和维护
- 导师激励政策的效果评估和优化

## 执行步骤

### 步骤1：辅导工作量核算
- 核算周期：按绩效考核周期（季度/半年度）
- 核算规则：
  - 完成一个完整辅导周期（从分配到新员工转正）：绩效加分0.5-1.0分
  - 辅导周期内辅导频率达标率≥85%：满分；70-85%：按比例折算；<70%：不加分
  - 新员工满意度≥4.0/5.0：额外加0.2分
  - 新员工试用期不通过（非导师责任）：照常计分
- 数据来源：辅导记录、满意度评分、转正结果

### 步骤2：优秀导师评选
- 评选周期：年度
- 评选维度：
  - 辅导投入度（辅导频率、沟通时长）权重30%
  - 辅导效果（学员成长速度、满意度评分）权重40%
  - 辅导经验（累计辅导人数、年资）权重15%
  - 方法创新（是否有可复制的辅导方法论）权重15%
- 评选流程：数据初筛→候选名单→评审委员会投票→结果公示
- 奖励：证书+奖金+优先培训/会议名额+公司内部宣传

### 步骤3：导师能力提升
- 必修培训（入池前完成）：
  - 辅导技巧基础课程（4小时）
  - 反馈给予方法（2小时）
  - 新员工心理适应理论（1小时）
- 选修提升：
  - 教练式辅导进阶
  - 跨代际沟通技巧
  - 情境领导力
- 导师交流会：每季度组织一次导师经验分享

### 步骤4：资源池维护
- 入池标准：绩效≥B、工作年限≥3年、完成必修培训、意愿确认
- 更新周期：每季度刷新一次
- 退出机制：绩效降至C以下、连续2次辅导满意度<3.0、本人申请退出
- 扩充策略：对高绩效员工主动邀请、将"担任导师"作为晋升参考项

### 步骤5：激励效果评估
- 年度评估指标：
  - 导师主动报名率（目标≥80%的符合条件员工愿意入池）
  - 辅导任务接受率（推荐时≥90%的导师愿意承接）
  - 导师留存率（年度退出率<15%）
  - 辅导效果趋势（学员满意度年度趋势）
- 对比分析：有导师 vs 无导师的新员工在适应速度、留存率上的差异
- 优化建议：根据数据调整激励力度和方式

## 输入规范

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 辅导完成数据 | 列表 | 是 | 各导师的辅导工作量数据 |
| 满意度评分 | 列表 | 是 | 新员工对导师的评分 |
| 辅导结果 | 列表 | 是 | 学员转正情况 |
| 导师基本信息 | 列表 | 是 | 资源池成员信息 |

## 输出规范

- 绩效加分核算表（按绩效周期）
- 优秀导师评选报告（年度）
- 导师资源池状态报告（季度）
- 激励体系效果分析报告（年度）
- 导师培训计划和执行记录

## 最佳实践

1. **物质+精神结合**：单纯的绩效加分不够，公开认可和荣誉感同样重要
2. **降低门槛感**：强调辅导也是导师自身的成长机会，而非额外负担
3. **标杆效应**：优秀导师的案例分享和公开表彰，激发更多人参与
4. **公平透明**：绩效加分规则公开透明，避免"辛苦付出但看不到回报"
5. **持续优化**：每年根据反馈调整激励方案，保持制度吸引力

## 约束条件

- 绩效加分需纳入正式绩效考核体系，经HR总监审批
- 优秀导师评选需有明确的量化标准，避免主观评定
- 导师激励政策变更需提前一个周期公告
- 辅导工作量核算以系统记录为准，不接受事后补录
- 导师培训的时间从工作时间中安排，不占用私人时间
