# 薪酬方案设计

## 技能概述
基于候选人评估结果、市场薪酬数据和内部薪酬体系，设计具有竞争力且合理的个性化薪酬方案，涵盖基本薪资、绩效奖金、长期激励和福利组合，确保在吸引人才与控制成本之间取得平衡。

## 适用场景
- 社招候选人通过终面后的薪酬方案制定
- 校招岗位的统一薪酬标准设计
- 候选人counter offer后的方案调整
- 特殊人才的超标薪酬方案设计（需特批）

## 执行步骤

### 第一步：信息收集
1. 获取候选人综合评估报告和录用建议等级
2. 了解候选人当前薪资结构（月薪×月数+奖金+股权+福利）
3. 了解候选人薪资期望和核心诉求
4. 获取目标岗位的薪资带宽（P25/P50/P75/P90）
5. 查询内部同岗位、同级别员工的薪资水平

### 第二步：薪资定位
基于候选人评估等级确定薪资定位：
| 评估等级 | 建议定位 | 说明 |
|----------|----------|------|
| 强烈推荐 | P50-P75 | 优秀候选人给予有竞争力的方案 |
| 推荐 | P25-P50 | 合格候选人定位在中等水平 |
| 保留（但录用） | P25 | 入门水平，通过绩效证明后快速调整 |

### 第三步：薪酬结构设计
1. **基本月薪**：按薪资定位确定月固定薪资
2. **绩效奖金**：按岗位等级确定绩效奖金月数（通常1-4个月）
3. **年终奖**：明确年终奖的目标月数和考核挂钩方式
4. **长期激励**（如适用）：股票期权/限制性股票的授予数量和归属期
5. **签字费**（如需要）：用于弥补候选人离职损失（如年终奖未发放）
6. **福利包**：五险一金基数、补充商业保险、年假天数、其他弹性福利

### 第四步：内部公平性检查
1. 对比同部门同级别员工薪资，确保无明显倒挂
2. 如新员工薪资高于同级别老员工10%以上，标记风险
3. 评估方案对团队薪酬体系的整体影响
4. 如有倒挂风险，提出缓解方案（如调整定级、设定绩效条件等）

### 第五步：方案文档化
1. 编制薪酬方案审批文档
2. 包含：候选人信息、评估结果摘要、薪酬方案详情、市场对标数据、内部对比分析
3. 如超出标准范围，附加说明和特批理由
4. 提交双签审批（用人部门负责人+HR总监）

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 候选人评估报告 | 文档 | 是 | 含录用建议等级 |
| 候选人当前薪资 | 结构化数据 | 是 | 完整的薪资结构 |
| 候选人薪资期望 | 数字 | 是 | 期望年度总包 |
| 岗位薪资带宽 | 数值区间 | 是 | P25-P75-P90 |
| 内部薪酬对标 | 数据 | 是 | 同级别员工薪资 |
| 市场薪酬数据 | 数据 | 是 | 外部薪酬报告 |

## 输出规范
- 薪酬方案文档：完整的薪酬结构拆解和对标分析
- 审批申请单：含所有审批所需信息
- 谈判策略建议：可协商项和底线说明
- Offer letter薪酬条款草案

## 最佳实践
- 总包涨幅30%是市场常见跳槽溢价，超过50%需谨慎评估候选人是否虚报当前薪资
- 薪酬方案设计要考虑长期激励的留人效果，不要过度依赖短期现金
- 在预算允许时，适当高于市场中位数可显著提升Offer接受率
- 非薪资吸引因素（弹性工作、发展空间、团队氛围）也是方案的重要组成部分

## 约束条件
- 薪资不得突破岗位带宽P90上限（除非CEO/CHRO特批）
- 同岗位新老员工薪资倒挂超过15%必须预警
- 试用期薪资不低于正式薪资的80%（劳动合同法要求）
- 薪酬方案涉及的税务影响需告知候选人（避免期望管理偏差）
