# 候选人初步评估（电话初面）

## 技能概述
对通过简历初筛的候选人进行15-20分钟的电话初面，确认候选人的求职意向、基本沟通能力、期望薪资和可到岗时间，作为是否推进至正式面试的决策依据。

## 适用场景
- 简历初筛通过后的常规电话确认
- 内推候选人的意向沟通
- 被动候选人的触达和意向激发
- 猎头推荐候选人的补充了解

## 执行步骤

### 第一步：电话前准备
1. 回顾候选人简历和初筛评估报告
2. 准备针对性问题（基于简历中需确认的要点）
3. 确认可面试时段信息以便当场约定
4. 确保通话环境安静、专业

### 第二步：开场与意向确认（2分钟）
1. 自我介绍（公司名称、HR身份、来电目的）
2. 确认对方是否方便接听
3. 初步了解候选人当前求职状态（在看机会/已有Offer/不考虑变动）

### 第三步：基本情况沟通（8-10分钟）
1. 当前工作内容和团队情况（验证简历真实性）
2. 离职/看机会原因（判断动机合理性）
3. 对新机会的核心诉求（薪资/平台/发展/稳定/地域）
4. 针对简历中的疑问进行确认（如职业空窗期、短期离职经历）
5. 快速评估沟通表达能力和逻辑思维

### 第四步：关键信息收集（3-5分钟）
1. 薪资期望：当前薪资结构（月薪×月数+奖金+股权）和期望总包
2. 到岗时间：当前在职需多久交接，最快可到岗日期
3. 其他在面试的公司和进度（了解竞争态势）
4. 对我们公司/岗位的了解程度和兴趣点

### 第五步：结束与后续安排（2-3分钟）
1. 简要介绍岗位亮点和公司优势（激发兴趣）
2. 告知后续流程和预计时间线
3. 如候选人意向明确，当场约定正式面试时间
4. 感谢并留下后续联系方式

### 第六步：初面记录归档
1. 完成电话初面评估表
2. 给出推进/暂缓/放弃的明确结论
3. 标记需要在正式面试中重点验证的问题点
4. 48小时内完成记录归档

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 候选人简历 | 文档 | 是 | 通过初筛的简历 |
| 初筛评估报告 | 文档 | 是 | 含匹配度评分和待确认问题 |
| 岗位关键信息 | 文本 | 是 | 用于向候选人介绍的岗位亮点 |
| 可面试时段 | 时间段 | 否 | 后续面试可安排的时间窗口 |

## 输出规范
- 电话初面评估表：意向程度（高/中/低）、沟通能力评价、薪资期望、到岗时间、核心诉求、推进建议
- 面试预约确认（如当场约定）
- 候选人状态更新：推进面试/暂缓/放入人才库/放弃

## 最佳实践
- 尊重候选人时间，控制通话在20分钟以内
- 不要在电话中做出任何薪资承诺或暗示
- 对于被动候选人，先建立信任再了解情况，不要过于功利
- 如候选人明确表示不考虑，礼貌结束并征求是否可存入人才库
- 初面后24小时内给候选人反馈后续安排，提升候选人体验

## 约束条件
- 不得在电话中询问婚育计划、宗教信仰等私密信息
- 不得向候选人透露其他候选人信息
- 通话录音需事先征得候选人同意（如有录音需求）
- 工作时间外致电需提前短信确认是否方便
