# 渠道效果分析

## 技能概述
对各招聘渠道的表现进行数据化分析，从简历数量、质量、转化率和成本效益四个维度评估渠道价值，为渠道策略优化和预算分配提供数据支撑。

## 适用场景
- 月度/季度渠道效果复盘
- 新渠道试用效果评估
- 年度招聘预算规划的数据支撑
- 猎头合作方绩效评估

## 执行步骤

### 第一步：数据采集
1. 统计各渠道的简历投递量（按时间周期）
2. 记录各渠道的初筛通过率
3. 追踪各渠道候选人的面试通过率和最终入职率
4. 计算各渠道的使用成本（平台费+人力投入+猎头费）

### 第二步：核心指标计算
| 指标 | 计算公式 | 标杆值 |
|------|----------|--------|
| 简历有效率 | 初筛通过数/总投递数 | 15-25% |
| 面试转化率 | 进入面试数/初筛通过数 | ≥80% |
| 录用转化率 | 发出Offer数/面试数 | ≥60% |
| 入职转化率 | 实际入职数/Offer数 | ≥85% |
| 单人招聘成本 | 渠道总成本/该渠道入职人数 | 视岗位级别 |
| 平均招聘周期 | 各渠道从简历到入职的平均天数 | ≤30天 |

### 第三步：渠道对比分析
1. 制作渠道效果对比矩阵（象限图：质量 vs 数量）
2. 识别高效渠道（高质量+大数量）和低效渠道（低质量+高成本）
3. 分析不同岗位类型下各渠道的表现差异
4. 趋势分析：各渠道效果的月度变化趋势

### 第四步：成本效益分析
1. 计算各渠道的ROI：(渠道贡献的入职员工年薪总额 × 系数) / 渠道总成本
2. 猎头渠道专项分析：各猎头公司的推荐质量、响应速度、成功率对比
3. 内推渠道分析：内推奖金投入 vs 内推入职人数和留存率
4. 评估渠道投入的边际效益递减点

### 第五步：优化建议输出
1. 渠道投入建议：增加/维持/减少/停止各渠道的投入
2. 预算重新分配方案
3. 新渠道试用建议（基于行业趋势和同行实践）
4. 猎头合作调整建议（续约/更换/新增）

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 分析周期 | 日期范围 | 是 | 月度/季度/年度 |
| 各渠道投递数据 | 数字 | 是 | 简历数量按渠道分类 |
| 转化漏斗数据 | 数字 | 是 | 各环节通过率 |
| 渠道费用数据 | 数字 | 是 | 各渠道的直接和间接成本 |
| 岗位分类标签 | 枚举 | 否 | 用于分岗位类型分析 |

## 输出规范
- 渠道效果分析报告：含核心指标计算结果、对比矩阵、趋势图表
- 成本效益分析表：各渠道ROI排名
- 优化建议方案：具体的渠道调整和预算重分配建议
- 猎头绩效评估卡：各猎头公司的综合评分和续约建议

## 最佳实践
- 分析时注意控制变量，不同岗位类型的渠道效果不可直接横向比较
- 关注长期趋势而非单月波动，避免因短期数据做出过激调整
- 内推渠道的隐性价值（文化匹配度高、留存率高）不容忽视
- 新渠道给予2-3个月的试用观察期再做效果判定

## 约束条件
- 渠道效果数据需真实准确，不允许选择性报告
- 猎头绩效评估需基于客观数据，避免主观偏见
- 渠道调整建议需考虑合同约束（如年度合作协议）
- 分析报告不对外公开，属于企业经营决策信息
