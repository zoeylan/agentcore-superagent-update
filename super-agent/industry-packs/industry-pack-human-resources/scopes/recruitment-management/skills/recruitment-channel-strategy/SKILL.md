# 招聘渠道策略制定

## 技能概述
根据岗位特征、紧急程度和预算约束，制定最优的招聘渠道组合策略。涵盖线上招聘平台、内部推荐、猎头合作、校园招聘、RPO外包等多种渠道的选择与组合优化。

## 适用场景
- 新岗位需确定首选招聘渠道
- 现有渠道效果不佳需调整策略
- 校招季启动需规划校园渠道布局
- 高端岗位需评估猎头合作方案
- 批量招聘需评估RPO外包方案

## 执行步骤

### 第一步：岗位特征分析
1. 判断岗位类型：技术类/管理类/销售类/运营类/职能类
2. 评估岗位级别：初级（1-3年）/中级（3-5年）/高级（5-10年）/总监+
3. 分析市场稀缺度：高度稀缺/一般稀缺/供给充足
4. 确认紧急程度：加急（≤15天）/正常（≤30天）/储备（≤60天）

### 第二步：渠道匹配矩阵
| 岗位特征 | 首选渠道 | 辅助渠道 | 预算考量 |
|----------|----------|----------|----------|
| 初中级技术岗 | Boss直聘、拉勾 | 内推、技术社区 | 低（平台年费） |
| 高级技术岗 | 猎聘、脉脉 | 猎头、内推 | 中高（猎头费20-25%年薪） |
| 管理岗 | 猎头 | 猎聘、内推 | 高（猎头费25-30%年薪） |
| 销售岗 | Boss直聘、智联 | 行业社群、内推 | 低 |
| 校招批量 | 校园宣讲、牛客网 | 学校就业网、实习转正 | 中（差旅+宣讲费用） |
| 基层批量 | 前程无忧、58同城 | RPO外包 | 按人头计费 |

### 第三步：渠道组合设计
1. 确定主渠道（1-2个）：预期贡献60-70%的有效简历
2. 确定辅助渠道（1-3个）：补充覆盖和多样性
3. 设定各渠道的预期产出指标（简历数量、质量预期）
4. 制定渠道启动时间线和资源投入计划

### 第四步：猎头管理（如涉及）
1. 从合作猎头池中选择擅长该领域的猎头公司
2. 确认费率（通常年薪的20-30%）和付款条件（入职付50%+过保付50%）
3. 签署委托协议并进行岗位briefing
4. 设定猎头简历推荐频率和质量要求
5. 简历去重机制：猎头推荐的候选人需与自有渠道已有简历核对

### 第五步：效果跟踪与优化
1. 每周统计各渠道简历投递量和初筛通过率
2. 渠道效果低于预期时（如2周内有效简历<5份）启动调整
3. 发现高效渠道时加大投入，低效渠道及时止损
4. 月度汇总渠道ROI分析报告

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 岗位画像 | 文档 | 是 | 完整的岗位画像信息 |
| 招聘预算 | 数字 | 是 | 本岗位可用的渠道预算 |
| 紧急程度 | 枚举 | 是 | 加急/正常/储备 |
| 历史渠道数据 | 数据 | 否 | 同类岗位过往各渠道表现 |
| 合作猎头池 | 列表 | 否 | 可用的猎头合作方信息 |

## 输出规范
- 渠道组合方案：主渠道+辅助渠道清单及各自预期产出
- 预算分配计划：各渠道的费用预算和时间安排
- 猎头委托方案（如涉及）：选定猎头、费率、timeline
- 渠道启动checklist：每个渠道的具体启动操作步骤
- 效果跟踪指标：各渠道的周度评估标准

## 最佳实践
- 内推是成本最低、质量最高的渠道，所有岗位都应同步开放内推
- 内推奖金设置需有吸引力（通常3000-20000元，按岗位级别阶梯）
- 猎头不宜同时委托超过3家，避免候选人被重复触达
- 校招渠道需提前2-3个月规划（与目标院校建立关系、确认宣讲排期）

## 约束条件
- 猎头合作需通过采购审批流程
- 招聘平台账号的发布内容需统一品牌形象
- 不得在渠道发布中包含歧视性信息
- 校招目标院校选择不得设置明确的院校歧视（如仅限985/211）
