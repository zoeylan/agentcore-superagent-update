# 面试安排与协调

## 技能概述
高效协调候选人与面试官的时间安排，确保候选人通过初筛后3个工作日内安排首轮面试，管理面试邀请发送、确认跟踪、改期处理等全部协调事务，最大化面试安排效率和候选人体验。

## 适用场景
- 候选人通过初筛后的面试安排
- 多轮面试的连续安排
- 面试改期/取消/重新安排
- 校招季批量面试排期
- 视频面试与现场面试的切换

## 执行步骤

### 第一步：面试时间协调
1. 查询面试官近期可用时间段
2. 联系候选人确认可参加面试的时间
3. 寻找双方最优匹配时段（优先候选人方便的时间）
4. 如3个工作日内无法匹配，启动备选面试官方案

### 第二步：面试邀请发送
1. 发送正式面试邀请（邮件+短信双通道）
2. 邀请内容包含：
   - 面试日期和时间
   - 面试地点/视频会议链接
   - 面试时长预估
   - 面试官姓名和职位
   - 需要准备的材料/内容
   - 公司地址和交通指引（现场面试）
   - HR联系方式（应急沟通）
3. 要求候选人24小时内确认出席

### 第三步：确认与提醒
1. 候选人确认后，同步通知面试官
2. 面试前1天发送提醒（候选人+面试官）
3. 面试当天提前1小时确认一切就绪
4. 未确认的候选人，通过电话再次联系

### 第四步：异常处理
| 异常情况 | 处理方案 |
|----------|----------|
| 候选人首次改期 | 尊重原因，3个工作日内重新安排 |
| 候选人二次改期 | 评估诚意，与用人经理确认是否继续 |
| 候选人爽约（有合理原因） | 给予一次重新安排机会 |
| 候选人爽约（无故缺席） | 标记放弃，通知相关方 |
| 面试官临时缺席 | 立即启动备选面试官，向候选人致歉 |
| 技术故障（视频面试） | 提供备选方案（电话面试/改期） |

### 第五步：面试后跟进
1. 面试结束后2小时内确认面试已完成
2. 提醒面试官24小时内提交评估反馈
3. 超48小时未提交的，升级提醒至面试官上级
4. 收集候选人面试体验反馈（可选）

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 候选人信息 | 结构化数据 | 是 | 姓名、联系方式、当前阶段 |
| 面试轮次 | 枚举 | 是 | 第几轮面试 |
| 面试官清单 | 列表 | 是 | 本轮需参与的面试官 |
| 面试形式 | 枚举 | 是 | 现场/视频/电话 |
| 面试时长 | 数字 | 是 | 预计面试时长（分钟） |

## 输出规范
- 面试安排确认单：时间、地点、参与者、准备事项
- 面试邀请函（候选人版+面试官版）
- 面试日程表（周度/月度面试安排总览）
- 异常处理记录

## 最佳实践
- 提供2-3个可选时间段让候选人选择，而非只给一个时间
- 面试间预留15-30分钟缓冲，避免连续面试的时间冲突
- 对于异地候选人优先提供视频面试选项，降低时间成本
- 面试安排确认后，在ATS系统中实时更新状态

## 约束条件
- 候选人通过初筛后3个工作日内必须安排首轮面试
- 面试时间尊重工作时间（9:00-18:00），特殊情况需候选人明确同意
- 面试邀请必须包含完整的信息，避免候选人需要反复询问
- 所有面试安排变更需留有书面记录
