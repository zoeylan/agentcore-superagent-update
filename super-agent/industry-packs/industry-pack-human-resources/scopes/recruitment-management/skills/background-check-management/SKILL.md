# 背景调查管理

## 技能概述
在候选人接受Offer后发起并管理背景调查全流程，通过合规的方式验证候选人提供信息的真实性，包括学历验证、工作经历确认、离职原因核实和必要的诚信检查，保障企业用人安全。

## 适用场景
- 社招候选人接受Offer后的标准背调
- 高管岗位的深度背调
- 财务/法务等敏感岗位的增强背调
- 背调结果异常时的风险评估和决策支持

## 执行步骤

### 第一步：背调授权获取
1. 向候选人说明背调的必要性和范围
2. 获取候选人签署的《背景调查授权书》
3. 授权书须明确：调查范围、信息用途、保密承诺
4. 确认候选人提供的证明人联系方式（至少2位前同事/上级）
5. 合规提醒：未获授权不得启动任何调查（《个人信息保护法》要求）

### 第二步：确定背调维度
根据岗位级别和敏感度确定背调深度：
| 岗位类型 | 背调维度 | 时效要求 |
|----------|----------|----------|
| 普通岗位 | 学历+最近1份工作经历 | 3个工作日 |
| 中层管理 | 学历+最近2份工作+离职原因 | 5个工作日 |
| 高管/敏感岗位 | 学历+全部工作+征信+诉讼记录 | 7个工作日 |

### 第三步：背调执行
1. **学历验证**：通过学信网验证学历学位真实性
2. **工作经历确认**：联系前公司HR或直属上级确认：
   - 在职时间是否属实
   - 职位和职责是否属实
   - 离职原因和离职方式
   - 工作表现评价（优势和不足）
3. **征信查询**（敏感岗位）：查询个人征信报告
4. **诉讼记录**（高管岗位）：通过企查查等工具查询涉诉情况
5. **竞业限制检查**：确认候选人是否处于竞业限制期

### 第四步：背调结果评估
| 结果分类 | 定义 | 处理方案 |
|----------|------|----------|
| 绿色（通过） | 信息全部属实，无异常 | 正常推进入职 |
| 黄色（关注） | 存在轻微偏差（如职位美化、时间±3个月） | 记录备注，与候选人确认 |
| 红色（异常） | 重大虚假信息（如学历造假、被辞退隐瞒） | 暂停入职，启动风险评估 |

### 第五步：异常处理
1. 发现轻微偏差：与候选人沟通确认，判断是否影响录用决策
2. 发现重大造假：
   - 暂停入职流程
   - 将情况如实通报用人部门负责人和HR总监
   - 评估撤回Offer的法律风险和操作方式
   - 如决定撤回，正式书面通知候选人并说明原因
3. 竞业限制风险：评估竞业限制的约束力和实际执行风险

### 第六步：背调归档
1. 完成背调报告撰写
2. 报告存档（保密管理，仅限HR内部查阅）
3. 通知Offer管理专员背调结果
4. 通过背调→推进入职时间确认和合同签署

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 候选人简历 | 文档 | 是 | 用于核对的基准信息 |
| 背调授权书 | 文档 | 是 | 候选人签署的授权文件 |
| 证明人联系方式 | 列表 | 是 | 至少2位前同事/上级 |
| 岗位敏感度 | 枚举 | 是 | 决定背调深度 |
| Offer信息 | 文档 | 否 | 用于核对薪资申报是否属实 |

## 输出规范
- 背景调查报告：各维度验证结果、总体结论（通过/关注/异常）
- 异常风险评估（如有）：问题描述、影响分析、处理建议
- 背调完成通知：告知相关方背调已完成及结论

## 最佳实践
- 背调应在Offer发出后尽快启动，不要等到入职前才做
- 证明人优先选择候选人的直属上级，而非同级同事
- 对于证明人的评价要辨别"客套话"和真实评价
- 背调是验证而非调查，不要超出授权范围刺探隐私

## 约束条件
- 必须先获得候选人书面授权才能启动背调
- 背调内容仅限与工作相关的信息，不得调查个人私生活
- 背调结果属于保密信息，不得向无关人员透露
- 背调过程中获取的第三方评价需匿名处理
- 背调结果不实导致撤回Offer时需有法务参与评估法律风险
