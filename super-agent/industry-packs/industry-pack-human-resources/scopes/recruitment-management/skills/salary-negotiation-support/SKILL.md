# 薪资谈判支持

## 技能概述
为招聘团队提供薪资谈判的策略支持和话术指导，处理候选人counter offer、竞对抢人、薪资期望超预算等复杂谈判场景，在维护企业薪酬体系完整性的同时最大化Offer接受率。

## 适用场景
- 候选人薪资期望超出预算上限
- 候选人收到竞对Offer要求匹配
- 候选人提出额外条件（签字费、特殊福利）
- 候选人犹豫不决需要推动决策
- 候选人拒绝Offer后的挽回尝试

## 执行步骤

### 第一步：谈判情景评估
1. 明确候选人的核心诉求：纯薪资驱动/发展空间/工作生活平衡/其他
2. 了解竞对Offer详情（如有）：总包对比、岗位级别、公司平台差异
3. 评估候选人的不可替代性：该候选人流失后的替代成本和时间
4. 确定企业的谈判底线和可让步空间

### 第二步：谈判策略制定
| 场景 | 策略 | 可用工具 |
|------|------|----------|
| 薪资差距≤10% | 直接匹配 | 调整基本薪资至区间内上限 |
| 薪资差距10-20% | 结构化弥补 | 签字费+绩效奖金提高+加速晋升承诺 |
| 薪资差距>20% | 综合价值陈述 | 长期激励+发展空间+平台价值叙事 |
| 竞对Offer | 差异化竞争 | 强调我方独特优势（非薪资维度） |
| 候选人犹豫 | 限时推动 | 设定回复期限+展示团队期待 |

### 第三步：谈判执行支持
1. 准备谈判话术要点（开场白、价值陈述、让步条件、收尾推动）
2. 设定让步节奏：不要一次性亮出全部底牌，分2-3轮逐步让步
3. 强调非薪资价值：团队氛围、技术挑战、成长速度、管理者风格
4. 建立"最后方案"的紧迫感：明确这是最终方案，超出需要走特批

### 第四步：升级处理
当标准谈判无法达成一致时：
1. 评估是否值得启动特批流程
2. 特批判断标准：候选人不可替代性高+岗位紧急度高+溢价幅度可控
3. 准备特批申请材料：候选人价值论述、市场稀缺性证据、业务影响评估
4. 特批路径：HR总监→CHRO→CEO（视超标幅度）

### 第五步：谈判结果处理
1. **接受**：确认薪酬方案无误，推进Offer letter出具
2. **拒绝**：记录拒绝原因，评估是否有挽回空间
3. **犹豫**：给予2-3天思考时间，期间保持适度跟进
4. 无论结果如何，保持专业和尊重，维护雇主品牌

## 输入规范
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 原始薪酬方案 | 文档 | 是 | 企业最初的方案 |
| 候选人诉求 | 文本 | 是 | 候选人的具体要求和关注点 |
| 竞对Offer信息 | 文本 | 否 | 竞争对手提供的方案 |
| 岗位紧急度 | 枚举 | 是 | 影响让步空间的决策因素 |
| 候选人不可替代性 | 枚举 | 是 | 高/中/低 |

## 输出规范
- 谈判策略方案：策略选择、话术要点、让步节奏
- 调整后的薪酬方案（如需调整）
- 特批申请材料（如需特批）
- 谈判结果记录和原因分析

## 最佳实践
- 谈判的目标是"双赢"而非"压价"，候选人入职后的满意度影响长期留任
- 不要承诺无法兑现的内容（如"半年后一定升职"），诚信是基础
- 适当展示诚意（如VP亲自致电表达重视）可以超越薪资本身的吸引力
- 拒绝时保持优雅，今天的候选人可能是未来的人才

## 约束条件
- 不得做出超出审批权限的薪资承诺
- 不得诋毁竞对公司来影响候选人决策
- 谈判记录需如实记录，不得虚报候选人诉求来争取内部预算
- 特批申请次数过多时需反思岗位薪资带宽的合理性
