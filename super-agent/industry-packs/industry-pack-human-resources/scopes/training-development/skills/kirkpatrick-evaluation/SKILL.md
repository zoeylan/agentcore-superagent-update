# 柯氏四级评估 (Kirkpatrick Evaluation)

## 技能概述

柯氏四级评估（Kirkpatrick Model）是全球企业培训领域最广泛使用的效果评估框架，通过四个递进层次（反应→学习→行为→结果）系统化衡量培训的实际价值。本技能要求掌握每个层级的评估方法论、工具设计和数据分析能力，能够科学地证明培训投入的业务回报。

### 适用场景
- 每次培训结束后的即时评估（Level 1&2）
- 训后1-3个月的行为转化追踪（Level 3）
- 重点培训项目的ROI深度分析（Level 4）
- 培训质量改进的数据依据收集
- 向管理层汇报培训价值时的数据支撑

## 详细执行步骤

### Level 1: 反应层评估 (Reaction)
1. 设计满意度问卷（推荐5-7个核心维度）：
   - 课程内容相关性
   - 讲师专业度和表现力
   - 教学方法有效性
   - 培训组织和后勤
   - 学习意愿提升度
   - 推荐意愿（NPS）
2. 培训结束当天即时收集（纸质或在线）
3. 回收率要求≥95%
4. 目标评分≥4.2/5.0
5. 48小时内完成数据统计和初步分析
6. 评分低于3.5的维度标记为改进重点

### Level 2: 学习层评估 (Learning)
1. 根据课程类型选择评估方式：
   - 知识类：笔试/在线测试
   - 技能类：实操考核/模拟演练
   - 态度类：情景判断测验/案例分析
2. 设计前测-后测对比（如条件允许）
3. 在培训结束时或培训后1周内完成
4. 目标通过率≥85%
5. 未通过者安排补学/复训机会
6. 分析知识点掌握的薄弱环节，反馈给课程开发

### Level 3: 行为层评估 (Behavior)
1. 训后1-3个月实施
2. 评估方法组合：
   - 上级观察和评价（结构化访谈）
   - 工作成果/产出对比（训前vs训后）
   - 同事/客户反馈收集
   - 学员自评行为变化
3. 抽样覆盖≥30%参训人员
4. 目标行为改变率≥60%
5. 行为未变化时的原因分析：
   - 学习未消化 → 安排复训/辅导
   - 工作环境不支持 → 向管理层建议环境改善
   - 培训内容不实用 → 反馈给课程开发优化
6. 制定针对性的后续干预措施

### Level 4: 结果层评估 (Results/ROI)
1. 训后3-6个月实施
2. 确定与培训目标相关的业务指标：
   - 生产效率指标（产量、质量、速度）
   - 客户满意度指标
   - 财务指标（收入、成本、利润）
   - 人员指标（离职率、晋升率）
3. 计算培训ROI：
   - ROI = (培训带来的收益 - 培训总成本) / 培训总成本 × 100%
4. 隔离培训效果（排除其他变量影响）：
   - 控制组对比法
   - 趋势线分析法
   - 专家估算法
5. 每年至少对3个重点项目进行完整ROI分析
6. 编制ROI分析报告提交管理层

## 输入规格
| 输入项 | 来源 | 格式要求 |
|--------|------|----------|
| 培训项目信息 | 课程开发专员 | 学习目标、参训名单、课程内容 |
| 问卷/测评工具 | 自行设计 | 标准化评估工具 |
| 业务绩效数据 | 业务系统/绩效模块 | 培训相关的KPI数据 |
| 培训成本数据 | 预算执行记录 | 直接成本+间接成本 |

## 输出规格
| 输出项 | 格式 | 接收方 |
|--------|------|--------|
| Level 1评估报告 | 数据报表+改进建议 | 讲师/课程开发专员 |
| Level 2评估报告 | 通过率+薄弱点分析 | 课程开发专员 |
| Level 3评估报告 | 行为变化分析+干预建议 | 培训需求分析师/管理层 |
| Level 4 ROI报告 | 投资回报分析 | HR总监/管理层 |
| 综合评估总结 | 年度培训效果白皮书 | 全公司 |

## 最佳实践
- 不是所有培训都需要做到Level 4，根据投入规模和战略重要性选择评估深度
- Level 1满意度高不代表培训有效，要警惕"happy sheet"陷阱
- Level 3的行为评估需要上级配合，提前与管理者沟通评估计划
- ROI计算要保守估计培训贡献，确保数据可信度
- 评估发现的问题要快速闭环——反馈给课程开发或需求分析进行改进

## 约束条件
- Level 1和Level 2回收率≥95%
- Level 3抽样覆盖≥30%参训人员
- Level 4每年至少完成3个重点项目的ROI分析
- 评估结果48小时内完成初步统计
- Level 3评估需获得被评估人员直属上级的配合承诺
- 评估数据保密管理，个人层面数据不对外公开
