# 年度培训计划制定 (Annual Training Planning)

## 技能概述

年度培训计划制定是将培训需求诊断结果转化为可执行的系统性规划方案的关键技能。它涵盖课程体系架构设计、培训日历编排、资源配置方案、以及与组织目标对齐的效果预期设定。一份高质量的年度培训计划是培训部门全年工作的行动纲领和资源分配依据。

### 适用场景
- 每年Q4制定次年年度培训计划
- 年中因重大业务调整需修订计划
- 新事业部/分公司成立需制定专项培训计划
- HR总监要求的培训规划汇报

## 详细执行步骤

### Step 1: 需求整合与分类
1. 汇总培训需求诊断的输出结果
2. 按培训类型分类：新员工培训、专业技能培训、管理能力培训、领导力发展
3. 按岗位序列分类：技术序列、业务序列、职能序列、管理序列
4. 按紧急程度分类：必修（合规/上岗必备）、重点（战略关联）、选修（发展性）

### Step 2: 课程体系架构设计
1. 建立分层课程体系（基础层→进阶层→高级层→专家层）
2. 为每个岗位序列设计学习路径图（Learning Path）
3. 明确每门课程的定位：内部开发 vs 外部采购
4. 设计课程间的前置关系和推荐学习顺序
5. 标记管理岗晋升必修课程（领导力系列）

### Step 3: 培训日历编排
1. 避开业务高峰期（如销售Q4冲刺期、财务年终结算期）
2. 必修课程优先排期，确保全员可参加
3. 新员工培训按月度入职人数规划开班频率
4. 领导力项目按学员群体的时间可用性集中安排
5. 线上课程保持常年可访问，线下面授按季度滚动排期

### Step 4: 资源需求规划
1. 内部讲师工作量预估和分配（避免单一讲师过度负荷）
2. 外部讲师/培训机构采购需求清单
3. 场地和设备需求（培训教室、会议室、在线平台许可）
4. 预算需求总体测算（详见培训预算分配技能）

### Step 5: 计划审批与发布
1. 编制年度培训计划书（含执行摘要、详细计划、预算概览）
2. 提交HR总监审批（次年1月中旬前完成）
3. 根据审批意见修订（常见：预算削减→削减低优先级项目）
4. 审批通过后向全公司发布培训日历

## 输入规格
| 输入项 | 来源 | 格式要求 |
|--------|------|----------|
| 培训需求优先级清单 | 需求诊断输出 | 分类排序表格 |
| 上年度培训执行总结 | 培训效果评估师 | 含改进建议的分析报告 |
| 次年业务规划 | 业务部门 | 关键业务节奏信息 |
| 人员编制计划 | HR规划 | 各部门增员/调整计划 |
| 可用预算范围 | 财务/HR总监 | 预算框架 |

## 输出规格
| 输出项 | 格式 | 接收方 |
|--------|------|--------|
| 年度培训计划书 | 正式文档（含附件） | HR总监审批 |
| 培训日历 | 日历/甘特图 | 全公司公示 |
| 课程体系架构图 | 可视化图表 | 课程开发专员执行 |
| 资源需求清单 | 明细表格 | 采购/行政/IT |

## 最佳实践
- 计划需留有10-15%的弹性空间应对临时需求
- 将培训计划与公司业务日历对齐，避免冲突
- 明确每个培训项目的成功标准和评估方式（在规划阶段就确定）
- 高频低时长的微学习项目可以补充正式培训的覆盖盲区
- 年中需进行一次计划执行回顾，根据实际情况微调下半年安排

## 约束条件
- 计划必须确保全员年度培训学时≥40学时
- 管理岗必修课程必须在计划中明确标注
- 培训预算不超过人力成本的2-3%（除非获得特批）
- 次年1月中旬为最终审批截止日
- 计划变更超过原计划20%需重新报批
