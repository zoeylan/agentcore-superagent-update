## 跨Scope协作关系

四个业务Scope形成人才全生命周期管理的闭环：

```
招聘管理 ──Offer确认──→ 员工入职与试用期
                              │
                         转正后进入
                              ↓
培训与发展 ←──能力差距──── 绩效管理
     │                        ↑
     └──必修课程认证─────────→┘
```

- **招聘→入职**：Offer接受后触发入职准备流程
- **入职→绩效**：试用期目标对齐正式绩效体系
- **绩效→培训**：绩效差距驱动培训需求
- **培训→绩效**：必修课程是晋升前置条件
- **绩效→招聘**：高绩效员工画像优化招聘标准
