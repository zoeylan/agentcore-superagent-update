# 薪酬体系设计与优化

## 技能描述
提供全面薪酬体系（Total Rewards）的设计、诊断与优化建议，涵盖岗位价值评估、薪酬结构设计、宽带薪酬搭建、短期激励（奖金）与长期激励（股权/期权）方案设计、薪酬竞争力分析及调薪策略。

## 触发场景
- 用户需要搭建或重构薪酬等级体系
- 用户需要设计年度调薪方案或奖金分配方案
- 用户需要评估薪酬竞争力并制定薪酬策略
- 用户需要设计股权激励/期权计划
- 用户面临薪酬公平性投诉或薪酬倒挂问题

## 输出格式

### 薪酬体系设计方案结构：
1. **薪酬策略定位** — 领先型/跟随型/混合型，对标市场分位值（P50/P75/P90）
2. **岗位价值评估** — 选择方法论（Hay/Mercer IPE/自研因素计分法），评估维度与权重
3. **薪酬结构设计** — 等级数量、带宽（Bandwidth）、重叠度（Overlap）、中位值级差（Midpoint Progression）
4. **薪酬组合比例** — 固浮比设计逻辑（按层级/按职能差异化）
5. **调薪机制** — 年度普调 + 晋升调薪 + 特殊调薪的触发条件与预算分配
6. **实施路线图** — 分阶段过渡方案，含red circle/green circle处理策略

### 激励方案设计结构：
1. **激励目标与对象** — 覆盖范围、分层逻辑
2. **激励模式选择** — 现金奖金/限制性股票(RSU)/期权/虚拟股/利润分享
3. **授予与归属规则** — Vesting schedule、绩效条件、退出机制
4. **测算模型** — 不同业绩场景下的支付水平模拟
5. **税务与合规** — 个税处理、股份支付会计处理、工商变更要点

## 关键规则
- 岗位价值评估必须"对岗不对人"，避免因人设岗导致体系扭曲
- 宽带薪酬带宽建议控制在40%-60%，过宽则失去管理意义
- 中位值级差（相邻职级间中位值差距）通常在15%-25%之间
- 薪酬数据对标需注意口径一致性：总现金（TC）vs 总薪酬包（TP）vs 基本工资
- 股权激励方案需注意《公司法》、《证券法》相关限制，上市公司需遵守证监会/交易所规则
- 调薪预算分配切忌"撒胡椒面"，应向关键岗位、高绩效人才倾斜
- 注意同工同酬原则，派遣员工与正式员工的薪酬差异需有合理岗位价值依据
- 年终奖发放规则需在制度中明确约定，避免"13薪是否属于工资"的争议
