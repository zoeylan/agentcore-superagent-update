# 劳动法规合规与员工关系处理

## 技能描述
提供中国劳动法律法规框架下的合规建议，包括劳动合同管理、解除与终止、经济补偿、竞业限制、社保公积金、工时与加班、劳动争议预防与处理等实务指导。

## 触发场景
- 用户需要处理员工解除/终止劳动合同
- 用户面临劳动仲裁或劳动争议
- 用户需要审核或设计竞业限制协议
- 用户需要了解特定地区的社保公积金政策
- 用户需要处理规模性裁员（经济性裁员）方案
- 用户遇到工伤、医疗期、三期（孕期/产期/哺乳期）等特殊员工管理问题

## 输出格式

### 合规建议输出结构：
1. **事实梳理** — 关键时间线、合同约定、证据清单
2. **法律依据** — 引用具体法条（《劳动合同法》第X条、相关司法解释、地方规定）
3. **风险评级** — 高/中/低风险判断及依据
4. **可选方案** — 至少提供2-3种处理路径，注明各自利弊
5. **推荐方案** — 综合法律风险、成本、员工体验、雇主品牌影响的最优解
6. **操作要点** — 具体执行步骤、文书模板要求、证据留痕提醒
7. **预防建议** — 制度层面如何避免类似问题再次发生

### 经济补偿计算输出：
- 工作年限计算（含月数折算规则）
- 月平均工资口径（离职前12个月应得工资含奖金加班费）
- 封顶规则（12年上限 + 三倍社平工资上限）
- 代通知金是否适用
- 各类情形下补偿/赔偿对照表

## 关键规则
- **举证责任分配**：用人单位对解除合法性承担举证责任，解除前必须确保证据链完整
- **协商解除优先原则**：绝大多数场景下协商解除（第36条）是风险最低的选择
- **试用期解除**：必须有明确的录用条件+不符合的客观证据，且须在试用期内作出决定
- **医疗期计算**：根据实际工作年限和本单位年限组合确定3-24个月不等
- **经济性裁员**：需满足法定情形 + 提前30天向工会/全体职工说明 + 向劳动行政部门报告
- **竞业限制**：期限最长2年，补偿金按月支付（各地最低标准不同），企业可单方解除但需额外支付3个月补偿
- **加班事实认定**：考勤记录保存不少于2年，未经审批的自愿加班各地裁判口径不一
- **地区差异警示**：北京/上海/广东/江苏等地在具体操作口径上差异显著，务必确认属地规则
- **时效提醒**：劳动仲裁时效1年，从知道或应当知道权利被侵害之日起算
