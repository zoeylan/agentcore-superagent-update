# 外呼话术生成技能 (Outreach Script Generation)

## 概述

基于线索的行业背景、联系人职位、行为信号和痛点假设，为SDR自动生成个性化的多渠道沟通话术（电话开场白、邮件模板、微信消息），确保每次触达都有针对性的价值主张。个性化话术相比通用模板可提升接通后转化率30-50%。

## 适用场景

- 新分配MQL线索的首次触达话术准备
- 后续跟进沟通的话术迭代（基于上次沟通反馈）
- 特定行业/场景的话术批量生成
- SDR新人的话术培训素材准备
- A/B测试不同话术效果对比

## 执行步骤

### 步骤1：线索背景分析
深度分析线索可用信息：
1. 企业信息：行业特征、发展阶段、近期动态（融资/扩张/裁员/上市）
2. 联系人画像：职位层级、关注焦点（技术人关注功能/业务人关注ROI/管理人关注效率）
3. 行为信号：下载了什么内容、参加了什么活动、浏览了哪些页面
4. 痛点假设：基于行业+规模+阶段推断可能的业务挑战

### 步骤2：价值主张定制
基于分析结果，确定核心沟通策略：
1. 选择最相关的价值主张角度（降本/增效/合规/竞争优势）
2. 准备行业案例（同行业/同规模客户的成功故事）
3. 设计钩子（Hook）——引起兴趣的关键一句话
4. 明确Call-to-Action（下一步动作邀请）

### 步骤3：多渠道话术生成

**电话开场白**（30秒版本）：
- 自我介绍（简短）
- 连接点（为什么打给对方——基于行为信号或行业洞察）
- 价值钩子（一句话点明可带来的价值）
- 确认时间可用性
- 开放式提问引出痛点

**邮件模板**（3段式）：
- 主题行：个性化+价值预告（目标打开率>30%）
- 正文第一段：建立相关性（提及对方的行业/角色/动态）
- 正文第二段：价值主张+社会证明（同行案例数据）
- 正文第三段：轻量级CTA（非压迫性的下一步邀请）

**微信/企业微信消息**：
- 简短友好，非正式语气
- 提供即时价值（行业报告/工具/洞察）
- 适合中国B2B场景的沟通风格

### 步骤4：异议预案准备
针对常见拒绝场景准备应答：
- "我们已有供应商" → 差异化价值强调 + 建议性对比了解
- "暂时不需要" → 行业趋势切入 + 轻度保持联系
- "没有预算" → 价值量化 + 灵活方案 + 下一财年规划
- "发个资料看看" → 约定回访时间 + 定制化资料

### 步骤5：输出与迭代
1. 生成完整话术包（电话+邮件+微信）
2. 标注使用场景和最佳发送时间
3. 记录话术使用效果，为后续迭代提供数据

## 输入规格

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| lead_profile | Object | 是 | 线索完整画像（企业+联系人+行为） |
| product_context | Object | 是 | 产品/解决方案信息 |
| channel | String | 是 | 目标渠道（phone/email/wechat/all） |
| attempt_number | Number | 否 | 第几次触达（影响话术策略），默认1 |
| previous_interaction | Object | 否 | 上次沟通记录（用于迭代话术） |

## 输出规格

| 字段 | 类型 | 说明 |
|------|------|------|
| phone_script | Object | 电话话术（开场白+提问框架+异议应答） |
| email_template | Object | 邮件模板（主题行+正文+CTA） |
| wechat_message | Object | 微信消息模板 |
| objection_handlers | Array | 异议处理话术列表 |
| best_timing | Object | 建议触达时间窗口 |
| personalization_notes | String | 个性化要点说明 |

## 最佳实践

1. **真实感优先**：话术要像真人在说话，避免机械化和模板化痕迹
2. **价值导向**：每次触达必须为对方提供某种价值（信息/洞察/工具），而非纯推销
3. **渐进式深入**：首次触达轻度接触（了解需求），后续逐步深入（方案探讨），不要首次就过于aggressive
4. **本地化适配**：中国B2B场景下，微信是核心渠道，话术语气需符合微信沟通习惯
5. **数据驱动迭代**：追踪各话术版本的接通率/回复率/转化率，持续优化high-performer话术

## 约束条件

- 话术不可包含虚假承诺或夸大产品能力的内容
- 不可冒充客户的同事、合作伙伴或熟人进行欺骗性开场
- 邮件必须包含退订选项，符合反垃圾邮件法规
- 电话外呼需遵守当地通信管理规定（如非工作时间禁止外呼）
- 话术中不可使用攻击竞品的不实言论
