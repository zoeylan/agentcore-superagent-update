# 定价计算技能

## 概述
定价计算技能负责根据产品配置方案执行精准的价格计算，包括标准定价查询、批量折扣阶梯计算、捆绑优惠规则应用、年框协议价格适配，以及折扣权限判断和审批路由决策。

## 适用场景
- 标准产品的目录价查询和总价计算
- 批量采购的阶梯折扣自动计算
- 多产品捆绑购买的组合优惠计算
- 年框协议客户的协议价适配
- 折扣幅度判断和审批层级路由
- 竞品报价对标的价格竞争力分析

## 执行步骤

### 步骤1：基础价格查询
- 从价格本中查询各SKU的标准目录价（List Price）
- 确认价格版本有效性（价格调整后旧版本报价需标注）
- 区分一次性费用和经常性费用（订阅/年费）
- 计算产品明细小计和总计

### 步骤2：折扣规则应用
- 检查是否适用批量折扣阶梯（如：10-49用户95折，50-199用户9折，200+用户85折）
- 检查是否适用捆绑优惠（如：A+B+C组合购买额外优惠5%）
- 检查客户是否有年框协议价格
- 检查是否有限时促销活动价格
- 多重折扣规则的优先级处理（不叠加，取最优）

### 步骤3：折扣权限判断
- 计算最终折扣率 = (目录价 - 报价) / 目录价 × 100%
- 判断折扣率所属审批层级：
  - ≤10%：销售经理审批
  - ≤20%：销售总监审批
  - ≤30%：VP审批
  - >30%：CEO审批
- 输出审批路由建议和预计审批时间

### 步骤4：利润率计算
- 计算成本价（产品成本+实施成本+支持成本）
- 计算毛利率 = (报价 - 成本) / 报价 × 100%
- 与利润率红线对比（通常毛利率不低于60%）
- 利润率低于红线时生成预警

### 步骤5：税费和总价计算
- 根据合同主体和交易类型确定税率
- 计算含税价和不含税价
- 如有多期付款，计算各期金额

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| productConfiguration | Array | 是 | 产品配置清单（SKU、数量） |
| customerType | String | 是 | 客户类型（新客户/续约/年框） |
| requestedDiscount | Number | 否 | 销售申请的折扣率 |
| frameworkAgreementId | String | 否 | 年框协议编号 |
| paymentTerms | String | 否 | 付款条件 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| listPrice | Number | 标准目录总价 |
| discountDetails | Object | 折扣明细（类型、比例、金额） |
| netPrice | Number | 折后净价 |
| taxAmount | Number | 税费金额 |
| totalPrice | Number | 含税总价 |
| grossMargin | Number | 毛利率 |
| discountApprovalLevel | String | 折扣审批层级 |
| marginAlert | Boolean | 是否触发利润率预警 |

## 最佳实践
1. **价格一致性**：同一客户同一时期的报价必须使用相同价格本版本
2. **折扣透明化**：报价单上分别展示目录价和折扣金额，让客户感知优惠力度
3. **竞品价格参考**：在审批决策包中附加竞品价格区间，辅助审批人判断
4. **历史价格对比**：老客户续约时参考历史购买价格，避免涨幅过大引发不满
5. **多年合同激励**：鼓励多年期合同，通过年限折扣（2年额外5%/3年额外10%）锁定客户

## 约束条件
- 价格本每季度审核一次，价格调整需提前30天通知已报价客户
- 折扣叠加规则：批量折扣与促销折扣不可叠加，取最优
- 年框协议价格优先于其他折扣规则
- 利润率红线以下的报价需VP层级特批
- 所有计算结果精确到分（两位小数）
