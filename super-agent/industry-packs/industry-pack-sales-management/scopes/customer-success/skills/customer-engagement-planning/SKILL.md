# 客户互动规划 (Customer Engagement Planning)

## 概述
为每个客户制定差异化的互动节奏和沟通计划，确保客户关系保持活跃和深入。合理的互动规划是预防客户流失的关键手段——研究表明，与CSM保持规律互动的客户续约率比低互动客户高出30%以上。

## 适用场景
- 新客户上线后的互动节奏初始化
- 季度互动计划制定和更新
- 客户层级变化（升级/降级）后的节奏调整
- 健康度变色后的互动频率调整
- 年度客户关怀计划制定

## 执行步骤

### 步骤1：客户分层与互动标准设定
- **动作**：
  - 按客户价值和战略重要性分层：
    - **Tier 1（战略大客户）**：ARR>100万或行业标杆客户
      - 月度check-in + 季度QBR + 半年高层互访
      - CSM 1:1 专属服务
    - **Tier 2（重要客户）**：ARR 30-100万
      - 月度check-in + 半年度QBR
      - CSM 1:15 服务配比
    - **Tier 3（标准客户）**：ARR<30万
      - 双月check-in + 年度review
      - CSM 1:50 服务配比（偏自动化触达）
  - 根据健康度颜色动态调整频率
- **输出**：客户分层表、互动标准规范

### 步骤2：制定月度互动日历
- **动作**：
  - 汇总当月所有客户的计划互动事件
  - 安排QBR准备时间（提前2周启动）
  - 标注高层互访时间窗口
  - 预留应急互动时间（客户突发需求/投诉）
  - 平衡CSM工作负载，避免互动集中在月末
- **输出**：月度互动日历

### 步骤3：互动内容准备
- **动作**：
  - **日常check-in**：准备使用数据摘要、近期产品更新、行业资讯
  - **QBR**：准备完整回顾材料（详见QBR准备技能）
  - **高层互访**：准备executive briefing（公司战略+合作价值+未来规划）
  - **产品更新通知**：个性化推送与客户相关的功能更新
  - **行业洞察分享**：推送客户所在行业的趋势报告和最佳实践
- **输出**：互动内容素材包

### 步骤4：执行与记录
- **动作**：
  - 按日历执行各类互动活动
  - 每次互动后在CRM中记录：
    - 沟通时间、参与人、沟通方式
    - 关键讨论内容和客户情绪
    - 客户提出的需求或问题
    - 约定的下一步行动和时间
  - 跟踪action item完成情况
- **输出**：互动记录、action item清单

### 步骤5：效果评估与优化
- **动作**：
  - 月末统计互动计划完成率（目标>90%）
  - 分析互动与客户健康度的关联性
  - 收集客户对互动频率和内容的反馈
  - 优化下月互动计划
- **输出**：互动效果分析报告

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 客户列表 | array | 是 | CSM管理的客户组合 |
| 客户分层 | object | 是 | 每个客户的Tier等级 |
| 健康度状态 | object | 是 | 每个客户的当前健康度颜色 |
| CSM日程 | object | 是 | CSM可用时间和已有安排 |
| 产品更新计划 | object | 否 | 近期产品发布路线图 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| 互动日历 | document | 月度/季度互动排期 |
| 互动标准 | document | 分层互动频率和内容规范 |
| 计划完成率 | number | 互动计划执行完成百分比 |
| 优化建议 | document | 基于效果分析的计划优化 |

## 最佳实践
1. **质量>数量**：每次互动都要有明确目的和为客户带来的价值
2. **个性化**：避免群发模板化内容，根据客户业务定制沟通内容
3. **多渠道组合**：电话+邮件+微信+现场拜访结合，适应客户偏好
4. **留白**：不要占满客户时间，过度互动反而产生骚扰感
5. **记录是财富**：详细的互动记录是CSM交接和团队协作的基础

## 约束条件
- Tier 1客户QBR不可缺席或取消，需要提前2周确认
- 高层互访需经CSM总监审批并准备briefing
- 客户明确表示不希望被打扰时需尊重（调整为低频邮件触达）
- 互动记录必须在互动后24小时内完成录入
