# QBR季度业务回顾准备 (QBR Preparation)

## 概述
为客户季度业务回顾会议（Quarterly Business Review）准备高质量的会议材料和议程，确保QBR成为展示价值、深化关系、发现机会的战略性互动，而非流于形式的例行会议。QBR是CSM展示专业度和客户获取洞察的最重要场景。

## 适用场景
- Tier 1客户每季度一次的QBR
- Tier 2客户半年度的业务回顾
- 续约前的年度总结回顾
- 客户高层变更后的重新介绍会议

## 执行步骤

### 步骤1：启动准备（QBR前2周）
- **动作**：
  - 确认QBR时间和参会人员（客户侧需有决策层参加）
  - 明确本次QBR的核心目标：
    - 价值展示（常规目标）
    - 关系深化（新高层需求）
    - 续约铺垫（合同到期前1-2个季度）
    - 增购探索（使用数据表现好的客户）
  - 收集所需数据和素材
  - 了解客户近期业务动态和关注点
- **输出**：QBR准备检查清单

### 步骤2：数据采集与分析
- **动作**：
  - **使用数据**：本季度登录次数、活跃用户数、功能使用分布、对比上季度变化
  - **价值量化**：通过产品帮助客户节省的时间/成本/效率提升数据
  - **服务记录**：本季度工单数量、解决率、满意度
  - **行业对比**：客户使用深度与同行业标杆对比（脱敏数据）
  - **目标达成**：对比上季度QBR设定的目标，评估达成率
- **输出**：QBR数据分析包

### 步骤3：材料制作
- **动作**：
  - 制作QBR演示材料（PPT），标准结构：
    1. **执行摘要**（1页）：本季度关键数据和核心结论
    2. **价值交付回顾**（2-3页）：使用数据→业务成果→ROI
    3. **服务回顾**（1页）：服务质量指标和改进措施
    4. **上季度行动项闭环**（1页）：对比约定事项的完成情况
    5. **下季度规划**（1-2页）：推荐的优化方向和新功能adoption
    6. **行业洞察**（1页）：行业趋势和同行实践分享
    7. **讨论议题**（1页）：需要客户输入的开放式议题
  - 确保PPT风格专业、数据可视化清晰
  - 准备针对可能问题的FAQ
- **输出**：QBR PPT材料、FAQ准备

### 步骤4：内部预审
- **动作**：
  - 与CSM经理review QBR材料
  - 确认关键数据准确性
  - 对齐QBR中可能涉及的商务话题（如续约/增购）的口径
  - 如有VP参加，准备简版executive brief
- **输出**：预审通过的最终版材料

### 步骤5：QBR执行
- **动作**：
  - 按议程主导会议流程（控制在60-90分钟）
  - 重点展示业务成果而非产品功能
  - 记录客户反馈、新需求、关切点
  - 与客户共同确认下季度目标和行动项
  - 会议结束确认下次QBR时间
- **输出**：QBR会议纪要

### 步骤6：会后跟进
- **动作**：
  - 24小时内发送会议纪要和承诺行动项
  - 将客户新需求录入产品反馈系统
  - 将新发现的机会信号传递给续约增购策略师
  - 更新客户互动记录和健康度参考信息
  - 设定action item跟踪提醒
- **输出**：跟进邮件、action item跟踪表

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 客户ID | string | 是 | 客户唯一标识 |
| QBR类型 | enum | 是 | 常规/续约铺垫/增购探索 |
| 参会人员 | array | 是 | 客户侧和我方参会人列表 |
| 上次QBR行动项 | array | 否 | 上次约定的行动项及完成状态 |
| 本季度使用数据 | object | 是 | 产品使用统计数据 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| QBR PPT | document | 完整演示材料 |
| 会议纪要 | document | QBR会议记录和决议 |
| 行动项清单 | array | 双方承诺的后续行动 |
| 机会信号 | array | 会议中识别的增购/升级信号 |

## 最佳实践
1. **客户视角**：所有数据和洞察从客户业务目标出发，不是产品功能秀
2. **不回避问题**：如果有未解决的issue，主动提出并展示解决计划
3. **互动而非单讲**：预留30%时间给客户提问和讨论
4. **层级匹配**：确保我方参会人层级与客户方对等
5. **节奏控制**：每个section有严格时间控制，不要拖堂

## 约束条件
- QBR材料需提前3天完成内部审核
- 涉及竞品对比数据需经市场部确认后使用
- 客户敏感业务数据在PPT中需脱敏处理
- QBR不是续约谈判会议，商务议题适度带出但不深入
