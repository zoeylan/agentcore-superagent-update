# NPS满意度调研管理 (NPS Survey Management)

## 概述
执行和管理NPS（Net Promoter Score，净推荐值）满意度调研的全流程，从问卷设计、发送执行、结果回收到分析报告和改进跟踪。NPS是衡量客户忠诚度的核心指标，也是健康度评分的重要输入维度。行业目标：NPS≥40分为优秀，响应率>40%确保统计有效性。

## 适用场景
- 半年度全量NPS调查执行
- 关键里程碑后的触发式NPS收集（如上线后30天、续约后）
- NPS分数下滑后的专项深度调研
- 新产品/新功能发布后的用户满意度采集

## 执行步骤

### 步骤1：调研设计与准备
- **动作**：
  - 确定调研范围：全量客户 vs 特定群体
  - 设计问卷结构：
    - 核心NPS问题："您有多大可能性向同事或朋友推荐我们？（0-10分）"
    - 开放式追问："您给出这个分数的主要原因是什么？"
    - 可选补充维度：产品满意度、服务满意度、价值感知
  - 确定发送渠道（邮件/产品内弹窗/微信）
  - 准备个性化邀请文案（不同Tier客户使用不同语气）
  - 设定调研周期（通常开放2周收集）
- **输出**：调研方案、问卷模板

### 步骤2：执行发送
- **动作**：
  - 按客户分层分批发送（避免同时大量发送影响响应率）
  - Tier 1客户：CSM个人发送+电话提醒
  - Tier 2客户：CSM个人邮件发送
  - Tier 3客户：系统自动发送+产品内提醒
  - 发送后3天对未响应客户进行一次友好提醒
  - 实时监控响应率，如低于预期及时调整策略
- **输出**：发送记录、响应率追踪表

### 步骤3：数据回收与分析
- **动作**：
  - 回收截止后汇总所有响应数据
  - 计算NPS分数：(Promoter比例 - Detractor比例) × 100
    - Promoter（推荐者）：9-10分
    - Passive（中立者）：7-8分
    - Detractor（贬损者）：0-6分
  - 多维度分析：
    - 按客户规模/行业/CSM/产品线细分
    - 与上次调研对比趋势
    - 开放式反馈的主题聚类分析
  - 识别高风险detractor和关键verbatim
- **输出**：NPS分析报告

### 步骤4：跟进行动
- **动作**：
  - **Detractor（0-6分）**：48小时内由CSM经理安排回访
    - 了解具体不满原因
    - 制定改善行动计划
    - 2周后跟进改善进展
  - **NPS<40的客户**：制定针对性的满意度提升计划
  - **Promoter（9-10分）**：请求推荐/案例/评价
  - 汇总产品反馈传递给产品团队
  - 汇总服务反馈用于CSM团队改善
- **输出**：跟进行动计划、产品反馈汇总

### 步骤5：结果同步与闭环
- **动作**：
  - 将NPS分数更新到客户健康度评分模型
  - 生成管理层报告（整体NPS趋势、关键洞察、改进进展）
  - 向参与调研的客户发送感谢和"您的反馈我们如何改进"的闭环通知
  - 记录本次调研经验，优化下次执行
- **输出**：健康度更新记录、管理层报告、客户闭环通知

## 输入规格
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 调研类型 | enum | 是 | 半年度全量/触发式/专项 |
| 目标客户范围 | array | 是 | 参与调研的客户清单 |
| 上次NPS数据 | object | 否 | 上次调研的基线数据 |
| 发送渠道 | enum | 是 | 邮件/产品内/微信/组合 |

## 输出规格
| 字段 | 类型 | 说明 |
|------|------|------|
| NPS分数 | number | 整体NPS值(-100到100) |
| 响应率 | number | 有效响应数/发送总数 |
| 分群分析 | object | Promoter/Passive/Detractor分布 |
| 改进行动计划 | document | 基于反馈的具体改进措施 |
| 产品反馈汇总 | document | 传递给产品团队的需求汇总 |

## 最佳实践
1. **简短是王道**：核心问卷控制在3个问题以内，完成时间<2分钟
2. **时机选择**：避免在客户有未解决问题时发送调研（会拉低分数且无新信息）
3. **闭环最重要**：客户愿意给反馈是因为相信反馈有用，必须告知改进行动
4. **不cherry-pick**：不能只向满意客户发送调研来人为提高NPS
5. **趋势>绝对值**：单次NPS分数波动正常，关注持续趋势更有价值

## 约束条件
- 半年度调研发送间隔不少于5个月，避免调研疲劳
- Detractor回访48小时SLA是硬性要求
- NPS数据不对外公开，内部使用需标注统计口径
- 调研过程中不得附带任何营销或促销内容
