# 收入预测建模

## 技能概述

本技能用于生成准确、可靠的季度和年度收入预测，综合运用加权管线法(Stage-Weighted Pipeline)、历史趋势法和AI修正模型三种预测方法论，输出含置信区间的预测报告。收入预测是CEO和CFO最关注的销售管理输出之一，直接影响公司的招聘计划、市场投入预算和产品研发优先级决策。

## 适用场景

- 每月收入预测更新（持续收窄置信区间）
- 季度首月10号前的正式季度预测报告
- 管理层要求的临时预测刷新（如大单变动后）
- 年度规划时的全年收入预测
- 预测偏差超过±10%后的模型调整和重新预测

## 执行步骤

### 步骤1：数据准备与清洗
1. 获取当前季度所有活跃商机的完整信息（金额、阶段、预计关单日期、赢率、负责人）
2. 获取已签约但在本季度确认收入的合同数据
3. 获取过去4-8个季度的历史业绩数据（用于趋势分析）
4. 清洗数据：排除明显异常值（如金额为0或明显虚报的商机）
5. 标记"高置信度"商机（口头承诺或合同在途）和"低置信度"商机（早期阶段）

### 步骤2：加权管线法预测
1. 为每个商机分配阶段赢率权重：
   - 需求确认：15%
   - 方案演示：30%
   - 报价阶段：50%
   - 商务谈判：70%
   - 口头承诺：90%
   - 合同签署中：95%
2. 计算加权金额 = Σ(商机金额 × 阶段权重)
3. 对预计关单日期在本季度内的商机进行汇总
4. 输出加权管线预测值

### 步骤3：历史趋势法预测
1. 分析过去4-8个季度的实际签约数据
2. 计算季度增长率（CAGR）和季节性系数
3. 识别增长/衰退趋势并外推
4. 考虑团队规模变化的影响（人均产出 × 当前团队人数）
5. 输出趋势预测值

### 步骤4：AI修正与综合预测
1. 对比两种方法的预测结果差异
2. 应用修正因子：
   - 销售人员历史承诺兑现率（识别habitually over-commit或sandbagging的个人）
   - 大单集中度风险（如预测高度依赖少数几个大单）
   - 季度末冲刺效应系数（历史上最后一个月签约占比）
   - 宏观环境修正（如经济下行期需降低乐观预测）
3. 生成综合预测值及置信区间（通常为±10-15%）

### 步骤5：风险标注与敏感性分析
1. 识别对预测影响最大的TOP 5商机（单一商机占预测>10%即标记为集中度风险）
2. 进行场景分析：Best Case / Most Likely / Worst Case
3. 标注关键风险因素（如某大单决策人变动、竞品压价等）
4. 计算预测的上行空间（upside）和下行风险（downside）

### 步骤6：报告输出与对比
1. 与上月预测进行对比，分析预测变动原因
2. 与年度计划目标对比，计算达成概率
3. 生成正式预测报告（含数据、方法、假设、风险）
4. 标注需要管理层关注的关键决策点

## 输入规格

| 输入项 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| 活跃商机数据 | Dataset | 是 | 本季度pipeline完整数据 |
| 已签约合同数据 | Dataset | 是 | 已确认的本季度收入 |
| 历史季度业绩 | Dataset | 是 | 至少过去4个季度的实际数据 |
| 季度/年度目标 | Number | 是 | Quota数值 |
| 团队规模数据 | Number | 是 | 当前销售团队人数 |
| 上月预测值 | Number | 否 | 用于对比和偏差分析 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| 综合预测值 | Number | 最终预测收入金额 |
| 置信区间 | Range | 预测的上下限（如±10%） |
| 场景分析 | Table | Best/Most Likely/Worst Case三种场景 |
| 预测达成概率 | Percentage | 达成Quota目标的概率估算 |
| 风险清单 | List | 按影响力排序的风险因素 |
| 对比分析 | Table | 与上期预测、年度目标的对比 |

## 最佳实践

- **预测频率**：每月正式更新一次，但Pipeline发生重大变化（大单签约/丢失）时应立即刷新
- **权重校准**：每季度末根据实际数据回测阶段赢率权重，确保反映真实转化概率
- **避免锚定效应**：不要因为年初设定的目标就倾向于预测接近目标的数字，预测必须基于数据
- **多声音输入**：结合一线销售经理的定性判断作为AI修正的输入之一，但需标注为"管理层判断"而非数据结论
- **预测偏差追踪**：建立每季度的Forecast Accuracy跟踪表，识别系统性偏差方向并修正

## 约束条件

- 预测准确率目标为±10%以内，超出此范围需进行根因分析并调整模型
- 不得为迎合管理层期望而人为调高或调低预测值
- 新市场/新产品线前3个季度因缺乏历史数据，预测置信度需明确降级标注
- 单一大单对预测贡献>30%时必须标注为"高集中度风险"
