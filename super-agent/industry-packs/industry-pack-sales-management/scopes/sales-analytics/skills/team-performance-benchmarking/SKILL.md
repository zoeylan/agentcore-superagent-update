# 团队效能对标分析

## 技能概述

本技能用于对销售团队和个人的多维度效能指标进行系统性对标分析，通过横向对比（团队间/个人间）、纵向对比（历史趋势）和外部对标（行业基准）三个维度，识别效能差距、提炼成功模式、发现改善机会。该技能是驱动销售组织从"经验管理"走向"数据管理"的关键分析能力。

## 适用场景

- 月度业绩排名和效能对比分析
- 团队扩张时的人效基准建立
- Top Performer成功模式提炼和复制
- Underperformer改善方向诊断
- 销售策略调整效果评估（A/B对比）
- 薪酬激励方案设计的数据支撑

## 执行步骤

### 步骤1：效能指标体系构建
1. 结果指标：
   - 签约金额和达标率
   - 赢率（Win Rate = Won / (Won + Lost)）
   - 平均交易规模（Average Deal Size）
   - 销售周期（Average Sales Cycle）
2. 过程指标：
   - 外呼量/有效通话数
   - 会议/演示数量
   - 新建商机数和金额
   - Pipeline增长率
3. 效率指标：
   - 人均产出（Revenue per Rep）
   - 活动到商机转化率
   - 商机到签约转化率
   - 时间利用率（销售活动时间占比）

### 步骤2：多维度对比分析
1. 团队内对比：同一团队成员间的排名和差异分析
2. 团队间对比：不同区域/产品线团队的效能对标
3. 时间序列对比：个人/团队的效能趋势（改善or衰退）
4. 公平性调整：考虑区域成熟度、客户质量分布、产品难度等因素

### 步骤3：成功模式识别
1. 定义Top Performer标准（综合排名前20%）
2. 分析Top Performer的共性行为特征：
   - 活动节奏（外呼频率、会议安排模式）
   - 商机管理方式（阶段推进速度、多线程策略）
   - 客户选择策略（目标客户画像偏好）
   - 时间分配模式
3. 提炼可复制的最佳实践建议

### 步骤4：改善机会诊断
1. 定义Underperformer标准（综合排名后20%或连续2月未达标）
2. 逐一分析效能短板：
   - 活动量不足 → 建议增加外呼/会议数量
   - 转化率低 → 建议提升话术/产品知识/需求挖掘能力
   - 赢率低 → 分析丢单原因（竞品/价格/需求匹配）
   - 周期长 → 分析阶段卡点并建议加速策略
3. 为每个改善方向给出具体建议和参考标杆

### 步骤5：输出对标报告
1. 生成团队效能仪表盘（关键指标排名+趋势）
2. 输出个人效能卡片（各维度得分+雷达图描述）
3. 提炼TOP 3团队级改善建议
4. 输出最佳实践案例和推荐的coaching重点

## 输入规格

| 输入项 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| 销售活动数据 | Dataset | 是 | 各类销售活动记录（通话/会议/演示等） |
| 成交数据 | Dataset | 是 | 签约/丢单记录含金额和周期 |
| 团队组织结构 | Config | 是 | 人员归属团队/区域/产品线 |
| 个人目标/配额 | Dataset | 是 | 各人的Quota设置 |
| 行业基准数据 | Config | 否 | 外部对标参考值 |
| 分析周期 | DateRange | 是 | 月度/季度 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| 效能排名表 | Table | 综合排名及各维度得分 |
| 团队对比分析 | Report | 团队间效能差异和原因分析 |
| 成功模式摘要 | List | Top Performer的可复制行为特征 |
| 改善建议 | List | 具体到个人/团队的改善方向 |
| 趋势报告 | Chart | 效能指标随时间的变化趋势 |

## 最佳实践

- **公平对标**：对比时需控制变量——新人vs老人、成熟区域vs新市场、大客户vs中小客户，不能简单直接排名
- **过程指标先行**：结果指标是滞后指标，通过过程指标可以提前2-3个月预判结果
- **正向激励为主**：分析结果用于帮助改善而非惩罚，强调"向Top Performer学习"而非"批评Underperformer"
- **动态基准**：基准线应随团队成熟度和市场变化动态调整，避免一成不变的考核标准
- **行业对标谨慎**：中国B2B市场缺乏公开的行业基准数据，外部对标仅供参考，内部趋势更有指导意义

## 约束条件

- 个人绩效数据属于敏感信息，分发范围严格限制为直接管理者和HR
- 不对入职不满3个月的新人进行排名对比（Ramp-up期不适用正常基准）
- 效能分析结论不直接用于淘汰决策——这是管理层综合判断的职责
- 排名和对比必须附带公平性调整说明，避免产生不当激励
