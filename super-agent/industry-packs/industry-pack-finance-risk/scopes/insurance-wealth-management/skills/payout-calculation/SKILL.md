# 赔付金额精算 (Payout Calculation)

## 概述

本技能负责根据保险合同条款、理赔材料核验结果和适用的赔付规则，精确计算保险理赔应付金额。覆盖医疗险费用型赔付、意外险定额给付、重疾险一次性赔付、车险定损赔付等多种计算模式，确保赔付金额误差率控制在0.1%以内。

## 适用场景

- 医疗险报销金额计算（扣除免赔额、社保已报销部分、赔付比例）
- 意外险伤残给付金额计算（按伤残等级比例给付）
- 重疾险保额给付计算（确诊即付或分组多次赔付）
- 车险维修/全损赔付计算（定损金额、折旧、残值）
- 多份保单的赔付顺序和分摊计算

## 执行步骤

### 步骤1：确定赔付规则
1. 从结构化条款数据中获取适用的赔付公式
2. 确认赔付类型：费用补偿型/定额给付型/损失赔偿型
3. 获取关键参数：保额、免赔额、赔付比例、赔付限额、免赔天数
4. 确认是否有特殊约定（加成赔付、降低赔付比例等）

### 步骤2：费用归集与分类
1. **医疗险**：将医疗费用按合同定义分类（门诊/住院/手术/药品/检查）
2. 区分社保范围内/范围外费用
3. 扣除非保险责任费用（如美容整形、既往症相关费用）
4. 确认每个费用类别的适用赔付比例和限额

### 步骤3：赔付金额计算
1. **费用补偿型**计算公式：
   ```
   应赔金额 = MIN(（合格费用总额 - 免赔额 - 已报销部分）× 赔付比例, 最高赔付限额)
   ```
2. **定额给付型**计算公式：
   ```
   应赔金额 = 基本保额 × 给付比例（按伤残等级/住院天数）
   ```
3. **损失赔偿型**计算公式：
   ```
   应赔金额 = MIN(实际损失金额 - 免赔额, 保额) × (1 - 折旧率 × 使用年限)
   ```
4. 处理多份保单赔付分摊（费用型按比例分摊，给付型各自独立赔付）

### 步骤4：扣减项计算
1. 扣除年度免赔额（累计已抵扣部分）
2. 扣除社保/商保已报销金额
3. 扣除第三方赔偿（如车险中对方全责已赔付部分）
4. 适用等待期/免赔天数扣除

### 步骤5：结果验证与输出
1. 执行计算结果的合理性校验（应赔金额不超过保额、不为负数、不超过实际损失）
2. 与历史同类案件的赔付金额进行横向比对
3. 对计算过程每一步附详细审计轨迹
4. 生成赔付计算明细书（可展示给客户和审核人员）

## 输入规格

| 输入字段 | 类型 | 必填 | 说明 |
|---------|------|------|------|
| claim_id | String | 是 | 理赔案件编号 |
| policy_terms | Object | 是 | 结构化的保单条款数据（来自合同解析Agent） |
| verified_expenses | Array | 是 | 核验通过的费用/损失明细 |
| deductions | Object | 否 | 已知扣减项（社保报销、第三方赔偿等） |
| multi_policy_info | Array | 否 | 多份保单信息（多保单赔付场景） |

## 最佳实践

1. **分步可追溯**：每一步计算都必须留有审计轨迹，支持客户质疑和监管审查
2. **边界条件测试**：特别注意免赔额刚好用完、保额恰好用尽等边界情况
3. **币种一致性**：涉及境外就医时注意汇率转换的时间点和汇率来源
4. **四舍五入规则**：统一使用银行家舍入法（四舍六入五成双），精确到分

## 约束条件

- 赔付金额计算误差率必须<0.1%
- 计算结果必须附完整的审计轨迹
- 超过100万元的赔付计算结果必须标记为需人工复核
- 计算逻辑不得偏离合同条款的明确约定
