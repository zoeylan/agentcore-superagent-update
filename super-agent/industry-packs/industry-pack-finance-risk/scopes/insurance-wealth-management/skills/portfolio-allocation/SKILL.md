# 资产配置方案生成 (Portfolio Allocation)

## 概述

本技能负责基于客户画像和市场环境，运用现代投资组合理论生成个性化的资产配置方案，包括大类资产比例确定、具体产品组合推荐、风险收益预估和适当性合规校验。配置方案必须在满足监管适当性要求的前提下，追求风险调整后收益最大化。

## 适用场景

- 新客户首次资产配置建议
- 存量客户的定期组合再平衡
- 市场环境重大变化时的策略调整建议
- 客户风险测评结果变更后的配置更新
- 大额资金入账后的增量配置建议

## 执行步骤

### 步骤1：配置约束确定
1. 从客户画像获取核心约束条件：
   - 风险等级上限（C1-C5对应R1-R5）
   - 可投资金额和投资期限
   - 流动性需求（需预留流动资金比例）
   - 特殊偏好/禁忌（如不投资特定行业）
2. 确认监管合规约束：
   - 适当性匹配铁律（C1/C2不可配R4/R5产品）
   - 单只私募认购下限100万元
   - 合格投资者门槛校验

### 步骤2：大类资产配置
1. 根据客户风险等级确定基础配置比例：
   - C1（保守型）：现金30%/固收60%/权益5%/另类5%
   - C2（稳健型）：现金20%/固收50%/权益20%/另类10%
   - C3（平衡型）：现金15%/固收35%/权益35%/另类15%
   - C4（积极型）：现金10%/固收20%/权益50%/另类20%
   - C5（激进型）：现金5%/固收10%/权益60%/另类25%
2. 根据当前市场环境进行战术调整（偏离基准不超过±10%）
3. 考虑客户特定需求进行个性化微调

### 步骤3：产品选择与组合构建
1. 在每个资产类别内选择具体产品：
   - 现金类：货币基金、银行存款、国债逆回购
   - 固收类：债券基金、银行理财、信托计划
   - 权益类：股票基金、混合基金、ETF、股票
   - 另类：私募股权、商品基金、REITs、结构化产品
2. 产品选择标准：风险等级匹配、历史业绩、基金规模、流动性、费率
3. 组合内产品数量控制在5-15只，避免过度分散

### 步骤4：风险收益预估
1. 计算组合预期年化收益率（基于历史数据和市场预期）
2. 计算组合最大回撤估计（历史最大回撤/蒙特卡洛模拟）
3. 计算组合VaR（95%/99%置信水平）
4. 执行压力测试（极端市场情景下的组合表现）
5. 计算夏普比率等风险调整收益指标

### 步骤5：适当性合规校验
1. 逐只产品校验风险等级≤客户可承受上限
2. 校验组合整体风险水平与客户分类匹配
3. 校验私募产品是否满足合格投资者要求
4. 校验集中度限制（单只产品不超过组合30%）
5. 任何违规自动拦截，不允许例外

### 步骤6：配置建议书生成
1. 输出标准化配置建议书，包含：
   - 大类资产配置方案及理由
   - 具体产品组合清单（附产品说明书摘要）
   - 风险收益预估和压力测试结果
   - 适当性匹配说明
   - 风险提示和免责声明
2. 附历史回测数据（过去3年/5年该配置方案的模拟表现）

## 输入规格

| 输入字段 | 类型 | 必填 | 说明 |
|---------|------|------|------|
| customer_profile | Object | 是 | 完整客户画像 |
| investable_amount | Number | 是 | 本次可配置金额 |
| market_environment | Object | 是 | 当前市场环境数据 |
| product_universe | Array | 是 | 可选产品池（含产品参数） |
| existing_holdings | Array | 否 | 现有持仓（再平衡场景） |
| special_requirements | Object | 否 | 特殊要求（偏好/禁忌） |

## 最佳实践

1. **适当性优先于收益**：任何配置方案首先必须满足适当性要求，然后才考虑收益最大化
2. **流动性匹配**：客户的流动性需求必须被充分考虑，避免全部配置长期锁定产品
3. **分散化原则**：避免过度集中于单一产品/管理人/行业/地区
4. **定期再平衡**：建议客户按季度/半年度检视组合，必要时再平衡

## 约束条件

- 适当性匹配违规拦截率必须100%（零容忍）
- 配置建议必须附风险提示和历史回测数据
- 单一产品集中度不超过组合总额的30%
- 不得向客户承诺或暗示保本保收益
- 所有配置建议留痕可追溯
