# 还款能力分析

## 技能概述
本技能对贷款申请人的还款能力进行量化评估，包括个人客户的DTI（债务收入比）测算和收入稳定性评估，以及企业客户的偿债能力、流动性和现金流分析。还款能力分析是判断借款人"能不能还"的关键依据，与信用评分（判断"愿不愿还"）互为补充。

## 适用场景
- 个人信贷申请的还款能力评估
- 企业信贷申请的偿债能力分析
- 额度核定时的最大可承受贷款金额测算
- 贷后复审中的还款能力重新评估

## 执行步骤

### 1. 个人客户——收入认定
- 验证收入来源：工资收入（银行流水中的工资入账）、经营收入（对公/对私流水中的经营性收入）、投资收入、租金收入等
- 计算月均收入：取近6个月银行流水中收入类交易的算术平均值（剔除大额非经常性收入）
- 收入稳定性评估：
  - 连续在同一单位发薪≥12个月：高稳定性
  - 6-12个月：中稳定性
  - <6个月：低稳定性
- 交叉验证：收入证明金额与银行流水的一致性（偏差>20%标记异常）

### 2. 个人客户——DTI测算
- 汇总现有月度债务：
  - 征信报告中的在贷月供总额
  - 信用卡已用额度的最低还款额（通常按已用额度的10%估算）
  - 多头查询中发现的网贷月供估算
- 加入本笔贷款的预计月供（按申请金额、期限和预估利率计算）
- DTI = （现有月度债务 + 本笔预计月供）/ 月均收入 × 100%
- DTI阈值：≤30%（优秀）、30-40%（良好）、40-50%（可接受，需关注）、>50%（超限，建议降额或拒绝）

### 3. 企业客户——财务分析
- **偿债能力指标**：
  - 资产负债率 = 总负债 / 总资产（警戒线：70%）
  - 流动比率 = 流动资产 / 流动负债（最低要求：1.2）
  - 速动比率 = （流动资产 - 存货）/ 流动负债（最低要求：0.8）
- **现金流分析**：
  - 经营性现金流净额：近12个月必须为正值
  - EBITDA利息覆盖倍数 = EBITDA / 利息费用（最低要求：2倍）
  - 现金流对新增贷款月供的覆盖能力
- **盈利能力**：
  - 净利润率趋势（近3年是否持续下降）
  - 营收增长率（负增长标记风险）

### 4. 企业客户——行业风险调整
- 查询申请企业所在行业的景气度指数（PMI、行业增速等）
- 行业处于上行周期：还款能力评估可适当乐观
- 行业处于下行周期或面临重大政策调整：还款能力评估需偏保守
- 特殊行业（如周期性行业）需考虑行业周期位置

### 5. 综合还款能力结论
- 输出还款能力等级：充足/适中/不足/严重不足
- 计算最大可承受贷款金额（基于DTI上限或现金流覆盖能力反推）
- 标记关键风险点（如DTI接近上限、行业下行、收入不稳定等）

## 输入规格
| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| applicationId | string | 是 | 贷款申请编号 |
| applicantType | string | 是 | personal/corporate |
| incomeData | object | 是 | 收入数据（流水/收入证明/财务报表） |
| existingDebtData | object | 是 | 现有负债数据（征信+多头） |
| loanRequest | object | 是 | 本笔贷款要素（金额、期限、预估利率） |
| industryCode | string | 否 | 企业所在行业代码（企业信贷必填） |

## 输出规格
| 字段 | 类型 | 描述 |
|------|------|------|
| repaymentCapacityGrade | string | 还款能力等级 |
| dtiRatio | number | DTI比率（个人客户） |
| financialRatios | object | 财务指标计算结果（企业客户） |
| maxAffordableAmount | number | 最大可承受贷款金额 |
| keyRisks | array | 关键风险点 |
| incomeStability | string | 收入稳定性评级 |

## 最佳实践
- DTI计算中应包含"隐性负债"（多头查询发现的网贷等征信未覆盖的负债）
- 企业财务报表应要求提供经审计的报表，未经审计的报表需打折考虑
- 收入认定以银行流水为主、收入证明为辅，两者不一致时以流水为准
- DTI计算误差须控制在1%以内

## 约束条件
- 不得仅凭收入证明认定收入（需银行流水交叉验证）
- 不得忽略征信报告中的担保责任对还款能力的潜在影响
- 企业财务分析须基于近3年数据（至少2年），不接受仅1年数据
- 行业风险调整因子须有据可依，不得主观随意调整
