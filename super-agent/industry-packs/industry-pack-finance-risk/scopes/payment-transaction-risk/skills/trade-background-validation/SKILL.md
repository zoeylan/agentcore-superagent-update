# 贸易背景真实性验证 (Trade Background Validation)

## 概述
贸易背景真实性验证是跨境合规审核的核心技能，对贸易项下跨境支付的交易背景材料（合同、发票、报关单等）进行真实性和一致性审核，防范虚假贸易背景下的资金跨境转移、洗钱和逃汇行为。

## 适用场景
- 贸易项下跨境汇出汇款的背景材料审核
- 服务贸易支付的合同和完税证明审核
- 预付货款/分期付款的合理性审核
- 转口贸易（离岸转手买卖）的背景核实
- 大额跨境支付的增强背景审查

## 执行步骤

### 步骤1：材料收集与分类
- 根据交易类型确定所需材料清单：
  - **货物贸易**：购销合同、商业发票（Commercial Invoice）、报关单/提单、装箱单
  - **服务贸易**：服务合同、完税证明（代扣代缴税费凭证）
  - **资本项目**：投资协议、商务部备案、外汇登记证明
- 检查材料是否齐全，缺失时生成补件清单

### 步骤2：材料真实性核验
- **合同核验**：
  - 合同要素完整性（买卖双方、标的、金额、付款条件、签署日期）
  - 签署方与汇款人/收款人一致性
  - 合同金额与付款金额的匹配性（允许合理误差±5%）
  - 合同签署时间的逻辑合理性（不应晚于付款时间）
- **发票核验**：
  - 发票金额与合同金额的一致性
  - 品名/数量/单价的合理性（参照行业价格区间）
  - 发票编号格式和连续性
- **报关单核验**：
  - 报关单号的真实性验证（对接海关系统查验）
  - 申报金额与付款金额的匹配度
  - 贸易方式和监管条件码的合规性
  - 进出口时间线与付款时间的逻辑性

### 步骤3：交叉一致性分析
- 合同↔发票↔报关单三者之间的交叉比对：
  - 品名/规格描述一致
  - 数量和金额匹配
  - 时间线逻辑合理（合同签署→发货→报关→付款）
- 检测异常模式：
  - 同一合同重复付款（重复使用背景材料）
  - 付款金额大幅超出合同/报关金额
  - 贸易方向与资金方向不匹配
  - 转口贸易中无合理商业逻辑的中间环节

### 步骤4：审核结论输出
- **通过**：材料真实完整，交易背景合理
- **不通过**：材料存在明显虚假或重大矛盾
  - 详细说明不通过原因
  - 建议拒绝交易或上报可疑
- **需补充**：材料部分缺失或存在轻微疑点
  - 列出具体补充要求
  - 设定48小时补件时限

## 输入规格
```json
{
  "transaction_id": "string",
  "trade_type": "string - GOODS/SERVICE/CAPITAL",
  "payment_amount": "number - 付款金额",
  "payment_currency": "string",
  "documents": [
    {
      "type": "string - CONTRACT/INVOICE/CUSTOMS_DECLARATION/BOL/TAX_CERT",
      "content_summary": "object - 文件关键信息提取结果",
      "file_reference": "string"
    }
  ],
  "payer": "object - 付款方信息",
  "payee": "object - 收款方信息"
}
```

## 输出规格
```json
{
  "validation_result": "string - PASS/REJECT/PENDING_SUPPLEMENT",
  "consistency_score": "number - 一致性评分(0-100)",
  "check_details": [
    {"check_item": "string", "result": "string", "finding": "string"}
  ],
  "anomalies_detected": "array - 检测到的异常",
  "supplement_required": "array - 需补充的材料",
  "risk_indicators": "array - 风险指标标记"
}
```

## 最佳实践
- 建立行业价格区间数据库，辅助判断商品定价的合理性
- 对老客户（历史合规记录良好）实施差异化审核（简化材料要求）
- 转口贸易应重点关注商业合理性（是否存在真实的贸易需求）
- 与海关系统实时对接可大幅提升报关单验真效率

## 约束条件
- 审核时限48小时内完成
- 不得仅凭材料形式通过审核，必须进行实质性核验
- 发现虚假贸易背景必须上报合规部门和AML监测Agent
- 审核记录和支撑材料保留≥5年
