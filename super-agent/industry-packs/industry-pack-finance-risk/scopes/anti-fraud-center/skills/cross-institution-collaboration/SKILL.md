# 跨机构协同 (Cross-Institution Collaboration)

## 技能概述

跨机构协同是欺诈情报分析Agent协调与外部机构（公安、同业金融机构、行业协会、第三方数据服务商）进行欺诈信息共享和联防联控的能力。金融欺诈尤其是团伙欺诈通常跨越多家金融机构，单一机构的防御视角存在天然局限，跨机构协同是提升整体防御能力的关键。

### 适用场景
- 向公安机关提交欺诈案件线索和协助调查
- 与同业金融机构的欺诈案例和黑名单共享
- 参与行业协会的联防联控机制
- 与第三方数据服务商的情报订阅和数据合作
- 接收和响应公安协查函/冻结函

## 执行步骤

### 第一步：协同机制管理
1. 维护跨机构协同关系清单：
   - **公安渠道**：属地公安经侦部门联络人、网安部门联络人、报案/协查流程
   - **同业网络**：已建立信息共享协议的金融机构清单、共享范围和频率
   - **行业协会**：参与的行业联防联控机制（如银联风控联盟、支付清算协会反欺诈委员会）
   - **第三方供应商**：数据服务商/安全厂商的合作协议和接口
2. 每半年审查协同关系的有效性和活跃度

### 第二步：信息共享输出
1. 确定共享内容和脱敏级别：
   - 按TLP（Traffic Light Protocol）标准分级：
     - TLP:RED：仅限指定接收方，不可二次分享
     - TLP:AMBER：限行业内部共享
     - TLP:GREEN：可在金融行业社区内共享
     - TLP:WHITE：公开信息
2. 共享内容类型：
   - **黑名单数据**：确认欺诈账户/设备/IP（脱敏后的特征信息）
   - **欺诈模式预警**：新型欺诈手法描述（不含客户具体信息）
   - **案例通报**：典型案例摘要（完全脱敏）
   - **IOC指标**：可疑IP段、设备特征指纹、恶意域名等
3. 信息输出前必须通过合规审查和脱敏处理

### 第三步：信息共享接收
1. 接收同业和行业共享的欺诈信息：
   - 其他机构通报的欺诈案例和新型手法
   - 行业黑名单更新
   - 公安机关的预警通告
2. 对接收信息执行与情报收集相同的结构化处理和威胁评估流程
3. 涉及本机构客户的信息优先处理

### 第四步：公安协作
1. **主动报案**：
   - 对确认的重大欺诈案件（涉案>50万元或涉及团伙），准备报案材料
   - 材料包括：案件概述、涉案交易明细、账户信息、调查分析报告、电子证据清单
   - 配合公安机关后续调查取证
2. **协查响应**：
   - 收到公安协查函/冻结函后，立即确认函件真实性
   - 在法定时限内完成账户查询/冻结操作
   - 准备回复材料并按时反馈
3. **联合行动**：
   - 配合公安机关组织的专项打击行动
   - 提供技术支持（如数据分析、资金追踪）

### 第五步：协同效果评估
1. 每季度评估跨机构协同效果：
   - 信息共享的双向活跃度
   - 基于共享情报发现/阻止的欺诈案件数和金额
   - 协查响应时效达标率
   - 联防联控的覆盖度和响应速度
2. 输出协同效果报告，优化协同策略

## 输入/输出规范

### 输入
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| collaborationType | enum | 是 | SHARE_OUT/SHARE_IN/POLICE_REPORT/POLICE_REQUEST/JOINT_ACTION |
| partnerInstitution | string | 是 | 协作机构标识 |
| content | object | 是 | 协作内容 |
| tlpLevel | enum | 否 | 信息共享级别 RED/AMBER/GREEN/WHITE |
| urgency | enum | 否 | 紧急程度 URGENT/NORMAL/LOW |

### 输出
| 字段 | 类型 | 说明 |
|------|------|------|
| collaborationId | string | 协作记录ID |
| status | enum | INITIATED/IN_PROGRESS/COMPLETED/PENDING_RESPONSE |
| sharedContent | object | 已共享/接收的内容摘要 |
| complianceCheckResult | object | 合规检查结果 |
| responseDeadline | datetime | 响应截止时间 |
| effectivenessMetrics | object | 协同效果指标 |

## 最佳实践
- 建立"快速协查"机制——紧急协查函24小时内完成响应
- 同业信息共享采用"对等互惠"原则，保持双向信息流动
- 定期参加行业反欺诈峰会和工作组，维护人脉网络
- 建立电子化协作平台，替代传统的函件传递方式提升效率

## 约束条件
- 所有对外信息共享必须经过合规审查和脱敏处理
- 客户个人信息的共享必须符合《个人信息保护法》规定（有合法依据）
- 公安协查函必须验证真实性后方可执行
- 信息共享协议必须明确各方的权利义务和数据安全责任
- 跨机构共享的数据不得用于约定范围外的用途
