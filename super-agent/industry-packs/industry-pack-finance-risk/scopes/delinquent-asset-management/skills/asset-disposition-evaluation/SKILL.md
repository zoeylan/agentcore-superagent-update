# 资产处置评估技能 (Asset Disposition Evaluation)

## 技能概述

本技能用于对不适合诉讼的M3+逾期案件评估替代处置方案，包括委外催收（Outsourced Collection）、不良资产包转让（NPL Package Transfer）和核销申报（Write-off Application）三种主要路径。通过对各方案的预期回收金额、时间周期、处置成本进行量化对比分析，为每个案件推荐最优处置方案。

本技能同时承担NPL资产包的定价分析职能，基于历史回收率数据和市场可比案例（Comparable Transactions），为资产包转让提供科学的定价依据。

### 适用场景

| 场景 | 说明 |
|------|------|
| 诉讼不可行案件处置 | 法务可行性评估结论为"不适合诉讼"的案件（无可执行财产、成本收益比不达标等），需要评估替代处置方案 |
| 委外催收到期评估 | 委外催收期满后仍未回收的案件，评估是否续期委外、转入NPL资产包或申请核销 |
| 季度NPL资产包组包 | 每季度对存量待处置案件进行批量评估和分组，形成NPL资产包进行转让 |
| 核销条件审查 | 对拟核销案件进行核销条件合规性审查，确保满足监管和内部核销标准 |
| 执行终本案件再评估 | 诉讼执行终本（终结本次执行）的案件，重新评估后续处置方案 |
| 处置方案调整 | 市场环境变化（如AMC收购价格波动）时，对存量待处置案件重新评估 |

## 执行步骤

### 步骤一：案件分类与处置条件判定

根据案件特征进行初步分类：

| 案件特征 | 适用处置方案 | 判断依据 |
|----------|-------------|----------|
| 有一定催收价值、借款人有部分还款能力或意愿 | 委外催收 | 最后联系可达、有收入来源但不配合内部催收 |
| 金额较大、批量同质案件、市场有需求 | NPL资产包转让 | 单笔或批量案件，AMC/投资者有收购兴趣 |
| 确认无回收可能、满足核销条件 | 核销申报 | 借款人死亡/失踪/破产且无可执行财产 |
| 仍有潜在回收可能但短期无法执行 | 暂缓处置（继续监控） | 终本案件、借款人年轻有未来恢复还款能力的可能 |

### 步骤二：委外催收方案评估

**评估维度：**

1. **预期回收金额计算**：
   ```
   委外预期回收 = 委外案件本金 x 委外回收率(历史同类案件) x (1 - 佣金率)
   ```
   - 委外回收率参考值（按案件类型）：
     - 信用卡逾期：8%-15%
     - 个人消费贷款：5%-12%
     - 经营贷款：3%-8%
   - 佣金率范围：15%-25%（根据逾期天数和回收难度调整）

2. **佣金率定价模型**：

| 逾期天数 | 案件难度 | 建议佣金率 |
|----------|----------|------------|
| 60-90天 | 低（有联系方式、曾有还款记录） | 15%-18% |
| 90-180天 | 中（部分失联、拒绝还款） | 18%-22% |
| 180-360天 | 高（完全失联、无还款记录） | 22%-25% |
| 360天以上 | 极高（长期失联、多轮催收无效） | 不建议委外（转NPL转让或核销） |

3. **时间周期**：委外催收标准周期为 3-6个月，最长不超过12个月。

4. **成本分析**：
   - 直接成本：佣金（按实际回收金额的比例支付，无回收则无成本）
   - 管理成本：委外机构管理、合规审计费用（分摊至每案约200-500元）

### 步骤三：NPL资产包转让方案评估

**1. 资产包组包策略**：

按以下维度对待转让案件进行分组：

| 分组维度 | 分组方式 | 目的 |
|----------|----------|------|
| 贷款类型 | 个人贷款包 / 信用卡包 / 经营贷款包 | 不同类型买方偏好不同 |
| 地域分布 | 按省份/区域分组 | 地方AMC通常偏好本地资产 |
| 逾期天数 | M3-M6包 / M6-M12包 / M12+包 | 逾期时间影响定价 |
| 金额区间 | 小额包(<5万) / 中额包(5-50万) / 大额包(>50万) | 不同买方的投资偏好 |
| 有无担保 | 纯信用包 / 有担保包 | 有担保资产包溢价更高 |

**2. 资产包定价模型**：

```
资产包定价 = 资产包本金总额 x 回收率折扣
```

回收率折扣定价参考（占本金比例）：

| 资产类型 | 逾期60-180天 | 逾期180-360天 | 逾期360天以上 |
|----------|-------------|---------------|--------------|
| 信用卡逾期 | 5折-8折 | 3折-5折 | 2折-3折 |
| 个人消费贷款 | 4折-7折 | 2折-4折 | 1.5折-3折 |
| 经营贷款（有担保） | 5折-8折 | 4折-6折 | 3折-5折 |
| 经营贷款（无担保） | 3折-5折 | 2折-3折 | 1折-2折 |

> 注：1折 = 10% of principal

**3. 定价调整因子**：

| 因子 | 影响方向 | 调整幅度 |
|------|----------|----------|
| 借款人有房产（已查封/未查封） | 上调 | +1折至+2折 |
| 借款人有稳定收入 | 上调 | +0.5折至+1折 |
| 借款人已进入失信被执行人名单 | 上调 | +0.5折（限高令有催收效果） |
| 借款人失联超过1年 | 下调 | -1折至-2折 |
| 诉讼时效已过或即将过期 | 下调 | -1折至-3折 |
| 批量案件数量溢价（>100笔） | 下调 | -0.5折（买方议价能力增强） |
| 市场流动性好（多家AMC竞标） | 上调 | +0.5折至+1折 |

**4. 底价线管理**：
- 每个资产包必须设定底价线（Reserve Price）
- 底价线 = 资产包本金 x 最低回收率折扣（由风控委员会审批）
- 实际转让价格不得低于底价线
- 底价线的制定须基于历史回收率数据（至少参考近3年同类资产包的实际回收表现）

**5. 潜在买方**：
- 四大AMC（中国华融、中国长城、中国东方、中国信达）
- 地方AMC（各省市持牌资产管理公司）
- 持牌金融资产投资公司
- 其他合规投资者（需符合监管准入要求）

**6. 时间周期**：NPL资产包转让通常需要 2-4个月（尽职调查1-2个月 + 交易谈判1个月 + 交割1个月）。

### 步骤四：核销申报方案评估

**核销条件审查**（须同时满足）：

| 核销条件 | 审查内容 |
|----------|----------|
| 已穷尽催收手段 | M1/M2催收记录完整、已尝试法务催收或委外催收 |
| 无可执行财产 | 财产查询结果显示无可供执行的资产 |
| 符合监管规定的核销情形 | 借款人死亡且无遗产/继承人、借款人依法宣告破产/关闭/解散、借款人失踪超过2年且无财产可执行、法院裁定终结执行等 |
| 内部审批流程 | 经办人→部门负责人→风控委员会→董事会（按金额分级审批） |
| 拨备已足额计提 | 该笔贷款已100%计提拨备 |

**核销金额分级审批权限**（参考）：

| 核销金额 | 审批层级 |
|----------|----------|
| 单笔 <= 5万元 | 部门负责人审批 |
| 5万 < 单笔 <= 50万元 | 风控委员会审批 |
| 50万 < 单笔 <= 500万元 | 行长/总经理办公会审批 |
| 单笔 > 500万元 | 董事会审批 |

**核销后管理**：
- 核销不等于放弃债权，核销后仍应保留追索权
- 核销案件纳入账销案存管理，定期评估是否有回收可能
- 核销回收的金额计入当期损益

### 步骤五：方案对比与推荐

为每个案件（或案件批次）生成处置方案对比表：

| 对比维度 | 委外催收 | NPL资产包转让 | 核销申报 |
|----------|----------|---------------|----------|
| 预期回收金额 | [计算值] | [计算值] | 0元（账面核销） |
| 回收率 | [X]% | [X]折 | 0% |
| 时间周期 | 3-6个月 | 2-4个月 | 1-3个月（审批周期） |
| 处置成本 | 佣金[X]%（仅在回收时支付） | 尽调/交易成本（约资产包金额的0.5-1%） | 内部人力成本 |
| 净回收金额 | [计算值] | [计算值] | 0元 |
| 后续管理负担 | 中（需持续监管委外机构） | 低（一次性转让完毕） | 低（账销案存定期复查） |
| 风险 | 合规风险（委外机构违规催收） | 定价风险（转让价格过低） | 审批风险（内部合规审查不通过） |

**推荐逻辑优先级**：
1. 净回收金额最大化
2. 时间周期最短（加速不良资产出清）
3. 管理成本最低
4. 合规风险最小

## 最佳实践与约束

### 最佳实践

1. **定价基于数据而非直觉**：NPL资产包定价必须基于历史回收率数据（内部历史数据 + 市场可比交易数据）。建议建立内部NPL定价数据库，至少积累3年以上的资产包转让和回收数据作为定价基准。

2. **组包策略影响定价**：资产包的组包方式直接影响买方兴趣和出价。建议按"贷款类型 + 地域 + 逾期时间"进行分组，形成特征相对均一的资产包，方便买方进行尽职调查和估值。避免将差异极大的案件混合打包。

3. **多买方竞价机制**：NPL资产包转让应采用公开招标或多方询价方式，确保至少有3家以上潜在买方参与竞价，避免定向转让导致价格偏低。

4. **核销材料充分性**：核销申报被内部合规审查退回的主要原因是"催收记录不完整"和"核销依据不充分"。在申请核销前，务必确认催收记录链条完整（内部催收→委外/诉讼→穷尽催收手段的证明）。

5. **动态再评估机制**：处置方案不是一成不变的。建议对已确定处置方案但尚未执行的案件，在执行前进行最终确认（如检查是否有新增财产线索、市场价格是否有变化等）。

### 约束条件

| 约束项 | 要求 |
|--------|------|
| 定价底线 | 资产包转让价格 **不得低于风控委员会审批的底价线** |
| 定价依据 | 须基于历史回收率数据和市场可比案例，禁止主观定价 |
| 核销合规 | 核销申报材料必须 **100%通过内部合规审查** |
| 转让合规 | NPL资产包转让须符合银保监会关于不良资产转让的监管规定 |
| 买方准入 | 资产包买方必须具备合法资质（持牌AMC或符合监管要求的合格投资者） |
| 信息披露 | 资产包尽职调查材料中的客户信息须按《个人信息保护法》要求进行脱敏 |
| 拨备要求 | 核销案件须已100%计提拨备 |
| 审批权限 | 处置方案须在授权范围内审批，超权限须逐级上报 |

## 示例

### 示例一：单案处置方案对比

**案件信息：**
- 案件编号：CASE-2026-041872
- 借款人李某某，信用卡透支
- 逾期本金52,000元，总欠款58,700元，逾期210天
- 无可执行财产，长期失联
- 法务评估结论：不适合诉讼（成本收益比0.70:1）

**处置方案对比：**

| 对比维度 | 委外催收 | NPL资产包转让 | 核销申报 |
|----------|----------|---------------|----------|
| 预期回收率 | 6%（同类信用卡失联案件历史回收率） | 3折（本金的30%） | 0% |
| 预期回收金额 | 3,522元 | 15,600元 | 0元 |
| 佣金/成本 | 佣金25% = 881元 | 交易成本约0.5% = 294元 | 内部成本约500元 |
| 净回收金额 | 2,641元 | 15,600元（转让即回收） | 0元 |
| 时间周期 | 6个月 | 纳入下批资产包，约3个月 | 审批约2个月 |
| 风险 | 回收不确定性高 | 定价风险（市场波动） | 审批不通过风险 |

**推荐方案：** NPL资产包转让
- 理由：委外催收预期净回收仅2,641元且不确定性高，NPL转让可一次性获得15,600元（确定性高）。核销虽然流程简单但回收为零。综合回收金额和确定性，建议纳入下一批信用卡逾期NPL资产包转让。

### 示例二：批量NPL资产包定价

**资产包概况：**
- 资产包名称：2026年Q2信用卡逾期资产包
- 案件数量：238笔
- 本金总额：1,850万元
- 总欠款金额：2,120万元
- 逾期天数分布：180-360天占65%，360天以上占35%
- 借款人特征：95%失联，5%可联系但拒绝还款
- 有财产线索案件：12%（主要为小额存款或低值车辆）

**定价分析：**

```
基础定价：
  180-360天部分（本金1,203万元）：基准折扣 3.5折 → 421万元
  360天以上部分（本金647万元）：基准折扣 2折 → 129万元
  合计基础定价：550万元

调整因子：
  (+) 12%案件有财产线索：+0.3折 → +56万元
  (-) 95%借款人失联：-0.5折 → -93万元
  (-) 批量折扣（>200笔）：-0.3折 → -56万元
  (+) 当前市场AMC需求旺盛：+0.3折 → +56万元

调整后定价：513万元（约2.77折）

底价线：本金的2折 = 370万元（经风控委员会审批）
建议挂牌价：550万元（约2.97折）
预期成交价区间：480-530万元（约2.6折-2.9折）
```

1. XX省地方AMC（对本地信用卡资产包有持续收购需求）
