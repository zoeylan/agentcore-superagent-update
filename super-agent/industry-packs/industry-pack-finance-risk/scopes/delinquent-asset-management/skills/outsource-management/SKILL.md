# 委外催收管理技能 (Outsource Management)

## 技能概述

本技能用于管理委托外部催收机构（Third-party Collection Agencies）的全生命周期，包括机构准入评估、案件移交、绩效KPI跟踪、合规审计、合同管理以及机构评级与续约/终止决策。确保委外催收在提高不良资产回收率的同时，严格遵守监管合规要求，防范因第三方催收机构的不当行为给金融机构带来声誉风险和监管处罚风险。

委外催收是M3+逾期资产处置的重要补充手段。当案件不适合直接诉讼、或作为诉讼前的过渡措施时，通过专业的外部催收机构进行催收可以有效扩展催收产能、利用机构的属地化优势提升回收率。

### 适用场景

| 场景 | 说明 |
|------|------|
| 委外催收日常管理 | 对在催案件进行回收绩效跟踪，监控各机构的回收率和成本效率 |
| 新案件移交 | 将法务可行性评估/资产处置评估推荐委外催收的案件移交给外部机构 |
| 季度合规审计 | 每季度对委外催收机构进行合规性审计，检查催收行为是否符合监管要求 |
| 机构绩效评估与评级 | 基于KPI数据对委外机构进行绩效排名，形成机构评分卡（Scorecard） |
| 合同续签/终止决策 | 基于机构评分卡和绩效数据，决定委外合同的续签、调整或终止 |
| 投诉与违规事件处理 | 处理因委外催收引发的客户投诉或合规违规事件 |

## 执行步骤

### 步骤一：委外催收机构准入与合同管理

**1. 机构准入评估维度**：

| 评估维度 | 评估内容 | 权重 |
|----------|----------|------|
| 资质合规 | 营业执照、催收行业从业资质、是否有行政处罚记录 | 25% |
| 经营规模 | 注册资本、员工人数、催收座席数、覆盖区域 | 15% |
| 历史业绩 | 近3年催收回收率、服务客户数、行业口碑 | 25% |
| 合规管理体系 | 内部合规制度、投诉处理机制、录音质检体系、信息安全措施 | 20% |
| 信息安全 | 数据保护措施、等保认证级别、人员背景审查制度 | 15% |

- 评估总分 >= 80分：可列入合格供应商名录
- 评估总分 60-79分：有条件准入（需整改后复评）
- 评估总分 < 60分：不予准入

**2. 合同核心条款**：

| 条款类别 | 关键内容 |
|----------|----------|
| 佣金条款 | 佣金率（15%-25%，按案件难度分档）、结算周期（月度/季度）、佣金计算基数（回收本金/本息） |
| 服务期限 | 委外催收期限（通常3-6个月/批次）、到期未回收案件的退回机制 |
| 合规要求 | 催收时间限制（08:00-21:00）、联系频次上限（日3次）、禁止暴力催收、禁止联系无关第三方 |
| 信息安全 | 客户数据保密义务、数据使用范围限制、数据销毁要求（合同终止后30天内） |
| 违约责任 | 合规违规的违约金条款、重大违规的即时终止权、投诉超标的处罚机制 |
| 绩效考核 | 最低回收率要求、KPI考核指标及权重、绩效不达标的后果 |
| 退出机制 | 合同到期不续签的案件交接流程、提前终止的通知期和交接要求 |

**3. 合同管理台账**：
- 记录所有委外合同的签订日期、到期日期、续签情况
- 到期前60天自动提醒，启动续签评估或终止准备
- 合同变更（如佣金率调整）须书面签署补充协议

### 步骤二：案件移交与回收

**1. 案件移交协议（Case Handover Protocol）**：

移交前准备：
- 按案件分配策略选择对应的委外机构（根据机构的区域优势、案件类型专长等匹配）
- 准备移交数据包（加密传输）

移交数据内容：

| 数据字段 | 说明 | 脱敏要求 |
|----------|------|----------|
| 案件编号 | 内部案件唯一标识 | 不脱敏 |
| 借款人姓名 | 全名 | 不脱敏（催收必需） |
| 联系电话 | 借款人及紧急联系人电话 | 不脱敏（催收必需） |
| 联系地址 | 住所地/工作地 | 不脱敏（催收必需） |
| 身份证号 | 全号 | 部分脱敏（中间段加*） |
| 欠款金额 | 本金、利息、罚息明细 | 不脱敏 |
| 催收历史 | 前期催收摘要 | 不脱敏 |
| 特殊标记 | 投诉客户、VIP客户、敏感案件等 | 不脱敏 |

移交确认：
- 委外机构须在收到数据后 **3个工作日** 内确认接收并开始催收
- 系统记录移交时间戳，作为催收期限的起算点

**2. 回收入账流程**：
- 委外机构催收到的还款由借款人直接还至金融机构指定账户
- 禁止委外机构代收代付（防止资金截留风险）
- 回款到账后系统自动匹配案件并更新回收记录
- 月度与委外机构对账，确认回收金额及佣金结算

### 步骤三：绩效KPI跟踪

**核心KPI指标体系：**

| KPI指标 | 定义 | 计算公式 | 目标值 |
|---------|------|----------|--------|
| 回收率 (Recovery Rate) | 已回收金额占委外案件总金额的比例 | 回收金额 / 委外案件总金额 x 100% | >= 8%（按案件难度调整） |
| 佣金成本率 (Cost Efficiency) | 佣金支出占回收金额的比例 | 佣金支出 / 回收金额 x 100% | <= 25% |
| 案件触达率 (Contact Rate) | 成功联系到借款人的案件比例 | 已联系案件数 / 委外案件总数 x 100% | >= 40% |
| 承诺还款率 (Promise-to-Pay Rate) | 承诺还款的案件比例 | 承诺还款案件数 / 已联系案件数 x 100% | >= 30% |
| 承诺履约率 (PTP Kept Rate) | 承诺还款后实际还款的比例 | 实际还款案件数 / 承诺还款案件数 x 100% | >= 50% |
| 投诉率 (Complaint Rate) | 因催收引发投诉的比例 | 投诉案件数 / 催收案件总数 x 1000 | <= 2%o（千分之二） |
| 回收时效 (Time to Recovery) | 从移交到首笔回款的平均天数 | 各案件(首笔回款日-移交日)的平均值 | <= 45天 |

**绩效跟踪周期：**

| 跟踪频率 | 跟踪内容 |
|----------|----------|
| 每日 | 回款金额、新增联系记录（系统自动汇总） |
| 每周 | 周回收率趋势、本周新增承诺还款案件 |
| 每月 | 全量KPI指标计算、与目标值对比、环比/同比分析 |
| 每季度 | 综合绩效评估、机构评分卡更新、合规审计 |

### 步骤四：合规审计

**季度合规审计内容：**

| 审计类别 | 审计项目 | 审计方式 |
|----------|----------|----------|
| 催收行为合规 | 催收时间是否在08:00-21:00之间 | 抽查催收系统时间戳记录（抽样率>=10%） |
| 催收行为合规 | 单日联系频次是否 <= 3次 | 系统自动检测，全量扫描 |
| 催收行为合规 | 是否存在暴力催收、威胁恐吓行为 | 抽听催收录音（抽样率>=5%） |
| 催收行为合规 | 是否违规联系无关第三方 | 抽查联系记录（抽样率>=10%） |
| 话术合规 | 催收话术是否使用审核通过的模板 | 抽听催收录音、查看短信/函件模板 |
| 信息安全 | 客户数据存储是否符合安全要求 | 现场检查或远程审计（年度至少1次现场） |
| 信息安全 | 是否存在数据泄露或超范围使用 | 数据访问日志审查 |
| 投诉处理 | 投诉件是否及时响应和处理 | 查看投诉处理台账、核实处理结果 |
| 人员管理 | 催收人员是否经过合规培训 | 查看培训记录和考核成绩 |
| 人员管理 | 是否有员工离职后的数据权限回收记录 | 查看离职交接记录 |

**审计发现的分类与处理：**

| 严重程度 | 定义 | 处理要求 |
|----------|------|----------|
| 一般问题 | 操作瑕疵但未造成实质影响（如个别催收时间略超限） | 书面通知整改，15个工作日内反馈整改措施 |
| 较严重问题 | 可能引发投诉或轻微合规风险（如话术不规范、频次超标较多） | 限期整改（10个工作日），扣减当期佣金的5%-10% |
| 严重违规 | 暴力催收、信息泄露、代收代付等重大合规违规 | **立即暂停移交新案件**，30天内退回全部在催案件，追究违约责任，必要时终止合同 |

### 步骤五：机构评分卡 (Agency Scorecard)

**评分卡维度与权重：**

| 维度 | 权重 | 评分指标 | 满分 |
|------|------|----------|------|
| 回收绩效 | 40% | 回收率达标情况、回收金额排名、回收时效 | 100分 |
| 合规表现 | 30% | 季度审计结果、投诉率、违规事件数 | 100分 |
| 运营能力 | 15% | 案件触达率、承诺履约率、数据反馈及时性 | 100分 |
| 配合程度 | 15% | 问题响应速度、整改配合度、沟通协作质量 | 100分 |

**评级标准：**

| 评级 | 评分范围 | 管理措施 |
|------|----------|----------|
| A级（优秀） | >= 85分 | 优先分配案件、可适当增加委外量、续签优先 |
| B级（良好） | 70-84分 | 正常委外案件分配、保持合作 |
| C级（一般） | 60-69分 | 减少案件分配量、加强月度绩效沟通、列入观察名单 |
| D级（不达标） | < 60分 | 暂停新案件移交、限期整改、整改不达标则终止合作 |

**评分卡更新频率**：每季度更新一次，年度形成年度综合评分。

### 步骤六：续约/终止建议

**续约评估流程**（合同到期前60天启动）：

```
1. 汇总近4个季度评分卡数据
2. 计算年度综合评分
3. 对比同类委外机构的市场表现
4. 综合评估续约/终止/调整建议
```

**决策矩阵：**

| 年度综合评分 | 市场替代性 | 建议 |
|-------------|------------|------|
| A级 | - | 续约，可适当提升佣金率以激励 |
| B级 | 有替代机构 | 续约，维持或适当下调佣金率 |
| B级 | 无替代机构 | 续约，维持佣金率 |
| C级 | 有替代机构 | 不续约，启动替换流程 |
| C级 | 无替代机构 | 有条件续约（签订绩效改善计划），6个月后复评 |
| D级 | - | 不续约，启动案件退回和新机构接替流程 |

**终止合同案件交接流程：**
1. 提前30天书面通知终止决定
2. 冻结新案件移交
3. 委外机构在30天内退回所有在催案件及相关数据
4. 确认客户数据已完全销毁（要求出具销毁证明）
5. 完成最终佣金结算
6. 将退回案件重新分配至其他委外机构或转入其他处置通道

## 最佳实践与约束

### 最佳实践

1. **多机构竞争机制**：建议同时合作2-3家委外催收机构，将同批次案件分拆分配，形成良性竞争。不同机构的回收数据可作为绩效考核的对标基准，也能降低对单一机构的依赖风险。

2. **案件-机构精准匹配**：不同委外机构通常有不同的区域优势和案件类型专长。应根据案件的地域分布、金额大小、借款人类型等特征，将案件分配给最匹配的机构。例如：属地化机构处理本地案件、大型机构处理跨区域案件。

3. **佣金结构激励设计**：可采用阶梯式佣金率激励更高回收表现：
   - 回收率 <= 5%：基础佣金率（如18%）
   - 回收率 5%-10%：中档佣金率（如20%）
   - 回收率 > 10%：高档佣金率（如23%）
   这种结构将委外机构的利益与金融机构的回收目标绑定。

4. **合规审计前不通知**：季度合规审计建议采用"不预先通知"的方式（至少部分审计项目），以获取更真实的合规状态。录音抽听应包含不同时段（上午/下午/晚间）的样本。

5. **投诉闭环管理**：因委外催收引发的客户投诉，金融机构承担最终责任。应建立"投诉即暂停"机制——收到投诉后立即暂停该案件的催收动作，调查确认后再决定是否恢复。投诉处理结果纳入委外机构的合规评分。

6. **数据安全全生命周期管控**：从数据移交（加密传输）到使用中（权限管控、日志审计）到退出销毁（数据删除确认），建立全生命周期的数据安全管控机制。合同终止后必须获取数据销毁证明。

### 约束条件

| 约束项 | 要求 |
|--------|------|
| 合规审计频率 | 委外催收机构须 **每季度** 进行一次合规审计 |
| 投诉率上限 | 委外催收投诉率不得超过 **千分之二（2‰）**，连续2个月超标须暂停新案件移交 |
| 佣金上限 | 委外催收佣金率不得超过 **25%**，特殊案件超标须法务主管审批 |
| 催收时间 | 委外机构催收时间严格限制在 **08:00-21:00** |
| 联系频次 | 单日催收联系不超过 **3次** |
| 禁止行为 | 严禁暴力催收、威胁恐吓、联系无关第三方、泄露借款人信息 |
| 资金安全 | 禁止委外机构代收代付，借款人还款须直接进入金融机构指定账户 |
| 数据安全 | 移交数据须加密传输，委外机构须签署数据保密协议，合同终止后30天内完成数据销毁 |
| 案件退回时效 | 委外催收期满或合同终止后，**30天内** 退回全部在催案件及相关数据 |
| 评分卡更新 | 委外机构评分卡须 **每季度更新**，年度形成综合评分 |

## 示例

### 示例一：委外机构月度绩效评估

**评估对象：** A催收公司（机构编号：AGC-001）
**评估期间：** 2026年3月

**绩效数据：**
```
委外案件总数：520件
委外案件总金额：3,280万元
本月回收金额：38.6万元
本月支付佣金：7.72万元（佣金率20%）
成功联系案件：228件
承诺还款案件：72件
承诺后实际还款：41件
客户投诉：1件
平均回收天数：38天
```

**KPI计算结果：**

| KPI指标 | 实际值 | 目标值 | 达标情况 |
|---------|--------|--------|----------|
| 回收率 | 1.18%（月度） | >= 2%（月度） | 未达标 |
| 佣金成本率 | 20.0% | <= 25% | 达标 |
| 案件触达率 | 43.8% | >= 40% | 达标 |
| 承诺还款率 | 31.6% | >= 30% | 达标 |
| 承诺履约率 | 56.9% | >= 50% | 达标 |
| 投诉率 | 1.92‰ | <= 2‰ | 达标（但接近红线） |
| 回收时效 | 38天 | <= 45天 | 达标 |

**评估结论：**
```
综合评分：72分（B级-良好）
趋势：较上月下降3分（上月75分）

亮点：
- 案件触达率和承诺履约率表现良好
- 回收时效较快（38天，优于目标）

关注点：
- 月度回收率1.18%，未达到2%的月度目标，需关注
- 投诉率1.92‰接近2‰红线，需加强话术合规管控
- 连续2个月回收率低于目标值

行动项：
1. 与A催收公司召开月度绩效复盘会议
2. 要求提交回收率改善计划（10个工作日内）
3. 针对投诉案件进行个案分析，强化催收人员合规培训
4. 如下月回收率仍未达标，考虑减少案件分配量
```

### 示例二：季度合规审计报告

**审计对象：** B催收公司（机构编号：AGC-002）
**审计期间：** 2026年Q1（1月-3月）

**审计结果：**

```
审计抽样量：
- 催收录音抽听：300通（总催收量的6%）
- 催收记录核查：500条（总记录的12%）
- 联系时间合规性检查：全量扫描
- 信息安全现场检查：1次

审计发现：

1. [较严重] 催收时间违规
   发现：2月期间有8通催收电话在21:00后拨出（最晚21:23）
   占比：抽样的2.7%
   处理：要求10个工作日内整改，扣减Q1佣金的5%

2. [一般问题] 话术不规范
   发现：部分催收人员未使用标准开场白（未表明机构身份和来电目的）
   占比：抽样的4.3%
   处理：书面通知整改，要求加强培训

3. [一般问题] 催收记录不完整
   发现：约7%的催收记录缺少通话结果分类标记
   处理：要求优化系统，强制填写通话结果

4. [无问题] 信息安全
   现场检查未发现数据安全隐患，数据存储加密、访问权限管控正常

审计总评：
  合规评分：78分（较上季度82分下降4分）
  主要风险：催收时间违规虽然数量不多但性质较严重，需高度重视

整改要求：
  1. 催收系统增加21:00自动锁定外呼功能（15个工作日内完成）
  2. 催收人员全员合规复训（10个工作日内完成）
  3. 催收记录系统优化（30个工作日内完成）
  4. 30日内提交整改报告
```

### 示例三：年度续约决策

**评估对象：** 三家委外催收机构年度综合评估

```
┌──────────────────┬────────┬────────┬────────┐
│     评估维度      │ A催收公司 │ B催收公司 │ C催收公司 │
├──────────────────┼────────┼────────┼────────┤
│ 回收绩效(40%)    │  68分   │  82分   │  75分   │
│ 合规表现(30%)    │  85分   │  72分   │  90分   │
│ 运营能力(15%)    │  78分   │  80分   │  70分   │
│ 配合程度(15%)    │  82分   │  75分   │  85分   │
├──────────────────┼────────┼────────┼────────┤
│ 年度综合评分      │  76.5分 │  78.1分 │  80.5分 │
│ 评级              │  B级    │  B级    │  B级    │
│ 上年评级          │  B级    │  A级    │  C级    │
│ 评级变化          │  维持   │  降级   │  升级   │
└──────────────────┴────────┴────────┴────────┘

续约决策：

A催收公司（AGC-001）：
  决策：续约
  理由：综合表现稳定，合规表现优秀（85分），但回收绩效偏低（68分）。
  调整：佣金率维持20%不变，但增加阶梯激励（回收率>10%部分佣金率提升至23%），
       激励其提升回收表现。
  附加条件：签订Q1绩效改善目标，月度回收率须达到2%。

B催收公司（AGC-002）：
  决策：有条件续约
  理由：回收绩效较好（82分）但合规表现下滑（从上年A级降至B级，
       Q1审计发现催收时间违规问题）。合规风险是核心关注点。
  调整：续约6个月（非标准12个月），签订合规改善承诺书。
       6个月后合规评分须提升至80分以上，否则终止合作。
  附加条件：Q2合规审计增加1次专项审计（催收时间合规性）。

C催收公司（AGC-003）：
  决策：续约
  理由：从上年C级提升至B级，表现明显改善。合规表现优秀（90分），
       回收绩效和运营能力仍有提升空间。
  调整：适当增加案件分配量（增加20%），佣金率维持18%。
  附加条件：下一年度如综合评分达到85分以上，升级为A级合作伙伴。
```

### 示例四：违规事件处理

**事件描述：**
收到借款人王某投诉（投诉编号：COMP-2026-0312），称B催收公司催收人员在催收电话中使用威胁性语言，声称"不还款就通知你的工作单位和邻居"。

**处理流程：**
```
1. 即时响应（收到投诉后2小时内）：
   - 暂停该案件的催收动作
   - 通知B催收公司暂停对该借款人的一切催收行为
   - 调取该案件的全部催收录音

2. 调查核实（48小时内）：
   - 听取涉事催收录音，确认催收人员确实存在以下违规行为：
     a) 威胁联系借款人工作单位（违反禁止联系无关第三方规定）
     b) 使用恐吓性语言（违反禁止暴力催收规定）
   - 调查结论：投诉属实，构成"较严重"合规违规

3. 处理措施：
   - 要求B催收公司立即解除涉事催收人员对该案件的催收权限
   - 向借款人王某致歉，承诺已停止不当催收行为
   - 对B催收公司处罚：
     a) 扣减当月佣金10%
     b) 要求涉事人员停岗培训
     c) 要求5个工作日内提交整改报告
   - 将此事件纳入B催收公司季度合规评分
   - 在投诉管理系统中关闭投诉件，48小时内向借款人反馈处理结果

4. 后续跟进：
   - 确认B催收公司整改措施落实
   - 下一季度对B催收公司增加催收录音抽听比例（从5%提升至10%）
   - 事件记录纳入B催收公司评分卡
```
