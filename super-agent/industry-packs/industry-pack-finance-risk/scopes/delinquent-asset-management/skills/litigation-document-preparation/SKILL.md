# 诉讼材料准备技能 (Litigation Document Preparation)

## 技能概述

本技能用于为通过法务可行性评估、确认进入诉讼通道的M3+逾期案件生成标准化、合规的诉讼材料包。涵盖起诉状（Complaint）、证据清单（Evidence List）、证据说明、送达地址确认书等核心法律文书的自动化生成，确保文书内容的法律准确性和格式的法院合规性。

本技能是法务处置Agent的核心执行技能之一，直接衔接法务可行性评估技能的输出结果，将评估通过的案件转化为可提交法院的完整诉讼材料包。所有材料须在 **5个工作日** 内完成准备。

### 适用场景

| 场景 | 说明 |
|------|------|
| 标准诉讼立案 | 可行性评估推荐诉讼的M3+案件，生成完整起诉材料包提交管辖法院 |
| 批量诉讼 | 针对同类型小额案件（如信用卡逾期），批量生成格式统一的起诉材料 |
| 担保追偿诉讼 | 主债务人和保证人同时起诉的案件，生成包含多被告的起诉材料 |
| 抵押物执行 | 有抵押担保的案件，生成实现担保物权的特别程序申请材料 |
| 支付令申请 | 对于债权债务关系明确、金额确定的案件，生成支付令申请书（比诉讼更快捷） |
| 诉讼保全申请 | 发现被执行人有转移财产风险时，紧急生成财产保全申请材料 |

## 执行步骤

### 步骤一：案件信息核验与材料收集

1. **核验评估报告**：确认法务可行性评估已通过审核，获取评估报告中的案件数据和推荐方案。

2. **收集原始材料**：
   - 借款合同原件/扫描件
   - 放款凭证（银行转账记录、放款确认单）
   - 借款人身份证明文件
   - 担保合同（如有）及担保人身份证明
   - 抵押登记证明（如有）
   - 逾期催收记录（短信记录、电话录音摘要、催收函送达凭证）
   - 逾期通知书/催款函及送达凭证
   - 利息及罚息计算明细

3. **法律要素确认**：
   - 确定管辖法院（被告住所地或合同履行地基层/中级法院，按标的额确定级别管辖）
   - 确定诉讼请求金额（本金 + 合法利息 + 罚息，利率不超过LPR的4倍）
   - 确定被告（借款人、担保人是否一并起诉）
   - 确定案由（金融借款合同纠纷 / 信用卡纠纷 / 保证合同纠纷等）

### 步骤二：起诉状生成

根据案件类型选择对应模板并填充内容：

**个人贷款案件起诉状模板要素：**

```
民事起诉状

原告：[金融机构全称]
住所地：[地址]
法定代表人：[姓名]，职务：[职务]
委托诉讼代理人：[律师姓名]，[律师事务所名称]律师

被告：[借款人姓名]
身份证号码：[号码]
住所地：[地址]
联系电话：[电话]

诉讼请求：
一、判令被告偿还原告借款本金人民币[金额]元；
二、判令被告支付逾期利息人民币[金额]元（暂计至[日期]，
    此后按合同约定利率计算至实际清偿之日止）；
三、判令被告支付罚息人民币[金额]元；
四、判令被告承担本案诉讼费用。

事实与理由：
[按时间线叙述：合同签订→放款→还款情况→逾期发生→催收过程→起诉必要性]

此致
[管辖法院名称]

具状人：[金融机构全称]（盖章）
[日期]
```

**经营贷款案件起诉状**补充要素：
- 如涉及企业借款人，需列明企业信息（名称、统一社会信用代码、注册地址、法定代表人）
- 贷款用途说明及是否存在贷款挪用情况

**担保贷款案件起诉状**补充要素：
- 增加保证人/抵押人作为共同被告
- 诉讼请求中增加"判令被告二（保证人）对上述债务承担连带保证责任"
- 或"判令原告对[抵押物描述]享有优先受偿权"

### 步骤三：证据清单编制

按以下标准格式编制证据清单：

| 序号 | 证据名称 | 证据形式 | 页数 | 证明目的 |
|------|----------|----------|------|----------|
| 1 | 借款合同 | 原件/复印件 | X页 | 证明原被告之间存在合法有效的借款合同关系 |
| 2 | 放款凭证 | 银行转账记录原件 | X页 | 证明原告已按合同约定履行放款义务 |
| 3 | 还款明细 | 系统导出打印件 | X页 | 证明被告的还款情况及逾期事实 |
| 4 | 利息罚息计算表 | 原告制作 | X页 | 证明诉讼请求中利息罚息金额的计算依据 |
| 5 | 催收记录 | 短信/电话记录打印件 | X页 | 证明原告已履行催收义务，被告拒绝还款 |
| 6 | 逾期催款通知书 | 原件/送达凭证 | X页 | 证明原告已通知被告逾期事实并催告还款 |
| 7 | 担保合同（如有） | 原件/复印件 | X页 | 证明保证人的担保责任 |
| 8 | 抵押登记证明（如有） | 原件 | X页 | 证明原告对抵押物享有优先受偿权 |

**证据说明书**：对每份证据的来源、真实性、关联性、合法性进行逐一说明。

### 步骤四：送达地址确认与辅助文书

1. **送达地址确认书**：
   - 核实被告在合同中确认的送达地址
   - 如合同中有"确认送达地址条款"，可直接使用该地址
   - 如无法确认地址或被告失联，准备公告送达的相关材料

2. **辅助文书准备**：
   - 授权委托书（委托律师代理诉讼）
   - 法人代表身份证明书
   - 营业执照副本复印件（加盖公章）
   - 财产保全申请书（如需同步申请保全）
   - 证据保全申请书（如电子证据需保全）

### 步骤五：质量审核与交付

1. **文书质量审核清单**：

| 审核项 | 检查内容 | 是否通过 |
|--------|----------|----------|
| 当事人信息准确性 | 姓名、身份证号、地址是否与原始材料一致 | [ ] |
| 诉讼请求计算准确性 | 本金、利息、罚息计算是否正确，利率是否合规 | [ ] |
| 事实陈述完整性 | 合同签订、放款、逾期、催收过程是否完整叙述 | [ ] |
| 证据链完整性 | 证据是否能支撑全部诉讼请求，有无缺失 | [ ] |
| 管辖法院正确性 | 级别管辖、地域管辖是否正确 | [ ] |
| 格式规范性 | 是否符合法院收文格式要求（纸张、字体、装订） | [ ] |
| 签章完整性 | 起诉状盖章、证据加盖核对章、授权委托书签章 | [ ] |
| 份数要求 | 是否按"法院1份 + 被告人数各1份 + 自留1份"准备 | [ ] |

2. **材料交付**：
   - 电子版上传至案件管理系统
   - 纸质版按法院要求装订成册
   - 交付律师或内部法务团队
   - 在案件管理系统中更新状态为"材料已准备，待立案"

## 最佳实践与约束

### 最佳实践

1. **模板化与标准化**：针对不同案件类型（个人贷款、经营贷款、担保贷款、信用卡）维护独立的起诉状模板库。模板应经法务部门审定，并根据法律法规变化定期更新（至少每半年一次）。

2. **利率计算合规性优先**：诉讼请求中的利率是法院审查重点。务必确保：
   - 合同利率不超过签订时LPR的4倍
   - 逾期罚息计算方式符合合同约定
   - 复利计算（如有）合法合规
   - 提供完整的利息计算明细表供法官核对

3. **证据链闭环**：确保证据能形成完整的证据链——从合同成立（签约）到合同履行（放款）到违约事实（逾期）到催收程序（催款）。任何一个环节的证据缺失都可能影响胜诉。

4. **送达地址前置确认**：起诉前应尽量确认被告的有效送达地址。如合同中有约定送达地址条款，应在起诉状中明确引用。公告送达会显著延长诉讼周期（增加约60天），应尽量避免。

5. **批量诉讼效率优化**：对于同类型小额案件的批量诉讼，可采用以下策略提升效率：
   - 使用统一模板、批量填充的方式生成起诉状
   - 与法院沟通集中立案
   - 申请适用小额诉讼程序（标的额不超过各省年度就业人员年平均工资30%的案件）

6. **电子证据固化**：对于APP签约、线上放款等电子证据，建议通过公证处进行证据保全或使用可信时间戳服务，增强证据效力。

### 约束条件

| 约束项 | 要求 |
|--------|------|
| 准备时效 (SLA) | 起诉材料准备须在接收评估报告后 **5个工作日** 内完成 |
| 质量审核 | 所有起诉材料必须经法务主管或委托律师审核后方可提交法院 |
| 格式规范 | 起诉状格式须符合《民事诉讼法》及各地法院立案要求 |
| 利率合规 | 诉请利率不得超过合同签订时LPR的4倍，超出部分法院不予支持 |
| 签章完备 | 起诉状须加盖公章，证据材料须加盖"与原件核对无异"章 |
| 份数要求 | 按法院要求准备正本（法院留存）+ 副本（按被告人数）+ 自存副本 |
| 保密要求 | 涉及客户个人信息的材料须按《个人信息保护法》要求进行脱敏处理（对外展示时） |
| 归档要求 | 电子版材料须同步上传案件管理系统，纸质版按档案管理规范归档 |

## 示例

### 示例一：个人贷款案件起诉材料

**案件背景：**
- 案件编号：CASE-2026-038291
- 借款人张某某逾期95天，欠款总额298,500元
- 可行性评估结果：推荐诉讼催收
- 管辖法院：XX市XX区人民法院

**生成材料包：**

1. **起诉状**（民事起诉状）
```
原告：XX银行股份有限公司XX分行
被告：张某某

诉讼请求：
一、判令被告偿还借款本金280,000元；
二、判令被告支付截至2026年4月10日的逾期利息及罚息共计18,500元，
    并支付自2026年4月11日起至实际清偿日止按年利率X%计算的利息；
三、判令被告承担本案全部诉讼费用。

事实与理由：
2025年1月15日，被告与原告签订编号为LOAN-2025-XXXX的《个人消费
贷款合同》，约定借款金额280,000元，期限24个月，年利率X%，按月
等额本息还款。合同签订后，原告于2025年1月17日将借款280,000元
发放至被告指定账户（附放款凭证）。

被告自2026年1月起未按合同约定偿还月供，截至起诉日已连续逾期3期，
逾期本金280,000元，逾期利息及罚息合计18,500元。原告多次通过短信、
电话催收（附催收记录），并于2026年2月20日向被告寄送《逾期催款
通知书》（附送达回执），但被告至今未履行还款义务。

根据合同第X条约定，借款人连续逾期超过X期，贷款人有权宣布贷款
提前到期并要求借款人立即偿还全部剩余贷款本息。
```

2. **证据清单**（共7项证据）

3. **送达地址确认书**（引用合同约定的送达地址条款）

4. **授权委托书**、法人代表身份证明书、营业执照副本

**SLA达标情况：** 2026年4月11日接收 -> 2026年4月15日完成，用时3个工作日，满足5工作日SLA。

### 示例二：担保贷款案件起诉材料（多被告）

**案件背景：**
- 案件编号：CASE-2026-045103
- 借款人王某某（经营贷款），保证人陈某某提供连带保证
- 逾期本金150,000元，总欠款165,200元
- 管辖法院：XX市人民法院

**生成材料包特殊要素：**

1. **起诉状**增加被告二：
```
被告一：王某某（借款人）
被告二：陈某某（保证人）

诉讼请求增加：
三、判令被告二对被告一的上述全部债务承担连带保证责任；

事实与理由增加：
2025年X月X日，被告二与原告签订《保证合同》（编号GUAR-2025-XXXX），
约定被告二对被告一在上述借款合同项下的全部债务承担连带保证责任，
保证期间为主债务履行期届满之日起两年。
```

2. **证据清单**增加：
   - 保证合同原件
   - 保证人身份证明文件
   - 保证人签署的相关确认文件

3. **份数调整**：所有材料按3份准备（法院1份 + 被告一1份 + 被告二1份 + 自留1份）

### 示例三：批量信用卡案件起诉材料

**案件背景：**
- 本批次共15件信用卡透支案件
- 单案金额范围：30,000-80,000元
