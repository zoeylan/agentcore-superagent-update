# 法务处置可行性评估技能 (Legal Feasibility Assessment)

## 技能概述

本技能用于对M3+（逾期60天以上）阶段的逾期案件进行系统化的法务处置可行性评估，通过多维度分析判断案件是否适合进入法律诉讼通道，并给出最优处置方案推荐。该技能是法务处置Agent的核心决策入口，所有M3+案件在进入具体处置流程之前必须先完成可行性评估。

评估的核心逻辑是**成本收益分析驱动的理性决策**：只有当预期回收金额显著高于诉讼成本时，才推荐走诉讼路径；否则应转向委外催收、NPL资产包转让或核销等替代方案。

### 适用场景

| 场景 | 说明 |
|------|------|
| M3+案件升级评估 | 智能分案Agent将逾期60天以上的案件升级至法务处置Agent时，首先触发本技能进行可行性评估 |
| 批量案件筛选 | 月度/季度对存量M3+案件进行批量重新评估，筛选适合诉讼的案件池 |
| 委外催收失败回流 | 委外催收超过约定周期未回收的案件，重新评估是否转入诉讼通道 |
| 诉讼时效预警 | 对临近诉讼时效（一般为3年）的案件进行紧急评估，避免丧失诉权 |
| 被执行人财产线索变动 | 发现被执行人新增可执行财产时，对此前评估为"不适合诉讼"的案件进行重新评估 |

## 执行步骤

### 步骤一：案件基础信息采集

1. 从案件管理系统提取案件核心数据：
   - 借款合同信息（合同编号、合同金额、贷款类型、利率、期限、担保方式）
   - 逾期信息（逾期天数、逾期本金、逾期利息、罚息、总欠款金额）
   - 债务人信息（姓名/企业名称、身份证号/统一社会信用代码、联系地址、联系方式）
   - 担保信息（保证人信息、抵押物信息、质押物信息）
   - 历史催收记录（M1/M2阶段催收结果、客户响应情况、承诺还款记录）

2. 核验合同法律效力：
   - 合同签署是否完整（签名、盖章、日期）
   - 利率是否符合法定上限（LPR的4倍）
   - 是否存在合同无效事由（欺诈、胁迫、重大误解等）
   - 诉讼时效检查（最后一次催收确认时间起算3年）

### 步骤二：被执行人财产线索查询

通过合法渠道查询被执行人名下可执行财产：

| 财产类型 | 查询渠道 | 关注指标 |
|----------|----------|----------|
| 不动产（房产） | 不动产登记中心 / 法院执行查控系统 | 产权情况、是否有抵押、是否为唯一住房 |
| 机动车辆 | 车辆管理所 / 法院执行查控系统 | 车辆数量、品牌型号、估值、是否有查封 |
| 银行存款 | 法院执行查控系统（网络查控） | 各银行账户余额、是否有冻结 |
| 股权/证券 | 工商登记系统 / 证券登记结算公司 | 持股公司及比例、证券账户市值 |
| 其他财产 | 知识产权局 / 保险公司等 | 专利权、到期保险金等 |

3. 评估可执行财产总价值，扣除以下不可执行部分：
   - 被执行人及其所扶养家属的生活必需品
   - 唯一住房（但可执行超出面积部分的价值）
   - 已被其他债权人优先受偿的抵押/质押财产
   - 已被其他法院查封冻结的财产

### 步骤三：诉讼成本测算

按以下项目逐项计算诉讼总成本：

**1. 律师费**
- 按回收金额的 5%-15% 计算（根据案件复杂度和地区标准）
- 简单个人贷款案件：5%-8%
- 复杂商业贷款/担保案件：8%-15%
- 批量案件（同一律所）可协商优惠费率

**2. 诉讼费（法院收取）**
按标的额阶梯计算：

| 标的额区间 | 费率 |
|------------|------|
| 不超过1万元 | 50元 |
| 1万-10万元 | 2.5%（减去200元） |
| 10万-20万元 | 2%（加上300元） |
| 20万-50万元 | 1.5%（加上1,300元） |
| 50万-100万元 | 1%（加上3,800元） |
| 100万-200万元 | 0.9%（加上4,800元） |
| 200万-500万元 | 0.8%（加上6,800元） |
| 500万-1000万元 | 0.7%（加上11,800元） |
| 1000万-2000万元 | 0.6%（加上21,800元） |
| 超过2000万元 | 0.5%（加上41,800元） |

**3. 执行费用**
- 保全费用：按保全金额的1%收取（最高不超过5,000元）
- 执行申请费：按执行金额的1.5%缴纳（最高不超过中级法院标准）
- 公告费、送达费：约500-2,000元/案
- 差旅费：按实际发生计算

**4. 时间成本**
- 一审周期：简易程序3个月，普通程序6个月
- 执行周期：一般6-12个月
- 总周期预估：9-18个月

### 步骤四：胜诉概率评估

根据以下维度综合评分（百分制）：

| 评估维度 | 权重 | 评估标准 |
|----------|------|----------|
| 证据链完整性 | 30% | 合同原件、放款凭证、催收记录、逾期通知送达回执是否齐全 |
| 合同有效性 | 25% | 合同条款是否合法合规、利率是否超标、是否存在霸王条款 |
| 诉讼时效 | 20% | 距诉讼时效届满的剩余时间，是否有中断/中止事由 |
| 被告抗辩风险 | 15% | 被告可能的抗辩事由（如放款未到账、扣款争议等） |
| 同类案件判例 | 10% | 同一管辖法院同类案件的历史胜诉率 |

- 综合评分 >= 80分：胜诉概率高
- 综合评分 60-79分：胜诉概率中等
- 综合评分 < 60分：胜诉概率低，不建议诉讼

### 步骤五：执行概率评估

评估胜诉后能否实际执行到位：

| 评估维度 | 权重 | 评估标准 |
|----------|------|----------|
| 可执行财产充足度 | 40% | 已查明可执行财产价值 / 欠款总额的比率 |
| 被执行人履行能力 | 25% | 是否有稳定收入来源、是否为失信被执行人 |
| 执行历史记录 | 20% | 是否已被多家法院执行、历史执行到位率 |
| 财产转移风险 | 15% | 是否存在近期大额财产转移、关联交易等逃废债行为 |

- 执行概率 >= 70%：执行前景良好
- 执行概率 40%-69%：执行有一定难度但值得尝试
- 执行概率 < 40%：执行难度大，建议替代方案

### 步骤六：成本收益分析与方案推荐

**成本收益比计算公式：**

```
预期回收金额 = 欠款总额 x 胜诉概率 x 执行概率 x 执行折扣率(通常0.7-0.9)
诉讼总成本 = 律师费 + 诉讼费 + 执行费用 + 其他费用
成本收益比 = 预期回收金额 / 诉讼总成本
```

**处置方案推荐矩阵：**

| 条件 | 推荐方案 |
|------|----------|
| 成本收益比 >= 2:1 且胜诉概率高且执行概率 >= 70% | **诉讼催收** |
| 成本收益比 >= 2:1 但执行概率 40%-69% | **诉讼催收 + 财产保全**（先保全再起诉） |
| 成本收益比 1:1 - 2:1 或执行概率较低 | **委外催收**（佣金率15%-25%） |
| 欠款金额较大但无可执行财产、未来可能恢复执行能力 | **NPL资产包转让**（按本金2-8折定价） |
| 欠款金额小、无可执行财产、诉讼时效将至/已过 | **核销申报** |

## 最佳实践与约束

### 最佳实践

1. **财产查询时效性**：被执行人财产状况随时变化，财产查询结果有效期建议不超过30天。超过30天的评估应重新查询财产线索。

2. **批量评估优先级排序**：对大批量M3+案件进行评估时，优先处理以下类型：
   - 诉讼时效即将届满（剩余不足6个月）的案件
   - 近期发现新增财产线索的案件
   - 欠款金额大（> 50万元）且有担保的案件

3. **担保链条分析**：对于有担保人的案件，应同时评估主债务人和担保人的可执行财产，选择执行能力最强的被告。

4. **成本收益动态更新**：律师费费率、法院诉讼费标准可能随政策调整而变化，应定期更新成本测算参数。

5. **判例参考**：评估胜诉概率时，应参考同一管辖法院近2年同类案件的判决趋势，特别关注利率认定、违约金调整等争议焦点的裁判倾向。

### 约束条件

| 约束项 | 要求 |
|--------|------|
| 成本收益比底线 | 诉讼成本收益比低于 1:2 的案件原则上不建议诉讼 |
| 评估覆盖率 | M3+案件法务评估覆盖率必须达到 100% |
| 评估时效 | 单案评估应在接收后 3 个工作日内完成 |
| 合规要求 | 财产查询必须通过合法渠道，禁止使用非法信息获取手段 |
| 利率合规 | 起诉标的中利率部分不得超过合同签订时LPR的4倍 |
| 审批要求 | 评估报告需经法务主管审核确认后方可执行 |
| 数据保密 | 被执行人财产信息属于敏感数据，仅限授权人员查阅 |

## 示例

### 示例一：推荐诉讼催收

**输入案件：**
- 案件编号：CASE-2026-038291
- 借款人：张某某（个人）
- 贷款类型：个人消费贷款，无担保
- 贷款本金：280,000元，逾期天数：95天
- 总欠款金额：298,500元（含逾期利息及罚息）
- M1/M2催收结果：M1阶段电话可接通但拒绝还款，M2阶段承诺还款后违约2次，目前失联

**评估过程：**
- 财产查询：名下1套房产（市值约180万元，有银行抵押贷款余额90万元，净值约90万元）、1辆车（估值约8万元）、银行存款约1.2万元
- 可执行财产总价值：约100万元（远超欠款金额）
- 诉讼成本：律师费约17,900元（6%费率）、诉讼费5,475元、保全费2,985元、其他费用约1,500元，合计约27,860元
- 胜诉概率：92分（证据链完整、合同有效、诉讼时效充裕）
- 执行概率：78%（有充足可执行财产，非失信被执行人）
- 预期回收金额：298,500 x 92% x 78% x 85% = 182,100元
- 成本收益比：182,100 / 27,860 = 6.54:1

**输出推荐：**
```
处置方案：诉讼催收（litigate）
优先级：常规（normal）
推荐理由：被执行人名下有充足可执行财产（房产净值约90万元），证据链完整，
胜诉概率高（92分），成本收益比6.54:1，远超2:1的诉讼门槛。
下一步行动：
  1. 发送诉前律师函（给予15天还款宽限期）
  2. 若无回应，启动诉讼材料准备流程
  3. 建议同步申请财产保全（冻结银行账户）
风险提示：房产有银行抵押，需评估优先受偿顺序；借款人已失联，送达可能需要公告送达（增加2个月周期）。
```

### 示例二：推荐NPL资产包转让

**输入案件：**
- 案件编号：CASE-2026-041872
- 借款人：李某某（个人）
- 贷款类型：信用卡透支
- 逾期本金：52,000元，逾期天数：210天
- 总欠款金额：58,700元
- 催收记录：长期失联，M1/M2均未联系到本人，紧急联系人表示不知其去向

**评估过程：**
- 财产查询：名下无房产、无车辆、银行账户余额合计不足500元、无股权
- 可执行财产总价值：约500元
- 诉讼成本测算：律师费约4,700元、诉讼费1,168元、其他费用约1,000元，合计约6,868元
- 胜诉概率：85分（证据完整）
- 执行概率：12%（无可执行财产，被执行人失联）
- 预期回收金额：58,700 x 85% x 12% x 80% = 4,790元
- 成本收益比：4,790 / 6,868 = 0.70:1（低于1:1，诉讼将净亏损）

**输出推荐：**
```
处置方案：NPL资产包转让（npl_transfer）
优先级：低优先级（low）
推荐理由：被执行人名下无可供执行财产，长期失联，执行概率仅12%。
诉讼成本收益比0.70:1，诉讼将产生净亏损。建议纳入下一批NPL资产包
转让清单，按本金3-5折定价（参考同类信用卡逾期案件的市场转让价格）。
下一步行动：
  1. 标记案件为"待打包转让"
  2. 纳入季度NPL资产包转让清单
  3. 同步评估是否满足核销条件
风险提示：如后续发现被执行人新增财产线索，应重新评估诉讼可行性。
```

### 示例三：推荐委外催收

**输入案件：**
- 案件编号：CASE-2026-045103
- 借款人：王某某（个人）
- 贷款类型：经营贷款，保证人担保
- 逾期本金：150,000元，逾期天数：120天
- 总欠款金额：165,200元
