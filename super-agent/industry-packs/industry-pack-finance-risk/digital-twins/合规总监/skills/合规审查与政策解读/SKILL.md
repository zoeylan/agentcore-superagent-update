# 合规审查与政策解读

## 技能描述
对信贷业务方案、产品设计、合作模式等进行全面合规审查，并对最新监管政策进行专业解读，评估对现有业务的影响。

## 适用场景
- 新产品/新业务上线前的合规审查
- 监管新规发布后的影响评估与业务调整建议
- 助贷、联合贷款等合作模式的合规性审查
- 利率定价、费用收取方式的合规性分析
- 营销宣传材料的合规审核
- 格式合同条款的合规性审查

## 工作流程

### 第一步：业务场景梳理
请用户提供需要审查的业务方案、产品设计文档或需要解读的监管政策文件。明确以下信息：
- 业务类型（信贷、助贷、联合贷、催收等）
- 涉及的主要业务环节
- 合作方情况（如有）
- 目标客群（如适用）

### 第二步：法规映射与合规分析
针对业务方案的每个关键环节，逐一进行合规审查：

1. **准入合规**：审查业务资质、经营范围、地域限制
2. **产品合规**：利率定价（IRR口径≤LPR×4）、期限、额度、还款方式
3. **流程合规**：客户身份验证、授信审批、合同签署、资金划付
4. **披露合规**：年化利率明示（须用IRR）、费用透明化、风险提示
5. **合作合规**：助贷机构准入、核心风控不外包、出资比例要求
6. **催收合规**：催收方式、时间、频次、外包催收机构管理

### 第三步：出具合规审查意见
按以下结构输出：

```
## 合规审查意见书

### 一、审查结论
[合规 / 基本合规(需整改) / 存在重大合规风险]

### 二、适用法律法规及监管文件
[逐一列出适用的法规及具体条文]

### 三、合规风险点分析
[按风险等级排序：高/中/低]
每个风险点包含：
- 风险描述
- 违反条款
- 可能的监管后果
- 整改建议

### 四、合规建议
[可操作的具体整改措施和时间建议]

### 五、关注事项
[灰色地带提示、监管趋势预判、后续跟踪事项]
```

## 关键知识要点

### 信贷业务核心法规清单
- 《商业银行互联网贷款管理暂行办法》（银保监发〔2020〕9号）
- 《关于进一步规范商业银行互联网贷款业务的通知》（银保监办发〔2021〕24号）
- 《关于加强商业银行互联网贷款业务管理提升金融服务质效的通知》（2024年）
- 《个人贷款管理暂行办法》（中国银行业监督管理委员会令2010年第2号）
- 《消费金融公司试点管理办法》及历次修订

### 利率合规红线
- 贷款年化利率必须以IRR（内部收益率法）计算并向借款人明示
- 不得通过收取咨询费、服务费等方式变相提高实际利率
- 民间借贷利率司法保护上限：LPR×4（当前约为13.8%）
- 持牌机构虽不直接适用民间借贷利率上限，但实务中法院判决趋势倾向于参照适用

### 助贷/联合贷合规要点
- 商业银行与合作机构共同出资发放贷款，单笔中合作方出资比例不低于30%
- 商业银行不得将贷款风险评估、授信审批等核心风控环节外包
- 与单一合作方的联合贷款余额不超过本行一级资本净额的25%
- 助贷机构不得以任何形式向借款人收取息费

## 注意事项
- 合规审查意见不替代正式法律意见书，重大事项建议同步咨询外部律所
- 监管政策存在地方差异，需关注属地监管局的窗口指导
- 合规审查应贯穿产品全生命周期，非一次性行为
