## 业务范围（Scopes）

### 📝 合同审查与管理

覆盖合同从需求发起到终止归档的全生命周期管理，是企业法务最高频、最核心的工作场景。包括合同起草、条款审查、审批流转、签署管理、履约跟踪、到期管理等环节。

**Agent 团队：**

| Agent | 职责 |
|-------|------|
| 合同条款审查专家 | 对合同文本逐条审查，识别不利条款并标注风险等级，输出修改建议和替代条款方案 |
| 合同起草助手 | 基于业务需求和标准模板库生成合同初稿，管理和维护合同模板库 |
| 合同风控评估师 | 综合条款审查结果、交易对手信用等信息，出具整体风险评级和法律意见书 |
| 履约跟踪管理员 | 管理合同签署后的履约监控、到期预警和归档管理 |

**核心技能：** 合同起草、条款分析、风险评级、法律意见书撰写、合同台账管理、到期续约管理、交易对手风险评估、履约监控等 13 项技能。

### 🛡️ 合规审计

企业合规体系的建设、运行和持续改进。覆盖反商业贿赂、数据合规（GDPR/个保法/数据安全法）、反洗钱、出口管制与制裁、行业特定合规要求等核心合规领域，遵循 ISO 37301 合规管理体系标准和"三道防线"模型。

**Agent 团队：**

| Agent | 职责 |
|-------|------|
| 合规风险扫描引擎 | 持续监控法规变更和企业行为，自动识别合规风险暴露点并分级预警 |
| 合规审计执行官 | 执行定期和专项合规审计，出具审计报告和整改建议，跟踪整改直至关闭 |
| 合规制度架构师 | 设计和维护企业合规制度体系（总纲→专项制度→操作指引三级架构） |
| 合规培训督导 | 管理全员合规培训计划的执行、考核和效果评估 |

**核心技能：** 合规风险评估、法规变更监控、审计计划制定、审计执行管理、审计报告撰写、合规制度设计、合规政策起草、异常检测分析、举报调查、培训计划制定等 13 项技能。

### 💡 知识产权管理

企业知识产权（IP）资产的全生命周期管理，包括专利申请与维护、商标注册与监控、著作权登记、商业秘密保护体系建设、知识产权侵权应对。目标是构建企业 IP 资产组合的竞争护城河。

**Agent 团队：**

| Agent | 职责 |
|-------|------|
| 专利战略分析师 | 负责专利检索分析、布局策略制定和申请流程管理，提供 FTO 分析 |
| IP 资产运维管理员 | 管理全部 IP 资产的日常运维、费用管理和状态监控，确保零遗漏 |
| IP 维权应对专家 | 处理知识产权侵权的发现、证据保全和维权策略制定 |

**核心技能：** 专利检索分析、专利申请策略、竞争对手专利监测、商标注册管理、IP 资产台账管理、时限监控、侵权监测、证据保全、维权策略、OA 答复管理等 12 项技能。

## 定制化建议

本解决方案包基于通用法律合规场景设计，建议根据企业实际情况进行以下调整：

### 合同审查模块
- **审批层级**：调整合同金额审批阈值（当前设置：≥50万法务审核，≥200万法务总监审核）以匹配企业实际授权体系
- **合同模板库**：补充企业特有的合同类型模板（如行业专属协议）
- **审查时效**：根据团队人力调整标准合同/非标合同的审查时限承诺

### 合规审计模块
- **合规领域优先级**：根据企业所在行业调整重点合规领域（如金融企业加强反洗钱、互联网企业加强数据合规）
- **审计频次**：根据企业风险偏好调整高/中/低风险领域的审计周期
- **礼品招待限额**：调整反商业贿赂的金额审批线（当前设置：500元）
- **培训计划**：根据员工规模和岗位分布定制培训频次和考核标准

### 知识产权模块
- **IP 类型侧重**：根据企业类型调整（科技公司侧重专利、消费品牌侧重商标、软件公司侧重著作权）
- **地域策略**：根据海外市场布局调整 PCT 申请路径和商标注册地域
- **代理机构对接**：配置与外部专利/商标代理机构的协作接口
- **年费策略**：根据专利组合价值制定差异化的年费缴纳/放弃策略

### 数字孪生
- **行业知识补充**：根据企业所在行业补充行业特定法规和实务经验
- **决策偏好调整**：根据企业风险偏好调整法务总监的风险评估阈值
- **企业制度导入**：将企业现有的法务制度和历史案例导入数字孪生知识库
