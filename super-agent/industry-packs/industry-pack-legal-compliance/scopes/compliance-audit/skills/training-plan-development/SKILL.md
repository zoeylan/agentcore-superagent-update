# 合规培训计划制定 (Training Plan Development)

## 概述

本技能用于制定年度合规培训计划，涵盖培训需求分析、课程体系设计、培训日历编排、讲师和资源安排。确保合规培训系统性覆盖全员通识和关键岗位专项需求，与合规风险评估和审计发现紧密挂钩。

## 适用场景

- 年度合规培训计划编制（每年Q4为次年制定）
- 新法规发布后的专项培训快速部署
- 合规事件发生后的针对性培训安排
- 新员工入职合规培训体系设计
- 关键岗位轮岗/晋升的合规培训要求制定

## 执行步骤

### 步骤1：培训需求分析
- 输入来源：
  - 年度合规风险评估结果（高风险领域需加强培训）
  - 审计发现（合规意识薄弱的部门/领域）
  - 法规变更（新法规的宣贯需求）
  - 合规事件（事后教育的需求）
  - 管理层要求和员工反馈
- 确定培训受众分类：
  - 全员（通识培训）
  - 管理层（合规领导力）
  - 关键岗位（采购/销售/财务/IT/HR等）
  - 新员工（入职培训）

### 步骤2：课程体系设计
- 通识必修课程：
  - 企业合规管理基本制度（年度）
  - 反商业贿赂基本规则（年度）
  - 数据保护与信息安全基础（年度）
  - 举报渠道和保护制度（入职+年度）
- 关键岗位专项课程：
  - 采购岗：供应商合规审查、利益冲突管理
  - 销售岗：反商业贿赂实务、客户尽职调查
  - 财务岗：反洗钱识别、异常交易报告
  - IT岗：数据安全操作规范、访问控制
  - HR岗：劳动合规、反歧视/骚扰
- 管理层课程：合规治理责任、合规文化建设、合规风险决策

### 步骤3：培训日历编排
- 月度培训主题规划（与业务周期匹配）
- 培训形式组合：
  - 线上课程（E-learning，灵活安排）
  - 线下集中培训（季度，互动讨论）
  - 微学习（每周5分钟合规知识推送）
  - 案例研讨会（结合真实案例）
  - 模拟演练（合规决策场景模拟）
- 时长控制：通识培训2-4小时/年，专项培训8-16小时/年
- 避免与业务高峰期冲突

### 步骤4：资源配置
- 讲师安排：内部合规团队/外部律所/行业专家
- 教材开发计划和预算
- 培训平台/系统需求
- 考核工具和题库准备

### 步骤5：计划审批与执行准备
- 编制年度培训计划书
- 预算编列和审批
- 与各部门沟通协调（参训时间确认）
- 建立执行监控机制（完成率跟踪）

## 输入规格

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 风险评估结果 | 年度合规风险评估 | 是 |
| 审计发现 | 合规薄弱环节 | 是 |
| 员工名册 | 含部门/岗位/入职日期 | 是 |
| 法规变更清单 | 年度新增法规要求 | 是 |
| 上年培训评估 | 效果和反馈数据 | 否 |

## 输出规格

| 输出项 | 格式 | 说明 |
|--------|------|------|
| 年度培训计划书 | 正式文档 | 含目标、课程、日历、预算 |
| 月度培训日历 | 日历/表格 | 具体时间安排 |
| 课程大纲集 | 文档集 | 各课程的教学设计 |
| 预算方案 | 表格 | 分项预算明细 |

## 最佳实践

1. **风险驱动**：培训重点与合规风险等级挂钩，而非均匀分配
2. **分层施教**：不同受众用不同深度和语言
3. **实用为王**：案例教学优于理论灌输，让学员知道"遇到这种情况怎么做"
4. **持续而非突击**：微学习+集中培训结合，避免一年一次的走过场
5. **效果导向**：培训目标不是"完成培训"而是"改变行为"

## 约束条件

- 培训覆盖率必须达到100%（含补训机制）
- 新员工入职30天内必须完成首次合规培训
- 关键岗位专项培训频率不低于年度一次
- 培训记录须完整保存，作为合规体系有效性证明
- 培训内容须与最新制度保持同步
