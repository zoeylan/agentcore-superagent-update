# 时限节点监控与提醒

## 概述

时限节点监控是IP资产运维中最关键的风险控制环节。知识产权领域存在大量法定时限，一旦逾期可能导致权利丧失且不可逆转（如专利年费逾期超过6个月宽限期、商标续展逾期超过6个月宽展期）。本技能通过多层次的预警提醒机制，确保所有法定时限的零逾期管理。

## 适用场景

- 专利年费缴纳期限管理
- 商标续展/宽展期限管理
- OA答复法定期限监控
- PCT进入国家阶段30个月期限
- 优先权期限管理（12个月/6个月）
- 驳回后复审3个月期限
- 无效宣告答辩期限
- 域名续费到期管理

## 执行步骤

### 步骤1：时限录入与计算
- 从台账中提取所有需要监控的时限节点
- 计算精确的到期日（注意法定节假日顺延规则）
- 区分硬性deadline（逾期丧失权利）和弹性deadline（可延期/恢复）
- 对于新入库的资产，立即计算并录入所有相关时限
- 特别注意：
  - 专利年费：按授权日对应的每年日期计算
  - 商标续展：注册有效期满前12个月至满后6个月
  - OA答复：以通知书发文日起算（非收到日）

### 步骤2：多层提醒机制设置
- **第一层提醒**（提前90天）：列入待办清单，确认缴费预算
- **第二层提醒**（提前60天）：正式提醒通知，启动缴费流程
- **第三层提醒**（提前30天）：紧急提醒，要求确认执行状态
- **第四层提醒**（提前7天）：最后期限警告，升级通知管理层
- **逾期处理**：立即触发恢复程序（如适用），同时上报
- 对于OA答复等短期限事项，调整为：收到即通知→提前30天→提前15天→提前7天

### 步骤3：执行状态跟踪
- 每个时限节点设置状态：待处理→处理中→已完成→已确认
- 跟踪缴费/续展的实际执行情况（是否已递交、是否已收到回执）
- 对于委托代理机构办理的事项，确认代理机构已收到指令并执行
- 定期（每周）检查所有"处理中"状态的事项进展

### 步骤4：异常处理
- **逾期预防**：任何到期前7天仍未完成的事项，自动升级至管理层
- **逾期恢复**：
  - 专利年费：6个月宽限期内缴纳滞纳金恢复
  - 商标续展：6个月宽展期内缴纳宽展费续展
  - OA答复：可申请恢复权利（需正当理由）
- **权利丧失评估**：如确实逾期无法恢复，评估影响并记录为管理事故
- **根因分析**：任何逾期事件必须进行根因分析并改进流程

### 步骤5：统计与报告
- 月度时限管理报告：本月到期事项总数、完成率、预警次数
- 季度趋势分析：时限管理效率趋势、费用支出趋势
- 未来90天预测：预告未来三个月将到期的事项清单
- 年度费用预算：基于台账数据预测来年年费/续展总费用

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| IP资产台账 | 包含所有资产到期日信息 | 是 |
| 新增时限事项 | OA通知、续展提醒等 | 是 |
| 执行反馈 | 缴费/续展的完成确认 | 是 |
| 法定节假日日历 | 用于到期日顺延计算 | 是 |

## 输出

| 输出项 | 说明 |
|--------|------|
| 到期提醒通知 | 分级别的预警通知 |
| 时限管理日历 | 可视化的时间线视图 |
| 月度时限管理报告 | 本月执行统计 |
| 未来到期预告 | 滚动90天的到期清单 |
| 年费/续展预算 | 费用预测 |

## 最佳实践

1. **零逾期原则**：这是不可妥协的底线，任何逾期都是管理事故
2. **双保险机制**：关键时限同时设置系统提醒和人工日历提醒
3. **提前量充足**：留出足够缓冲时间应对突发情况（代理机构延误、节假日等）
4. **确认闭环**：提醒发出后必须获得"已执行"确认才能关闭
5. **费用预拨**：根据年度费用预测提前做好预算安排
6. **代理机构协同**：与代理机构建立双重提醒机制，互为备份

## 约束条件

- 年费/续展缴纳逾期率目标：0%
- OA答复时效达标率目标：100%
- 提醒通知必须在设定时间点的当天发出
- 逾期事件必须在24小时内启动应急处理
- 所有时限计算必须考虑法定节假日顺延
- 海外资产的时限需按当地时区和假日计算
