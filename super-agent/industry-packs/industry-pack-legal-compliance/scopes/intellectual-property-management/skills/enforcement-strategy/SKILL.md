# 维权策略制定

## 概述

维权策略制定是在完成证据保全后，综合评估法律、商业、成本等多维度因素，选择最优维权路径并制定详细执行计划的决策环节。维权不是简单的"起诉"决定，而是需要在多种维权手段中做出最符合企业利益的战略选择。

## 适用场景

- 确认专利侵权后的维权路径选择
- 商标侵权/假冒的维权决策
- 著作权侵权的维权方案制定
- 商业秘密泄露的法律应对
- 多侵权方/多地域的系统维权规划
- 维权与商业合作的平衡决策

## 执行步骤

### 步骤1：维权基础评估
- **权利稳定性评估**：
  - 我方专利被无效的风险有多大？（检索现有技术、审查历史）
  - 商标是否存在被撤三或无效的风险？
  - 权利是否在有效期内？年费/续展是否正常？
- **侵权确定性评估**：
  - 侵权比对分析的确信度（全面覆盖/等同侵权/边界情况）
  - 对方可能的不侵权抗辩（现有技术抗辩/先用权/合法来源）
- **证据充分性评估**：
  - 证据链是否完整？能否证明侵权行为存在？
  - 能否证明侵权规模和损害程度？
  - 证据的合法性是否可能被挑战？

### 步骤2：维权路径分析
- **行政投诉路径**：
  - 优势：处理速度快（3-6个月）、成本低、可实现快速制止
  - 劣势：无经济赔偿、处罚力度有限、对方可能行政复议
  - 适用：需要快速制止侵权、侵权规模小、诉讼成本过高的情况
  - 渠道：市场监管局（商标）、知识产权局（专利）、版权局（著作权）
- **民事诉讼路径**：
  - 优势：可获经济赔偿（惩罚性赔偿最高5倍）、可申请禁令、判决强制执行
  - 劣势：周期长（12-18个月一审）、成本高、结果不确定
  - 适用：侵权规模大、需要经济赔偿、需要建立判例先例
  - 管辖：知识产权法院/知识产权法庭
- **刑事报案路径**：
  - 适用条件：假冒注册商标（≥5万元）、侵犯商业秘密（≥50万元损失）
  - 优势：威慑力最强、公安机关调查取证能力强
  - 适用：故意侵权、规模巨大、造成严重损害
- **协商和解/许可**：
  - 优势：速度最快、保护商业关系、可获即时利益
  - 劣势：可能被视为软弱、无法建立威慑
  - 适用：对方有合作价值、侵权非故意、双方有互补资源

### 步骤3：成本收益分析
- **成本预算**：
  - 律师费（按小时/按件/风险代理）
  - 官费（诉讼费、保全费、鉴定费）
  - 取证费用（已发生和预计追加的）
  - 企业内部人力和时间成本
  - 可能的反诉应对成本
- **预期收益**：
  - 经济赔偿预期（法定赔偿/实际损失/侵权获利/许可费倍数）
  - 禁令效果（制止侵权带来的市场份额恢复）
  - 威慑效应（对其他潜在侵权者的震慑）
  - 商业价值（许可费收入/和解金额）
- **风险因素**：
  - 败诉风险及后果
  - 被反诉/被提无效的风险
  - 舆论风险（被指"专利流氓"）
  - 商业关系破裂的损失

### 步骤4：策略方案制定
- 确定推荐的维权路径（可以是组合策略）
- 制定执行时间表（关键节点和里程碑）
- 明确所需资源和预算
- 设定谈判底线和最优目标
- 制定Plan B（如首选路径受阻的备选方案）
- 明确需要管理层审批的事项

### 步骤5：策略审批与执行启动
- 撰写《维权策略建议书》提交管理层
- 内容包含：案件概述、权利基础、侵权事实、路径分析、成本收益、推荐方案
- 获得审批后启动执行：
  - 行政路径→准备投诉材料
  - 诉讼路径→委托律师/律所
  - 和解路径→发出律师函/启动谈判
- 设定阶段性评估节点，根据进展动态调整策略

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 证据保全结果 | 完整的证据材料 | 是 |
| 侵权比对分析 | 技术特征对比结论 | 是 |
| 侵权方信息 | 企业规模、财务状况、商业关系 | 是 |
| 我方权利文件 | 专利/商标/著作权证书和审查档案 | 是 |
| 预算限额 | 可投入的维权预算 | 否 |

## 输出

| 输出项 | 说明 |
|--------|------|
| 维权策略建议书 | 完整的策略分析和推荐方案 |
| 成本收益分析表 | 各路径的费用和预期收益对比 |
| 执行时间表 | 维权行动的关键节点计划 |
| 风险评估报告 | 各种风险及缓释措施 |

## 最佳实践

1. **组合策略**：多路径并行往往效果最好（如行政+诉讼+谈判同时推进）
2. **先礼后兵**：除非紧急情况，可先发律师函试探对方态度
3. **权利先行**：启动诉讼前确保权利基础稳固（主动做稳定性检索）
4. **战略诉讼**：选择有示范效应的案件重点投入，形成行业威慑
5. **商业思维**：维权是手段不是目的，最终服务于商业利益
6. **动态调整**：根据对方反应和案件进展及时调整策略

## 约束条件

- 维权预算超过10万元必须提交管理层审批
- 发出律师函前需经法务负责人确认
- 启动诉讼前必须完成权利稳定性评估
- 和解方案（涉及许可/赔偿金额）需管理层批准
- 涉及刑事报案的决策需法务总监和管理层共同批准
- 所有维权行动记录归档保存不少于10年
