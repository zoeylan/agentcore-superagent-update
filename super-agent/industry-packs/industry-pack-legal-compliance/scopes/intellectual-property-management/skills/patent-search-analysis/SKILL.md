# 专利检索分析

## 概述

专利检索分析是专利申请和侵权判定的基础性工作，通过系统化的检索策略在全球专利数据库中查找相关现有技术（Prior Art），评估技术方案的新颖性和创造性，为专利申请决策和FTO（自由实施）分析提供依据。

## 适用场景

- 研发成果提交技术交底书后的可专利性评估
- 专利申请前的新颖性/创造性检索
- 产品上市前的FTO（Freedom to Operate）分析
- 专利无效宣告中的现有技术证据收集
- 竞争对手技术趋势分析

## 执行步骤

### 步骤1：技术方案理解与分解
- 仔细阅读技术交底书或待分析的技术方案
- 提取核心技术特征（必要技术特征 vs 附加技术特征）
- 确定技术领域、技术问题和技术效果
- 识别可能的关键创新点（区别技术特征）

### 步骤2：检索策略制定
- 确定检索数据库（至少3个）：
  - 中国：CNIPA专利检索系统
  - 美国：USPTO PatFT/AppFT
  - 国际：WIPO PATENTSCOPE、Espacenet
  - 商业库：Derwent Innovation、Innography（如有权限）
- 构建检索式：
  - 关键词检索：技术术语 + 同义词 + 上下位概念
  - 分类号检索：IPC/CPC分类号锁定技术领域
  - 组合检索：关键词 AND/OR 分类号
  - 引用检索：从已知相关专利的引用/被引出发
- 每次检索至少构建5组以上不同的检索式

### 步骤3：检索执行与筛选
- 按照检索策略逐一执行检索
- 对检索结果按相关度初步排序
- 快速浏览摘要，筛选高相关文献（通常控制在20-50篇）
- 对高相关文献进行全文阅读
- 记录每条检索式的命中数量和筛选结果

### 步骤4：对比分析
- **新颖性分析**：将技术方案的每个技术特征与对比文件逐一对比
  - 单独对比原则（一份文件公开全部技术特征→丧失新颖性）
  - 注意公开方式：文字描述、附图隐含公开、数值范围公开
- **创造性分析**：
  - 确定最接近的现有技术
  - 识别区别技术特征
  - 分析区别技术特征解决的技术问题
  - 判断是否存在技术启示（其他文件是否公开了将区别特征应用于最接近现有技术的启示）

### 步骤5：检索报告编写
- 报告结构：检索目的→技术方案摘要→检索策略→检索结果→对比分析→结论建议
- 列出所有相关对比文件（按相关度A/X/Y分级）
- 给出明确的新颖性和创造性评价结论
- 提供申请建议（建议申请/修改后申请/不建议申请）

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 技术交底书 | 包含技术方案详细描述的文档 | 是 |
| 技术领域说明 | IPC/CPC分类号建议 | 否 |
| 检索范围要求 | 时间范围、地域范围、数据库指定 | 否 |
| 已知相关文献 | 发明人已知的相关专利或论文 | 否 |

## 输出

| 输出项 | 说明 |
|--------|------|
| 专利检索报告 | 完整的结构化检索报告 |
| 对比文件清单 | 相关文献列表及相关度评级 |
| 新颖性/创造性评估结论 | 三性评价结果 |
| 申请建议 | 可申请性建议及权利要求调整方向 |

## 最佳实践

1. **检索式迭代**：初次检索后根据结果调整检索式，扩展或缩小范围
2. **注意非专利文献**：学术论文、产品手册、标准规范也构成现有技术
3. **时间节点把控**：检索截止日以申请日/优先权日为准
4. **多语言检索**：核心技术需要跨中英日韩语言检索
5. **检索记录留痕**：每条检索式、每个数据库的检索日期必须记录
6. **避免确认偏差**：不能只检索支持"有创造性"的证据，必须全面客观

## 约束条件

- 检索数据库覆盖不少于3个
- 每次检索构建检索式不少于5组
- 检索报告必须在收到技术交底书后5个工作日内出具
- 对比文件相关度评级必须使用标准分类（X/Y/A/E）
- 涉及国防专利或保密技术时需走特殊审批流程
