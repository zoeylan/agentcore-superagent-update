# 证据保全

## 概述

证据保全是知识产权维权成功的关键前置环节。在确认侵权后48小时内启动证据保全程序，通过法律认可的方式固定侵权事实、侵权范围和损害程度的证据，确保后续行政投诉或司法诉讼中证据的合法性、真实性和关联性。证据保全的质量直接决定维权的成败。

## 适用场景

- 确认专利侵权后的技术证据固定
- 商标假冒/仿冒产品的购买取证
- 线上侵权网页/店铺的证据固定
- 软件著作权侵权的代码比对证据
- 商业秘密泄露的证据收集
- 诉前证据保全申请（紧急情况）

## 执行步骤

### 步骤1：证据保全方案制定
- 分析需要固定的证据类型：
  - 侵权行为存在的证据（产品/网页/广告）
  - 侵权规模的证据（销售量/下载量/覆盖范围）
  - 侵权获利或我方损失的证据（价格/销售数据）
  - 侵权方身份信息的证据（工商信息/网站备案）
- 选择证据保全方式（按证明力从高到低）：
  - 公证取证（证明力最强，法院直接采信）
  - 诉前证据保全（法院裁定，适用于可能灭失的证据）
  - 行政执法取证（市场监管局现场检查）
  - 可信时间戳存证（快速、低成本、电子证据）
  - 区块链存证（新兴方式，部分法院认可）
- 制定保全时间表和预算

### 步骤2：公证取证执行
- **实物购买公证**（产品侵权）：
  - 联系公证处预约公证员
  - 公证员全程见证购买过程
  - 记录：购买时间、地点、店铺信息、价格、交易凭证
  - 产品封存并由公证处保管或出具保管证明
  - 公证书应包含完整的照片记录
- **网页公证**（线上侵权）：
  - 公证员在公证处使用公证处电脑操作
  - 记录完整URL、访问时间、页面内容
  - 逐页截屏并打印
  - 注意：需要显示电脑时间和网络连接状态
- **现场公证**（制造/仓储端侵权）：
  - 配合行政执法或律师调查
  - 记录现场环境、侵权产品数量、生产设备

### 步骤3：电子证据固定
- **可信时间戳存证**：
  - 使用TSA（时间戳认证机构）对电子文件加盖时间戳
  - 适用于：网页截图、电商交易记录、聊天记录
  - 优势：即时操作、成本低、可远程完成
  - 注意：需要选择司法认可的时间戳服务机构
- **区块链存证**：
  - 将证据哈希值上链，确保不可篡改
  - 部分互联网法院（如杭州、北京、广州）已认可
- **录屏取证**：
  - 全程录屏记录网页浏览和操作过程
  - 需要显示系统时间和网络信息
  - 作为公证的补充或临时保全手段

### 步骤4：证据链完整性检查
- 检查证据是否覆盖侵权认定要件：
  - 专利侵权：侵权产品+技术特征对比+我方专利有效证明
  - 商标侵权：侵权标识+使用商品/服务+我方商标注册证
  - 著作权侵权：侵权作品+对比分析+我方著作权登记证
- 确保证据来源合法（不得通过违法手段获取）
- 确保证据时间线逻辑自洽
- 确保每份证据与待证事实之间的关联性清晰
- 制作证据清单和编号目录

### 步骤5：证据保管与移交
- 实物证据：密封保管，标注取证时间和经手人
- 电子证据：原始文件+哈希校验值，存储在安全介质
- 公证书：原件妥善保管，复印件用于日常参考
- 建立证据保管链记录（Chain of Custody）
- 移交给维权执行人员或律师时签署交接记录

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 侵权线索报告 | 已确认的侵权事实 | 是 |
| 侵权方信息 | 侵权者的身份和联系方式 | 是 |
| 我方权利文件 | 专利证书/商标注册证/著作权证 | 是 |
| 维权策略方向 | 决定证据保全的重点和范围 | 否 |

## 输出

| 输出项 | 说明 |
|--------|------|
| 证据保全方案 | 保全方式、时间计划、预算 |
| 公证书 | 公证处出具的正式公证文书 |
| 电子存证证书 | 时间戳/区块链存证凭证 |
| 证据清单 | 全部证据的编号目录 |
| 证据链分析报告 | 证据完整性和证明力评估 |

## 最佳实践

1. **速度第一**：线上证据随时可能被删除，发现后尽快保全
2. **多重保全**：重要证据采用公证+时间戳双重固定
3. **全流程记录**：取证过程本身也需要被记录（录像/拍照）
4. **合法合规**：陷阱取证（引诱侵权）可能不被法院采信
5. **专业协助**：重大案件建议聘请专业取证律师参与
6. **预算控制**：根据案件价值合理控制取证成本

## 约束条件

- 确认侵权后48小时内必须启动证据保全
- 公证取证必须选择有资质的公证机构
- 电子存证必须使用司法认可的平台
- 证据获取过程必须合法（不得入侵、窃取、引诱）
- 所有原始证据保存期限不少于诉讼时效期间（3年）+诉讼周期
- 涉及跨境取证需评估外国法律要求和公证认证程序
