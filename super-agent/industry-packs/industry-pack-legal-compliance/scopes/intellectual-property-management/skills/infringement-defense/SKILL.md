# 被控侵权应对

## 概述

被控侵权应对是当企业收到侵权警告函、被提起侵权诉讼或面临行政执法调查时的紧急应对能力。有效的被控侵权应对需要在极短时间内完成风险评估、制定防御策略，并在必要时启动设计规避以确保业务连续性。

## 适用场景

- 收到知识产权侵权警告函/律师函
- 被提起专利侵权民事诉讼
- 被提起商标侵权/不正当竞争诉讼
- 面临行政执法部门的知识产权检查
- 收到海关知识产权扣货通知
- 客户/合作伙伴转达的侵权指控

## 执行步骤

### 步骤1：紧急响应（72小时内）
- 收到侵权指控后立即评估紧急程度：
  - 紧急（已被诉/被扣货/行政调查）→24小时内响应
  - 重要（律师函/正式警告）→72小时内响应
  - 一般（非正式提醒/客户转达）→1周内评估
- 初步收集信息：
  - 对方权利基础（专利号/商标号/著作权登记号）
  - 指控的具体侵权行为/产品
  - 对方诉求（停止使用/赔偿/许可）
  - 法律程序状态（是否已立案/是否有时限要求）
- 通知管理层和相关部门（研发、销售、供应链）
- 评估是否需要立即暂停相关产品/服务

### 步骤2：权利有效性审查
- 核实对方权利的法律状态：
  - 专利是否在有效期内？年费是否正常缴纳？
  - 专利是否曾被提无效？结果如何？
  - 商标是否在有效期内？是否有被撤三风险？
- 审查对方权利的稳定性：
  - 专利：检索可能影响其新颖性/创造性的现有技术
  - 评估提起无效宣告的可行性和成功概率
  - 审查其审查历史（禁止反悔原则的适用）
- 核实对方权利人身份和授权链（是否有权起诉）

### 步骤3：侵权比对分析
- **不侵权分析**：
  - 逐一对比我方产品/技术的技术特征与对方权利要求
  - 寻找缺少的必要技术特征（全面覆盖原则的反向适用）
  - 评估等同侵权的风险（功能-方式-效果三要素测试）
  - 分析捐献原则（对方说明书公开但权利要求未覆盖的方案）
- **抗辩事由分析**：
  - 现有技术抗辩（我方技术方案属于公知技术）
  - 先用权抗辩（在对方申请日前已经使用或准备使用）
  - 合法来源抗辩（适用于销售者，证明合法进货渠道）
  - 权利用尽抗辩（产品经权利人或许可人首次销售后）
  - 专利默示许可（基于交易关系推定的许可）

### 步骤4：防御策略制定
- **策略选项**：
  - **不侵权主张**：我方技术不落入对方保护范围
    - 适用：技术比对结果确实不构成侵权
    - 行动：回函阐明不侵权理由/诉讼中提交不侵权抗辩
  - **无效宣告**：攻击对方权利基础
    - 适用：找到有力的现有技术证据
    - 行动：向国家知识产权局提起专利无效宣告请求
    - 可与诉讼答辩同步进行
  - **设计规避**：修改技术方案绕开对方权利
    - 适用：侵权风险确实存在且无法有效抗辩
    - 行动：研发团队制定技术替代方案
    - 注意：规避方案需确保不再落入对方保护范围
  - **协商许可**：获取对方的使用许可
    - 适用：无法有效规避且需要继续使用该技术
    - 行动：谈判合理的许可条件
  - **交叉许可**：以我方专利作为谈判筹码
    - 适用：我方拥有对方可能需要的专利
    - 行动：提出交叉许可方案

### 步骤5：执行与跟踪
- 按确定的策略执行防御行动
- 诉讼案件的关键节点管理：
  - 答辩期限（收到起诉状15日内）
  - 举证期限（法院指定）
  - 开庭准备
  - 一审判决→是否上诉评估
- 无效宣告程序跟踪（通常5-8个月出结论）
- 设计规避方案的实施进度
- 定期向管理层报告案件进展

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 侵权指控文件 | 警告函/起诉状/行政通知 | 是 |
| 对方权利文件 | 对方的专利/商标证书和权利要求 | 是 |
| 我方产品/技术说明 | 被指控侵权的产品或技术方案 | 是 |
| 我方现有IP资产 | 可用于交叉许可谈判的资产 | 否 |
| 先用权证据 | 在对方申请日前使用的证据 | 否 |

## 输出

| 输出项 | 说明 |
|--------|------|
| 紧急风险评估报告 | 72小时内出具的初步风险判断 |
| 侵权比对分析报告 | 详细的技术特征对比和结论 |
| 防御策略方案 | 推荐的应对策略及备选方案 |
| 无效检索报告 | 攻击对方权利的现有技术证据 |
| 设计规避方案 | 技术方案修改建议 |

## 最佳实践

1. **不要恐慌**：收到警告函不等于侵权成立，冷静评估是第一步
2. **不要忽视**：任何正式的侵权指控都需要认真对待和正式回应
3. **证据保全**：保留所有我方在先使用的证据（研发记录、购买凭证、邮件等）
4. **业务连续**：制定应急计划确保即使被判侵权也有替代方案
5. **攻防结合**：无效宣告是最有力的"釜底抽薪"策略
6. **商业视角**：有时获取许可是最经济的选择，不要为了面子而过度对抗

## 约束条件

- 紧急响应必须在72小时内完成初步评估
- 诉讼答辩必须在法定期限（15日）内提交
- 无效宣告申请需在答辩策略确定后10日内提交
- 设计规避方案需经专利战略分析师确认不再落入保护范围
- 涉及暂停产品/服务的决策需管理层批准
- 所有与对方的沟通必须经法务负责人确认
