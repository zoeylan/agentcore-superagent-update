# IP资产台账管理

## 概述

IP资产台账管理是企业知识产权运维的基础性工作，通过建立和维护一份完整、准确、实时更新的IP资产清单，确保企业对自身知识产权资产的全面掌握。台账是所有IP管理决策（维护/放弃/许可/转让/维权）的数据基础。

## 适用场景

- 新IP资产获得授权/注册后的入账登记
- IP资产法律状态变更的及时更新
- 管理层查询企业IP资产概况
- IP尽职调查中的资产清点
- 年度IP资产盘点和价值评估
- IP资产转让/许可/质押交易的权属核实

## 执行步骤

### 步骤1：资产入账登记
- 新专利授权：录入专利号、名称、权利人、发明人、申请日、授权日、技术领域（IPC）、摘要、权利要求数、保护期限、代理机构
- 新商标注册：录入注册号、商标图样、指定商品/服务类别、注册人、注册日、有效期至、代理机构
- 新著作权登记：录入登记号、作品名称/软件名称、著作权人、创作完成日、首次发表日、登记日
- 域名资产：录入域名、注册商、注册日、到期日、DNS解析状态
- 关联信息：与业务线/产品线的关联标记、与其他IP资产的关联关系（同族专利、系列商标）

### 步骤2：状态监控与更新
- 每日检查各IP局官方公告（CNIPA专利公告、商标公告）
- 专利状态变更：受理→公开→实审→授权→有效→失效
- 商标状态变更：申请→初审公告→注册→有效→待续展→失效
- 异常状态处理：被提无效、被异议、被撤三→标记为"争议中"并通知相关专家
- 所有状态变更须在2个工作日内更新台账

### 步骤3：数据质量管理
- 月度数据校验：核对台账记录与官方数据库的一致性
- 缺失字段补全：确保每条记录的必填字段完整度≥99%
- 重复记录清理：识别和合并重复登记的资产记录
- 历史数据归档：已失效/放弃的资产移入历史库但保留可查询

### 步骤4：资产分类与标签
- 按类型分类：发明专利/实用新型/外观设计/商标/著作权/域名/商业秘密
- 按地域分类：中国/美国/欧洲/日本/韩国/其他
- 按技术领域：与企业技术分类体系对齐
- 按业务线：与企业组织架构和产品线对齐
- 按价值等级：核心资产/重要资产/一般资产/待评估
- 按生命周期阶段：申请中/有效/即将到期/已失效

### 步骤5：报表与查询服务
- 提供多维度查询接口（按类型/地域/时间/业务线/状态等）
- 定期生成统计报表（月度增减变动、年度资产概览）
- 支持临时性的数据查询需求（如并购尽调、融资需要）
- 为其他Agent提供资产数据支撑

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| IP授权/注册通知 | 官方下发的授权/注册证书或通知 | 是 |
| 法律状态变更信息 | 缴费回执、状态变更通知等 | 是 |
| 业务关联信息 | 资产对应的产品线/业务线 | 否 |
| 价值评估结果 | 资产价值等级划分 | 否 |

## 输出

| 输出项 | 说明 |
|--------|------|
| IP资产台账 | 完整的结构化资产清单 |
| 资产变动报告 | 月度/季度资产增减变动统计 |
| 数据质量报告 | 台账完整性和准确性指标 |
| 查询结果 | 按条件筛选的资产信息 |

## 最佳实践

1. **唯一标识**：每项资产使用唯一内部编号，避免混淆
2. **变更留痕**：所有修改记录操作人、操作时间、修改前后值
3. **双重校验**：关键数据（如到期日）录入后需二次校验
4. **定期盘点**：每年至少一次全面盘点，确保账实相符
5. **权限管控**：台账数据按密级设定查阅和修改权限
6. **关联维护**：维护专利族（同族专利）、系列商标等关联关系

## 约束条件

- 台账准确率必须≥99%
- 新资产入账须在获权后5个工作日内完成
- 状态变更更新须在2个工作日内完成
- 必填字段不得留空（申请号、权利人、法律状态、到期日等）
- 涉及商业秘密的资产信息需要额外的访问控制
