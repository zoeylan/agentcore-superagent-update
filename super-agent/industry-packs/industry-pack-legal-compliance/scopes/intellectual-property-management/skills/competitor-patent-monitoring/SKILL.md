# 竞争对手专利动态监控

## 概述

竞争对手专利动态监控是企业专利战略情报工作的核心组成部分，通过持续追踪竞争对手的专利申请、授权、转让、许可等动态，及时发现竞争威胁和合作机会，为企业技术研发方向调整和IP布局策略优化提供情报支持。

## 适用场景

- 定期（月度/季度）竞品专利动态跟踪
- 新产品/新技术立项前的竞争格局分析
- 并购/投资标的的IP尽职调查
- FTO（自由实施）分析中的风险识别
- 行业技术趋势判断和白色空间（White Space）发现

## 执行步骤

### 步骤1：监控对象确定
- 识别核心竞争对手名单（直接竞争者、潜在进入者、上下游企业）
- 确定监控的技术领域范围（IPC/CPC分类号）
- 设置监控关键词（技术术语、产品名称、品牌名称）
- 确定监控频率：核心对手每周监控，一般对手每月监控

### 步骤2：数据采集
- 新申请监控：追踪竞品最新公开/公告的专利申请
- 授权动态：竞品新获授权的专利
- 法律状态变更：专利转让、许可备案、质押、无效、放弃
- 诉讼动态：竞品作为原告/被告的专利诉讼案件
- 人才动态：核心发明人的流动（通过发明人检索追踪）

### 步骤3：情报分析
- **技术方向分析**：从新申请中判断竞品的研发重点和技术路线
- **专利质量评估**：分析竞品专利的权利要求保护范围和稳定性
- **布局策略推断**：从申请地域、类型、时间规律推断其IP策略
- **威胁评估**：识别可能影响我方产品/技术的竞品专利（FTO风险）
- **机会发现**：发现竞品放弃的领域或薄弱环节

### 步骤4：预警与报告
- 紧急预警：发现直接影响我方业务的竞品专利，24小时内发出预警
- 月度简报：竞品专利动态汇总（新申请数/授权数/关键专利摘要）
- 季度深度报告：技术趋势分析、竞争格局变化、策略建议
- 年度综述：全年竞品IP战略回顾和来年预判

### 步骤5：策略建议输出
- 针对发现的威胁提出应对建议（规避设计/无效准备/交叉许可）
- 针对发现的机会提出布局建议（空白领域抢先申请/外围包围策略）
- 更新企业专利布局路线图

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 竞争对手名单 | 监控对象的企业名称和变体名称 | 是 |
| 技术领域范围 | IPC/CPC分类号或技术关键词 | 是 |
| 企业产品/技术清单 | 用于FTO风险匹配 | 否 |
| 上期监控报告 | 用于对比变化 | 否 |

## 输出

| 输出项 | 说明 |
|--------|------|
| 竞品专利动态简报 | 定期的动态汇总 |
| FTO风险预警 | 针对具体威胁的紧急通知 |
| 技术趋势分析报告 | 行业技术方向判断 |
| 策略建议 | 应对措施和布局建议 |

## 最佳实践

1. **建立监控档案**：为每个核心竞品建立专利档案并持续更新
2. **多维度监控**：不只看专利数量，关注质量（引用次数、同族规模）
3. **关注发明人**：核心发明人的动态可以揭示技术人才流动和研发方向
4. **结合商业情报**：将专利动态与竞品的产品发布、融资、并购等商业动态关联分析
5. **定期回溯验证**：检查过去的预判是否准确，优化监控策略

## 约束条件

- 核心竞争对手监控频率不低于每周一次
- FTO风险预警必须在发现后24小时内发出
- 监控报告中不得包含通过非法手段获取的商业秘密信息
- 所有分析必须基于公开可获取的专利文献和法律状态信息
