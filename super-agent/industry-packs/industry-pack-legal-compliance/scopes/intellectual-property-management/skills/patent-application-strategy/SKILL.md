# 专利申请策略制定

## 概述

专利申请策略制定是将技术创新转化为有效法律保护的核心决策环节，涵盖专利类型选择、申请路径规划、权利要求布局设计、以及与企业整体IP战略的匹配。优质的申请策略能够以合理成本获取最大保护范围。

## 适用场景

- 技术交底书通过可申请性评估后的策略规划
- 企业年度专利布局规划
- 核心技术的多层次保护方案设计
- 国际市场拓展时的海外专利布局
- 竞争对手专利包围时的反制策略

## 执行步骤

### 步骤1：技术价值评估
- 评估技术方案的商业价值（市场规模、技术领先程度、可替代性）
- 判定技术成熟度（已实现/可实现/概念阶段）
- 评估保护紧迫性（是否即将公开、竞品是否在研发类似技术）
- 划分优先级：S级（核心技术，最高优先）→A级（重要技术）→B级（一般技术）→C级（防御性布局）

### 步骤2：专利类型选择
- **发明专利**：
  - 适用：核心创新、方法发明、算法/软件、新材料/配方
  - 保护期20年，审查严格（新颖性+创造性+实用性）
  - 审查周期18-24个月，适合高价值技术方案
- **实用新型**：
  - 适用：产品结构/形状改进
  - 保护期10年，初步审查即授权（6-8个月）
  - 适合快速获权形成防御屏障，但权利稳定性较低
- **外观设计**：
  - 适用：产品外观造型、GUI界面设计
  - 保护期15年（新法），局部外观设计可申请
- **组合策略**：同一技术方案同日申请发明+实用新型（"双报策略"）

### 步骤3：申请路径规划
- **国内直接申请**：
  - 仅需国内保护时选择
  - 费用最低，流程简单
- **巴黎公约途径**：
  - 在优先权期限内（发明/实用新型12个月，外观6个月）逐国申请
  - 适合目标国家明确且数量较少（≤5个）的情况
- **PCT途径**：
  - 通过一份国际申请保留进入多国的选择权
  - 30个月内决定进入哪些国家（延长决策时间）
  - 适合目标国家不确定或数量较多的情况
  - 注意：PCT只是程序途径，最终仍需各国审查
- **时间线规划**：标注每个关键时间节点（优先权期限、PCT进入国家阶段期限、各国审查预估时间）

### 步骤4：权利要求布局设计
- **独立权利要求**（核心保护范围）：
  - 用最上位概念概括核心技术方案
  - 仅包含必要技术特征（实现发明目的的最少特征集合）
  - 考虑方法权利要求+产品权利要求+系统权利要求多维度覆盖
- **从属权利要求**（层次化限缩）：
  - 每层增加一个附加技术特征进行限缩
  - 通常设计5-10项从属权利要求
  - 优选实施例的技术特征必须被某项从属权利要求覆盖
  - 考虑"伞状"结构：多个独立权利要求+各自的从属链
- **防御性权利要求**：
  - 覆盖竞争对手可能的规避设计路径
  - 考虑等同替换的可能方案

### 步骤5：策略文档输出
- 输出《专利申请策略书》，包含：
  - 技术方案概要及价值评估结论
  - 推荐的专利类型及理由
  - 申请路径选择及时间线
  - 权利要求框架设计（独立权利要求草案+从属链结构）
  - 预算估算（官费+代理费+翻译费）
  - 风险提示（可能的审查障碍、无效风险）

## 输入

| 输入项 | 说明 | 必需 |
|--------|------|------|
| 检索分析报告 | 包含新颖性/创造性评估结论 | 是 |
| 技术交底书 | 技术方案详细描述 | 是 |
| 企业IP战略规划 | 年度布局方向和预算 | 否 |
| 目标市场信息 | 产品销售/制造的目标国家 | 否 |
| 竞争对手专利地图 | 竞品在相关领域的专利布局 | 否 |

## 输出

| 输出项 | 说明 |
|--------|------|
| 专利申请策略书 | 完整的策略规划文档 |
| 权利要求框架 | 独立权利要求草案和从属链结构 |
| 时间线计划 | 各阶段时间节点和截止日期 |
| 预算估算 | 申请和维护的费用预测 |

## 最佳实践

1. **先宽后窄**：独立权利要求尽量概括，审查中再逐步限缩
2. **多维度保护**：产品+方法+用途+系统全方位覆盖
3. **预留规避空间**：预判对手可能的规避方案并提前布局
4. **成本效益平衡**：不是所有技术都需要全球布局，聚焦核心市场
5. **关注公开时机**：利用18个月公开期进行竞争情报分析
6. **双报策略运用**：对核心技术同日申请发明+实用新型，确保快速获权

## 约束条件

- 策略书必须在检索报告完成后3个工作日内出具
- 对于S级技术，策略书需经IP管理负责人审批
- 涉及PCT路径的策略必须明确30个月期限的关键日期
- 预算估算偏差不得超过实际费用的20%
- 涉及保密发明的技术需先完成保密审查再确定策略
