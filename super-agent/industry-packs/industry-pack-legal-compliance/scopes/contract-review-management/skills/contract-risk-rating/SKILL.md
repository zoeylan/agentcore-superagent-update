# 合同风险评级 (Contract Risk Rating)

## 技能概述

本技能用于对合同进行整体风险评级，综合条款审查结果、交易对手信用、历史数据和行业环境等多维度信息，输出A/B/C/D四级风险评级结论。风险评级直接决定合同的审批层级和后续管控力度，是合同管理中的关键决策节点。

## 适用场景

- 新合同签署前的风险评级决策
- 续约合同的风险重新评估
- 合同条款重大变更后的风险等级调整
- 交易对手信用状况发生重大变化时的风险重评
- 批量合同的风险组合分析

## 执行步骤

### 步骤1：信息汇集
收集并整理以下评级输入信息：
- **条款风险数据**：来自条款审查专家的逐条审查结果（高/中/低风险条款数量及分布）
- **交易对手信息**：工商登记信息、信用评级、财务状况、诉讼记录、历史合作表现
- **合同要素信息**：合同金额、履行期限、标的物特征、付款安排
- **外部环境信息**：行业政策变化、市场行情波动、监管趋势

### 步骤2：多维度评分
按照评分矩阵对各维度打分（每维度0-25分，总分100分）：

**维度一：条款风险（25分）**
- 无高风险条款且低风险≤3项：25分
- 无高风险但中风险≤2项：20分
- 存在1项高风险（已有缓释方案）：15分
- 存在2项以上高风险或1项无法缓释的高风险：≤10分

**维度二：交易对手风险（25分）**
- 长期合作、信用良好、无诉讼记录：25分
- 首次合作、信用一般、规模较大：20分
- 存在诉讼记录但非恶意违约、财务一般：15分
- 信用不良、有多起诉讼、财务困难：≤10分

**维度三：交易结构风险（25分）**
- 标准交易模式、付款方式有保障：25分
- 存在一定复杂性但有成熟处理方案：20分
- 涉及关联交易/对赌/跨境等复杂结构：15分
- 创新交易模式无先例可循或涉及重大资金垫付：≤10分

**维度四：履约风险（25分）**
- 履行标的明确可控、双方产能充足：25分
- 存在一定履行不确定性但可监控：20分
- 履行期较长或依赖第三方或存在技术风险：15分
- 履行严重依赖外部条件或对方历史履约不良：≤10分

### 步骤3：综合评级
根据总分确定风险等级：
- **A级（85-100分）**：低风险，可签署。合同条款公平、对手可信、结构清晰、履约有保障
- **B级（70-84分）**：较低风险，建议签署。存在个别关注点但风险可控
- **C级（55-69分）**：中等风险，附条件签署。需落实具体风险缓释措施后方可签署
- **D级（<55分）**：高风险，建议拒绝。存在重大不可接受风险，需对方实质修改或放弃交易

### 步骤4：审批层级确定
根据风险评级和合同金额确定审批路径：
- A/B级 + 金额<50万 → 业务负责人审批
- C级 或 金额≥50万 → 法务审核
- D级 或 金额≥200万 → 法务总监审核

### 步骤5：评级报告输出
生成结构化的评级报告，包含评级结论、各维度评分详情、关键风险因素和审批建议。

## 输入规格

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| clause_review_result | 结构化数据 | 是 | 条款审查结果（含风险标注） |
| counterparty_profile | 结构化数据 | 是 | 交易对手信用档案 |
| contract_elements | 结构化数据 | 是 | 合同基本要素（金额、期限、类型） |
| historical_data | 结构化数据 | 否 | 与该对手的历史合作数据 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| risk_grade | 枚举(A/B/C/D) | 综合风险评级 |
| total_score | 数字 | 综合评分（0-100） |
| dimension_scores | 结构化数据 | 各维度评分明细 |
| approval_level | 枚举 | 对应的审批层级 |
| key_risk_factors | 列表 | 核心风险因素摘要 |
| mitigation_requirements | 列表 | 风险缓释要求（C级时必填） |

## 最佳实践

1. **客观量化**：评级过程尽量使用可量化的指标，减少主观判断空间
2. **一票否决机制**：存在法律强制性违规的，无论总分如何直接评D级
3. **动态调整**：评级不是一次性的，重大情况变化时需重新评估
4. **历史对标**：参考同类合同的历史评级分布，确保标准一致性
5. **充分说理**：评级结论必须有充分的事实和逻辑支撑，经得起质疑

## 约束条件

- 评级过程必须覆盖所有四个维度，不得省略
- C级和D级评级必须附带具体的风险缓释建议或拒绝理由
- 评级结论不受业务部门签约压力影响，保持独立性
- 评级结果需存档备查，作为后续履约监控和复盘的参考基准
