# 法律风险识别 (Legal Risk Identification)

## 技能概述

本技能用于在合同条款分析的基础上，对每个偏差条款和潜在问题条款进行法律风险识别和评级。通过对照法律强制性规定、行业惯例和我方利益保护标准，判断条款可能给我方带来的法律风险，并按高/中/低三级进行分类标注。这是合同审查的核心价值输出环节。

## 适用场景

- 识别对方合同版本中的不利条款和法律陷阱
- 评估非标条款偏离标准模板后的风险影响
- 发现合同中违反法律强制性规定的无效或可撤销条款
- 识别条款之间的逻辑矛盾和权利义务失衡

## 执行步骤

### 步骤1：确定审查标准
根据合同类型加载对应的风险识别规则：
- **通用必审规则**：适用于所有合同类型的基础风险检查
- **类型专用规则**：特定合同类型的重点关注领域
- **行业特殊规则**：特定行业的监管合规要求
- **金额敏感规则**：大额合同的额外审查要求

### 步骤2：五大核心条款风险审查
对以下五类核心条款进行深度风险分析：

**2.1 违约责任审查**
- 违约情形是否对等（双方触发条件是否平衡）
- 违约金比例是否合理（是否超过损失的30%——参照《民法典》第585条）
- 是否存在单方免责条款（我方承担全部责任风险）
- 赔偿范围是否包含间接损失和预期利益
- 责任限额是否合理设定

**2.2 知识产权归属审查**
- 委托开发成果归属是否明确（《民法典》第859条）
- 合作开发中各方贡献与权利分配是否对应
- 背景IP和前景IP是否区分清晰
- 许可范围（独占/排他/普通）是否与商业意图匹配
- 改进成果归属是否约定

**2.3 保密义务审查**
- 保密信息范围界定是否合理（避免过于宽泛）
- 保密期限是否合理（通常2-5年，不应无限期）
- 违约金金额是否过高或过低
- 例外情形是否包含法定披露义务
- 返还或销毁义务是否明确

**2.4 争议解决审查**
- 仲裁条款是否有效（仲裁机构是否明确、是否存在或裁或审）
- 管辖法院约定是否有效（是否符合法定连接点要求）
- 准据法选择是否对我方有利（跨境合同）
- 是否约定了前置协商/调解程序

**2.5 不可抗力审查**
- 不可抗力范围是否合理列举
- 是否包含情势变更条款
- 通知义务和证明义务是否合理
- 不可抗力的法律后果是否明确（解除/中止/减免）

### 步骤3：风险等级标注
对识别出的每个风险点标注等级：
- **高风险（红色）**：可能导致重大经济损失（>合同金额20%）、法律责任（行政处罚/刑事风险）、或条款可能被认定无效/可撤销
- **中风险（橙色）**：存在争议隐患，可能在履行中引发纠纷，但可通过补充约定或履行注意事项控制
- **低风险（黄色）**：建议优化但不影响合同效力和核心权益，属于谈判优化空间

### 步骤4：风险关联分析
- 分析各风险点之间的关联关系（如：违约金过高+无责任限额=风险叠加）
- 评估风险集中度（是否多个高风险集中在同一方向）
- 识别系统性风险模式（如：合同整体有利于对方的倾斜设计）

## 输入规格

| 输入项 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| clause_analysis_result | 结构化数据 | 是 | 条款分析技能的输出结果 |
| contract_amount | 数字 | 是 | 合同金额（万元） |
| our_role | 枚举 | 是 | 我方在合同中的角色（买方/卖方/委托方/受托方等） |
| counterparty_info | 文本 | 否 | 交易对手基本信息 |

## 输出规格

| 输出项 | 类型 | 说明 |
|--------|------|------|
| risk_findings | 列表 | 每个风险点的详细分析（条款定位、风险描述、风险等级、法律依据） |
| risk_summary | 统计 | 高/中/低风险数量统计 |
| critical_alerts | 列表 | 需要立即关注的高风险警示 |
| risk_correlation | 文本 | 风险关联分析和系统性风险判断 |

## 最佳实践

1. **法条依据精确引用**：每个风险判断必须有明确的法律依据，引用到具体条款号
2. **区分效力性与管理性强制规定**：违反效力性强制规定导致无效，违反管理性仅导致处罚
3. **考虑司法实践**：不仅看法条规定，还要参考最高法指导案例和地方司法实践
4. **立场明确**：始终站在我方立场评估风险，同时客观分析对方可能的抗辩理由
5. **商业敏感度**：理解条款背后的商业意图，区分商业风险和法律风险

## 约束条件

- 风险识别必须有法律依据支撑，不得主观臆断
- 高风险标注需谨慎，避免过度风险提示导致业务停滞
- 不得遗漏五大核心条款中任何一类的审查
- 对于可能违反法律强制性规定的条款必须标注为高风险
