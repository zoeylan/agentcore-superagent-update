# 公司治理与股权事务

## 技能描述
为企业提供公司治理结构设计、股东会/董事会运作、股权架构规划、并购重组、信息披露等公司法事务的全面法律支持，确保公司决策程序合法合规，防范治理风险。

## 触发场景
- 用户需要设计或优化公司治理架构
- 用户面临股权融资、增资扩股、股权转让等事项
- 用户需要召开股东会/董事会的程序性指导
- 用户涉及并购重组交易的法律架构设计
- 用户需要处理股东纠纷或公司僵局
- 用户作为上市公司需要信息披露合规指导

## 工作流程

### 第一步：治理现状诊断
1. 审查现有公司章程、股东协议、议事规则
2. 梳理股权结构和实际控制关系
3. 评估决策机制的合理性和效率
4. 识别现有治理架构中的法律漏洞

### 第二步：方案设计
根据企业阶段和需求设计治理方案：
- **初创期**：创始人控制权保护、股权预留、反稀释
- **成长期**：引入投资人、董事会席位分配、一票否决权
- **成熟期**：规范化治理、独立董事制度、专业委员会
- **上市期**：符合监管要求的治理架构、信息披露制度

### 第三步：文件起草与审查
- 公司章程（定制化条款设计）
- 股东协议/投资协议
- 董事会/股东会议事规则
- 关联交易管理制度
- 信息披露管理制度
- 高管薪酬与激励方案

### 第四步：实施与维护
- 决策程序的合规性把关
- 重大事项的法律合规审查
- 定期治理评估和优化建议
- 监管政策变化的跟踪和应对

## 核心知识模块

### 股权架构设计
- **有限责任公司 vs 股份有限公司**选择
- **境内架构**：直接持股 / 持股平台（有限合伙）/ 员工持股平台
- **境外架构**：VIE协议控制 / 红筹架构 / WFOE直接持股
- **控制权工具**：AB股（差异化表决权）/ 一致行动协议 / 投票权委托 / 有限合伙GP控制
- **反稀释保护**：优先认购权 / 加权平均反稀释 / 棘轮条款
- **退出机制**：回购条款 / 拖售权(Drag-along) / 随售权(Tag-along) / 领售权

### 董事会运作
- 董事提名与选任程序
- 董事的忠实义务和勤勉义务
- 董事会决议的有效性要件（通知/出席/表决）
- 独立董事的独立性认定
- 专业委员会设置（审计/薪酬/提名/战略）
- 董事责任保险（D&O Insurance）

### 股东权利保护
- 股东知情权的范围和行使程序
- 利润分配请求权
- 优先购买权与同意权
- 股东代表诉讼的前置程序
- 异议股东回购请求权
- 小股东退出机制

### 并购重组
- **尽职调查**：法律/财务/业务/HR/IT/环境全方位调查
- **交易结构**：股权收购 / 资产收购 / 合并 / 分立 / 各方案税务比较
- **估值与对价**：估值方法选择 / 对价支付方式 / 业绩对赌/补偿
- **交割条件**：先决条件设计 / MAC条款 / 政府审批
- **交割后管理**：管理层过渡 / 竞业限制 / 陈述保证存续期 / 赔偿机制

### 上市公司治理
- 信息披露义务（定期报告/临时公告/内幕信息）
- 内幕交易防控（知情人登记/窗口期管理/信息隔离墙）
- 关联交易的认定、审批和披露
- 重大资产重组程序
- 股份减持规则（大股东/董监高）
- 要约收购与反收购策略

## 关键决策框架

### 关联交易审查流程
```
识别关联关系 → 判断是否构成关联交易 → 评估交易公允性
→ 确定审批层级（总经理/董事会/股东会）→ 关联方回避表决
→ 独立董事意见 → 信息披露（如适用）→ 备案存档
```

## 注意事项
- 公司章程是公司的"宪法"，设计时必须考虑各种可能场景
- 股东协议与章程冲突时，对外以章程为准，对内按约定处理
- 有限公司的人合性特征意味着股权转让需要其他股东同意
- 工商登记具有公示效力，但不等于真实权属，注意"显名"与"隐名"问题
- 一人有限公司务必做到财产独立，否则股东承担连带责任
- 对赌条款效力：与目标公司对赌的效力存在争议（注意最高院九民纪要规则）
- 国有企业公司治理有特殊规定（"三重一大"决策、党委前置研究等）
- 境外架构拆除（红筹回归）涉及复杂的税务和外汇问题
